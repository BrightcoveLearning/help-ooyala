URL Parameters

Modify the player with URL parameters.

You add these types of query parameters to the http://player.ooyala.com/v3/.. .. <script> tag. These player URL-based parameters include:

  • namespace=(unique javascript id)

  • platform=platform=(flash-only|flash|html5-fallback|html5-priority) ?

For more information about the use of parameters for localization and closed captioning see the topic “Closed Captioning and Localization” in this guide.
