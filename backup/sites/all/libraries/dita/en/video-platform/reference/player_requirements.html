Player V3 System Requirements by Platform (Deprecated)

Player V3 is designed to deliver the highest quality experience for live and on demand video.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).

Player V3 provides consumers with the highest quality stream possible based on factors such as available bandwidth, throughput, CPU, Flash version, and device. If you are seeing less than ideal quality, make sure that your environment meets these minimum requirements.

Note: These configurations undergo regular testing with each release. Other environments (such as Linux or Chrome beta channels) might work, but are not actively tested.

Windows

Requirement Description
Processor Intel Pentium 4 or equivalent with 1 GHz processor
Operating System
  • Windows 10 with all available updates
  • Windows 8 with all available updates
  • Windows 7 with all available updates
RAM 512 MB minimum (1 GB recommended)
Video Card 32 MB minimum (128 MB recommended)
Browser
  • Microsoft Edge (Windows 10)
  • Internet Explorer in Standard Document Mode (v9+). Note that fullscreen support is enabled in Internet Explorer v11+.
  • Firefox v38+
  • Chrome v40+
Flash
  • For Real Time Messaging Protocol (RTMP) playback: 10.0 or higher (10.1 or later recommended)
  • For HTTP Dynamic Streaming (HDS) playback: 10.1.0 or higher
Screen Resolution 800 x 600 minimum (1280 x 1024 or greater recommended)
Internet Connection Broadband/High Speed (DSL, Cable, T1, etc.). Dial-up connections are insufficient for video playback.
Playback Type Flash and HTML5
Note: Internet Explorer v8-v10 does not support true fullscreen mode in JavaScript.

Mac OS X

Requirement Description
Processor Intel Core2 Duo 1.83GHz or faster processor
Operating System Mac OS X 10.5 or greater with Intel Processors of 1GHz or better. No PowerPC support.
RAM 512 MB minimum (1 GB recommended)
Video Card 32 MB minimum (128 MB recommended)
Browser
  • Safari v8+
  • Firefox v38+
  • Chrome v40+
Flash 10.0 or higher (10.1 or later recommended)
Screen Resolution 800 x 600 minimum (1280 x 1024 or greater recommended)
Internet Connection Broadband/High Speed (DSL, Cable, T1, etc.). Dial-up connections are insufficient for video playback.
Playback Type Flash and HTML5

Apple iOS

Requirement Description
Operating System iOS 7+
Browser Safari
Internet Connection WiFi or 3G
Playback Type HTML5

Android

Requirement Description
Operating System Android 2.3+
Browser Chrome for Android. Best Effort support for pre-installed browser on Android 2.3.x.
Internet Connection WiFi or 3G
Playback Type HTML5
Note: Android device testing is limited to popular Android devices from Samsung, Google, and HTC.
