Assigning Ad Sets with setEmbedCode in Player V3 (Deprecated)

You also have the option of assigning ad sets using the setEmbedCode function.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).
With this function you can set up playlists with different ad sets or set up ad sets for different contexts.
Note: This function requires that you associate an ad set with a movie in Backlot as shown in Assigning an Ad Set with OO.Player.create in Player V3 (Deprecated).

The setEmbedCode function takes an additional options parameter that must be an object and must belong to the same provider as the embed code. In the options hash, use the adSetCode parameter, as in the following example.

<script>
player.setEmbedCode(embedCode, {
    adSetCode: 'yourAdSetCode'
});
<script>
