TV Ratings Backlot Metadata Configurations

Table 1. TV Ratings Metadata
Custom Metadata Description Valid Values Required Example
tvrating Specifies audience ratings or recommended audience. TV-X - Where X can be one of the following:
  • Y
  • Y7
  • G
  • PG
  • 14
  • MA
Note: For a description of these rating see FCC - The TV Parental Guidelines.
YES 
tvrating: “TV-PG”
tvsubratings Specifies content label or nature of content. The position of the of the rating watermark on the video asset. This parameter can have one of the following four values:
  • V: violence
  • S: sexual situations
  • L: coarse, crude, or indecent language
  • D:suggestive dialogue
  • FV:fantasy violence
 NO 
tvsubratings:”D,S”
tvratingurl A click through URL. A URL string. No 
tvratingurl= "http://www.parentstv.org/ptc/parentsguide/tvratings2.html"
