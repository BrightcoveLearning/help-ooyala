Ooyala IQ SDK for Roku Integration

You can now integrate Roku channels with Ooyala IQ with the Ooyala IQ SDK for Roku. The Ooyala IQ SDK for Roku is an out-of-the-box SDK that sends JSON data to the Ooyala IQ JSON API endpoint and automatically follows the best practices for what events should be sent when, and makes sure that all required fields are present.

The Ooyala IQ SDK for Roku supports the Roku OS v7+.
Note: The Ooyala IQ SDK for Roku currently does not support the Roku Scene Graph API.

If you use the Ooyala IQ SDK for Roku, Roku will appear as an OS in the Device panel of the Ooyala IQ Business Intelligence interface.

Hosting

Ooyala hosts the Ooyala IQ SDK for Roku library on GitHub (where you can download the library). The Ooyala IQ SDK for Roku must be an integral part of your Roku channel package, because it cannot be downloaded to a Roku box on the fly as you can with the JavaScript SDK.

Implementation Steps

You can integrate with the Ooyala IQ SDK for Roku yourself for free, or Ooyala Professional Services can help you integrate with the Ooyala IQ SDK for Roku solution for a fee.

To implement the Ooyala IQ SDK for Roku, do the following:

  1. Download the Ooyala IQ SDK for Roku library from GitHub.
  2. Integrate the Ooyala IQ SDK for Roku with your Roku channel (as described in the Ooyala IQ Roku Library API Documentation).
  3. Certify your Roku application with Roku.
  4. Publish your Roku application to your customers.
  5. Repeat steps 1-4 every time Ooyala releases a new version of the Ooyala IQ SDK for Roku. We will notify you every time a new release occurs.
