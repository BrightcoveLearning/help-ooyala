Program Theme Builder Customizations for Player

Theme Builder lets you create embeds of custom combinations of players and playlists.

Prerequisites

Before you use Theme Builder, ensure you have a collection of videos in Backlot. If you want to use labels to create playlists automatically, define any required labels in Backlot.

To learn how to use the Theme Builder UI, see Logging into Theme Builder and the topics that follow.

Players

Players let your viewers watch your video content.

Playlists

Playlists are ordered groupings of video content.

Playlist Carousels (also known as Pods)

Playlist Carousels display your custom playlists with the player.

Theme Builder Embeds

A Theme Builder embed is one or more playlists embedded within a player. You must create the embed yourself.

Theme Builder Playlist CSS Class Names

Theme Builder relies on Cascading Style Sheets (CSS) to stylize the player and playlists.

Players

Players let your viewers watch your video content.

To create the players you want to use with Theme Builder, you can use either of the following:
  • The Backlot GUI. See Setting up Players Using the Backlot REST API (Deprecated).
  • The Backlot API /v2/players route. See Players (Backlot REST API).

Playlists

Playlists are ordered groupings of video content.

You can build playlists by specific video assets or by label. With playlists, you can control the content your viewers can access within a theme. You can also organize the content in the playlist. For example, you can create a western theme and have it contain a playlist filled with western movies, ordered by release date.

Note:
  • The maximum number of videos you can add to a playlist is 64.
  • The maximum number of labels you can associate with a playlist is 16.
  • Allow some time for the playlist you create to be propagated through the Ooyala systems. If you are doing a demonstration, it is best to use playlists you have created previously.
Note: Playlists are not visible by default. If you do not enable playlist carousels for your player, playlists are not visible to users. To make playlists visible to users, enable playlist carousels by associating the playlist and playlist carousel with a player.
