Updating a Discovery Profile

Note: Use the Backlot UI to create a discovery profile (see Creating Discovery Profiles Using the Backlot UI). A default discovery profile is created automatically.

To update a discovery profile, use the following PUT request. The default_countdown is the time (in seconds) before the next video in a series plays. All other properties retain their defaults.

[PUT]/v2/players/profile_id/discovery/profile{  
   "discovery_interface":{  
      "default_countdown":"4"
   }
}

Backlot returns a response similar to the following example:
