Alternative Routes

The Backlot API has several ways to accomplish the same tasks.

Note: For more information about Backlot REST API commands, see the Backlot API Reference.

In addition to the /v2/assets route and its special-purpose qualifiers, there are other routes to accomplish similar tasks.

/v2/assets Alternative Described in...
/v2/assets/asset_id/labels /v2/labels Labels
/v2/assets/asset_id/publishing_rules /v2/publishing_rules Publishing Rules
/v2/assets/asset_id/ad_set /v2/ad_set Working with Your Own Ad Assets, Ad Sets, and Ad Sources
/v2/assets/asset_id/players /v2/players Working with Your Own Ad Assets, Ad Sets, and Ad Sources
