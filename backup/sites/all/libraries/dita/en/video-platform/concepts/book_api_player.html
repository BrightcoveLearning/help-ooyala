Player V3 JavaScript API Reference (Deprecated)

The Player V3 API is a JavaScript API for customizing your Player v3 functionality and UI elements that also provides message bus event handling.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).
Important: The topics in this section are being migrated to the Ooyala API Docs page.
  • Overview of the Player V3 JavaScript API
    Welcome to the Ooyala Player API Reference. Ooyala offers a customizable, feature-rich Player used to provide a top-notch video experience across industries and across multiple devices.
  • Player V3 JavaScript API Functions
    Player V3 JavaScript functions can be used to customize the player behavior and appearance.
  • Player V3 JavaScript API Parameters (Deprecated)
    Player V3 parameters can be used to customize the player ads, behavior and to change player styles.
  • Player V3 Message Bus Events (Deprecated)
    Use message bus events to subscribe to or publish player events from video to ad playback.
  • Flash-based Player V3 Error Messages (Deprecated)
    Error messages are displayed to the video viewer or audio listener during playback. Errors can be categorized as general errors, Flash-based errors and Pay-Per-View errors, which are prefixed with a PPV.
