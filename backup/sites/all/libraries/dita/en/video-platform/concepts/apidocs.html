apidocs.ooyala.com

apidocs.ooyala.com is a new site for developers that includes API references, SDK documentation, and sample applications.

http://apidocs.ooyala.com currently contains the following:
  • Video Platform API Documentation
    • Player V4 JavaScript API Reference
    • Player V4 JSON Schema
    • Player Gallery
    • Player V3 JavaScript API Reference
    • Mobile SDK for Android API Reference
    • Mobile SDK for iOS API Reference
    • Ad Manager Controller API Reference
    • Ad Manager Template API Reference
    • VAST and VPAID Plugin API Reference
    • Google IMA Plugin API Reference
    • FreeWheel Plugin API Reference
    • Analytics Framework API Reference
    • Analytics Framework Template API Reference
    • Usage API Reference
    • Ooyala Live REST API Reference
  • Analytics Products API Documentation
    • Ooyala IQ JavaScript SDK
    • Ooyala IQ SDK for Roku
  • Ad Products API Documentation
    • Ad Product SDK References
    • Pulse REST API Reference

Additional API references and new sample applications will be available in upcoming releases.
