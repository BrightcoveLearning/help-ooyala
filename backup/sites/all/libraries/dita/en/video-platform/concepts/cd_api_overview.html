Discovery REST APIs

To use these Discovery APIs, you make Discovery requests and return user action information to train the algorithms.

For these types of Ooyala Discovery recommendations, your code should send requests when results are displayed and when recommended videos are clicked. If you are using personalized recommendations, you should also include a device ID with each request.

  • Summary of Ooyala Discovery REST API Requests
    Use the Discovery REST API to configure various aspects of the service, obtain recommendations, and obtain other user-related data.
  • Trending Content Recommendations
    The Ooyala Discovery trending algorithms are designed to find content that is popular on your site now and reinforce its popularity.
  • Related Content Recommendations
    The related content algorithms return content related to the current asset, which you can display on a web page or within in App.
  • Similar Assets Recommendations
    Learn about using the Discovery APIs to provide recommendations for similar videos.
  • Discovery Editorial Query String Parameters
    To enable you to adjust the results of the Ooyala Discovery recommendations, Ooyala provides query string parameters on the baseline APIs.
  • Top Titles
    Use the Top Titles route to return metrics about top trending videos.
