Player V4 JavaScript API

The Player V4 API is a JavaScript API for customizing your player functionality and UI elements, along with message bus event handling. See docs.brightcove.com/apidocs-ooyala/ for the Player V4 JavaScript API Reference.
  • About the Player V4 JavaScript API
  • Programming Closed Captioning Using Player V4 JavaScript
    Closed Captioning (CC) APIs enable you to get/set supported closed captions and localize the Player UI language.
  • Event Model for Player V4
    Player V4 incorporates the use of a message bus to handle events.
  • Programming Bitrates and Buffering for Player V4 Using JavaScript
    Ooyala provides bitrate and buffering functions and events. In addition, Ooyala uses a number of heuristics to make playback as smooth as possible.
  • Handling Errors for Player V4 Using JavaScript
    Error Handling functions and events enable you to get information about Player errors.
