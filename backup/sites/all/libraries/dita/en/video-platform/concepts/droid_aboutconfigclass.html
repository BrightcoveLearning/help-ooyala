About the Android Options Configuration Class

Learn about using the Ooyala SDK for Android configuration class.

The SDK now includes the Options builder class to contain configuration data.

Use this class when your app needs the following configuration options:

Table 1. Options Configuration Implementations
Implementation Function Details
TV Ratings FCCTVRatingConfiguration getTVRatingConfiguration For more information about this implementation see About using Android FCC Ratings.

For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
Cue Point Markers VisualOnConfiguration getVisualOnConfiguration For more information about this implementation see Cuepoint Markers.

For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
Live Content Scrubber getShowLiveControls

For more information about this implementation see Configuring VisualOn Live Scrubbing.

For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
Ad Controls getShowAdsControls For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
Promo Image getShowPromoImage For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
Connection Time Out getConnectionTimeoutInMillisecond For more information see the com.ooyala.android.configuration.Options Class Reference and the Options Sample app on the Ooyala ios-sample-apps git hub repository.
