Manual Publishing

The most common way to publish content is to copy the HTML embed code to a web page.

You can manually embed all types of assets, including:

  • Videos
  • Audio assets
  • Ads
  • Live streams
  • Channels
  • Channel sets
  • YouTube videos
  • Remote assets
  • Configuring Player Embed Settings in Backlot
  • Masking the Player URL
    You can change all instances of Ooyala URLs from player.ooyala.com to your own web site subdomain (player.mycompany.com, video.mycompany.com, content.mycompany.com, and so on..
  • Manually Embedding an Asset
    To make an asset available on a web page, you can manually embed it.
