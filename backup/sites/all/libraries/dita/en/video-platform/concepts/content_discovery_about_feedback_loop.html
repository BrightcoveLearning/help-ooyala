Discovery Feedback Loop

Improve recommendations for your customers by feeding behavior activity from the user to Ooyala Discovery.

Use the Feedback Loop routes to improve the value of recommended content over time. When these routes are included in a Discovery implementation, viewer usage statistics are recorded and leveraged to provided future recommendations influenced by known user choices.

Maintaining the feedback loop is beneficial in several ways:

  • It increases the basic, "raw" metrics about your content.
  • It helps more finely train the machine learning algorithm that powers Ooyala Discovery.
  • The more feedback, the better the recommendations for your viewer.

Every time a user consumes a video based on a Discovery recommendation, or follows a recommendation link presented by Ooyala Discovery, the feedback loop routes the information is stored and used to further refine the user's recommendation experience.
