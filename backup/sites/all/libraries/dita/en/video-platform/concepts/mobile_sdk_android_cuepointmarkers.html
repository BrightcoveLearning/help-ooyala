Cuepoint Markers

Cuepoint visual markers appear on the scrubber to show ad or segment breaks. This feature is enabled by default and compatible with Android 2.3 and newer devices. Cuepoints appear for FreeWheel, VAST,l and Google IMA ads.

To turn off this feature set ShowCuePoints to false in the project main activity. This is shown in the following code example: 

options:options]; Android: Options options = new
  Options.Builder().setShowCuePoints(false).build();
    playerLayoutController = new OptimizedOoyalaPlayerLayoutController(playerLayout, PCODE,domain, options); 
 ...
Important: This is a one time configuration, once the cuepoint markers are disabled in the project the feature cannot be re-enabled.
