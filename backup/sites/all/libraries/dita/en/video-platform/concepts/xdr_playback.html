Resuming Playback in Your Application

After your intermediate server has returned the last playhead position and the formatted OPT request to your application, pass these values to the desired device or platform:

  • Mobile SDK for iOS: [OOOoyalaPlayer playWithInitialTime:initialTime]
  • Mobile SDK for Android: OoyalaPlayer.play(initialTime)
  • Ooyala Web Player: initialTime as part of the third argument on the player invocation.
Note: Regardless of platform, you need to write a function that retrieves the value of initialTime which is the timestamp returned in Step 2 in the general architecture. In these examples, we call this function getInitialTime. Because this function depends on how your servers are programmed, we do not show the actual definition of the function itself.
