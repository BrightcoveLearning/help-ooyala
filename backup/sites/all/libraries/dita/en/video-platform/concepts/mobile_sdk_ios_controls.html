Layout Controls

The DefaultControlsSource directory contains the default Ooyala player skin.

This directory includes source files that control the layout of the video player on the device, both for the default and for inline or fullscreen display, including the following:
OOControlsViewController This method that invokes the player layout controls.
OOFullScreenViewController This method invokes the fullscreen video mode controls.
OOFullScreenControlsView This method invokes options for the controls for fullscreen video mode.
OOImages This method invokes options to control the shape of icons displayed in the player.
OOInlineControlsView This method invokes controls for inline video mode.
OOInlineViewController This method invokes the controller for inline video mode.
OOClosedCaptionsSelectionViewController This method invokes options for closed captions.
OoyalaPlayerViewController This method invokes the primary program for layout control configuration.
OOUIProgressSlider This method invokes options to set characteristics of the progress slider (also called the "scrubber"), such as the image for the slider itself, the display of time elapsed or remaining.
OOUIUtils This method invokes options to configure interface utilities.
OOVolumeView This method invokes options to manipulate the volume slider color and shape.
OOClosedCaptionsSelectorBackgroundViewController The method invokes options for closed caption background selection mode.
OOClosedCaptionsSelectorViewController This method invokes the controller for the closed captions background selector.
