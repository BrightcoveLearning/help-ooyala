Using a Defined Discovery Profile on API Requests

To use a discovery profile in any Discovery API read request, add the following query string parameter:

discovery_profile_id=profile_id

where profile_id is the "player ID" of a previously created profile. This query string parameter supersedes other query string parameters included on the request.

The following example is a request for personalized recommendations asssociated with the discovery_profile_id query string parameter.

[GET]/v2/discover/trending?discovery_profile_id=profile_id
