Player V3 Multi-feature Samples

The Player V3 multi-feature samples combine multiple features to enable you to start coding more quickly.

The code samples currently include:

  • Secure XDR Player Sample—an advanced Player V3 sample that features cross-device resume (XDR), Google IMA integration, Secure Player Token security, and a message bus.
  • HTML5 Sample—a web page and mobile web page sample that features a video embed, search engine optimization (SEO), Twitter card integration, and Ooyala Discovery recommendations.

The code samples consist of JavaScript on the client and PHP on the server. If you are not using PHP, you will need to write equivalent code in your language of choice.

  • HTML5 Player V3 Starting Sample (Deprecated)
    This sample produces web pages with several Ooyala features.
  • Secure XDR Player V3 Sample (Deprecated)
    Secure XDR code sample can help you get started in creating a multi-feature playback experience with Ooyala Cross-Device Resume, Google IMA ads integration, and Ooyala Player Token authentication.
