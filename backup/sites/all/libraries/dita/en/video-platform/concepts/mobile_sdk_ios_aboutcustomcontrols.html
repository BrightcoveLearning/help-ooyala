About Custom Controls for iOS

The Ooyala Mobile SDK folder includes a directory named DefaultControlsSource. This folder contains the base code for all player controls. For more information about customizing controls see Layout Controls and Implementing Inline Custom Controls for iOS.

Note: The SDK does not have an extension or sub-classing mechanism for customizing the provided controls.
