Mobile SDK Ad Integration

The Ooyala mobile SDKs for iOS and Android integrate with ad managers and certain IAB standards.

See the Mobile SDK for Android documentation and Mobile SDK for iOS documentationfor full details on how to integrate ads with the mobile SDKs.

The mobile SDKs support the following ad integrations and IAB standards.

Android Mobile SDK
  • Google IMA (Google DFP)
  • FreeWheel
  • VAST 2.0 and 3.0
iOS Mobile SDK
  • Ooyala Pulse
  • Google IMA (Google DFP)
  • FreeWheel
  • VAST 2.0 and 3.0
