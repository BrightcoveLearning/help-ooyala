Analytics and Labels

Label Queries and Removed Labels

Analytics data will only be returned for non-deleted videos that are associated with the label at report query time. This means that if you remove a label from a video or delete a video, analytics reports will not return data for the video when a label filter is applied.

Example:
  • June 1: Video ‘Movie Trailer’ is uploaded to Backlot, label ‘A’ is added, and the video is published.
  • June 2: Video has 10 plays.
  • June 3: Label ‘B’ is added to the video.
  • June 4: Video has 5 plays.
  • June 5: Label ‘C’ is added to the video.
At this point, if a report is run for the month of June, you will get the following results when applying filters.
Table 1. Plays Returned
Filter Plays for the video 'Movie Trailer'
Label 'A' 15
Label 'B' 15
Label 'C' 15
At this point, if Label ‘A’ is removed from the video and a report is run for the month of June, you will get the following results when applying filters.
Table 2. Plays Returned After Label A Removed
Filter Plays for the video 'Movie Trailer'
Label 'A' No results returned as Label ‘A’ is no longer associated with the video ‘Movie Trailer'
Label 'B' 15
Label 'C' 15

Label Queries and Label Heirarchy

Ooyala IQ currently supports two heirarchy use cases for filtering by label:
  • Filtering by the root label and all of its child labels in the tree.
    Note: If one video asset belongs to multiple child labels, it is only counted once in analytics metrics (there is no double counting).
  • Filtering by a single leaf (child) label.

Example:

You create the following label heirarchy in Backlot:
  • Sports
    • Basketball
      • Spurs
      • Celtics
      • Pistons
    • Football
    • Cricket
If you apply the filter "Sports" to the IQ UI (or query with the IQ API), you will see results for videos that have the "Sports", "Basketball", "Spurs", "Celtics", "Pistons", "Football", or "Cricket" labels applied. Videos with multiple labels (such as "Pistons" and "Celtics") will not be double counted.

If you apply the filter "Basketball" to the IQ UI (or query with the IQ API), you will see results for videos that have the "Basketball", "Spurs", "Celtics", or "Pistons" labels applied. Videos with multiple labels (such as "Pistons" and "Celtics") will not be double counted.

If you apply the filter "Spurs" to the IQ UI (or query with the IQ API), you will see results for videos that have the "Spurs" label applied.
