Managing Closed Captions

You can add closed captions to your videos.

  • About Closed Captions
    Closed captions are additional text shown within a video during playback. The captions typically contain an audio transcript of the video.
  • Uploading a Closed Caption File in Backlot
    After preparing a closed caption file, you can upload it to Backlot.
  • Deleting a Closed Caption File
    You can delete a closed caption file at any time. If you need to edit a closed caption file, make any changes on your computer, delete it within Backlot, and upload the new file.
  • FCC 708 Styling for Closed Captions in Flash Player V3 (Deprecated)
    FCC 708 Compliance supported for Closed Captioning (CC) to allow users to style closed caption appearance on the V3 Flash player.
  • Ingesting Closed Caption Files
    You can ingest closed caption files via Backlot or the Ooyala Ingestion Service (OIS).
  • Deleting a Closed Caption File
    You can delete a closed caption file at any time.
