Using Top Titles

The following example shows the base call for the top_titles mapping route:

href="http://api.ooyala.com/v2/discover/trending?score_type=top_titles

The following example shows a request to return the seven most popular series assets in the last week.

Request:

GET http://api.ooyala.com/v2/discover/trending?score_type=top_titles&show_type=Series
&window=hour&limit=7&signature=xxxxxxxxxxxxxxxxxxxxxxxxxx
&expires=1408138735&api_key=xxxxxxxxxxxxxxx
