Data Migration (Deprecated)

Warning: The v2 Analytics API has been deprecated. See Ooyala IQ Analytics for more details on the new v3 Analytics API.
All customers will have reprocessed data starting from January 1, 2014 to query with Ooyala IQ.

You will still be able to use the v2 Analytics API for single dimensional queries on all of your historical data until June 30, 2016.

Warning: Customers will not have access to the v2 analytics API after June 30, 2016 . As such, you will be required to move any code associated with the v2 API to the corresponding v3 Analytics (Ooyala IQ) API before that date.
To save your historical data from Backlot as CSV using the Backlot UI:
  1. Log in to Backlot.
  2. Click the ANALYZE tab.
  3. Set filters and dimensions so that your report contains the desired content.
  4. Click Save as CSV next to the date selector in the upper right portion of the UI.

To save your historical data from Backlot as CSV using the Backlot API, please see How to Export Your Data With the v2 Analytics API (Deprecated).
