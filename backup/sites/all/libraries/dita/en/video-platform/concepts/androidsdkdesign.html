General Design of the SDK

In general, the SDK has two parts:

  • The content management functions include loading a video and its associated metadata from embed code (content identifiers), and querying for more videos. This function includes controlling video playback on a predefined player skin.

    These functions are essential for playback.

  • The user interface layout controls that include the provided skins, and positioning of the video surface within the application.

    These controls can be replaced entirely, either from scratch or with the provided controls as a starting point. To run custom controls over the Ooyala Player, see Implementing Custom Controls for Android.
