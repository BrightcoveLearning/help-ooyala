Recovering from Transcoding Errors

The error status of an asset usually indicates problems in transcoding.

If the status of an asset is "error," the most likely reason is a problem transcoding from the file that was uploaded. Whenever a movie fails in transcoding for some reason and after all possible retries have been attempted, the error status is set.

Attempt to upload the file again to see if the error was only transitory due to telecommunications of data storage problems.

If the error state persists, contact Technical Support.
