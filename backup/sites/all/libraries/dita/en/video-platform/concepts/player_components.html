Player V3 Components (Deprecated)

Visual components of Ooyala Player V3.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).

This topic describes the visual components of Ooyala Player V3.

Key Components

Here are the key components of the player.

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
display area Where the video is displayed.
scrubber bar Enables viewers to manually move forward or backward through a video. The scrubber bar can be positioned at the top, bottom, left side, or right side of the player.
buttons Click to share the video on social networks, copy the embed code, view additional information, or show the channel associated with the video. Buttons can be positioned in various places on the player.

Scrubber Bar

The scrubber (also known as a slider) enables viewers to manually move forward or backward through a video.

Here are the visual elements of the scrubber bar in the READY or PAUSED state:

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
Play button Click to start video or audio playback.
video quality (bitrates) Click to display the current video quality. The bitrate is the number of bits delivered in a particular unit of time, typically measured in seconds. In general, the higher the bit rate, the higher the quality of the playback experience.
fullscreen Click to display the video in fullscreen mode.
volume control Click the volume control and drag the slider to adjust the volume
Here are the visual elements of the scrubber bar in the PLAYING state:

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
Pause button Click to pause playback.
scrubber (slider) Drag and slide to manually move forward or backward through a video.
play head position Current position during playback (hours:minutes).
content duration Total length of the asset (hours:minutes).
To configure scrubber bar options in Backlot, see:
  • Backlot GUI: Configuring Basic Player Settings for Flash Player V3 (Deprecated)
  • Backlot API: Setting up Players Using the Backlot REST API (Deprecated)

Buttons

The player displays buttons that can provide options to share the video on social networks, copy the embed code, view additional information, or show the channel associated with the video.

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
Share button Click to see available options to share the asset via social media or URL.
Embed button Click to get an embeddable URL for the asset.
Info button Click to display additional information about the asset.
Channel button Click to show the channel associated with the asset.
To configure these buttons in Backlot, see:
  • Backlot GUI: Configuring Basic Player Settings for Flash Player V3 (Deprecated)
  • Backlot API: Setting up Players Using the Backlot REST API (Deprecated)

Start Screen

The start screen shows before a video starts.

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
Play button text Prompt below the play button.
Title Asset title.
Description Brief description of the asset.
To configure the appearance and functionality of the start screen in Backlot, see:
  • Backlot GUI: Configuring the Start Screen for Flash Player V3 (Deprecated)
  • Backlot API: Setting up Players Using the Backlot REST API (Deprecated)

End Screen

The end screen shows after a video completes playback, which can provide options to replay the video, view similar content, share the video on social networks, copy the embed code, and view additional information.

Note: The above example screen uses Ooyala default desktop web settings. Your screen might look different.

# Name Description
buttons Click to replay the video, share the video on social networks, copy the embed code, view additional information, or show the channel associated with the video.
more videos Click to scroll through related or similar assets.
To configure the appearance and functionality of the end screen in Backlot, see:
  • Backlot GUI: Configuring the End Screen for Flash Player V3 (Deprecated)
  • Backlot API: Setting up Players Using the Backlot REST API (Deprecated)
