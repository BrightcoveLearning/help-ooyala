Setting the Default Browser for Learn More Events

Use the set.UserDefaultBrowsermethod to control how the Learn More link opens when activated.

For Google IMA ads, with the Learn More feature, you can configure how the link opens when activated. This option is configured using the set.UserDefaultBrowser method. This method defaults to false by default. If the method is not directly called in the app the Learn More button will open in the native application. Configure the button to open in the default browser by adding the following methods to your main activity: 

-(void)setUseDefaultBrowser: true
