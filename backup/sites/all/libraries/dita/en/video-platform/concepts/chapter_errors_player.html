Player V3 Errors Overview (Deprecated)

Troubleshoot player issues using error lists and error handling topics.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).

Discussed here are the types of Player V3 errors and error handling provided by Ooyala.

  • Handling Errors for an HTML5 Player V3 Using JavaScript (Deprecated)
    Error Handling functions and events enable you to get information about Player V3
  • HTML5-based Player V3 Error Messages (Deprecated)
    From the OO.EVENTS.ERROR event, the Ooyala Player can show specific error messages.
  • Flash-based Player V3 Error Messages (Deprecated)
    Error messages are displayed to the video viewer or audio listener during playback. Errors can be categorized as general errors, Flash-based errors and Pay-Per-View errors, which are prefixed with a PPV.
  • Player Errors from Other Sources (Deprecated)
    The player displays errors that originate from other sources.
