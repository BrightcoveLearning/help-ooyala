Supported Operating Systems and Browsers

To use the Theme Builder UI, you must use a supported operating system and browser.

Supported Operating Systems

The Theme Builder UI supports the following operating systems:
  • Windows 10
  • Windows 8+
  • Windows 7
  • Mac OS X 10.9
  • Mac OS X 10.8
  • Mac OS X 10.7

Supported Browsers

The Theme Builder UI supports the following browsers:
  • IE 10
  • IE 11
  • Chrome 32
  • Chrome 33
  • Chrome 34
  • Firefox 26
  • Firefox 27
  • Firefox 28
  • Safari 6
  • Safari 7
