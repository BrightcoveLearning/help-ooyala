eCommerce Version 3 Release

New RAML API for eCommerce Version 3

The eCommerce API is now available in RAML. See the developer docs at https://docs.brightcove.com/apidocs-ooyala/. The eCommerce API is a generalized, open eComm framework. The APIs in the eCommerce Version 3 include the ability to register and confirm payment methods, as well as to create subscriptions and one-time rentals. Please contact your Ooyala Customer Support Manager for more information.
