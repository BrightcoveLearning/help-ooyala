2016-05-23 Release

Notable contents of this release include an update to the V3 ActionScript Flash Player and updated Player V4 documentation.

Software Feature Releases

V3 Flash Player Update

With today's release, the Ooyala V3 ASP (ActionScript) Flash player will always show the closed caption button even if the stream being played does not have closed captions. The closed caption button was implemented this way to quickly implement 608/708 closed captions. We are working on fixing this implementation in a future release.

Documentation Releases

  • Customizing the Player V4 Appearance Using CSS: Updated the examples to show the updated CSS file values.
  • Basic Tutorial for Player V4: Updated the basic tutorial to include a step about hosting font and image assets.
  • Programming Bitrates and Buffering for Player V4 Using JavaScript: Added a note about how ABR and bitrate selection are currently mutually exclusive.
