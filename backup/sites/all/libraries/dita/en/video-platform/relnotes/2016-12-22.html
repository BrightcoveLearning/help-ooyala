2016-12-22 Release

Notable contents of this release include mobile SDK enhancements for download to own (DTO) and download to rent (DTR), Player V4 support for Accelerated Mobile Pages (AMP), a new REST API to query the ingestion log for Ooyala Ingestion Services (OIS) v2.5, and a deprecation notice for the Discovery Insights weekly email report.

Software Feature Releases

Offline Playback: Download to Own (DTO) and Rentals (DTR): The Ooyala Android and iOS SDKs now provide users with the ability to download video assets so that they can be played offline. For more information, see Introduction to Offline Playback: Download to Own (DTO) and Rentals (DTR).

Player V4 support for Accelerated Mobile Pages (AMP) To optimize playback on mobile web browers, you can now use Ooyala's Accelerated Mobile Pages (AMP) plugin. See Using Ooyala Player V4 with Accelerated Mobile Pages (AMP) for details.

Ingestion Log REST API for Ooyala Ingestion Services (OIS) v2.5. If you use OIS 2.5, you can now query the ingestion log using REST API calls. For details, see Ingestion Log REST API Docs and Ingestion Log. The ooyala.log file exists only for OIS v1.
