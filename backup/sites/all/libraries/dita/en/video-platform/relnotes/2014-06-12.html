2014-06-12 Release

Notable changes include sharing Integration for Adobe Flash and documentation updates to the iOS and Android Sample Apps table.

Software Feature Releases

The following are new software release items.

Sharing Integration for Adobe Flash

The “Share URL by Email” feature will be removed for an interim period to implement robustness improvements. The following updates have been made to support this change:

  • The “Email a Friend” option is no longer displayed on the player share panel.

  • The “Share by Email” Backlot REST API has been removed.

  • The option to select “Email to a Friend” in the Theme Builder UI still exists. The option will be removed in the future.

    If the “Email to a Friend” option is selected in the UI, the player share panel will not display the “Email a Friend” button.
