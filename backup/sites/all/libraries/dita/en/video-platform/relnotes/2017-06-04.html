Ooyala Video Platform Deprecation Announcements

Ooyala is deprecating the following functionality. For more information, contact your Ooyala representative.
  • Ooyala Player V3 (Deprecated)

    Ooyala no longer adds or fixes Player V3 functionality. If you are still using Player V3, you need to upgrade to Player V4 instead (see Migrating to Player V4). After July 31, 2017, Player V3 will no longer be maintained under any service-level agreement.

  • Ooyala Player V2 (Deprecated)

    Ooyala no longer adds or fixes Player V2 functionality - see V2 Player APIs (Player V2 Deprecated). If you are still using Player V2, you need to switch to Player V4 instead. After July 31, 2017, Player V2 will no longer be maintained under any service-level agreement.

  • Channels and Channel Sets (Deprecated)

    Channels and Channel Sets are supported only in Ooyala Player V3, which has been deprecated. Ooyala no longer adds or fixes channel / channel set functionality. If you are still using channels / channel sets, you need to migrate to using Player V4 and Using Playlists in Player V4 instead. Channels and channel sets are scheduled to be disabled on July 31, 2017.

  • Akamai HD Video Plugin for Akamai Packaged HDS for Player V4 (Deprecated)

    Ooyala no longer adds or fixes Akamai plugin functionality. If you are still using the Akamai plugin, you need to use a different Player V4 video plugin instead. The Akamai plugin is scheduled to be disabled on August 31, 2017.

  • OSMF Flash Video Plugin for HDS for Player V4 (Deprecated)

    Starting August 31st, 2017, Ooyala will no longer add or fix OSMF plugin functionality. If you are still using the OSMF Flash plugin, you need to use a different Player V4 video plugin instead.

  • The Hook Mobile Player App (Deprecated)

    The Hook Mobile Player App will be removed from the Google Store on June 15, 2017. Ooyala will no longer add or fix functionality. If you are still using the Hook Mobile Player App, switch to using the Chrome Mobile browser instead. This app is scheduled to be disabled on June 30, 2017.

  • Backlot Live Streams (Deprecated)

    Backlot Live streaming (video and audio) will be removed from Backlot on June 30, 2017. Ooyala will no longer add or fix functionality. If you are still using Backlot Live streaming, you need to switch to Ooyala Live instead.

  • Adobe Access DRM (Deprecated)

    Ooyala integration with Adobe Access DRM has been deprecated and will be disabled on August 31, 2017. Ooyala will no longer add or fix functionality. If you are still using the Adobe Access DRM, you need to switch to one of our other Digital Rights Management solutions instead.

  • Widevine Classic DRM (Deprecated)

    Ooyala integration with Widevine Classic DRM has been deprecated and will be disabled on June 30, 2017. Ooyala will no longer add or fix functionality. If you are still using the Widevine Classic DRM, you need to switch to one of our other Digital Rights Management solutions instead.

  • Discretix Playready DRM (Disabled)

    Ooyala integration with Discretix Playready DRM for mobile apps has been disabled. Further transcoding of assets are no longer possible. If you are still using the Discretix Playready DRM, you need to switch to one of our other Digital Rights Management solutions instead.
