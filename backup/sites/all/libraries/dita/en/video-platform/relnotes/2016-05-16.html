2016-05-16 Release

Notable contents of this release include support for the Authorization API in Player V4 and updates to Widevine Modular Content Protection.

Documentation Releases

  • Widevine Modular Content Protection: Changed URL and updated contents of Widevine Modular Content Protection documentation.
  • Content Protection Options By Device: Updated list of supported content protection options by device.
  • Akamai Secure Token: Updated Akamai Secure Token to indicate support in Player V3.
  • PlayReady Content Protection: Updated PlayReady contents for Widevine Modular.
  • Player Security FAQ: Updated security FAQ contents for Widevine Modular.
  • Player Authorization API for Player V4: Added support in Player V4 for the Authorization API.
  • Player Authorization API for Player V3 (Deprecated): Updated the Authorization API for Player V3.
