2014 Video Platform Release Notes

Release Notes from 2014.

  • 2014-12-17 Release
    Notable changes include Google IMA improvements for the HTML5 player and the release of the mobile SDK v.3.2.1 for iOS and Android.
  • 2014-12-04 Release
    Notable changes include the migration from Google IMA 1.x and 2.x to Google IMA v3, new Flash player parameters, and a mobile SDK release.
  • 2014-11-10 Release
    Notable changes include the release of v.3.1.0 of the Ooyala mobile SDKs for iOS and Android.
  • 2014-11-03 Release
    Notable changes include the release of v.3.0.1 of the Ooyala mobile SDK for iOS.
  • 2014-10-27 Release
    Notable changes include the release of anonymous proxy controls and the adRequestTimeout Flash parameter.
  • 2014-10-13 Release
    Notable changes include the release of v.3.0.0 of the iOS and Android mobile SDKs.
  • 2014-09-22 Release
    Notable changes include the Ooyala Chromecast HTML5 player and updated Discovery documentation.
  • 2014-09-16 Release
    Notable changes include the release of the Ooyala mobile SDK for iOS v.2.7.1 and a documentation update to Player V3 error handling.
  • 2014-09-08 Release
    Notable changes include the new Tinypass integration and the change in location of the documentation PDFs from the SDKs tab to the Documentation tabs.
  • 2014-08-13 Release
    Notable changes include information on the upgraded Android SDK and the release of the Android and iOS SDKs v.2.7.0.
  • 2014-07-29 Release
    Notable changes include new default player features. In the Flash player you can now hide the control bar during ad playback and apply a TV rating watermark to your assets. These features will be available for HTML5 soon. In the HTML5 player you can now programmatically override any VAST ads or ad sets applied in Backlot.
  • 2014-07-17 Release
    Notable changes include the release of v.2.6.0 of the iOS and Android mobile SDKs.
  • 2014-07-02 Release
    Notable changes include the release of v.2.5.0 of the iOS and Android mobile SDKs and a new optional parameter for the destroy() function in the Player API (HTML5 only).
  • 2014-06-12 Release
    Notable changes include sharing Integration for Adobe Flash and documentation updates to the iOS and Android Sample Apps table.
  • 2014-06-03 Release
    Notable changes include software updates to iOS SDK v. 2.4.0, Android SDK v. 2.4.0, Discovery, and documentation updates for iOS SDK v. 2.4.0, Android SDK v. 2.4.0, TinyPass Paywall Integration, Google IMA Ad Rules, Facebook Active Sharing, Theme Builder supported operating systems and browsers, DFXP, and country code update in Backlot.
  • 2014-05-19 Release
    Updates include software updates to Google IMA, Player Debugging, and documentation updates for Mobile SDK Integration with FreeWheel, Google IMA macro support, Player Debugging and Player Message Bus events.
  • 2014-05-06 Release
    Updates include software updates to Backlot Syndication to YouTube, Flash Desktop Player FCC 708 Compliance for Closed Captions, and an update to Country and Location Codes.
  • 2014-04-22 Release
    Updates include software updates to the Player API and iOS and Android mobile SDKs, the addition of an Ad Integration guide, and an update to a Hook script snippet.
  • 2014-04-07 Release
    Updates include new Theme Builder documentation, Discovery documentation (including Discovery Profile), new FAQs, additions to the rights locker and Player API, Core Platform documents, and fixes in the Backlot API documentation.
