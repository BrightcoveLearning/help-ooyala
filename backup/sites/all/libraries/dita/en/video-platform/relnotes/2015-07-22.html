2015-07-22 Release

Notable changes for this release include the release of Hook v.1.5.0.

Software Feature Releases

Hook Release

Hook version 1.5.0 is now available for download from the Google Play Store. This release provides improved video playback and takedown security via AuthToken validation.

Mobile SDK for iOS 4.3.1

Release 4.3.1 Released 7/15/15

Resolved Issues

  • An issue where Google IMA ads sometimes played when the back button was activated is now resolved.
  • An issue where an ad fail event would end video playback is now resolved.
  • An issue where failure to release resources caused apps to crash is now resolved.
