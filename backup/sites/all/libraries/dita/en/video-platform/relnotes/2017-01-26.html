2017-01-26 Release

Ooyala Mobile SDKs now provide volume getter and setter methods.

Software Feature Releases

Ooyala Mobile SDKs now provide volume getter and setter methods that can be used, for example, in cases of Auto-Play with Mute. Note that these methods do not work for IMA or Freewheel ads, but do work for content and VAST and Ooyala ads. See the Mobile SDK release notes for Android and Mobile SDK release notes for iOS for more details.
