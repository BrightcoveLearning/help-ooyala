2016-01-08 Release

Notable contents of this release include new integration and customization documentation for the Ooyala integration with Chromecast.

Documentation Releases

  • Getting Started with Chromecast: Added a new getting started guide for the Ooyala integration with Chromecast.
  • Customizing the Sender Application for the Ooyala Chromecast Integration: Added new documentation that explains how to customize a sender application for the Ooyala integration with Chromecast.
  • Customizing the Receiver Application for the Ooyala Chromecast Integration: Added new documentation that explains how to customize a receiver application for the Ooyala integration with Chromecast.
  • Supported Browsers: Made it clearer that the IE browser is not supported.
  • DRM Attributes for Remote Assets (Including Live Streams): Fixed the formatting of a license URL so the entire URL appears in the text.
