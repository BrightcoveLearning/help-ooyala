2016-06-16 Release

Notable contents of this release include added properties for the Widevine Modular Policy to configurable DRM policies.

Documentation Releases

  • DRM Policies: Added properties for widevine_modular_policy.
  • Common Metric Definitions and Examples: Added a definition and example for the hours watched metric.
  • Supported Configurations for iOS: Updated the list of devices supported by the mobile SDK for iOS.
  • Ooyala-hosted Player V4 Resources: Updated notes related to the Pulse ad plugin for Player V4.
