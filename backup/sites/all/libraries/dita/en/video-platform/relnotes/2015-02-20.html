2015-02-20 Release

Notable changes for this release include updates to the Chromecast receiver player.

Chromecast Receiver Player

The Ooyala Chromecast Receiver player has been updated to use the Google Media Player Library (MPL) version 1.0.0. See Google's release notes for MPL at https://developers.google.com/cast/docs/release-notes under "Media Player Library".

Documentation Releases

  • Chromecast
    • Integrate with Ooyala Chromecast HTML5 Player - Updated to show that the Chromecast receiver player uses the Google Media Player Library v.1.0.0.
  • Analytics Delivery Report
    • Delivery (Deprecated) - Updated description of the report. The delivery report returns the number of VOD bytes delivered for all content within your account or for an individual asset.
