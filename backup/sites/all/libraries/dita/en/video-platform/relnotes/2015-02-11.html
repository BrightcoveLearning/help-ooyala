2015-02-11 Release

Notable changes for this release include documentation on Adobe Access and new default V3 player fallback behavior.

New Fallback Behavior for Default V3 Player

We have changed the default behavior of the V3 player. When Flash is not available, the player will now default to html5-fallback (if available).
