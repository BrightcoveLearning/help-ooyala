Backlot API v2: Get Channel Set Information

[GET] https://api.ooyala.com/v2/assets/<channelSetId>

Backlot API v2: Get Channel Set Lineup Information

[GET] https://api.ooyala.com/v2/assets/<channelSetId>/lineup

Examples

The following example gets information about a channel set:

[GET] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v

Backlot returns a response similar to the following:

{
  "asset_type": "channel_set",
  "duration": 0,
  "name": "My Channel Set",
  "preview_image_url": null,
  "created_at": "2011-08-10T17:45:24+00:00",
  "embed_code": "dvNTVxMjrRktSlb5v",
  "time_restrictions": null,
  "updated_at": "2011-08-10T17:45:24+00:00",
  "external_id": null,
  "hosted_at": null,
  "original_file_name": null,
  "description": null,
  "status": "live"
}

The following example gets the lineup for the specified channel set:

[GET] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v/lineup

Backlot returns a response similar to the following:

[
  "IzNnllMjphu2XF3",
  "Y1dTdvMjq9QtOMG"
]
