Player Pass-Through Parameters (Player V2 Deprecated)

This topic lists and describes the Query String Pass-Through Parameters for customizing the player.

Pass-Through Parameter Reference

Important: Ooyala Player V2 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V2 will no longer play your video or audio content. Customers still using Player V2 need to migrate to Player V4 (see Player V4.

The following table lists and describes the query string pass-through parameters.

Parameter Description
transition

Specify play to show the next movie or specify selector to show the channel browser after each video in a channel. The default is play.

view Specify whether the initial state of the player is normal (shows the video) or the channel. Default is normal.
width Specify the width, in pixels, of the player.
wmode Sets the transparency, layering and positioning in a browser. Possible values are one of the following:
  • window
  • opaque
  • transparent
Opaque players sit on the lowest level whereas transparent players sit at the top most level. Default is window.
