Embedding the Ooyala Upload Widget (Deprecated)

To embed the widget, you must include a code snippet in the body of a web page.

The Upload Widget loads the Flash component that handles the upload. The dimensions of the upload button are controlled by the width and height parameters and the label of the button is specified by the label parameter.

<script src="http://www.ooyala.com/partner/uploadButton?width=100&height=20&label=Browse%20Button"></script>

To include a custom image for the browse button, a URI-encoded URL can be specified using the skin parameter. This image should be composed of three separate, stacked images representing the button states of up, over, and down. Each image should be sized with the height and width specified in the embed string. In this example, the referenced image is at https://www.ooyala.com/static/partner/example_skins.png and is a total size of 180x75.

<script src="http://www.ooyala.com/partner/uploadButton?width=180&height=25&label=Browse%20Button&skin=http%3A%2F%2Fwww.ooyala.com%2Fstatic%2Fpartner%2Fexample_skins.png"></script>

Next, implement the onOoyalaUploaderReady function which will be called when the Upload Widget is ready. You can add event listeners, set your parameters, and add authentication in this function. To access the Uploader Widget from JavaScript, use the ooyalaUploader Flash object, but do not invoke methods directly upon ooyalaUploader.
