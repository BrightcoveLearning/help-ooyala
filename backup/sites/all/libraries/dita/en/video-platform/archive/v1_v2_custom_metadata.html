Custom Metadata API (Deprecated)

The custom metadata API is now part of the assets API. You can now view, update, replace, or delete custom metadata or an asset.

For more information about the custom metadata API, see Custom Metadata.

Backlot API v1

http://api.ooyala.com/partner/set_metadata
?pcode=<pcode>
&signature=<signature>
&expires=<expiration>
&embedCode=<embedCode>
&attribute1=<new value>
&attribute2=<new value>
&delete=writer%00copyright

Backlot API v2

[PATCH] /v2/assets/<asset_id>/metadata
{
  properties
}

Examples

This example creates custom metadata for the specified asset:

[PUT] /v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/metadata
{
 "director" : "Michel Gondry",
 "year" : "2004",
 "rating" : "R"
}

Backlot returns a response similar to the following:

{
 "director" : "Michel Gondry",
 "year" : "2004",
 "rating" : "R"
}

This example replaces the director for the specified asset:

[PATCH] /v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/metadata
{
 "director" : "Spike Jonze"
}

Backlot returns a response similar to the following:

{
 "director" : "Spike Jonze",
 "year" : "2004",
 "rating" : "R"
}
