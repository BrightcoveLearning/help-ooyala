Backlot API v2: Single Label

[PUT] /v2/assets/<asset_id>/labels/<label_id>

Backlot API v2: Multiple Labels

[POST] /v2/assets/asset_id/labels/
[
  "label_id",
  "label_id",
  ...
]

Example

The following example creates a label:

[PUT] /v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/labels
[
  "638aed2a18734375b3b332cb85999dd2", 
  "814efb109416490a98ee3f4fcd6784cf"
]

Backlot returns a response similar to the following:

{
  "items": [
    {
      "name": "Sports",
      "id": "814efb109416490a98ee3f4fcd6784cf",
      "parent_id": null,
      "full_name": "/Sports"
    },
    {
      "name": "Hobbies",
      "id": "638aed2a18734375b3b332cb85999dd2",
      "parent_id": null,
      "full_name": "/Hobbies"
    }
  ]
}
