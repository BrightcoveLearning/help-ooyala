Delivery (Deprecated)

The delivery report returns the number of VOD bytes delivered for all content within your account or for an individual asset. The delivery API has been migrated to the Usage REST API.

Warning: The v2 Analytics API has been deprecated. See Ooyala IQ Analytics for more details on the new v3 Analytics API. The delivery API has been migrated to the Usage REST API.

Get Video Analytics

[GET]/v2/analytics/reports/level/delivery/total/date_range

Examples

This example returns the bytes delivered for one day: 

[GET]/v2/analytics/reports/account/delivery/total/2011-09-01

which returns a response similar to the following:

{  
   "results":[  
      {  
         "id":"total",
         "metrics":{  
            "bytes_delivered":91625071328
         }
      }
   ]
}

This example returns the bytes delivered for one asset: 

[GET]/v2/analytics/reports/asset/dmbHViOgvgZMqXtY40DWmh4DuW6Ie6I6/delivery/total/2011-09-01

which returns a response similar to the following:

{  
   "results":[  
      {  
         "id":"total",
         "metrics":{  
            "bytes_delivered":625071328
         }
      }
   ]
}
