Examples

The following example replaces the current channel lineup:

[GET] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v

Backlot returns a response similar to the following:

[
  "IzNnllMjphu2XF3",
  "Y1dTdvMjq9QtOMG"
]

The following example adds an asset to the current channel lineup:

[PATCH] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v/lineup/rtsfds23xFTrejrtUe

Backlot returns a response similar to the following:

[
  "IzNnllMjphu2XF3",
  "Y1dTdvMjq9QtOMG"
  “rtsfds23xFTrejrtUe”
]
