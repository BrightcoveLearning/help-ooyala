Player JavaScript APIs (Player V2 Deprecated)

Ooyala Player JavaScript APIs enable you to use this popular scripting language to customize the Ooyala Player and video experience.

Important: Ooyala Player V2 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V2 will no longer play your video or audio content. Customers still using Player V2 need to migrate to Player V4 (see Player V4.

The following topics describe the Player JavaScript APIs. You can view all of the APIs in a single file, or select the API information by functional categories such as APIs used for ads, close captioning, Player and so on. A separate topic contains examples that illustrate the use of these JavaScript APIs.

  • JavaScript API Summary List (Player V2 Deprecated)
    Lists and describes all Player JavaScript APIs grouped by Functions/Methods, Objects, Properties and Events.
  • Ads JavaScript APIs (Player V2 Deprecated)
    Ads JavaScript APIs can be used to respond to events such as starting, clicking, completing and skipping ads.
  • Assets JavaScript APIs (Player V2 Deprecated)
    Assets ActionScript APIs can be used to handle ratings, videos (play/pause, describe), and channels.
  • Bitrate and Buffer JavaScript APIs (Player V2 Deprecated)
    The Bitrate Selection API allows developers to programmatically obtain buffer information, as well as get and set bitrates.
  • Close Captioning JavaScript APIs (Player V2 Deprecated)
    Close Captioning JavaScript APIs enable you to get supported closed captions, set close caption languages, and get the caption timing.
  • Embed Code JavaScript APIs (Player V2 Deprecated)
    Embed Code JavaScript APIs can be used to handle getting and setting embedCode for assets, getting asset descriptions, getting images and fetching metadata for assets.
  • Error JavaScript APIs (Player V2 Deprecated)
    Error JavaScript APIs can be used to get error codes and error text.
  • Player JavaScript APIs (Player V2 Deprecated)
    Player/Player UI JavaScript APIs can be used to handle aspects of the player such as fullscreen mode, player state, volume, playhead, player volume and more.
  • JavaScript Demo and Examples (Player V2 Deprecated)
    JavaScript Examples demonstrate the embedding capability of the Ooyala player and the capability to add additional features to the player using the Player API.
