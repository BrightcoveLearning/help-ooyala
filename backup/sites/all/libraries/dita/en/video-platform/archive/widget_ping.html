Upload Completion Ping URL (Deprecated)

Ooyala will notify you when videos are uploaded into your account if you set up a completion ping URL.

The ping will occur after content has finished processing. In the Backlot ACCOUNT tab, under the Developers subtab, there is a field for an API Ping URL. Here you can enter the URL to a script on your site that will receive a GET request when the processing of an API-uploaded video has completed. The request points to the ping url and appends the embed code of the video that has been uploaded. For example, if you enter the URL as http://www.ooyala.com/complete.php, the full request sent on completion would be http://www.ooyala.com/complete.php?embedCode=<embedCode>.
