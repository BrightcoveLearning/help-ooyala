Query String and Pass-Through Parameters (Player V2 Deprecated)

Ooyala Player Query String and Pass-Through Parameters enable you to pass basic customizations to the Ooyala Player.

Important: Ooyala Player V2 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V2 will no longer play your video or audio content. Customers still using Player V2 need to migrate to Player V4 (see Player V4.

The following topics describe these Query string and Pass-Through Parameters Player API Parameters. The All Query String and Pass-Through Parameters topic lists and describes all the parameters in one file, while the other topics describe the Query String and Pass-Through Parameters by functional categories such as APIs used for Ads, Assets, Player and so on.

  • Player Query String Parameters (Player V2 Deprecated)
    Lists and describes all Player Query String Parameters
  • Player Pass-Through Parameters (Player V2 Deprecated)
    This topic lists and describes the Query String Pass-Through Parameters for customizing the player.
  • Ads Pass-Through Parameters (Player V2 Deprecated)
    This topic lists and describes the Player Pass-Through Parameters for advanced ad tracking and targeting.
  • Query String Parameter Examples (Player V2 Deprecated)
    Examples of query string parameters that can be passed to the Ooyala Player for basic customizations.
