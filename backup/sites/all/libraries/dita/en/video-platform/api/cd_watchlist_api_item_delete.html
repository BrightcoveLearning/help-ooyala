Delete an Item from a Watchlist

This endpoint deletes the specified item from the specified watchlist.

HTTP Request

DELETE https://player.ooyala.com/watchlist/v3/<watchlist_id>/items/<item_id>?account_token=<account_token>

Query Parameters

Parameter Description Type Required
api_version Must be v3. String Yes (URL)
watchlist_id The ID of the watchlist containing the items to delete. Must be a valid UUID linked to an existing watchlist associated with the user's account. If not specified, items are deleted from the user's default watchlist. UUIDv4 OR String No (URL)
item_id ID of the item to delete. Must be a valid ID linked to an item that exists on the specified watchlist.
Note: After an item is deleted, the positions of all the other items in the watchlist are updated automatically.
 String Yes (URL)
account_token Used by the Watchlist Service to reference the watchlist associated with a particular end user (identified by their UUID). For details, see Ooyala Account Token API. String Yes (URL body)

Return Codes

Code Reason Description
200 OK DELETE request succeeded. The item was deleted.
400 Request error Failed due to any of the following issues:
  • Parameters are missing or formatted incorrectly.
  • The specified item does not exist in the watchlist.
  • The specified watchlist_id is not empty and does not exist.
401 Unauthorized The specified account_token is invalid.
404 Not found The server is down.
500 Internal server error An internal server error occurred.
