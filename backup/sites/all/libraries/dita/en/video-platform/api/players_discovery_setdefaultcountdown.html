Setting the Default Countdown

Discovery recommendations come as a sequence of recommended videos. After a video has been played, the Up Next appears in the player with the title of the next video. This setting in the profile specifies the amount of time between the two videos.

Note: Up Next does not appear during ad playback.

The following request and response example shows how to use the PATCH method and the default_countdown property to set the default countdown to 10 seconds:

Request:

[PATCH] /v2/players/{player_id}/discovery/profile
{
"discovery_interface": {
"default_countdown": "10"
}

Response:

{
"discovery_interface": {
"default_countdown": "10",
"show_pause_screen": "false",
"hide_thumbnail_title": "false",
"when_dosne": ""
},
...
