Using a Previously Created Discovery Profile

To leverage an existing discovery profile in a Discovery API request, add the query string parameter discovery_profile_id=profile_id. The profile_id is the identifier for a previously created profile. This query string parameter supersedes other query string parameters included on the request.

The following request and response example shows a request for personalized recommendations, modified by the discovery_profile query string parameter.

Request:

[GET] /v2/discover/trending?discovery_profile_id={profile_id}
