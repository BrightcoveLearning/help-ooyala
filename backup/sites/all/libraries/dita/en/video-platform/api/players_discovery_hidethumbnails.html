Hiding Thumbnails

The following request and response example shows how to how to update a specified profile to hide the player thumbnail title using the PATCH method and the hide_thumbnail_title property set to true.

Request:

[PATCH] /v2/players/{player_id}/discovery/profile
{ 
  "discovery_interface": { 
      "hide_thumbnail_title": "true" 
  } 
}

Response:

{
"discovery_interface": {
"default_countdown": "0",
"show_pause_screen": "false",
"hide_thumbnail_title": "true",
"when_done": ""
},
...
