Custom Analytics: Reporting (Deprecated)

The custom analytics report shows performance information for tags that you configured.

Warning: The v2 Analytics API has been deprecated. See Ooyala IQ Analytics for more details on the new v3 Analytics API.

Get Custom Analytics

[GET]/v2/analytics/reports/account/performance/tags/{  
   date_range
}
[GET]/v2/analytics/reports/asset/asset_id/performance/tags[  
   date_range
]
[GET]/v2/analytics/reports/asset/asset_id/performance/tags/tag_id[  
   date_range
]

Examples

This example returns the custom analytics for the account for a single day, sorted by displays: 

[GET]/v2/analytics/reports/account/performance/tags/2011-08-01...2011-08-02?order_by=displays
Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.
