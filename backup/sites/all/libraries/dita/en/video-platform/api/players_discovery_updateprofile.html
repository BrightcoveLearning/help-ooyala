Updating a Discovery Profile

After creating a profile in the Backlot UI, you can update it using the discovery profile properties. The API request strings require the player ID. The following request and response examples show a request to list of all available players and the response that includes each player with the associated player_id.

Request:

[GET] /v2/players

Response:

{
"items": [ {
"id": "{player_id}",
"name": "ForProfiles",
"watermark": {
"alpha": "1.0",
"image_url": null,
"scaling_percentage": null,
"click_url": null,
"position": "bottom-right",
"scaling_option": "no_scale"
},
...
