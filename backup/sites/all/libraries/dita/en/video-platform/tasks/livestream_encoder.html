Setting Up an Encoder

Setting up an encoder involves configuring it with the Backlot-provided URLs when you set up the live stream.

Note: Backlot Live streaming has been deprecated and is scheduled to be disabled. For details and alternatives, see the OVP Release Notes.
Before you can configure the encoder, you must have already create the video asset on Backlot or with the Backlot API as detailed in Adding a Live Stream.

To set up an encoder:

  1. Open the encoder's configuration interface.
  2. If you will be generating Flash streams, navigate to the page where you configure Flash streams and enter the primary and secondary URLs. On the Spinnaker 7100, do the following:
    1. Click the H264 tab.
    2. Click the MP4 Output subtab.
    3. Enter master in the Variant playlist filename(s) field.
    4. In the Streaming to Primary Flash Media Server section, enter the value of the primary_entry_point property received in the response with the live stream was created in the Server Location URL field. (See Adding a Live Stream for details.)
    5. In the Streaming to Secondary Flash Media Server section, enter the value of the backup_entry_point property received in the response with the live stream was created in the Server Location URL field. (See Adding a Live Stream for details.)
    6. Click Apply.
  3. If you will generate iOS streams, navigate to the page where you configure Flash streams and enter the upload and delivery URLs. On the Spinnaker 7100, do the following:
    1. Click the H264 tab.
    2. Click the iPhone Output subtab.
    3. Enter master in the Variant playlist filename(s) field.
    4. Enter the value of the ios_live_upload_url property received in the response with the live stream was created in the Storage URL field, removing "/master.m3u8" from the end of the URL. (See Adding a Live Stream for details.)
    5. Enter the value of the ios_live_delivery_url property received in the response with the live stream was created in the Publishing URL field, removing "/master.m3u8" from the end of the URL. (See Adding a Live Stream for details.)
    6. Click Apply.
