Deleting a Closed Caption File

You can delete a closed caption file at any time.

To delete a closed caption file:

Use DELETE, the /v2/assets route., the asset ID, and the /closed_captions qualifier. The following example deletes closed captions from the JxbzdkMjqB asset: 
[DELETE]/v2/assets/JxbzdkMjqB/closed_captions

Backlot returns a 200 response.

Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.
