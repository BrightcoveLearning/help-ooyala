Customizing the iOS Sender Application

Important: This topic contains older information and has been deprecated. For the latest information about Ooyala integration with Chromecast, see Chromecast Integration.
  1. Customized Mini Controller
    1. Open CustomizedMiniControllerView.m and adjust the values according to your application needs.
  2. Additional steps to take in the customized mini controller.
    1. Take the castManager as a property in your class.
    2. Instantiate play/pause the casting video through [self.castManager.castPlayer play]/[self.castManager.castPlayer pause].
    3. Update the play/pause button image.
    4. Manage cast mode video view (The UIView fills in the blank video view in OOOoyalaPlayer when casting video).
    5. Set the castModeVideoView in castManager as shown in the following code sample: [self.castManager setCastModeVideoView:videoView]; This view will be displayed in the videoView of OOOoyalaPlayer when casting the video.
