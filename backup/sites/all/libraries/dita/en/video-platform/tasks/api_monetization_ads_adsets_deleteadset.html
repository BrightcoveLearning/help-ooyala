Deleting an Ad Set

You can delete an ad set at any time. All settings for the ad set are deleted and cannot be recovered.

To delete an ad set:

Make a DELETE request to the /v2/ad_sets/ad_set_id, as in the following example. 
[DELETE]/v2/ad_sets/ed4a4c6abdfc4ab8aa3e7b8a05f31ec6
