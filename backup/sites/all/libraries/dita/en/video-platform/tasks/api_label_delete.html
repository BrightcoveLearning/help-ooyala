Deleting Labels

With the Backlot API, you can delete labels at any time.

To delete labels:

  1. If you don't have the ID of the label to delete, use GET with the /v2/labels route. The following is a GET example. 
    [GET]/v2/labels

    Backlot returns a response similar to the following.

    [  
   {  
      "name":"Hobbies",
      "id":"Yh98G7hsaß",
      "full_name":"/Hobbies"
   },
   {  
      "name":"Hockey",
      "id":"82d562682",
      "full_name":"/Hobbies/Hockey",
      "parent_id":"Yh98G7hsaß"
   },
   {  
      "name":"Disc Golf",
      "id":"55514522",
      "full_name":"/Sports/Disc Golf",
      "parent_id":"Yh98G7hsaß"
   }
]
    Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.
  2. Delete a label. The following example deletes the "Disc Golf" label. 
    [DELETE]/v2/labels/55514522

    Backlot returns a 200 response code.

    Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.
