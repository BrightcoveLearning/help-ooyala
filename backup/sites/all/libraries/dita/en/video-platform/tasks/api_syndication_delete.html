Deleting a Syndication

You can delete a syndication at any time. No new content will be sent to the syndication and any feeds will be deleted.

To delete a syndication:

Use delete with the syndications route. The following example deletes the ZnbWVpMjqToT1W syndication. 
[DELETE]/v2/syndications/ZnbWVpMjqToT1W

Backlot returns a 200 response.

Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.

The syndication is successfully deleted.
