Deleting an Asset: DELETE

To permanently remove an asset, use DELETE /v2/assets/asset_id.

Note: For more information about Backlot REST API commands, see the Backlot API Reference.

To edit YouTube video information:

Use a DELETE with the /v2/assets route and the asset ID. The following example performs a get to obtain the current settings for the asset with the BnMDNxMjqmPTu ID. 
[DELETE]/v2/assets/BnMDNxMjqmPTu

Backlot returns a 200 response without a response body.

Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.

The asset is successfully deleted.
