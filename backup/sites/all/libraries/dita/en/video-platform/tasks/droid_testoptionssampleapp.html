Testing the Android Options Sample App

To complete this tutorial have the following:

  • A system running Android Studio

Use the following steps to download, configure, and test the Ooyala Android Options sample app.

  1. Get the Options Sample App from the Ooyala Android Sample Apps Git repository and save it to your drive.
  2. Open Android Studio and select Open an existing Android Studio project.
  3. Navigate to the Options sample app folder and select Choose.
  4. Wait for the Gradle tasks to run and complete
  5. From Android Studio select File > Project Structure and navigate to the app properties tab.
  6. Set the Compile Sdk Version to API 22: Android 5.1 (Lollipop)

    The project will automatically rebuild. Wait for this process to complete before continuing to the step.

  7. Select Play to run the project in the emulator.
From the list of videos configured with options you can preview:
  • Cue Points and Ad Controls Options
  • Preload and PromoImage Options
  • Preload and Promo Options with Initial Tsime
  • Timeout Options
  • Server-Side TV Ratings
  • TV Ratings Configuration
