Implementing Inline Custom Controls for iOS

Learn how to customize player controls for iOS playback. Note that these steps are specific to inline controls. Use the same process for fullscreen controls but replace the inline source files with the fullscreen source files.

Note: The steps for implementing custom controls have been replaced with configuring the Player Skin JSON file, as described in Customizing the Player V4 Skin with skin.json and Customizing the Player V4 Appearance. For a legacy implementation using components that are no longer available, see https://github.com/ooyala/ios-sample-apps/blob/stable/AdvancedPlaybackSampleApp/AdvancedPlaybackSampleApp/Players/CustomControlsPlayerViewController.m.
