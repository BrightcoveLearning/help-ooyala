Ingesting Closed Captions Using XML Manifest Files

You can ingest closed caption files using XML manifest files.

To ingest closed caption files using an XML Manifest File:
  1. Create an XML manifest file with the format described in Ingesting Closed Caption Files. For closed caption ingestion to occur, the <media:subTitle> element must exist.
  2. Upload the closed caption file or video to the FTP/Web server.
  3. Upload the XML manifest file you created to the FTP/Web server.
When all the files are uploaded, Backlot begins ingesting the files.
