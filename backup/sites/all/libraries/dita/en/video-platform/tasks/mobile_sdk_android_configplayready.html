Configuring PlayReady for Android Apps (Disabled)

Note: Ooyala integration with the Playready DRM for mobile apps (Discretix variant) has been disabled. For details and alternatives, see the OVP Release Notes.

Have the following information ready for PlayReady configuration:

  • The Account API and Secret keys
    Note: For more information see Your API Credentials Your API Keys and Partner Code.
  • The VisualOn license
    Note: The VisualOn sample application includes a license for customer use. For more information see About VisualOn Integrations About VisualOn Integrations.
Warning:

As of April 1, 2016 the Ooyala Android SDKs have been shipping with Google ExoPlayer libraries instead of the VisualOn libraries. While there are steps that you will need to take, moving to ExoPlayer will provide a more consistent, higher quality HLS and MPEG-DASH playback experience for VOD and live streams on Android devices.

As of August 1, 2016, Ooyala will no longer support the HLS player integration with VisualOn or provide bug fixes or improvements specific to VisualOn libraries.

In order to upgrade, you will need to
  • Download the latest Android SDK 4.13.0+ (available at Ooyala Downloads) to start your integration.
  • Follow the integration instructions in Integration with Google ExoPlayer on Android.
  • Recompile your application and submit to Google Play Store.

Use the following steps to configure PlayReady for an Android app.

  1. Open the app MainActivity file for editing.
  2. Add the following libraries:
    • All JNI libraries
    • All JAR libraries
  3. Add the following code to your app: OoyalaPlayer.enableCustomPlayreadyPlayer = true;

Your app is not configured to use the PlayReady Secure Player. For a complete sample app including the PlayReady configuration contact your CSM.
