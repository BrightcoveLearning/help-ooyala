Using the QoS Solution Powered by NPAW for Desktop

The desktop implementation of the Ooyala QoS Solution powered by the NICE PEOPLE AT WORK (NPAW) YOUBORA platform provides analytics for the player events with Ooyala's HTML5 and Flash players.

Prior to enabling the Ooyala QoS Solution powered by YOUBORA, you must have the following:
  • A customer relationship with NPAW with accounts for YOUBORA. Please contact your Ooyala point of contact to set up your NPAW Account ID and User ID.
  • An Ooyala Backlot account.
To work with the Ooyala QoS Solution powered by NPAW, enable the plugin and start using the YOUBORA dashboard.
  1. Ask your Ooyala point of contact to enable your account for the Ooyala QoS solution powered by YOUBORA.
  2. Start using the YOUBORA dashboard. For information about the YOUBORA Analytics platform, see the YOUBORA product documentation.
