Status of an Asset

The Backlot UI MANAGE tab shows the status of an asset.

To view the status of an asset:
  1. Log in to the Backlot UI. The Backlot UI opens.
  2. Click MANAGE.
  3. On the list of assets, click a row to select it.
  4. On the far right, look at the color-coded status of the asset.
  5. From the upper left STATUS legend, match the color code of your asset.
    Status Description
    Live Asset is available for use.
    Paused Processing of asset is paused. An asset can be paused via the Backlot API or by the MANAGE tab's upper right Pause button.
    Processing Asset transcoding or other processing in progress.
    Uploading File uploading in progress.
    Error Problem in processing or other use.
    Recently Created Asset created within the last 48 hours.
    Approval Pending Asset needs to be approved before going live.
