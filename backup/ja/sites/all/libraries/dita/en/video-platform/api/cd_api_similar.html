同様の動画

Ooyala Discoveryの類似動画APIは、指定した動画に似た動画を返します。 たとえば、視聴者がスカイダイビングビデオを視聴している場合、Ooyala Discoveryアルゴリズムは他のスカイダイビングまたは極端なスポーツビデオを返す可能性があります。

注意： （要求ごとに個別のパラメーターを使用して）アドホック要求を作成するのではなく、ディスカバリー・プロファイルを使用して設定と推奨動作を一元管理することを検討してください。 検出プロファイルは、ユーザーが作成した設定の定義済み構成です。 のGET ルート 詳細については、 ディスカバリープロファイル.

同様の動画を入手

[GET] / v2 /発見/類似/資産/埋め込みコード？score_type = default？

同様のリモートアセットを入手する

[GET] / v2 / discover / similar / externalassets /埋め込みコード？score_type = default？

ライブストリームコンテンツに似た動画を取得する

[GET] / v2 / discover / similar / live_assets /virtual_asset_id？score_type = default？

編集クエリ文字列パラメータ

次のような社説のクエリ文字列パラメータの説明を参照してください。 discovery_profile_id in クエリ文字列パラメータ.
