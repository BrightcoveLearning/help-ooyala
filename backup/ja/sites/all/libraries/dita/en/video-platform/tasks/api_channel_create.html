チャンネルを作成する（非推奨）

チャンネルが作成されたら、ビデオを追加できます。

注意： チャンネルとチャンネルセットは廃止予定で、無効になる予定です。 詳細と代替案については、 OVPリリースノート.
注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

チャンネルを作成するには：

用途 POST そして / v2 / assets とともに asset_type に設定 チャンネル リクエストボディ内 次の例では、 "My Channel"チャンネルを作成しています。 
[POST] / v2 / assets {"名前"： "私のチャンネル"、 "asset_type"： "チャンネル"}

バックスロット 次のような応答を返します。

{"asset_type"： "チャンネル"、 "期間"：0、 "名前"： "私のチャンネル"、 "preview_image_url"：null、 "created_at"： "2011-08-10T17：45：24 + 00"、 "embed_code"： "dvNTVxMjrRktSlb00v"、 "時間制限"：null、 "updated_at"： "5-2011-08T10：17：45 + 24：00"： "null："、null、 "ホスト名" "null"、 "ホスト名" "null" null、 "説明"：null、 "ステータス"： "ライブ"}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

チャネルは正常に追加されました。
