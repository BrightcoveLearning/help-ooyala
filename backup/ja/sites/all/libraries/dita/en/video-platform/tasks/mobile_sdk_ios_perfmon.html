iOSアプリのパフォーマンスモニタリングの実装

アプリケーションのパフォーマンスに関する統計レポートを作成したい場合は、 OOPerformanceMonitor APIです。

  1. 次のimport文をコードに追加してください。 
    #import <OoyalaSDK / OOPerformanceMonitor.h>
  2. 作る OOPerformanceMonitor プロパティ： 
    @property（nonatomic）OOPerformanceMonitor * perf;
  3. あなたのインスタンス化 OOPerformanceMonitor 使用して getStandardMonitor API： 
    self.perf = [OOPerformanceMonitor getStandardMonitor];
  4. (任意バッファリングメトリックを取得するには 
    self.perf = [OOPerformanceMonitorBuilder getStandardMonitor];
  5. (任意）すべての広告の切り替えを追跡するには 
    self.perf = [OOPerformanceMonitorBuilder getStandardAdsMonitor];
見ます OOPパフォーマンス統計スナップショット 取得できる統計についての情報のためのAPI。
