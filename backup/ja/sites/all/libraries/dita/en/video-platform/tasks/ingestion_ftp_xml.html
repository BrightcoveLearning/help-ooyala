XMLマニフェストファイル

動画を取り込むときは、XMLマニフェストファイルを使用して、サムネイルおよびクローズドキャプションファイルだけでなく、追加のメタデータ（動画のタイトル、説明、フライト時間、ラベル、カスタムメタデータなど）を指定できます。 FTP、Aspera、またはMRSSでコンテンツを取り込むときにXMLマニフェストファイルを使用できます。

注意： このトピックは、OIS v2.5とOIS v1の両方に適用されます。 指示がある場合、特定のセクションは1つのバージョンにのみ適用されます。 OIS v1は無効になっています。 OIS v1を使用しているお客様は、OIS v2.5に切り替える必要があります。

XMLマニフェスト要素

XMLマニフェストファイルでは、次の要素を使用できます。

  • MRSSによる摂取の場合（ リモートでホストされているMRSSフィードからの取り込み）、動画がある場所のURL。 URLに含めることができます http:// or https://.

    長い （オプション） - クローズドキャプションファイルの言語。 2文字のコード 見る サポートされているクローズドキャプション.

注意： 指定されている場合、有効な値を指定するのはお客様の責任です。 長い そして フレームレート。 有効な値がDFXPファイルに追加されています。 準拠していない値は無視されます。 どちらの方法でも、クローズドキャプションファイルはBacklotに追加されてアセットにリンクされます。

XMLマニフェストファイルの作成

XMLメタデータファイルを作成するには

  1. ビデオやその他のコンテンツ（サムネイル、クローズドキャプションファイルなど）をアップロードします。
  2. テキストエディタ、できればXMLを検証するものを開きます。
  3. 必要なXMLヘッダーを追加してください。 例えば： 
    <？xml version = "1.0" encoding = "utf-8"？> <rss version = "2.0" xmlns：media = "http://search.yahoo.com/mrss/" xmlns：dcterms = "http：/ /purl.org/dc/terms/ "xmlns：fh =" http://purl.org/syndication/history/1.0 "xmlns：ooyala =" http://www.ooyala.com/mrss/ ">
  4. 作る チャンネル コンテナ。
  5. 作る 項目 上の表の目的の要素を含む、各ビデオのエントリ。

    次の例では、XMLは2つの動画のメタデータを指定しています。 最初のビデオでは、ファイルとサムネイルはアップロードサーバーにあり、そのサーバーから取得されます。 2番目のビデオでは、ファイルとサムネイルはすでにOoyalaサーバーにアップロードされています。 2番目のビデオにも飛行時間があります。
