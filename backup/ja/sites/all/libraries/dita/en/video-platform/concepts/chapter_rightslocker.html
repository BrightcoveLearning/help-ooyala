権利ロッカー

Rights Lockerは、デジタル資産への資格を定義および管理するための一連のアプリケーションプログラミングインターフェイス（API）です。 全体としてRights Lockerは、3セットのAPIとOoyalaサービスで構成されています。

API ロケーション 説明
バックスロットAPI バックスロットREST API
  • 資産を管理する（ビデオ）
  • ラベルを管理します
  • 公開ルールを管理します
Rights Locker API Rights Locker APIリファレンス
  • Backlot APIで定義された資産、ラベル、および公開ルールを使用して、顧客定義アカウントの資格を作成および管理します
プレイヤー認証API プレーヤV3用プレーヤ認証API（廃止予定）
  • 権限と資格の実行時チェック

Rights Lockerの使用には3つの部分があります。

  1. ライブストリームイベント開始時間 バックスロットAPI メソッド（ バックスロットAPI 参照）、アセット（ビデオ）、ラベル、および公開ルールを作成します。 （ バックスロットUI 使用することもできます。）
  2. ここで説明されているRights Locker APIを使用して、事前定義済みの資産、ラベル、および公開ルールを組み合わせて、個々のユーザーの資格を作成し、グループ化します。
  3. プレーヤーを使用して、再生時および必要に応じてデジタル著作権管理（DRM）システムのライセンスまたはその他の資格情報を発行する際の資格に対する承認を確認します。 ほとんどの場合、認可はプログラミングなしで行われます。
