資産の後処理ステータスの設定

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
アセットをアップロードしてトランスコードした後は、すぐに運用したくない場合があります。これがデフォルトです。 資産の後処理ステータスを設定することで、資産をすぐに公開しないようにシステムに指示できます。 状況は2つしかありません。
  • ライブ （デフォルト）
  • 一時停止
  1. 資産の後処理ステータスをに設定するには 一時停止. 
    [PATCH] / v2 / assets / BjMDVrOTrLeHriLAq43T5y {"ステータス"： "一時停止"}

    バックスロット 次のような応答を返します。

    { "期間"： "5038"、 "asset_type"： "ビデオ"、 "名"： "bleebleblobble.mov"、 "preview_image_url"： "http://ak.c.ooyala.com/w3ZHc0Njr33Tdp-RRcwfZMjaOrmzOP82/Ut_HKthATH4eww8X4xMDoxOmdtO40mAx" 、 "original_file_name"： "bleebleblobble.mov"、 "embed_code"： "w3ZHc0Njr33Tdp-RRcwfZMjaOrmzOP82"、 "hosted_at"： ""、 "publishing_rule_id"： "11a2b0702959459cb60db9cd5b7adb62"、 "説明"： "geebleガツガツ食べる"、 "ステータス"：「一時停止 "、" external_id "：" ABC124 "、"時間制限 "：null、" created_at "：" 2012-10-10T15：24：40Z "、"更新した_at "：" 2013-08：XUMX "： ad_set_id "：null}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. 資産の後処理ステータスをに設定するには ライブ. 
    [PATCH] / v2 / assets / BjMDVrOTrLeHriLAq43T5y {"ステータス"： "ライブ"}
