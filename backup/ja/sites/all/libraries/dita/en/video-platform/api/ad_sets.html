広告セット

Ad sets are ad configurations that control the ad experience of an asset during playback.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
The ad set routes create, delete, and change the names of ad sets and assign ad sets to assets. All other ad behavior must be defined in the バックスロットUI.

Create Ad Set

[POST]/v2/ad_sets{  
   properties
}

Add an Ad to Ad Set

[POST]/v2/ad_sets/ad_set_id/ads{
   properties
}

List Ad Sets

[GET]/v2/ad_sets

Get Ad Set

[GET]/v2/ad_sets/ad_set_id

Modify Ad Set

[PATCH]/v2/ad_sets/ad_set_id{  
   properties
}

Delete Ad Set

[DELETE]/v2/ad_sets/ad_set_id

Remove Ad from Ad Set

[DELETE]/v2/ad_sets/ad_set_id/ ads /ad_id

ルート属性

次の表は、ルートを通じて表現できるすべての属性を示しています。

ルート属性 説明
ad_id The ID of the ad.

タイプ：文字列

デフォルト：なし

Example: da1c1a4bd90c48c39d7ce3e8347bd758
ad_set_id

The ID of the ad set. To get a list of ad sets, perform a get against the / v2 / ad_sets ルート。

タイプ：文字列

デフォルト：なし

Example: /ad_sets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg
