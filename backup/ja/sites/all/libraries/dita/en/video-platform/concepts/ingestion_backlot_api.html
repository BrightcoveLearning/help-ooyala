Backlot APIを使った取り込み

プログラムでBacklotにコンテンツをアップロードするには、Backlot APIを使用します。 このアプローチにより、コンテンツ管理システム（CMS）またはワークフローをBacklotプラットフォームと直接統合することができます。

Backlot APIを使用して、ビデオ、サムネイル、およびクローズドキャプションファイルをプログラムでアップロードおよび管理することができます。
Backlot APIは、高レベルの統合とカスタマイズを提供します。 開発期間とリソースが必要です。 Backlot APIを使用してコンテンツを取り込む場合は、
  • ビデオとオーディオの資産 （動画とサムネイル用）
  • クローズドキャプションの使用

ファイルをアップロードするために使用される主な呼び出し（「資産」と呼ばれる）は、 / v2 / assets ルート。

注意： nullの動画を追加しないでください external_idつまり、 external_id 値なし（（""）またはの値 "ヌル"。 このようなNULL外部IDは後で検索することはできません。
