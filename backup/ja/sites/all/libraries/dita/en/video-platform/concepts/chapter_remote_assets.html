リモートアセット

リモートアセットは、Backlotの外部でホストされるコンテンツの一部です。 コンテンツをアップロード、トランスコーディング、およびBacklotに保存する代わりに、必要に応じてBacklotがコンテンツを見つけることができるように、そのURLやその他の説明情報をアップロードするだけです。

リモート資産管理は、管理、シンジケーション、収益化、分析、およびOoyala APIへのアクセスなど、Backlotのすべての利点を提供します。

リモートアセットでサポートされているストリーミング形式

Ooyalaは、リモートアセット用に次のストリーミング形式をサポートしています。
  • ダッシュ
  • mp4
  • HDS
  • HLS
  • スムーズ
  • rtmp
注意： iPadおよびiPhoneのストリームでは、MP4またはM3U8形式のみが有効です。

リモートアセットを追加する方法

以下の方法でリモート資産をBacklotに追加することができます。
  • バックスロットUI（を参照） リモートアセット)
  • Backlot API（を参照） リモートアセット)
  • MRSS経由（を参照） MRSSを介したリモートアセットの取り込み)
コンテンツがBacklotに追加された後、Ooyalaは期間とタイトルを含むそれに関する基本的な情報を保存します。 ルール、カスタマイズ、プレビュー画像、および追加のメタデータは、Backlot UIまたはAPIを介してリモートアセットに適用できます。

リモートアセットの配信

追加されたリモートアセットは、Ooyalaまたは選択したCDNで配信できます。
注意： リモート資産が作成された後、そのさまざまなCDNへの伝搬が60秒以上遅れることがあります。 作成後すぐにリモートアセットをリクエストした場合、結果はCDNによってキャッシュされます。クリアには数分かかることがあります。 ベストプラクティス：作成後、30または60秒待ってからBacklot API [GET] / v2 / assets / asset_idルートで照会し、リモートアセットの埋め込みコード（コンテンツIDまたはアセットID）を取得した後、アセットの埋め込みに進みます。 。
