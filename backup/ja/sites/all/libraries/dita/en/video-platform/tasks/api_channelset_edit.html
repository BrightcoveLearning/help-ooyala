チャンネルセットの編集（廃止予定）

チャンネルセットに関する情報はいつでも変更できます。

注意： チャンネルとチャンネルセットは廃止予定で、無効になる予定です。 詳細と代替案については、 OVPリリースノート.
注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

チャンネルセット情報を編集するには：

  1. チャンネルの情報を見るには、 GET / v2assets チャンネルセットのアセットIDでルーティングします。 次の例では、チャネルセットの現在の設定を取得するためにgetを実行します。 IwYTVxMjqLtYHmwqy. 
    [GET] / v2 / assets / IwYTVxMjqLtYHmwqy

    バックスロット 次のような応答を返します。

    {"asset_type"： "channel_set"、 "期間"：0、 "名前"： "マイチャンネルセット"、 "preview_image_url"：null、 "created_at"： "2011-08-10T18：20：03：00" 、 "embed_code"： "IwYTVxMjqLtYHmwqy"、 "time_restrictions"：null、 "updated_at"： "00-2011-08T10：18：20 + 04：00"、 "external_id"、 "null"、 "元のファイル"、 "hostname"。 ：null、 "説明"：null、 "ステータス"： "ライブ"}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. チャンネルの情報を変更するには、 PATCH 　 / v2 / assets、リクエストIDのアセットID、および更新されたプロパティ。 次の例では、名前、「ホスト先」のURL、および説明を変更します。 
    [PATCH] / v2 / assets / dvNTVxMjrRktSlb5v {"名前"： "私の素晴らしいチャンネルセット"、 "ホストされたチャンネル"： "http://www.mydomain.com"、 "説明"： "これはチャンネルのセットです。" }

    バックスロット 次のような応答を返します。

    {"asset_type"： "channel_set"、 "duration"：0、 "name"： "私の素晴らしいチャンネルセット"、 "preview_image_url"： "null"、 "created_at"： "2011-08-10T18：20：03：00： "、" embed_code "：" IwYTVxMjqLtYHmwqy "、" time_restrictions "：null、" updated_at "：" 00-2011-08T10：18：20 + 04：00 "、" external_id "、" http：// "、" http：// "、" http：// " /www.mydomain.com "、" original_file_name "：null、" description "："これはチャンネルのセットです。 "、" status "：" live "}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

チャンネルセットは正常に編集されました。
