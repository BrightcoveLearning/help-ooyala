プレーヤHTML5 API（プレーヤV2は廃止予定）

プレーヤ/プレーヤUI HTML5 APIを使用してプレーヤの側面を処理できます。

プレイヤー機能

重要： Ooyala Player V2は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V2はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V2を使用しているお客様は、Player V4に移行する必要があります（を参照）。 プレイヤーV4.

次の表は、プレーヤーに関連付けられているすべてのHTML5メソッドを示しています。 これらすべてのHTML5 APIを使用した例については、トピックを参照してください。 クエリ文字列とパススルーパラメータの例.

関数 説明
setEmbedCode

現在のプレーヤーのembedCodeを設定します。

タイプ：文字列

例： 

<input type = button value = "埋め込みコードの変更" onclick = "setEmbedCode（）" />
setPlayheadTime

再生ヘッドを数ミリ秒の精度で新しい位置に設定します。

タイプ：番号

例： 

   関数setPlayheadTime（）{get_player（）。setPlayheadTime（$（ '＃playhead_time'）。val（））;} }
