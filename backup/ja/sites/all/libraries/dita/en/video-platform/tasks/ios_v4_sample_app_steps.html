iOSスキンを使用した既存のアプリケーションの更新

新しいiOSスキンを使用して既存のアプリケーションを更新することができます。 以下は、iOSスキンをに統合する方法を説明するチュートリアルです。 BasicPlaybackSampleApp.

  1. まだ行っていない場合は、ダウンロードして解凍します。 OoyalaモバイルSDKの最新バージョン iOSの場合 またダウンロード iOS用Ooyala Skin SDK これは必要なフレームワークを含みます。 最後に、 IOSサンプルアプリケーション 次のコマンドを入力してリポジトリを作成します。 
    git clone https://github.com/ooyala/ios-sample-apps.git
  2. クローンから IOSサンプルアプリケーション ディレクトリに移動します BasicPlaybackSampleApp サブディレクトリ。 開く BasicPlaybackSampleApp.xcodeproj XCodeのファイル
  3. 加えます JavascriptCore, Social、および MessageUI フレームワーク たとえば、次のようにします。 JavascriptCore、をクリックしてください BasicPlaybackSampleApp 左上隅のプロジェクト（青いアイコン）。 に行く リンクされたフレームワークとライブラリ セクション。 クリック + 新しいフレームワークを追加するためのプラスボタン。 下にスクロールして JavascriptCore.framework フレームワーク。
  4. Add をクリックします。 libz.tbd、zip圧縮ライブラリ。 これを行うには、プロジェクトで、 構築フェーズ タブ。 下 バイナリとライブラリのリンク セクション、を押す + プラスボタン を検索して追加 libz.tbd.
  5. 加えます OoyalaSkinSDK.framework フレームワーク。 これを行う最も簡単な方法は、フレームワークを左側の列の適切なフォルダにドラッグすることです。 [必要に応じてアイテムをコピーする]チェックボックスがオンになっていることを確認してください。 必要に応じて、フォルダのパスを フレームワーク検索パス 下のスロット ビルド設定 の中に 検索パス設定。 新しいフレームワーク検索パスを追加するには、スロットをクリックし、既存のパスの後にコロン（スペースで区切って）を追加してから、新しいパスを挿入します。
  6. に行きます ビルド設定 タブ。 下 連結 セクション、追加 - オブジェクト 「その他のリンカーフラグ」と書かれているスロットに。
  7. ドラッグ main.jsbundle からのファイル OoyalaSkinSDK-iOS ディレクトリに BasicPlaybackSampleApp > Players の作成と管理 Xcodeプロジェクトの左側の列にあるディレクトリ構造。 [必要に応じてアイテムをコピーする]チェックボックスがオンになっていることを確認してください。
  8. あなた自身のスキン設定ファイルを作ることができます。 また、あります skin-config サブディレクトリ OoyalaSkinSDK-iOS フォルダ。 スキン設定ファイルから、 en.json, es.json, zh.json、および skin.json ファイルを Players の作成と管理 ディレクトリ。 [必要に応じてアイテムをコピーする]チェックボックスがオンになっていることを確認してください。
  9. In BasicPlaybackSampleApp > Players の作成と管理 > BasicSimplePlaybackViewController.m、変化 #import <OoyalaSDK / OOOoyalaPlayerViewController.h> 〜へ #import <OoyalaSkinSDK / OOSkinViewController.h>.
  10. Add をクリックします。 #import <OoyalaSDK / OODiscoveryOptions.h> そして #import <OoyalaSkinSDK / OOSkinOptions.h> インポートのリストへ
  11. 同じファイルで、のすべてのインスタンスを置き換えます。 OOOoyalaPlayerViewController 　 OOSkinViewController。 これは押すことによってすることができます CMD + F と演奏 見つけて交換する.
