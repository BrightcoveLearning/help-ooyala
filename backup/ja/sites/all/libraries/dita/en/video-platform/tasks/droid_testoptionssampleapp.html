Android Optionsサンプルアプリのテスト

このチュートリアルを完了するには、次のものがあります。

  • A system running Android Studio

Use the following steps to download, configure, and test the Ooyala Android Options sample app.

  1. Get the Options Sample App from the Ooyala AndroidサンプルアプリGitリポジトリ and save it to your drive.
  2. Android Studioを開いて選択 Open an existing Android Studio project.
  3. Navigate to the Options sample app folder and select 選択する.
  4. Wait for the Gradle tasks to run and complete
  5. From Android Studio select File > Project Structure and navigate to the app properties tab.
  6. をセットする Compile Sdk Version 〜へ API 22: Android 5.1 (Lollipop)

    The project will automatically rebuild. Wait for this process to complete before continuing to the step.

  7. 選択します 遊びます to run the project in the emulator.
