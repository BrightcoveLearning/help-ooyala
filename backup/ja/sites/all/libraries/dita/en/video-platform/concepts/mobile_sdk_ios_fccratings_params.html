iOS FCC評価設定パラメータ

The iOS rating configuration is set with the FCCTVRatingConfguration.Builder object and contains the following parameters:

パラメーター 説明
durationSeconds

評価が資産の透かしとして表示される時間（秒単位）。

This parameter takes any positive float value and does not assert a default value.
位置

The position of the of the rating watermark on the video asset. This parameter can have one of the following four values:

  • OOFCCTvRatingsPositionTopLeft （デフォルト）
  • OOFCCTvRatingsPositionTopRight
  • OOFCCTvRatingsPositionBottomLeft
  • OOFCCTvRatingsPositionBottomRight
階段

このパラメータは、ビデオアセットのサイズを基準にして、格付け透かしのスケールを設定します。

このパラメータは、0と1の間の任意のfloat値を取ります。 デフォルト値は0.2です。
不透明度

このパラメータは、格付け透かしの不透明度を設定します。

このパラメータは、0と1の間の任意のfloat値を取ります。 デフォルト値は0.9です。
