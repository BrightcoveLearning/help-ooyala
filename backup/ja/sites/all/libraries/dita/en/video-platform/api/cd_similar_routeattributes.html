Similar Assets Route Attributes

属性 説明
埋め込みコード

Specifies the embed code of the video or remote asset for which you want to collect related videos.

タイプ：文字列

デフォルト：なし

Examples: pmd2xzMjrCg4TpQd6cwz9HP4U-pcFSHH

score_type

Force results based on a consistent algorithm by using the score_type attribute when calling similar asset recommendations. The attribute takes the following values:

  • デフォルト (recommended for consistent results): Use the default value to return recommendations based on matching labels and metadata.
  • 確率: Use the probability value to return recommendations based on lift trends.
  • 貪欲な: Use the greedy value to return recommendations based on labels, metadata and lift trends.

タイプ：文字列

デフォルト：なし
virtual_asset_id

Specifies the ID of the virtual asset within the live stream. To specify a virtual asset ID, the Electronic Programming Guide (EPG) must be enabled for your account. For more information about EPG, contact Sales, your Customer Success Manager, or Technical Support.

タイプ：文字列

デフォルト：なし
