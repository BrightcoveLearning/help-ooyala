API使用状況のディスカバリUI CTRレポート（廃止予定）

注意： このレポートのサポートは近い将来中止されています。 Ooyalaでは、Discoveryによるエンゲージメントの増加を追跡するために、代わりにOoyala IQのDiscoveryモジュール（タブ）を使用することをお勧めします。 詳しくは、 ディスカバリーインサイトの表示

Discovery UI CTR Reportは、Discovery APIから要求され、サイトのコンテンツカルーセルまたはパネルに表示されているDiscovery推奨事項の有効性を測定します。 ディスカバリUIクリックスルー率（CTR）は、ディスカバリAPIリクエストからコンテンツを調達しているカルーセルのインプレッション数に対するクリック数の比率として定義されます。

レポートファイル名

レポートファイル名は Discovery_ui_ctr_ <yyyymmdd> <yyyymmdd> <PID> .csv.

レポートフィールド

レポートには次のフィールドがあります。
フィールド 説明
UI_TAG UIカルーセルまたはパネルを識別するために使用されるオプションの文字列。 UIタグなしでDiscovery Feedback Loopを通じて報告されたクリック数または表示回数は、DEFAULTとして分類されます。 0のUIタグは、集約CTRを提供するために使用されます。
日付 クリック率の日付。
クリック クリック数。
感想 インプレッション数
CTR クリック率
親トピック ディスカバリーインサイトレポート（利用不可）
