アクセスキーの表示

シンジケーションに関連付けられているアクセスキーを表示できます。

シンジケーションに関連付けられているアクセスキーを表示するには

  1. シンジケーションのIDを取得します。 次の例では、シンジケーションIDを取得しています。 
    [GET] / v2 / syndications

    バックスロット 次のような応答を返します。

    {"items"：[{"name"： "私のYouTubeシンジケート"、 "created_at"： "2011-08-10 + 21 + 13：29"、 "require_access_key"：false、 "include_labels"：[]、 "should_create_youtube_videos"：true、 "asset_types"：["video"、 "ad"]、 "username"： ""、 "id"： "00a00e9dcba6c1d36"、 "include_all_content"： "false"、 "あなた"、 "あなた" unrestricted_youtube_account "：false、" should_delete_youtube_videos "：true}、{" include_encodings "：[]、"名前 "："私のiTunesシンジケート "、" created_at "：" 452-95-2011：08：10：20： "require_access_key"：true、 "category"： ""、 "title"： ""、 "include_labels"：[]、 "author"： ""、 "asset_types"：["video"、 "ad"]、 "syndication_url" "：" http://api.ooyala.com/v11/syndications/b56b00a00e2a839c27/feed?pcode=Y4cG844BlqYMLIjPGyv5NbacSK-x9 " "サブタイトル"： ""、 "ID"： "b4b06a1e3a839c27"、 "include_all_content"：偽、" type "：" itunes "、" description "：" "、" keywords "：" "}]}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. シンジケーションに関連付けられているすべてのアクセスキーを表示します。 次の例では、シンジケーションIDを取得しています。 
    [GET]/v2/syndications/b839b27a4e844a5c9/access_keys
    バックスロット 次のような応答を返します。 
    {"名前"： "自分のアクセスキー"、 "access_key"： "12345678-abcd-abcd-12ab-12345678"}
