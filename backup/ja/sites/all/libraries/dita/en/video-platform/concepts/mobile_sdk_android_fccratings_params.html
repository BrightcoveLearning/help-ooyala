Android FCC評価設定パラメータ

TV Ratings機能は、次の設定クラスをメインアクティビティにインポートすることで有効になります。
  • com.ooyala.android.configuration.Options
  • com.ooyala.android.configuration.FCCTVRatingConfiguration

    この手順は、次のコードサンプルに示されています。

import com.ooyala.android.configuration.FCCTVRatingConfiguration; import com.ooyala.android.configuration.Options;

クラスをインスタンス化し、次の設定オプションを使用してそれを拡張します。

パラメーター 説明
setDurationSeconds

評価が資産の透かしとして表示される時間（秒単位）。

デフォルト値は10秒です。
setPosition

動画アセット上のレーティング透かしの位置。 このパラメータは、次の4つの値のいずれかを取ります。

  • Position.TopLeft （デフォルト）
  • Position.TopRight
  • Position.BottomLeft
  • Position.BottomRight
setScale

このパラメータは、ビデオアセットのサイズを基準にして、格付け透かしのスケールを設定します。

このパラメータは、0と1の間の任意のfloat値を取ります。 デフォルト値は0.2です。
setOpacity

このパラメータは、格付け透かしの不透明度を設定します。

このパラメータは、0と1の間の任意のfloat値を取ります。 デフォルト値は0.9です。

次のコード例は、デバイス画面上で評価が表示される位置を設定するために拡張されたクラスを示しています。

FCCTVRatingConfiguration fccConfig = new FCCTVRatingConfiguration.Builder（）。setPosition（getTVRatingPosition（））。setDurationSeconds（TVRATING_DURATION）.build（）; Options options = new Options.Builder（）。setTVRatingConfiguration（fccConfig）.build（）;
