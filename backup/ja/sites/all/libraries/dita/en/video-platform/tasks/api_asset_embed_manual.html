手動でアセットを埋め込む

Webページでアセットを利用可能にするには、手動で埋め込むことができます。

このセクションでは、基本的な埋め込みスニペットの作成方法について説明します。 高度な埋め込みスニペットの場合は、から自動的に生成されたスニペットを使用できます。 バックスロットUI またはシンジケーションフィードから。 からスニペットを取得する方法については バックスロットUIを参照してください。 バックスロット ユーザーガイド。 シンジケーションフィードからスニペットを取得する方法については、 シンジケーション.

資産を埋め込むには：

  1. 幅、高さ、および埋め込みコードを指定する埋め込みスニペットを作成します。 以下は、資産の最小埋め込みコードの例です。 Q3bDhiMToizUftO IDを指定します。 
    <script src = "http://player.ooyala.com/player.js？width = 640＆height = 480＆embedCode = Q3bDhiMToizUftO"> </script>
  2. スニペットをWebページに貼り付けます。
  3. Webページを開いて埋め込みを確認します。
