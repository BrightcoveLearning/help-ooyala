ウーヤラスケジュール

Ooyala Scheduleを使用すると、仮想アセットのトラックを組み合わせてライブストリームの上で再生できます。

スケジュールを操作するために、3つのAPIルートがあります。

  • / v2 / tracks
  • / v2 / track_segments
  • / v2 / virtual_assets
  • トラック
    トラックはウーヤラスケジュールの基本です。
  • トラックセグメント
    トラックセグメントは、トラック内のスケジュールされた開始/終了時刻にある仮想アセットのインスタンスとして定義されます。
  • バーチャルアセット
    仮想アセットは個々のプログラムまたはイベントを表します。
