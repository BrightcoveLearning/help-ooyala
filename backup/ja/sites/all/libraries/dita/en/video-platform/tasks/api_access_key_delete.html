アクセスキーを削除する

シンジケーションからアクセスキーを削除できます。

シンジケーションからアクセスキーを削除するには

deleteを使って v2 / syndications / syndication_id / access_keys / access_key ルート。 次の例では、 12345678-abcd-abcd-12ab-12345678 b839b27a4e844a5c9シンジケーションからのアクセスキー。 
[DELETE]/v2/syndications/b839b27a4e844a5c9/access_keys/12345678-abcd-abcd-12ab-12345678

バックスロット 200応答を返します。

アクセスはシンジケーションから正常に削除されました。
