tvOSでOoyala Pulseと統合する

あなたが必要なもの

Ooyala Pulseを統合するためのOoyala SDKを使用すると、デスクトップで作成したのと同じ広告エクスペリエンスを、サポートされているモバイルデバイスでも作成できます。

To get started with the integration between your Ooyala Player V4 and ad serving from Ooyala Pulse on tvOS, you will need the following:

  • Ooyala Mobile SDK for tvOS (available at Ooyala ダウンロード)
  • Ooyala Skin SDK for tvOS (available at Ooyala ダウンロード） 詳細については、 iOS用プレイヤースキン.
  • Ooyala SDK for Ooyala Pulse tvOS (available at Ooyala ダウンロード)
  • Ooyala Video Advertising tvOS 2.x SDK framework (available at Ooyala ダウンロード)
  • アップルXcode

サンプルアプリケーション

The Ooyala Pulse sample app is part of the TVOSSampleApp and is available for download at Ooyala Samples GitHub.

使ってみよう

A simple Getting Started tutorial is included in the SDK reference documentation for the iOS SDK, which is very similar to the tvOS SDK: Ooyala Player v4パルスインテグレーション.

必須パラメータ、カスタムメタデータ、およびビデオプラザ広告セット

Mobile SDKでOoyala Pulse広告配信を利用するには、次のタイプの広告セットを作成する必要があります。 ビデオプラザ Backlotで。 詳細はBacklotユーザーガイドを参照してください。 Ooyalaでは、Ooyala Pulse関連のパラメータをさまざまな場所に保存できます。 優先順位の高い順に、Pulseパラメータとその値は次のように定義できます。

  1. あなたのアプリ自体。
  2. バックスロットで 管理 タブで定義されたカスタムメタデータとしての各ビデオアセット用 カスタムメタデータ をクリックします。
  3. バックスロットで マネー タブ、 広告セット のサブタブ ビデオプラザ 広告セットの種類。
注意： アプリで直接パラメータを設定する場合は、Ooyala Pulse Sessionが作成される前に設定する必要があります。
次のパラメータは、アプリで直接設定するか、Backlotでアセットのカスタムメタデータとして設定するか、またはアセットに関連付けられたVideoplaza広告セット（推奨）で設定する必要があります。
  • パルスホスト：あなたのOoyala PulseアカウントへのURL。 アプリでは、このパラメータを setPulseHost メソッドを設定することもできます。 deviceContainer と persistentId。 設定することをお勧めします deviceContainer 〜へ ゼロ。 例えば：
    [OOPulse setPulseHost：@ "https://pulse-demo.videoplaza.tv" deviceContainer：nil persistentId：nil];
他の重要なパラメータは次のオブジェクトを通して設定されます。
  • リクエスト設定オブジェクト：Ooyala Pulseから要求する広告の掲載順位と広告の品質に関する情報。 このオブジェクトのプロパティは VPRequestSettings.
  • コンテンツメタデータオブジェクト：アプリでの再生用に選択された動画コンテンツに関する情報。OoyalaPulseからのターゲット広告のリクエストに役立ちます。 このオブジェクトのプロパティは VPContentMetadata.
各パラメータの概要、設定可能な場所、および基本値の場所については、を参照してください。 iOSのOoyala Pulseのパラメータ.
