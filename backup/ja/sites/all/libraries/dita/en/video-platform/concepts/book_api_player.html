プレーヤV3 JavaScript APIリファレンス（非推奨）

Player V3 APIは、Player v3機能とメッセージバスイベント処理も提供するUI要素をカスタマイズするためのJavaScript APIです。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
重要： このセクションのトピックは、 Ooyala APIドキュメント のページをご覧ください。
  • プレーヤV3 JavaScript APIの概要
    へようこそ Ooyala プレーヤー APIリファレンス Ooyalaはカスタマイズ可能で機能豊富なPlayerを提供しています。これは業界や複数のデバイスで最高のビデオ体験を提供するために使用されています。
  • プレーヤV3 JavaScript API関数
    プレーヤV3 JavaScript関数を使用してプレーヤの動作と外観をカスタマイズできます。
  • Player V3 JavaScript APIパラメータ（廃止予定）
    プレーヤV3パラメータは、プレーヤ広告、動作をカスタマイズし、プレーヤスタイルを変更するために使用できます。
  • プレーヤV3メッセージバスイベント（非推奨）
    メッセージバスイベントを使用して、動画から広告の再生までのプレーヤイベントを購読または発行します。
  • FlashベースのプレーヤV3エラーメッセージ（非推奨）
    再生中にエラーメッセージがビデオビューアまたはオーディオリスナに表示されます。 エラーは、一般エラー、Flashベースのエラー、ペイパービューエラーに分類されます。これらのエラーの先頭にはPPVが付きます。
