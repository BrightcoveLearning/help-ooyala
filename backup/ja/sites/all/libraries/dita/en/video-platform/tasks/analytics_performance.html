パフォーマンスレポートの表示

This report shows comprehensive performance data for all your videos. You can view data on an account level, by labels, or by individual videos.

To view the Performance report:

  1. にログインします。 バックスロットUI. Backlot UIが開きます。
  2. Video Cloud Studioで ANALYZE のページをご覧ください。 ANALYZE page opens with the 洗浄力 page showing.
  3. To specify a date range, click the date in the upper right hand corner and select a new date range.
  4. By default, the graph displays プレイズ during the specified period. To change the metrics to graph, click the graph list box and select the check boxes of one or more metrics to view. For example, you might want to compare 表示 そして プレイズ to view how many people play videos that are shown to them or プレイズ そして 100% Play-Through to see how many people play the content from beginning to end. By default, each point on the graph represents a day. You can also configure each point to represent a 週間 または 月.
  5. To reduce the number of videos in the list, you can filter the content. To only show a single video, channel, or channel set, enter its name in the Filter Content: field. To only view videos that have a specified label, click ラベルを選択 and specify a label.
  6. By default, the Performance report lists assets such as videos, channels, channel sets, ads, and YouTube videos. To change the type of information displayed, select one of the following from the 次元 リストボックス。
    オプション 説明
    デバイスの種類 Aggregates the performance information based on the device type (e.g., Desktop/Laptop, Tablet, Mobile, and so on).
    OS/Browser Aggregates the performance information based on the operating system (e.g., Desktop/Laptop - Mac - Firefox).
    ヶ国 Aggregates the performance information based on country (e.g., United States, Japan, France).
    都市 Aggregates the performance information based on cities (e.g., Los Angeles, New York, Tokyo).
    Designated Marketing Areas Aggregates the performance information based on the major regions (e.g., San Francisco - Oak - San Jose, Miami - Ft Lauderdale).
    ドメイン Aggregates the performance information based on the domains where the assets are hosted (e.g., yourdomain.com).
    URLは Aggregates the performance information based on the URLs where the assets are embedded (e.g., yourdomain.com/yourvideos).
  7. You can change the type of data that appears in the report. To specify the type of data that appears in the report, click Columns Displayed and select the check box for each type of data to display.

    For a thorough discussion of how plays, displays, and initial plays (or "play starts"), see 表示、再生、および再生の開始（廃止予定）.

    オプション 説明
    プレイズ Measures the number of times that the "Play" button is triggered either manually or automatically. The requested content could be ad content or the actual video content (a play is counted regardless of which type of content is requested to play). Plays currently don’t include the Replay event.
    イニシャルプレイ A video, channel, or channel set is played for the first time in a viewer session (a single browser session). For example, if a viewer replays a video three times, it counts as one initial play and three plays.
    表示 Measures the number of times that a piece of video content is loaded and displayed within the player before it gets played (displays are related to each individual asset). Each time an embed code is changed, this event gets triggered.
    25% Playthrough At least 25% of a video, channel, or channel set was watched by a viewer.
    50% Playthrough At least 50% of a video, channel, or channel set was watched by a viewer.
    75% Playthrough At least 75% of a video, channel, or channel set was watched by a viewer.
    100% Playthrough 100% of a video, channel, or channel set was watched by a viewer.
    Avg. Unique Users/Day Average number of unique viewers for the video, channel, or channel set per day.
    Avg. Videos/Users/Day Average number of videos per viewer for each day.
    Player Conversion Rate The number of times a video is initially loaded compared to the number of times the player is loaded on the page.
    動画変換率 The number of times a video is played compared to the number of times it is displayed. Videos that are displayed on high traffic pages, but are not played, will have low conversion rates.
    Hours Delivered The number of hours delivered during the specified period.
    平均 見た時間/ビデオ The average amount of time the video was watched during the specified period. This metric is calculated as follows: (hours delivered*60)*60/Total Plays
  8. To export the report to the comma-separated value (CSV) spreadsheet format, click CSVとして保存 and save the file to your computer.
    注意： The CSV report contains information from the upper graph. To change information that appears in the CSV report, select from the fields used to generate the graph.
