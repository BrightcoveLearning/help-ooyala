アーカイブ

へようこそ Ooyala アーカイブ。廃止予定の機能、ツール、およびAPIのドキュメントが含まれています。

  • v2 Analytics（非推奨）
    Ooyalaには、v2 Analytics APIを使用してデータを取得するという基本的な分析機能がありました。
  • Ooyala API V1（非推奨）からAPI V2への移行
    Ooyala V1 APIには限られた機能しかありません。 の Ooyala V2 APIを使用すると、リソースをより細かく制御でき、新しい機能がリリースされたときに更新が容易になります。
  • Ooyala Backlot APIリファレンスV1（非推奨）
    このバージョンのBacklot APIは非推奨です。 APIのバージョン2で新しい開発作業を行うことを強くお勧めします。 V2 APIの使用方法については、Ooyalaサポートセンターの開発者ドキュメントセクションを参照してください。
  • アップローダJavaScriptライブラリ（廃止予定）
    Uploader JavaScript Libraryには、参照サーバーの実装が含まれています。
  • 非推奨：アップロードウィジェットAPI v1リファレンス（非推奨）
    v1アップロードウィジェットはv1 APIに基づいているため、非推奨です。 代わりに、Uploader JavaScript Libararyを使用してください。
  • V2プレーヤAPI（プレーヤV2は廃止予定）
    Ooyala V2プレーヤーAPIリファレンスへようこそ。 このガイドには、Ooyala V2 Player APIを使用して実行できる操作に関する従来の情報が含まれています。
  • ウーヤラディスカバリーキュー（廃止予定）
  • Widevine Classic（廃止予定）
