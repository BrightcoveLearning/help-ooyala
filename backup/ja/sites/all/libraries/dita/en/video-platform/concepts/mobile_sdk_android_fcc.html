Android FCC 708字幕コンプライアンス

Android FCC 708 compliance has been applied using device level Accessibility settings. You need to enable Closed Caption (CC) settings on the device itself for it to be rendered in preferred styles on the video app. This one time setting, applies to any Ooyala SDK app. For more information, please see the Google's Android Accessibility Captions topic.

注意： Ooyala FCC 708 compliance is only supported on Android 4.4+ versions. HTML5 and Hook are not currently supported.
注意： The native SDKs do not have any styling settings within the Ooyala Player. For Android, the Android Skin SDK uses the operating system's closed caption styling information to render closed captions. The Android Skin SDK supports FCC closed caption styling.
