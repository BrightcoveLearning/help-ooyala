Android上のGoogle IMAとの統合

Google IMA（Interactive Media Ads）を統合するためのOoyala SDKを使用すると、デスクトップで作成したのと同じ広告エクスペリエンスをモバイルデバイスで作成できます。

注意： This software and documentation is for Google IMA with Ooyala's Mobile SDK for use on mobile devices. For Google IMA for the desktop player, see Google IMA Ad Plugin.

OoyalaのSDK for Google IMAの背後にある主な設計原則は次のとおりです。
  • OoyalaのIMA Managerオブジェクトには、Google IMA SDKの必要な機能がすべて組み込まれているため、Googleの通話を直接操作することに煩わされる必要はありません。
  • GoogleのモバイルSDKの上にOoyala SDK for Google IMAを構築したので、引き続きその基本機能を使用できます。

あなたが必要なもの

To get started with Ooyala’s Android SDK for Google IMA, download the following:

  1. Ooyala Mobile SDK for Android (available from Ooyala ダウンロード)
  2. Ooyala Google IMA SDK for Android (Ooyala Google IMA Android) (available from Ooyala ダウンロード)
  3. Google IMAライブラリ（ Googleデベロッパーウェブサイト)
    注意： While using the latest version of the Google IMA libraries please make sure the version you are using is compatible with the latest Ooyala integrations.
  4. Google Android SDK
  5. Android Studio IDE
  6. Google Play Services Project imported into Android Studio.

前提条件：関連するビデオ素材を使用したGoogle IMAの設定

Ooyala Google IMA SDKを使い始める前に、Google IMAが正しく機能することを確認してください。 https://developers.google.com/interactive-media-ads/.

注意： Mobile SDKが依存するOoyala Backlotの動画アセットに関連付けられた静的広告タグが必要です。 これらが本番のBacklotアカウントに存在しない場合は、それらをアプリにロードする必要があります。

Google IMAがすでにデスクトップ上で正常に機能している場合は、モバイルデバイスでも問題なく動作するはずです。

Structure of the Ooyala Google IMA SDK

The SDK has the following directories and files.

ディレクトリ/ファイル 説明
オンラインマニュアル Reference java docs for the OoyalaIMAManager クラス。
IMASampleApp A sample application implementing the Ooyala SDK.
OoyalaIMASDK.jar The Ooyala SDK jar file you need to add to any new Android project .
VERSION Version number of the Ooyala SDK for Google IMA.
