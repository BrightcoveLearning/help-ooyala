ラベル管理API（廃止予定）

新しいラベル管理APIには、Backlot API v1よりも多くの機能があり、それほど複雑ではありません。

古いラベルAPIは、次のことを可能にするモードを提供していました。

  • リストラベル
  • ラベルを作成する
  • ラベルを削除
  • ラベルを割り当てる
  • ラベルの割り当て解除
  • ラベル名を変更

新しいラベルAPIを使用すると、ラベルを一覧表示、作成、名前変更、および削除できます。 ラベルの割り当てと割り当て解除は、アセットAPIの一部になりました。

  • ラベル管理：リスト（非推奨）
  • ラベル管理：作成（非推奨）
  • ラベル管理：削除（廃止予定）
  • ラベル管理：割り当て（廃止予定）
  • ラベル管理：名前の変更（廃止予定）
