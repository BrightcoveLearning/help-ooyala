2015-12-02リリース

このリリースでの注目すべき変更には、Ooyala IQの新しいサードパーティプレーヤーレポート機能と新しいコミュニティログインプロセスが含まれます。

ソフトウェア機能リリース

Ooyala IQの新しいサードパーティプレイヤーレポート機能

あなたは今統合することができます Ooyala サードパーティ製システム（非Ooyala ビデオプレーヤー）。

UIの調整

ウーヤラIQ サードパーティのプレーヤー分析に合わせてUIが調整されました。 調整内容は次のとおりです。
  • プレーヤー サードパーティのプレイヤーからイベントが発生した場合、パネルにサードパーティのプレイヤーの名前が表示されます。 たとえば、JWプレーヤーからのイベントを分析すると、「JWプレーヤー」が表示されます。 プレーヤー パネル。
  • コンテンツのメタデータがBacklotに取り込まれない場合、Ooyala IQは適切なUIの低下をもたらします。 つまり、Ooyala IQは、サードパーティのCMSシステムからのメタデータの取り込みを必要とせずに、サードパーティのプレイヤーイベントに関する分析を提供できます。 たとえば、Ooyala IQがビデオアセットのタイトルメタデータを取得しない場合は、タイトルの代わりにビデオアセットの外部IDが使用されます（例については下の画像を参照）。
  • Ooyalaプレーヤーとサードパーティプレーヤーを使用してOoyalaビデオアセット（Backlotでホストされている）を再生している場合は、1つのBacklotアカウントを使用してビデオをOoyalaプレーヤーとサードパーティプレーヤーにストリーミングできます。 両方のタイプのプレーヤーについて、1つのダッシュボードに（1つのユーザーアカウントを介して）統合されたIQ分析指標を表示できます。プレーヤーパネルとデバイスパネルを見ることで、Ooyalaプレーヤーとサードパーティプレーヤーのさまざまな指標を見ることができます。 サードパーティのアセット（サードパーティのCMSでホストされている）の場合は、アセットメタデータを取り込むために別のBacklotプロバイダーアカウントを設定する必要があります。

メタデータ同期

Backlot APIを使用して、サードパーティのメタデータをBacklotおよびOoyala IQと同期させることができます。 見る Ooyala IQとサードパーティのメタデータを同期させる 詳細については。

バックスロットアカウント管理

サードパーティのプレーヤアナリティクスアカウント用に、独自のBacklotアカウントがあります。 Ooyala IQをスタンドアロンで使用するお客様にとって、Backlot UIをよりユーザーフレンドリーにするためのいくつかの変更が行われました。
  • [管理]タブで、[埋め込み]、[収益化]、[公開ルール]、[カスタムメタデータ]、および[エンコード]サブタブを削除しました。
  • [管理]タブ[詳細]サブタブで、[クローズドキャプション]、[公開先]、[コンテンツの置換]セクションを削除しました。
  • メインのBacklotパネルで、PauseボタンとSize列を削除しました。
  • アセットの[ステータス]セクションには、[ライブ]と[最近作成]のみが表示されます。
  • 左側のパネルのコンテンツタイプでは、リモートアセットとYouTubeビデオのみが利用可能です。
  • 「アカウント」タブで、「アカウント設定」サブタブと「ユーザー管理」サブタブを使用できます。
  • [アカウント]タブの[ユーザー管理]サブタブで、[APIシークレット]キーを使用できます。
  • [アカウント]タブでは、管理者、管理者、読み取り専用、および分析のみのユーザーロールがサポートされています。

新しいJavaScript SDK

サードパーティのプレイヤーのイベントをOoyala IQに伝えるJavaScript SDKを作成しました。 JavaScript SDKと統合するには、プレーヤーのイベントとAPIを変換して、Ooyala JavaScript SDKで定義されているイベントと関連付けるアダプタを使用する必要があります。

あなたはOoyalaアダプタを使うことも、独自のアダプタを作ることもできます。

既存のアダプターとの統合および独自のアダプターの作成について詳しくは、以下を参照してください。 JS SDKを使用したサードパーティ製プレイヤーとIQの統合.

使ってみよう

この新しい機能を使用するには、次の手順に従います。
  1. アカウントマネージャに新しいプロバイダアカウントを作成するよう依頼してください。 サードパーティのプレイヤーレポート機能付き。
  2. サードパーティのメタデータを同期する Ooyala IQとBacklotを使って。
  3. 新しいを使う Ooyala IQ JavaScript SDK サードパーティのプレイヤーイベントをOoyala IQと統合するため。 このステップでOoyalaプロフェッショナルサービスを利用したい場合は、Ooyalaの窓口にご相談ください。

コミュニティアップデート

Ooyalaコミュニティをよりオープンでアクセスしやすいものにするために、サインオンにLithiumプラットフォームを利用する新しいログインシステムを実装しました。 この変更は、12 / 2 / 15より前にアクティブになっている既存のコミュニティメンバーにのみ顕著です。 以前は、コミュニティのサインインはあなたのBacklotアカウントに関連付けられていました。 コミュニティにアクセスするには、12 / 2 / 15より前にアクティブなメンバーは新しいパスワードを生成する必要があります。 このプロセスの詳細については、 http://community.ooyala.com/t5/Ooyala-Blog/New-Community-quot-Sign-in-quot-coming-soon/ba-p/7821.
