Supported Browsers and Operating Systems for プレイヤーV4

プレイヤーV4 3つの製品に実装されている新しいスキンを備えています。
  • OoyalaプレイヤーV4 for HTML5 デスクトップとモバイルデバイスを渡る普及したブラウザで動くWebベースのプレーヤーです。
  • iOS用Ooyalaプレイヤースキン を統合することにより、iOS用のOoyala（ビデオプレーヤー）Mobile SDKを強化します。 プレイヤーV4 iOSネイティブアプリのスキン。
  • Android用Ooyalaプレイヤースキン 統合することで、Android用のOoyala（ビデオプレーヤー）Mobile SDKを強化します。 プレイヤーV4 Androidネイティブアプリのスキン。
Player V4と新しいプレイヤースキンは、以降の環境で現在サポートされています。

Windowsサポート

OoyalaプレイヤーV4 for HTML5 Windows環境で実行されている次のブラウザでサポートされています。
  • Windows上のChrome 47 + 7 +
  • Windows 11 +上のInternet Explorer 7 +（またはMPEG-DASHまたはHLSストリームパッケージが使用されている場合はWindows 8.1 +）
  • Windows 10上のMicrosoft Edge
  • Windows上のFirefox v48 + 7 +
注意： 現在IE / Edgeを実行しているWindowsタブレットでのタッチ処理に関する既知の問題がいくつかあります。これにはマイクロソフトの解決が必要です。

Mac OSのサポート

OoyalaプレイヤーV4 for HTML5 Mac OS 10.9 +環境で実行されている次のブラウザでサポートされています。
  • Chrome v47 +
  • Firefox v48 +
  • Safari v8 +

モバイルWebサポート

OoyalaプレイヤーV4 for HTML5 モバイルWebの次のブラウザでサポートされています。
  • iOS上のSafari v9 +
  • Android v4.4 +上のChrome

iOSアプリのサポート

iOS用Ooyalaプレイヤースキン iOS v9 +環境でサポートされています。

Androidアプリのサポート

Android用Ooyalaプレイヤースキン Android v4.4 +を実行しているデバイスでサポートされています。

注意： 2017-10-01以降、サポートされているAndroid環境はAndroidバージョン4.4から始まります。 このバージョンより前のAndroidバージョンに関する問題は、ケースバイケースで見直されます。
