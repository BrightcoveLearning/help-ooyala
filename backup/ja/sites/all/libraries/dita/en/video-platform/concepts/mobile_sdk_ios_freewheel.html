iOS上のFreeWheelとの統合

SDKを使用すると、モバイルデバイス向けの豊富なカスタムビデオアプリケーションエクスペリエンスを迅速に開発できます。 SDKには、ビデオの再生、VAST広告およびOoyala広告の表示、Ooyalaコンテンツのクエリなどを行うためのメソッドやクラスが含まれています。
  • Playback
  • プレイリスト/チャンネル - サムネイル
  • インフォ
  • 関連メディア
  • ライブ（新）
  • 広告
  • アナリティクス
  • モバイル使用状況を理解するための完全に統合された分析
  • クローズドキャプション
  • DFXP（現在はTTML）ファイルを介してクローズドキャプションをロードおよび表示するためのAPI

あなたが必要なもの

FreeWheelを統合するためのOoyala SDKを使用すると、デスクトップで作成したのと同じ広告エクスペリエンスを、サポートされているモバイルデバイスでも作成できます。

iOSでOoyalaのSDK for FreeWheelを使い始めるには、以下をダウンロードしてください。

  • iOS用Ooyala Mobile SDK Ooyala ダウンロード)
  • iOS用のOoyala FreeWheel SDK Ooyala ダウンロード）（Ooyala FreeWheel iOS）

    注意： Mobile SDKが依存するOoyala Backlotの動画アセットに関連付けられた静的広告タグが必要です。 これらが本番のBacklotアカウントに存在しない場合は、それらをアプリにロードする必要があります。

  • AdManager.framework FreeWheelのWebサイト（zipをダウンロードするにはFreeWheelの認証情報が必要です）
    注意： FreeWheelライブラリの最新バージョンを使用している間、あなたが使用しているバージョンが最新のOoyala統合と互換性があることを確認してください。
  • アップルXcode

基本パラメータとFreeWheel OPFモジュール広告セット

Mobile SDKでFreeWheelを利用するには、タイプの広告セットを作成する必要があります。 フリーホイールOPFモジュール Backlotで。 詳細はBacklotユーザーガイドを参照してください。 Ooyalaでは、FreeWheel広告関連のパラメータをさまざまな場所に保存できます。 優先順位の高い順に、FreeWheelパラメータとその値は次のように定義できます。
  1. あなたのアプリ自体。
  2. お客様のサクセスマネージャーを介して設定できる、内部Ooyala構成。
  3. バックスロットで マネー タブ、 広告セット のサブタブ フリーホイールOPFモジュール 広告セットの種類。
OoyalaのMobile SDK用のFreeWheel統合に含まれているサンプルアプリには、次の行が含まれています。これらの行には、アプリで設定する必要がある2つの重要なパラメーターが示されています。
注意： これらのパラメータは、アプリケーションで直接設定するか、Customer Success ManagerによってOoyalaに内部設定する必要があります。
  • fw_ios_ad_server：iOSで広告を配信するためのURL
  • fw_ios_player_profile：iOS上のプレーヤー用に定義されたプロファイル
// FreeWheelパラメータを設定します。 これらはオプションであり、BacklotまたはOoyala内部で設定されている設定を上書きすることに注意してください。
[fwParameters setObject：@ "http://demo.v.fwmrm.net/" forKey：@ "fw_ios_ad_server"]; [fwParameters setObject：@ "90750：ooyala_ios" forKey：@ "fw_ios_player_profile"];
[fwParameters setObject：@ "ooyala_test_site_section" forKey：@ "fw_ios_site_section_id"]; [fwParameters setObject：@ "ooyala_test_video_with_bvi_cuepoints" forKey：@ "fw_ios_video_asset_id"]; [fwParameters setObject：@ "channel = TEST; subchannel = TEST; section = TEST; mode = online; player = ooyala; beta = n" forKey：@ "FRMSegment"];
