無制限の資産データをクエリする方法はありますか

資産以外のディメンション、および1か月を超えるすべてのクエリでは、最大10,000行までしかアクセスできません。 ただし、クエリの日付範囲が1か月未満の場合は、IQ APIを通じて無制限の資産データを取得できます。

どのように v3アナリティクス

に対してクエリ 資産 ディメンション、1月以下（<32日）の日付範囲を設定し、 ページ そして 制限 パラメーター。 次の例では、データの最初の1000行を返します。 最後のページまですべてのページを繰り返してデータをポーリングします。 結果の合計行数と現在のページ番号を使用して、最後のページを決定できます。
[GET] /v3/analytics/reports/?
report_type=performance
&dimensions=asset
&metrics=plays_requested
&start_date=2015-10-10
&end_date=2015-10-20
&page=0
&limit=1000
&api_key=yourApiKey
