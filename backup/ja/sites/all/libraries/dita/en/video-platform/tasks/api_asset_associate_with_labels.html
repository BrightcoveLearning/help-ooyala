ラベルを使った作業

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

ビデオ、チャンネル、リモートアセット、またはチャンネルセットをラベルに関連付けた後、 バックスロット シンジケーションの構成方法に基づいて自動的に公開します。

それらを資産に関連付ける前に、ラベルを作成する必要があります。 見る .

資産のラベルを操作するには：

  1. ラベルをアセットに関連付けるには、 POST とともに / v2 / assets ルート、アセットID、 / v2 /ラベル 修飾子、およびラベルID。 複数のラベルを同時に関連付けることもできます。 次の例では、Motorcycle Racingラベルを JxbzdkMjqBEsO 資産。 
    [PUT]/v2/assets/JxbzdkMjqBEsO/labels/bace921fdea44cc18a5a273155514522

    バックスロット 次のような応答を返します。

    {"items"：[{"name"： "スポーツ"、 "id"： "814efb109416490a98ee3f4fcd6784cf"、 "フルネーム"： "/ Sports"、 "parent_id"：null}、{"name"： "バイクレース"、 " id "：" bace921fdea44cc18a5a273155514522 "、"フルネーム "：" /スポーツ/オートバイレーシング "、" parent_id "：" 814efb109416490a98ee3f4f6784fcdXNUMXcf "}]}}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. 資産の現在のラベルを表示する GET / v2 / assets /asset_id/ラベル. 次の例では、のラベルを取得します。 JxbzdkMjqBEsO 資産。 
    [GET] / v2 / assets / JxbzdkMjqBEsO / labels

    バックスロット 次のような応答を返します。

    {"items"：[{"name"： "スポーツ"、 "id"： "814efb109416490a98ee3f4fcd6784cf"、 "フルネーム"： "/ Sports"、 "parent_id"：null}]}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  3. アセットから単一のラベルを削除するには、 DELETE とともに / v2 / assets ルート、アセットID、 /ラベル 修飾子、およびラベルID。 資産からすべてのラベルの関連付けを解除するには、次のように指定します。 /ラベル ラベルIDなし。
    注意： ラベルを完全に削除してすべてのアセットから削除するには、 ラベルを削除する.

    次の例では、Motorcycle Racingのラベルを JxbzdkMjqBEsO 資産。

    [DELETE]/v2/assets/JxbzdkMjqBEsO/labels/814efb109416490a98ee3f4fcd6784cf

    バックスロット 200応答を返します。

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
