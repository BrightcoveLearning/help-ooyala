機能

Ooyalaは、オンラインビデオ管理、出版、分析、収益化のリーダーです。 当社の統合された一連の技術とサービスは、コンテンツ所有者に視聴者を拡大する力と、視聴者の関心を高めビデオからの収益を高める深い洞察をもたらします。

概要については、 Ooyalaビデオプラットフォーム そして オンラインビデオプラットフォームの紹介.

管 理

どのくらいの量のビデオを作成しても、Ooyalaがそれを処理します。 チーム全体にとってアップロード、管理、公開は簡単です。

  • 最適 - Ooyalaは、ほぼすべてのコンテンツ管理システム（CMS）または独自の社内システムと統合できます。 ビデオを個別に、またはまとめて管理します。
  • フルアクセス、フルコントロール - Backlotは、あなたが選択した人々に正確にアカウントアクセスを提供します。
  • すべての画面とサイズ - 一度アップロードしてリラックスしてください。 当社の分散型クラウドは、すべての主要なフォーマットおよび画面を迅速かつ確実に自動的にトランスコードします。
  • あなたのプレーヤー、あなたのルック - もちろんあなたのビデオにあなたのブランドを反映させたいです。 プレーヤーを真にカスタマイズするか、当社のテンプレートを使用してください。Silverlight、Flash、HTML5のプレーヤーをサポートしています。
  • リモートアセット - 他のリモートオンラインソースからビデオを簡単に取得できるので、機密コンテンツを自分のサーバーでホストできます。
  • カスタムメタデータ - 各動画にキーと値のペアを割り当てます。 カスタムメタデータを使用して、コンテンツを整理および分類したり、ビデオが埋め込まれているページに表示するビデオに関する情報を保存したり、ビデオの表示方法に関する情報（CSSデータなど）を指定したりできます。
  • ビデオラベル - ファッションやカスタマーサポートなどのトピックでビデオにラベルを付けるか、動物>ペット>犬などの複数のラベルを入れ子にします。 それからあなたのビデオをシンジケートするか、それらのラベルに基づいて動的なプレーヤーを作成します。 ラベルはまたあなたのサイトのセクションへの論理的なマッピングを提供してビデオ管理を改善するのを助けます。
  • より高速なトランスコーディングとエンコーディング - Ooyala独自のトランスコーディングクラウドは、迅速かつ効率的な並列処理のために着信ファイルを複数の部分に分割します。 デコードとエンコードは複数のノードにまたがって行われ、新しいノードが動的に追加されてビジネスニーズに合わせて拡張されます。 Ooyalaは、HTTPライブストリーミング、スムーズストリーミング、HTTPダイナミックストリーミング、RTMP、RTMPE、Akamai HD Network、プログレッシブダウンロードなどの最新のエンコードおよび配信フォーマットをサポートしています。

パブリッシュ

マドリードの携帯電話。 ボストンの大画面。 奥地のラップトップ。 Backlotはあなたのビデオを最高品質で世界中のあらゆるスクリーンに届けます。

  • 魅力的に構築 - HD品質のビデオに加えて、投票、買い物、またはチャイムインのためにビデオをクリックできるインタラクティブな要素を備えた最新の視聴者をつかみましょう。
  • どこでもTV - Ooyalaは、Roku、Boxee、その他のセットトップボックスを介して、またはPanasonic、LG、Samsungなどのインターネット接続テレビに直接HDビデオをTV画面に配信するためのテクノロジを構築しました。
  • フルシンジケーション - YouTube、Facebook、iTunes、およびRokuにビデオを公開します。 または、パートナーに直接公開するための独自のMRSSフィードを作成します。 Ooyalaは、リーチと収益を簡単に構築できます。
  • 視聴者はモバイルです - 適応型のビットレート配信と動的なプレーヤーのHTML5への切り替えにより、すべてのフォーマットとデバイスでビデオをトランスコードできます。 Ooyalaでは、統合モバイル広告を使ってモバイルビデオを収益化することもできます。
  • ライブストリーミング - Ooyalaのライブストリームのネイティブサポートにより、ライブストリーミングはオンデマンドビデオと同じくらい簡単になりました。 完全な分析、シンジケーション、およびアダプティブビットレート配信が含まれています。
  • アダプティブビットレートパワー - デバイスや接続速度に関係なく、視聴者に最高の視聴エクスペリエンスを提供するために、Flash用のアダプティブビットレートテクノロジを初めて開発しました。 私たちは、視聴者が世界中のあらゆるデバイスで最適なエクスペリエンスを得られるように、技術の開発を続けています。
  • あなたがコントロールしている - あなたのビデオを韓国でのみ公開したいですか？ 平日だけ？ 特定のドメインだけで？ いつでもどこでも？ 地域ターゲティングツールやその他のターゲティングツールを使用すると、完全に制御できます。
  • クローズドキャプション - セクション508準拠が必要ですか。 Ooyalaがお手伝いします。 そしてクローズドキャプションは聴覚障害のある視聴者を助けるだけでなく、あなたのコンテンツを検索エンジンにとってより見やすくします。
  • ソーシャルシェア - Facebook、Twitter、あらゆる種類のソーシャルメディアにわたって、視聴者がチャンスを与えれば、視聴者があなたのビデオを正確なシーンまで共有できます。 また、Ooyala分析を使用すると、ビデオがどのようにどこで共有されているかを確認できます。

分析する

ライブビデオとオンデマンドビデオのリアルタイム分析により、視聴者がコンテンツをどのように、いつ、どこで見て共有しているのかを真に理解するために必要なデータが得られます。 視聴者が何を見ているの？ 彼らはいつ興味を失いますか？ 彼らは何を共有しますか？ 実用的な分析は当て推量を終了します。 明確な数字を手に入れて、本当にうまくいくコンテンツと収益化戦略を構築します。

  • 効果的なコンテンツ戦略 - Ooyalaのパブリッシャは、自分のコンテンツが視聴者によってどのように、いつ、どこで見られ、共有されているかを本当に理解するために必要なデータを入手します。 パブリッシャーはそのリアルタイムデータを使ってエンゲージメントを高め、ビデオからより多くのお金を稼ぎます。 そのため、私たちはそれらをアクション可能分析と呼びます。
  • すべての動画からより多くの収益を得る - 最高の広告数 どのプレースメントで最も収益が上がるか あなたはビデオのためにいくら課金するべきですか？ 実用的な分析は、収入を最大にする方法をあなたに示します。
  • 誰が見ているの？ そして、その方法は？ - あなたの聴衆：彼らが居住する場所（国、州、指定マーケティングエリア）、ラップトップ、モバイルデバイス、リビングルームのテレビなど、さまざまな場所にいます。 都市レベルのパフォーマンスにドリルダウンして、バッファローまたはブエノスアイレスで視聴されているビデオを確認することもできます。
  • お客様と連携するAPI - 独自のカスタムダッシュボードを作成して視聴数を監視したり、エグゼクティブレポートを作成したり、自分の特別な関心事を追跡したりするための柔軟なAPIを使用します。 OmnitureとVisible Measuresには、Ooyalaのビデオ分析を直接レポートに取り込むプラグインがあります。
  • ドメインレポート - トップレベルドメインおよび個々のURLに関する完全なレポートを使用すると、ビデオが埋め込まれてWeb上で共有されている場所を正確に確認できます。
  • 地理レポート - 都市レベルのレポートでは、ビデオが影響を及ぼしている場所を正確に把握できます。
  • 指定マーケットエリア（DMA）レポート - これらの業界標準ゾーンで広告を購入することを好む大規模メディア企業にとって、DMAリーチと周波数数は非常に重要です。
  • エンゲージメントレポート - 1秒ごとのエンゲージメントと放棄のデータは、何がうまくいっているのか、そしてどこで広告が最も収益を上げられるのかを示しています。
  • ユニーク訪問者レポート - ビデオを閲覧している訪問者の数を追跡し、日ごと、週ごと、月ごとに複数のビデオの結果を比較します。
  • ソーシャルシェアレポート - Facebook、Twitter、その他のソーシャルメディアで視聴者がどのように動画を共有しているかを確認します。
  • デバイスレポート - PC、携帯電話、iPad、その他のデバイスからのビデオビューの数を調べます。 あなたは人々が見ている方法のためにあなたのコンテンツを最適化していますか？
  • 外部発行とYouTubeのレポート - あなたのビデオはあなたのサイトを超えた人生を送ります。 外部の景色を追跡していないのであれば、写真の一部しか見えていません。

収益化

Ooyalaは、ビデオソリューションと同じくらい収益化ソリューションです。 結果を測定して改善するための優れた分析とともに、広告および統合されたペイウォールの収益化オプションが利用可能です。 そしてドラッグアンドドロップのインターフェイスはそれをすべて容易にさせる。

  • 広告配信を簡単に - Backlotは、現在使用している可能性があるものも含め、20を超える大手広告ネットワークとの統合が組み込まれています。 ご希望の場合は、Backotの組み込み広告サーバーを使用して独自の在庫を販売してください。
  • シンプルなドラッグアンドドロップ - 広告を配置するのは、選択したビデオ内の任意の場所に広告をドラッグするのと同じくらい簡単です。 これにより、視聴者の離脱が最小限に抑えられ、収益が最大化される場所に広告を簡単に配置できます。
  • 一括で広告を管理する - テーマやコンテンツで独自の広告セットを作成し、それらすべての動画の広告をまとめて管理します。たとえば、すべてのスポーツやニュース動画にプレロールを追加します。
  • ビデオカウントダウン - プレーヤ広告カウンタは、各広告に表示される期間を視聴者に正確に示します。 それはより幸せな視聴者と少ないドロップアウトを意味します。
  • 広告API - Ooyalaと同様に、Googleのエンジニアリングチームは、広告ワークフローをカスタマイズおよび簡素化するための柔軟なAPIを構築しています。
  • ペイウォール統合 - サードパーティのペイウォールベンダとの連携。
  • 新たな獲得の機会 - ライブペイパービュー、オンデマンドレンタル、国や都市によるシンジケート：Ooyalaはそれをすべて可能にします。

モバイル

モバイルビデオは現在、すべてのモバイルデータトラフィックの最大の部分を占めており、100を通じて年間2015％の年複利率で成長し続けるでしょう。 世界で750の3Gモバイル加入者がすでに存在するため、堅実なモバイルビデオ戦略はあなたのビジネスにとって必須です。

  • モバイルビューアに到達する - Ooyalaは、HTML5互換プレーヤを自動的にスワップインして、iPhoneおよびiPadで再生します。 Flash 10.1のサポートにより、Androidおよび他のモバイルデバイスでの完璧な再生が保証されます。
  • iOSデバイスへの安全な配信 - パブリッシャは、自分のビデオを暗号化フォーマットでiPod、iPhone、およびiPadに配信できるようになり、コンテンツの違法コピーがより困難になりました。 Ooyalaは安全なiOS配信を提供する最初のオンラインビデオソリューションです。
  • ライブコンテンツをストリーミングする - ライブコンテンツをiOSデバイスにストリーミング配信すると、より多くの顧客にアクセスできます。 Ooyalaの統合されたターンキーソリューションを使用すると、数分でiOS向けのライブストリームを設定できます。
  • モバイルビューアの注目度を理解する - 個々のビデオレベルまで豊富なレポートを作成することで、モバイルビューアと、あらゆるビューからより多くの利益を得ることができるデザイン戦略について学ぶことができます。 高度なトレンドデータを使用して、Android、iPhone、iPadなどのモバイルデバイス間でのビデオ消費量を比較し、モバイル視聴者数が急増していることを確認します。
  • ライブストリーミング - Ooyalaのライブストリームのネイティブサポートにより、ライブストリーミングはオンデマンドビデオと同じくらい簡単になりました。 完全な分析、シンジケーション、およびアダプティブビットレート配信が含まれています。
  • 常に最高の視聴環境 - iOSデバイス用の適応ビットレートを使用すると、携帯電話ネットワークや接続速度に関係なく、視聴者は常に最高品質の視聴環境を体験できます。
  • モバイルコンテンツの収益化 - OoyalaはVASTに準拠しているため、広告は複数の広告ネットワークやサーバーからモバイルFlashまたはHTML5プレーヤーにストリーミング配信できます。彼らが見ている機器は何ですか。
  • iOSデバイスのコンテンツ制限 - 公開ルールでは、コンテンツの再生を特定の地域、ドメイン、および時間範囲に制限することができます。 コンテンツ所有者とライセンス契約を結んでいる場合は、公開ルールにより、配布制限を満たすことができます。

強力で拡張可能なAPI

  • Backlot API - 開発者がすべてのコアBacklot機能にアクセスできるようにします。 多くの出版社がサードパーティのCMSと統合するためにそれを使用しています。
  • Ingestion API - ほぼすべてのデバイスまたはWebサイトからコンテンツを取り込み、一括アップロードをプログラムで制御し、他のビデオ作成および取り込みシステムと統合します。
  • プレーヤAPI - プレーヤをブランド化してカスタマイズし、インタラクティブビデオや新しい収益化エクスペリエンスなどのリッチメディア機能を追加できます。
