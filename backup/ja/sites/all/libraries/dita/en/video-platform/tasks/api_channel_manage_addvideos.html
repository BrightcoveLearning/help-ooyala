チャンネルにビデオを追加する（廃止予定）

チャンネルを作成したら、ビデオの追加を開始できます。

チャンネルが期待どおりに再生できるように、チャンネルのすべての動画に同じプレーヤーを割り当てる必要があります。
注意： チャンネルとチャンネルセットは廃止予定で、無効になる予定です。 詳細と代替案については、 OVPリリースノート.
注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
注意： チャンネルにチャンネルを追加することはできません。 チャンネルのコレクションを作成するには、チャンネルセットを使用します。

動画をチャンネルに追加するには：

  1. 現在チャンネルに割り当てられているビデオを表示するには、 GET / v2 / assets、アセットID、および /並ぶ 修飾子 次の例では、 dvNTVxMjrRktSlb5v チャンネル。 
    [GET] / v2 / assets / dvNTVxMjrRktSlb5v / lineup

    バックスロット 次のような応答を返します。

    ["Argh3lMjphu25tH"、 "RQlbdvMjq9QtGGG"]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. 現在のビデオを新しいセットに置き換えるには、 PUT とともに / v2 / assets ルート、チャンネルのアセットID、 /並ぶ リクエストボディ内の動画のアセットID。 次の例では IzNnllMjphu2XF3 そして Y1dTdvMjq9QtOMG ビデオアセット dvNTVxMjrRktSlb5v 現在の動画を置き換えます。 
    [PUT] / v2 / assets / dvNTVxMjrRktSlb5v / lineup ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]

    バックスロット 次のような応答を返します。

    ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  3. 現在のセットを置き換えずに1つのビデオをチャンネルに追加するには、 PUT / v2 /アセット チャンネルのアセットIDで、 /並ぶ 修飾子、および追加する動画のアセットID。 次の例では BrdXVjMjrgtupU3M ビデオアセット dvNTVxMjrRktSlb5v チャンネル。 
    [PUT] / v2 / assets / dvNTVxMjrRktSlb5v / lineup / BrdXVjMjrgtupU3M

    バックスロット 次のような応答を返します。

    ["BrdXVjMjrgtupU3M"、 "IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]

    それを試してみてください

  4. チャンネルからビデオを削除するには、 DELETE / v2 / assets チャンネルのアセットIDで、 /並ぶ 修飾子、および削除する動画のアセットID。 次の例では、 BrdXVjMjrgtupU3M からのビデオアセット dvNTVxMjrRktSlb5v チャンネル。 
    [削除] / v2 / assets / dvNTVxMjrRktSlb5v /ラインナップ/ BrdXVjMjrgtupU3M

    バックスロット 次のような応答を返します。

    ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

チャンネル内の動画は正常に更新されました。
