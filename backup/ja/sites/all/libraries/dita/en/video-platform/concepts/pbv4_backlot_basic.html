基本的な埋め込み情報

This topic describes the information you will need to provide when you embed プレイヤーV4 on a web page. See for details.

Retrieve Your pcode

プレイヤーV4 requires that you provide your pcode, which you can retrieve from Backlot.
  1. Log into your Backlot account.
  2. ストリーミングを停止する場合は上部の アカウント をクリックします。
  3. ストリーミングを停止する場合は上部の 開発者 サブタブ
  4. Look for your API key on the left side of the screen.
  5. Copy your pcode, which is the alphanumeric string that precedes the period. For example, if the API key is xxxxxxxxxxxxxxxxxxxxxxxxxxxx.xxxxx, then use the first 28 characters (xxxxxxxxxxxxxxxxxxxxxxxxxxxx) for the pcode.

Retrieve a Player Branding ID

プレイヤーV4 requires that you provide a playerBrandingId, which references a player defined in your Backlot account. Backlot displays this as プレーヤーID. To retrieve this from Backlot:
  1. Log into your Backlot account.
  2. ストリーミングを停止する場合は上部の 管理 をクリックします。
  3. ストリーミングを停止する場合は上部の 埋め込む サブタブ
  4. Select your player from the Custom Player dropdown.
  5. Copy its プレーヤーID, which you need to provide in the playerBrandingId.

Alternatively, you can retrieve the Player ID with the Backlot API. See the Backlot API reference 詳細については。

You can use players that have already been defined for Player V4 (see プレーヤV4への移行), or you can create an entirely new player.

There are two ways in which to create a player in Backlot:

  • Backlot UI (Backlotでプレイヤーを作成する)
  • Backlot API（を参照） Creating a Player Using the Backlot REST API)

Retrieving the Asset ID

プレイヤーV4 requires that you provide an asset ID that uniquely identifies the video asset. An asset ID is the same as a コンテンツID (in the Backlot UI) and an 埋め込みコード （の中に プレイヤーV4 JavaScript API). To retrieve this from Backlot:
  1. Log into your Backlot account.
  2. ストリーミングを停止する場合は上部の 管理 をクリックします。
  3. ストリーミングを停止する場合は上部の 埋め込む サブタブ
  4. Select your content from the list of video assets.
  5. Copy its コンテンツID, which you need to provide in the assetId.

Retrieving a Default Embed Code

You can retrieve a default Player V4 embed code from Backlot and paste it into your web page.
  1. Log into your Backlot account.
  2. ストリーミングを停止する場合は上部の 管理 をクリックします。
  3. ストリーミングを停止する場合は上部の 埋め込む サブタブ
  4. 下 カスタムプレイヤー, select a Player V4 player.
  5. の中に 埋め込みコードを生成する section, configure any settings you want to change. For details, see バックスロットでのプレーヤー埋め込み設定.
  6. Copy the embed code and paste it into your web page.
