クローズドキャプションの使用

You can add closed captions to videos, ads, remote assets, or YouTube videos.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

Supported Formats for Uploading via the Backlot API

拡張 説明
.TTML （旧 DXFP) TTML（Timed Text Markup Languageファイル。一般に、クローズドキャプションファイルの拡張子は.dfxp（Distribution Format Exchange Profile）です。詳細については、を参照してください。 サポートされているクローズドキャプションDFXP（現在はTTML）形式.

Upload Closed Captions

[PUT] / v2 / assets /asset_id/closed_captions<dfxp_file_content>

Get Closed Captions

[GET]/v2/assets/asset_id/ closed_captions

Delete Closed Captions

[DELETE]/v2/assets/asset_id/ closed_captions

例

This example adds closed captions to the IzNnllMjphu2XF3_UgPROoCi9B2rtWs 資産： 

[PUT]/v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/closed_captions
<dfxp_file_contents>

Backlotは200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
