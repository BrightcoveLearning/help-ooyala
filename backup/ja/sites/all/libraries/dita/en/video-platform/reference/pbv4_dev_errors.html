JavaScriptを使用したPlayer V4のエラー処理

エラー処理機能とイベントを使用すると、Playerエラーに関する情報を取得できます。

The Ooyala Player V4 error functions and events are described in the following tables.

エラー関数

次の表に、サポートされているエラー処理機能を示します。

関数 説明
getErrorCode Displays the video error code
getState 現在のプレイヤーの状態を取得します。 可能な状態は、で定義されています。 OO.STATE 次のように
  • ローディング
  • READY
  • プレイ
  • PAUSED
  • バッファリング
  • ERROR
  • 破壊されました

エラーメッセージ

次の名前空間 プレーヤJavaScript APIリファレンス, contain the error messages that can be handled through the V4 JavaScript Message bus. Use message bus events to handle errors by subscribing to or intercepting the OO.EVENTS.ERROR イベント：

エラーネームスペース 説明
OO.ERROR Represents the Ooyala V4 Player Errors. This is the parent namespace for all errors, and includes errors for channel content.
OO.ERROR.API コンテンツ、メタデータ、およびサーバー接続に関連するエラーを表します。
OO.ERROR.API.SAS 承認関連のすべてのエラーを表します。
OO.ERROR.PLAYBACK 再生に関するすべてのエラーを表します。
OO.ERROR.VC Represents errors related to video element creation and encoding.

エラーイベント

次の表は、メッセージバスによって発行されたエラー処理イベントについて説明しています。

エラーイベント 説明
PLAYER_CREATED プレーヤが作成された後に発行される一番最初のメッセージ。初期化するようにすべてのプラグインに通知するために使用されます。
ERROR エラーが発生したときに公開されます。2番目のパラメータにはエラーの詳細があります。
破壊する プレイヤーが破壊されているときに公開されます。

エラーイベントの処理

OO.EVENTS.ERROR event is published to the message bus. The mb.subscribe triggers the function you specify (the callback function) when an error happens, and the error continues to propagate to other subscribers. Use mb.subscribe and handle the error as desired according to any existing error-handling functions you might already have in place (such as printing to the console, stopping playback, displaying an error screen, etc). In the following example illustrating the use of the Message Bus 申し込む（） method, the callback function YourErrorHandlingFunctionNameHere is invoked if the OO.EVENTS.ERROR イベントはメッセージバスに発行されます。
<script>
    function onCreate(player) {

        player.mb.subscribe(
            OO.EVENTS.ERROR,
            'test',
            function(event, payload) {
                YourErrorHandlingFunctionNameHere();
            }
        );
    }

    OO.ready(function() {
        window.pp = OO.Player.create('container', 'YOUR_ASSET_ID', {
            "playerBrandingId": "YOUR_PLAYER_ID",
            "pcode": "YOUR_PCODE",
            skin: {
                config: "url_where_hosted/skin-plugin/skin.json",
            },
            // Include the onCreate() function you defined above in these embedded parameters.
            onCreate: window.onCreate
        });
    });
</script>
