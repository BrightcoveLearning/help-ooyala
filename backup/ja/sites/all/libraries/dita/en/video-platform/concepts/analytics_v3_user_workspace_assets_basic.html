ビデオの詳細

ビデオの詳細を調べるには、[ビデオの詳細]ページを使用します。

ビデオの詳細 このページは大規模なコレクションの一部です。 ビジネス·インテリジェンス ページ。 ビデオ（またはアセット）の名前またはタイトルをクリックすると、 ビデオの詳細 のページをご覧ください。

ビデオの詳細 ページには、動画の掲載結果とエンゲージメントのデータがあります。

[ビデオの詳細]ページのレポートをCSVファイルとしてエクスポートするには、ダウンロードアイコンをクリックします。

[Video Details]ページは、4つの主要領域に分かれています。
  1. ビデオセクション。KPI（Key Performance Indicator）バー、ビデオ自体、およびビデオ情報が含まれています。
  2. [視聴セグメント]セクションには、ビデオの再生時間に対してプロットされた再生要求数が表示されます。 1セグメントは、ビデオの長さの2.5％として定義されています。 動画の開始数が、最初のセグメント全体を見た視聴者の数よりも多い場合があります。 一部のユーザーは、ビデオの最初の1 / 40番目が完了する前に中断することがあります。

    注意： ユーザーが同じセグメントをN回巻き戻して視聴した場合、そのセグメントに対して監視されたセグメントはN回としてカウントされます。

  3. の中央パネル 一意のユーザー, プレイズ, 視聴時間、および 実況プレイ.
    1. ユニークユーザー 一定期間にわたるユニークユーザー数（ページ上部に設定した日付範囲）をグラフィカルに表示します。
    2. 要求された演劇数： 一定期間に要求された再生回数（ページ上部に設定した日付範囲）をグラフィカルに表示します。
    3. 見た時間： 経過時間の経過時間をHH：MM：SS形式で表示します（ページ上部で設定した日付範囲）。
    4. プレイスルー： 要求された再生回数に対してプロットされた、アセットの再生完了のパーセンテージ、25％、50％、75％を表示します。
  4. 動画に関する以下の情報について、情報をグラフまたはリスト形式で表示するメトリックセクション。
    • デバイス：要求された再生、視聴時間、およびビデオが再生されたデバイスの再生を表示します。
    • 地理：要求された再生、視聴時間、およびビデオが再生されたさまざまな地域の再生を表示します。
    • プレーヤー：要求された再生、視聴時間、およびビデオが再生されたさまざまなビデオプレーヤーの再生を表示します。
    • トラフィックソース：要求された再生、視聴時間、およびビデオが再生されたさまざまなトラフィックソースの再生を表示します。

日付範囲を設定する

とともに 期間を選択 プルダウンすると、プロットされたグラフと表示されているメトリックの日付範囲を変更できます。 以下のオプションから選択できます。 週は月曜日 - 日曜日として定義されることに注意してください。

  • 今日
  • 昨日
  • 今週
  • 先週
  • 先週の営業日
  • 今月
  • 先月
  • 今年
  • 昨年
  • 最終12月間

タイムセグメント

チャートの右上には、表示されているデータの粒度（詳細レベル）を変更するために使用できる一連のボタンがあります。

以下の時間区分が利用可能です。

  • 15ミン
  • 時間
  • 日
  • 週間
  • 月

デフォルトの時間セグメントはDayです。

注意：

日時セグメントのデフォルトの日付範囲は、15分および時セグメントのデフォルトの日付範囲とは異なります。

日時セグメントの場合、デフォルトの日付範囲は、週の最初の日の12：00amから最後の日の11：59pm（8 24時間セグメント）です。 15の分と時の時間セグメントの場合、デフォルトの日付範囲は、週の最初の日の12：00pmから最後の日の12：00pm（7 24時間セグメント）です。

デフォルトの日付範囲の違いは、日の時間セグメントと15の分または時間のセグメントを切り替えると、メトリックデータが変わることを意味します。 今後のリリースでは、15の分と時間の時間セグメントの日付範囲の制限を日の時間範囲（8 24の時間セグメント）と一致するように変更します。
