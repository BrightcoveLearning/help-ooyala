プレイヤーV4 JavaScript API

プレイヤーV4 APIは、メッセージバスのイベント処理とともに、プレーヤの機能とUI要素をカスタマイズするためのJavaScript APIです。 見る docs.brightcove.com/apidocs-ooyala/ Player V4 JavaScript APIリファレンス用。
  • プレーヤV4 JavaScript APIについて
  • Player V4 JavaScriptを使用したクローズドキャプションのプログラミング
    クローズドキャプション（CC）APIを使用すると、サポートされているクローズドキャプションを取得/設定したり、プレーヤのUI言語をローカライズしたりできます。
  • プレーヤV4のイベントモデル
    プレーヤーV4はイベントを処理するためにメッセージバスの使用を取り入れています。
  • JavaScriptを使用したPlayer V4のビットレートのプログラミングとバッファリング
    Ooyalaはビットレートとバッファリングの機能とイベントを提供します。 さらに、Ooyalaは再生をできる限りスムーズにするために多くのヒューリスティックを使用しています。
  • JavaScriptを使用したPlayer V4のエラー処理
    エラー処理機能とイベントを使用すると、Playerエラーに関する情報を取得できます。
