ビデオ情報の編集

動画に関する情報はいつでも変更できます。

To edit video information:

  1. 使用 / v2 / assets ルート。 次の例では、getを実行して資産の現在の設定を取得します。 Y1dTdvMjq9QtOM IDを指定します。 
    [GET] /v2/assets/Y1dTdvMjq9QtOM
    注意： To get information about all assets, do not specify the asset ID in the request.

    バックスロット 次のような応答を返します。

    {
  "asset_type": "video",
  "duration": 30,
  "name": "My Video",
  "preview_image_url": null,
  "created_at": "2011-07-22T18:54:19+00:00",
  "embed_code": "Y1dTdvMjq9QtOM",
  "time_restrictions": null,
  "updated_at": "2011-07-22T20:51:07+00:00",
  "external_id": null,
  "hosted_at": null,
  "original_file_name": "my_video.avi",
  "description": null,
  "status": "live"
}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. PATCH要求で変更を加えます。 次の例では、名前、URLでホストされている、および説明を変更します。 
    [PATCH] /v2/assets/Y1dTdvMjq9QtOM
{
  "name": "My Awesome Video",
  "hosted_at": "http://www.example.com",
  "description": "The next Internet meme."
}

    バックスロット 次のような応答を返します。

    {
  "asset_type": "video",
  "duration": 30,
  "name": "My Awesome Video",
  "preview_image_url": null,
  "created_at": "2011-07-22T18:54:19+00:00",
  "embed_code": "Y1dTdvMjq9QtOM",
  "time_restrictions": null,
  "updated_at": "2011-07-22T20:51:07+00:00",
  "external_id": null,
  "hosted_at": "http://www.example.com",
  "original_file_name": "my_video.avi",
  "description": "The next Internet meme.",
  "status": "live"
}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
