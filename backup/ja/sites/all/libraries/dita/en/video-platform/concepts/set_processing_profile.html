処理プロファイルを設定する

Backlot UIを介して新しいコンテンツを追加するときに、処理プロファイルを選択できます。

注意： Backlot UIでのアセットのアップロード処理中の処理プロファイルの選択はオプションです。 必要に応じて、選択を自動処理のままにすることができます。

処理プロファイルを設定するには：

  1. [管理]ページに移動して[コンテンツの追加]をクリックします。
  2. ビデオまたはオーディオを選択します。 （デフォルトの処理プロファイルは2つあります。1つは音声と1つのビデオです）。
  3. [資産の追加]ポップアップウィンドウで、通常どおりに[参照]をクリックして資産を追加します。
  4. 以下に、処理プロファイルオプションがあることがわかります。
  5. Processing Profileドロップダウンメニューに、利用可能な処理プロファイルのリストが表示されます。
  6. 処理プロファイルを選択してください。 デフォルトのプロファイルは >>.
  7. 「アップロード」をクリックすると、この処理プロファイルは取り込み中にアセットに適用されます。
注意： あなたがVR 360アセットをアップロードしているならば、Backlotはあなたのために自動的に関連する処理プロファイルを選びます。
