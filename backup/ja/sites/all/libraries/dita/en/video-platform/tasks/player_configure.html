バックスロットでのプレイヤーの設定

プレーヤーを作成した後（参照 Backlotでプレイヤーを作成する）、あなたはその外観と動作を設定することができます。 このプレーヤーに関連付けられているすべての動画に変更が自動的に適用されます。
注意： Player v4.11.13以降、[Player Branding]タブでPlayer V4の表示設定を構成できます。

プレーヤーのブランドタブ

Backlot UIでプレーヤーを設定するには、[プレーヤーのブランド設定]タブに移動します。
  1. Backlotにログインします。 https://backlot.ooyala.com/.
  2. 「公開」タブを選択します。
  3. [プレーヤブランド]タブを選択します。
  4. 設定したいプレイヤーを選択してください。
  5. サブタブをクリックして、プレーヤーの特定の領域を設定します。 設定オプションは以下の通りです。
プレーヤーの表示オプションの概要については、以下を参照してください。
  • プレーヤV4スキンのユーザインタフェース
  • プレーヤV3コンポーネント（非推奨）

プレイヤーサブタブ

選択したプレーヤーの基本設定を構成します。
設定 説明
プレーヤの名前 このプレイヤーの説明的な名前。
Playerのバージョン このプレイヤーのプレイヤーバージョン。 プレーヤーのバージョンを変更するボタンを選択した場合は、続行することを確認する必要があります。 選択したプレーヤーのバージョンによって、Backlotで設定できるオプションが決まります。
注意： プレーヤーのバージョンを変更すると、ターゲットバージョンでサポートされていない既存の設定はすべて失われます。
プレーヤー
株式オプション） プレーヤーに[共有]ボタンを表示するように選択します。 選ぶ （オプション） 表示する共有オプションを指定します。
  • Facebook
  • Twitter
  • URL
埋め込む プレーヤーに埋め込みボタンを表示する場合に選択します。 視聴者は埋め込みボタンをクリックして、プレーヤーの埋め込みスクリプトを示すポップアップを表示できます。これを使用して、視聴者のWebサイト、ブログなどにアセットを表示できます。
情報（オプション） プレーヤーに情報ボタンを表示する場合に選択します。 視聴者は、現在再生中のアセットに関する情報を見るために情報ボタンをクリックすることができます。 選ぶ （オプション） 表示する情報オプションを指定します。
  • 職務 - アセットの - （MANAGEタブで編集できます）
  • 説明 - アセットの - （MANAGEタブで編集できます）
  • 提供は - 自分の組織（プロバイダ名）を表示するように選択します。
  • プロバイダ名 - 資産のため。 組織名を指定してください。
  • URL - 資産の、ここで指定できます。 URLの前に「http：//」を指定する必要はありません。
チャンネル 視聴者がクリックしてチャンネル内の他のアセットを表示できるチャンネルボタンを表示する場合に選択します。 Channels＆Channel Setsサブタブ（下記）をご覧ください。 チャンネルとチャンネルセット（廃止予定）.
広告カウントダウンメッセージ 動画の再生中にOoyala Ad（Backlotにアップロードされたアセット）が再生されたときに広告カウントダウンメッセージをプレーヤーに表示するには（チェックを付けます）。 以下のいずれかのメッセージ
  • "広告：あなたのビデオはX秒で再開します"
  • 「広告：あなたのビデオはまもなく再開されます」（長さが利用できない場合）
Ooyala Adsの詳細については、
  • プレイヤーV4: 広告を配信する
  • プレイヤーV3: 独自の広告資産、広告セット、および広告ソースを操作する
リリースバージョン Player V4 Webリソースのリリースバージョン：
  • 生産 - Ooyalaがホストするプロダクションパス（デフォルト）
  • 最新 - Ooyalaがホストするプロダクション/最新のパス
の「Ooyalaリソースパス」を参照してください。 OoyalaがホストするプレイヤーV4リソース 詳細については。
スクラバーバー スクラバーバーのオプションを設定します。
ボリューム このプレーヤーのスクラバーバーに音量コントロールを表示する場合に選択します。
フルスクリーン このプレーヤーのスクラバーバーにフルスクリーンコントロールを表示する場合に選択します。
ビットレート このプレーヤーのスクラバーバーにビットレートコントロールを表示する場合に選択します。
一時停止時のバッファ プレーヤーが一時停止している場合にアセットをバッファするように選択します。
スクラバーバーを常に表示 常にスクラバーバーを表示するように選択します。 スクラバーバーを自動的に非表示にするには、選択を解除します。
アクセントカラー プレーヤ全体で使用するアクセントカラーを変更してください。
プレーヤーのプレビュー 選択した設定をプレビューします。 Player V3の場合のみ、次のいずれかのオプションを選択してください。
  • 動画 - 資産のビデオ表示（デフォルト）。
  • オーディオ - 資産の音声表示

ブランディングサブタブ

選択したプレーヤーのブランド設定を構成します。
設定 説明
ストリーム上のブランディングオーバーレイ 再生中にブランディングオーバーレイとして表示するプレーヤーのカスタム画像を指定します アップロード オーバーレイ画像を選択します。 サポートされているフォーマット：PNG、JPG、およびJPEG。 Ooyalaは、透明な背景を持つ.PNGファイルを使用することをお勧めします。 オーバーレイ画像がアップロードされたら、クリックスルーURL、透明度、およびプレーヤ内の位置を指定できます。 Player V4の場合のみ、以下も参照してください。 プレーヤV4にカスタムブランディングオーバーレイを追加する.
スクラバーブランディング 視聴者がビデオをナビゲートしているときに表示する画像を選択するには、をクリックします。 アップロード そして画像を選択します。 .PNG、.JPG、および.GIF画像をアップロードできます。 画像の縦横比は4：3でなければならず、320x240を超えてはいけません。

開始画面サブタブ

選択したプレーヤーのスタート画面を設定します。 ビデオが再生を開始する前に開始画面が表示されます。
設定 説明
スタート画面に表示
職務 ビデオのタイトルを表示するときに選択します。
説明 ビデオの説明を表示する場合に選択します。
再生ボタンのテキスト 再生ボタンの下にテキストを表示する場合に選択します。
プレーヤーのプレビュー 選択した設定をプレビューします。

画面終了サブタブ

選択したプレーヤーの終了画面設定を構成します。 ビデオの再生が完了すると、終了画面が表示されます。
設定 説明
ビデオ共有とおすすめのオプションを使用して、エンドスクリーンをカスタマイズします。
リプレイ ビデオの再生が終了した後に再生ボタンを表示する場合に選択します。
もっと動画 ビデオの再生が終了したら、クリックして[その他のビデオ]ボタンを表示します。 下記の追加オプションを参照してください。
株式オプション） 共有ボタンを表示するように選択します。 選ぶ （オプション） 表示する共有オプションを指定します。
  • Facebook
  • Twitter
  • URL
埋め込む プレーヤーに埋め込みボタンを表示する場合に選択します。 視聴者は埋め込みボタンをクリックして、プレーヤーの埋め込みスクリプトを示すポップアップを表示できます。これを使用して、視聴者のWebサイト、ブログなどにアセットを表示できます。
情報（オプション） プレーヤーに情報ボタンを表示する場合に選択します。 視聴者は、現在再生中のアセットに関する情報を見るために情報ボタンをクリックすることができます。 選ぶ （オプション） 表示する情報オプションを指定します。
  • 職務 - ビデオの（MANAGEタブで編集できます）
  • 説明 - ビデオの（MANAGEタブで編集できます）
  • 提供は - 自分の組織（プロバイダ名）を表示するように選択します。
  • プロバイダ名 - ビデオ用。 組織名を指定してください。
  • URL - あなたがここで指定できるビデオの - 。 URLの前に「http：//」を指定する必要はありません。
職務 ビデオのタイトルを表示する（[管理]タブで編集可能）を選択します。
説明 ビデオの説明の表示を選択します（MANAGEタブで編集できます）。
「その他のビデオ」を表示 [その他のビデオ]オプションが選択されている場合は、どのビデオを表示するかを指定します。
  • 共有ラベル - 共有ラベルを持つ動画を使用する（デフォルト）
  • 指定ラベル - 特定のラベルの付いた動画を表示します。 選ぶ ラベルを選択 表示したいラベルを指定します。
「その他のビデオ」で並べ替え [その他の動画]オプションが選択されている場合は、動画の並べ替え方法を指定します。
  • アップロード時間（最新から最新） （デフォルト）
  • アップロード時間（古いものから新しいものへ）
クリック動作 [その他のビデオ]オプションが選択されている場合は、視聴者がクリックしたときにビデオを開く方法を指定します。
注意： このクリック動作では、「ホストされている」URLが暗室で設定されているビデオのみが[その他のビデオ]画面に表示されます。
  • 同じプレーヤーで開く - 同じプレーヤーでビデオを開きます。 （Player V4専用のオプションです。）
  • 同じページで開く - 同じページの新しいプレーヤーでビデオを開く。
  • 新しいページで開く - 新しいページで新しいプレーヤーでビデオを開く（デフォルト）
プレーヤーのプレビュー 選択した設定をプレビューします。
