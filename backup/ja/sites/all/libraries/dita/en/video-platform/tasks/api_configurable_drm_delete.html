DRMポリシーの削除

資産またはアカウントからDRMポリシーをすばやく簡単に削除できます。

資産からDRMポリシーを削除するには、次の手順を実行します。

注意： 資産のDRMポリシーを置き換えたい場合は、単にその資産に新しいポリシーを割り当てます。
  1. 資産からDRMポリシーを削除します。 例えば： 
    [DELETE]/v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/drm_policy

    バックスロット 200応答を返します。

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. DRMポリシーをアカウントのデフォルトにする場合は、アカウントレベルでポリシーを割り当てることができます。 例えば： 
    [削除] / v2 / provider_drm_policy

    バックスロット 200応答を返します。

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
