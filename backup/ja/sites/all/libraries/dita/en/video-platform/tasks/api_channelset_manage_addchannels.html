チャンネルセットにチャンネルを追加する（廃止予定）

チャンネルセットを作成したら、チャンネルの追加を始めることができます。

注意： チャンネルとチャンネルセットは廃止予定で、無効になる予定です。 詳細と代替案については、 OVPリリースノート.
注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
注意： ビデオをチャンネルセットに直接追加することはできません。 ビデオのコレクションを作成するには、チャンネルを使用します。 見る チャンネルを作成する（非推奨）.

チャンネルセットにチャンネルを追加するには：

  1. 現在チャンネルセットに割り当てられているチャンネルを表示するには、 GET / v2 / assetsチャネルセットのアセットID /並ぶ 修飾子 次の例では、チャンネルの一覧を示します。 dvNTVxMjrRktSlb5v チャンネル。 
    [GET] / v2 / assets / dvNTVxMjrRktSlb5v / lineup

    バックスロット 次のような応答を返します。

    ["Argh3lMjphu25tH"、 "RQlbdvMjq9QtGGG"]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. チャンネルセットにチャンネルを追加したり、現在のチャンネルセットを新しいセットに置き換えるには、 PUT とともに / v2 / assets ルート、チャンネルセットのアセットID、 /並ぶ リクエストボディのチャネルのアセットID。 次の例では IzNnllMjphu2XF3 そして Y1dTdvMjq9QtOMG へのチャンネル dvNTVxMjrRktSlb5v チャンネルセット 
    [PUT] / v2 / assets / dvNTVxMjrRktSlb5v / lineup ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]

    バックスロット 次のような応答を返します。

    ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  3. 現在のものを置き換えずに、単一のシングルをチャンネルセットに追加するには、次のようにします。 PUT / v2 /アセット チャンネルセットのアセットIDで、 /並ぶ 修飾子、および追加するチャンネルのアセットID。 次の例では BrdXVjMjrgtupU3M へのチャネル dvNTVxMjrRktSlb5v チャンネルセット 
    [PUT] / v2 / assets / dvNTVxMjrRktSlb5v / lineup / BrdXVjMjrgtupU3M

    バックスロット 次のような応答を返します。

    ["BrdXVjMjrgtupU3M"、 "IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
  4. チャンネルセットからチャンネルを削除するには、 DELETE / v2 / assets チャンネルセットのアセットIDで、 /並ぶ 修飾子、および削除するチャンネルのアセットID。 次の例では、 BrdXVjMjrgtupU3M からのチャンネル dvNTVxMjrRktSlb5v チャンネルセット 
    [削除] / v2 / assets / dvNTVxMjrRktSlb5v /ラインナップ/ BrdXVjMjrgtupU3M

    バックスロット 次のような応答を返します。

    ["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]

チャネルセット内のチャネルは正常に更新されました。
