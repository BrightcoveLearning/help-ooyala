次の表は、API V2で使用可能な新機能について説明しています。

表2 新しいV2機能
特徴 説明
資産 GET、POST、DELETE、PUT、およびPATCHコマンドを使用してすべてのアセットタイプを作成、読み取り、更新、および削除（CRUD）するための多数の改善があります。
ラベル API V2はラベルの構造化とネストをサポートします。
ライブラリ Ooyalaは現在、いくつかの言語でラッパーSDKを提供しています。
Players の作成と管理 API V2は、プレーヤーのブランド設定オプションの構成をサポートしています。
公開ルール API V2は、公開ルールの作成、読み取り、更新、および削除のオプションを実行できます。
クエリAPI 新しいクエリAPIを使用すると、メタデータ、ラベル、およびアセットを検索できます。
スクラッチパッド スクラッチパッドは、Webブラウザを介して直接APIリクエストを送信できるようにすることで開発を簡素化するツールです。 スクラッチパッドにアクセスするには、にアクセスしてください。 Ooyala APIスクラッチパッド.
ユニバーサルシンジケーション ユニバーサルシンジケーションはAPI V2でのみ利用可能です。 詳細については、 ユニバーサルシンジケーション.
