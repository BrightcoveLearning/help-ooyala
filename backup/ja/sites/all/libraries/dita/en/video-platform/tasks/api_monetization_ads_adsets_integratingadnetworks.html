広告ソースと統合するための広告セットの作成

自分の資産を使用する代わりに、広告ネットワークを利用することができます。

このセクションでは、広告ソース固有の広告セットを設定するための一般的な手順について説明します。
  1. Google IMA V3
  2. フリーホイール
  3. LiveRail
  4. Adap.TV
  5. 振戦メディア
注意： / v2 / ad_setsルートを使用して広告ソース名の指定を含む広告セットを作成することはできますが、 バックスロットUI 広告ソースの設定を編集します。
注意： 必要な情報は広告ソースによって異なります。 フィールドの正確な意味については、常に広告ソースのドキュメントを確認してください。
  1. 作る POST 要求する / v2 / ad_sets以下に示すように、 
    [POST] / v2 / ad_sets {"名前"： "私の新しい広告セット"、 "デフォルト"：true}
    バックスロットAPI 次のような応答で応答します。 
    {"default"： "true"、 "id"： "ed4a4c6abdfc4ab8aa3e7b8a05f31ec6"、 "ads"：[]、 "私の新しい広告セット"}
  2. 自分のアセットからの、または任意の数の広告ソースからの広告を広告セットに関連付けます。 
    [POST] / v2 / ad_sets / ed4a4c6abdfc4ab8aa3e7b8a05f31ec6 /広告{ "ad_source"： "資産"、 "AD_TYPE"： "プリロール"、 "embed_code"： "01M2F0MTr3SWBB9dfAPT5xEsg8Gpy-JP"、 "plays_before"：0、 "plays_between"：1、 "click_url"： "http://www.myclickurl.com"、 "tracking_pixel_urls"：["http://www.mytrackingpixel.com"]}
  3. とともに バックスロットUI 『Backlotユーザーガイド』の説明に従って、特定の広告ソースに必要なフィールドに入力します。
