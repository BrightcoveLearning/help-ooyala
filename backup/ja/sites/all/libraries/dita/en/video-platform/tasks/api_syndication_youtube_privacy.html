YouTubeシンジケートのプライバシー設定およびその他の考慮事項

Youtubeでシンジケートするアセットのデフォルトのプライバシー設定を設定できます。

YouTubeに配信される動画のデフォルトのプライバシー設定は、 / v2 /シンジケーション ルート。

  1. シンジケーションの現在のプライバシー設定を見るには [GET] / v2 / syndications シンジケーションID を探す asset_defaults 応答をブロックします。 応答本体は次のようになります。 
    [GET] / v2 / syndications / fbdd2472css9476496cdaa5dc05ce6fb {"name"： "YouTube"、 "created_at"： "2011-06-15T23：05Z"、true：はい..._replay "true_replay"、true：はい... "true"、 "true"、 "true"、... ["51df64faas5fcx3d27383ac80744345"]、 "asset_types"：["video"、 "ad"]、
   "asset_defaults"：{"プライベート"：true}、
   "username"： "GeorgeBush"、 "id"： "fbdd2472css9476496cdaa5dc05ce6fb"、 "include_all_content"：false、 "type"： "youtube"、 "unrestricted_youtube_account"：false、 "should_delete_outcount"： "false"
  2. プライバシー設定を変更するには、 パッチ/ v2 /シンジケーション シンジケーションIDと asset_defaults でブロック プライベート 財産。 このプロパティの有効な値はtrueまたはfalseです。 
    [PATCH] / v2 / syndications / syndication_id {"assets_default"：{"private"：false}}
