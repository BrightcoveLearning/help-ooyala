サポートされている出力形式

Backlotは多数の一般的な出力フォーマットをサポートしています。

Backlotは以下の出力フォーマットをサポートします。

  • mp264コンテナ内のH.4ビデオ+ AACオーディオ
  • 264gpコンテナー内のH.3ビデオ+ AACオーディオ
  • iOS ABR用HTTPライブストリーミング（HLS）
  • Android用HTTP Live Streaming（HLS）（サポートはOSのバージョンによって異なります。詳細については、を参照してください。 Android Web用のHLSの有効化)
  • HTTPを介した動的適応ストリーミング（DASH）
  • Microsoft Smooth Streaming（Silverlight）形式（ISMVコンテナ内のH.264 / AACのみ）
  • アドビHTTPダイナミックストリーミング（HDS）形式
    注意： Akamai HDS（またはHD2）ストリーミングプロトコルを含むHTTP Dynamic Streaming（HDS）はFlashを使用します。 ブラウザはFlashプラグインを無効にしてブロックしています。 結果として、Ooyalaは、お客様がHLSとDashを今後もエンコードして使用することを強くお勧めします。
Ooyala Playerの推奨出力フォーマットについては、 プレーヤV4のための推奨エンコーディング.
