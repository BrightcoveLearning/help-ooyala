プレーヤV4の統合

あなたのコンテンツとの関わりを強めるために、あなたはあなたのコンテンツと OoyalaプレイヤーV4 for HTML5 他の環境やサービスと一緒に。

Ooyala製品との統合

OoyalaプレイヤーV4 for HTML5 他のOoyala製品と統合します。

現在統合可能なOoyala製品 OoyalaプレイヤーV4 for HTML5 次のとおりです。
  • Ooyala IQアナリティクスユーザーガイド
  • Ooyala Discovery（コンテンツの推奨）
  • ウーヤラコンテンツ保護
    • との認証 プレイヤーV3のOoyalaプレイヤートークン（廃止予定）
    • の権利管理 権利ロッカー
    • 承認（ビューアごとの同時ストリーム数の制限, 不正なストリームの停止） とともに プレーヤV3用プレーヤ認可API
    • アクセス制御 コンテンツ発行
  • ウーヤラバックスロット（資産運用）
    • でプレイヤーを作成する バックスロットREST API または Backlotでプレイヤーを作成する
    • で資産を管理する バックスロットREST API または コンテンツのアップロードと管理
    • で公開ルールを適用する バックスロットREST API or 公開ルールを作成する
    • でシンジケーション制御を適用する バックスロットREST API or シンジケーション制御セキュリティ機能とプログラミングとの関係
    • Player V4埋め込みコードの生成 バックスロットでのプレーヤー埋め込み設定.
    注意： [プレーヤーのブランド設定]サブタブの設定は影響を受けるだけです フラッシュプレイヤーV3。 のために プレイヤーV4を参照してください プレーヤV4外観のカスタマイズ。 のために HTML5プレーヤーV3JavaScript APIを使用してください。 Player V3 JavaScript APIを使用した開発（非推奨） そして docs.brightcove.com/apidocs-ooyala/).
    注意： Backlot APIを使用して行われたプレーヤーのカスタマイズは影響を与えるだけです フラッシュプレイヤーV3。 のために プレイヤーV4を参照してください プレーヤV4外観のカスタマイズ。 のために HTML5プレーヤーV3JavaScript APIを使用してください。 Player V3 JavaScript APIを使用した開発（非推奨） そして docs.brightcove.com/apidocs-ooyala/).
  • Ooyala Pulseユーザーガイド

また、 iOS用モバイルSDK そして Android用モバイルSDK.
