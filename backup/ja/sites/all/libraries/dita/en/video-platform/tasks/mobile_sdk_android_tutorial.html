基本的なAndroidアプリの作成

基本的なプレーヤーアプリを素早く作成する方法を学びましょう。

このチュートリアルでは、次のものが必要です。
  • 提供されたOoyala PCODE。 あなたを得るために PCODE、Backlotアカウントにログインして、 アカウント > 開発者。 あなたに APIキーピリオドの前の英数字（ "。"）は、 PCODE チュートリアルで使用する定数です。 たとえば、APIキーが xxxxxxxxxxxxxxxxxxxxxxxxxxxx.0000その後、最初の28文字（xxxxxxxxxxxxxxxxxxxxxxxxxxxx） のために PCODE。 詳しくは、 あなたのAPIクレデンシャル.
  • 再生したい動画の埋め込みコード（コンテンツIDまたはアセットID）。
  • Androidのメーカー システムにインストールおよび設定されている。
  1. GitHubレポジトリからBasic Playback Sample Appを入手します。

    Androidサンプルアプリ用のOoyala Mobile SDK

  2. Android Studioを開いて選択 フィレット > プロジェクトをインポート.
  3. BasicPlaybackSampleAppに移動して、 OK.
  4. 編集のためにBasicPlaybackVideoPlayerActivityを開きます。 
    BasicPlaybackSampleApp / app / java / com.ooyala.sample / players / BasicPlaybackVideoPlayerActivity.java

    importステートメントに注目してください。 次のステートメントは、あらゆるプレーヤーアプリの標準的なインポートです。

    • import com.ooyala.android.OoyalaPlayer;
    • import com.ooyala.android.OoyalaPlayerLayout
    • import com.ooyala.android.PlayerDomain
    • import com.ooyala.android.OoyalaPlayerLayoutController;
  5. 独自のEMBEDおよびPCODE値を入力してください。 
      final String EMBED = "yourEmbedCodeこちら"; //埋め込みコード、またはコンテンツID final String PCODE ="yourPcodeHere";
    注意： 出て DOMAIN 変わらない定数 http://www.ooyala.com。 これはと連携して動作します シンジケーション制御 Backlotの（公開ルール） Backlotのビデオにインターネットドメイン制限を設定している場合は、 ドメイン ここの変数はそれらの許可されたドメインの1つに設定することができます。 設定していない場合 シンジケーション制御、 DOMAIN ここでの定数は効果がありません。
  6. アプリをテストします。
    1. ナビゲーションヘッダーの 遊びます.
  7. テストデバイスまたはエミュレータを選択して、 OK.
  8. テストデバイスのロック解除オプションに移動して、[基本再生]オプションを選択します。

ビデオが再生されます。
