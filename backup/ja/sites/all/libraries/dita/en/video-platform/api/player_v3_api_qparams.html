プレーヤV3クエリ文字列パラメータ（廃止予定）

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
このような種類のクエリ文字列パラメータを、Player V3ライブラリとビデオを呼び出すURLに追加します。 name1=value1 ＆name2=value2... 次の例では 
     <スクリプト> http://player.ooyala.com/v3/player_branding_id?name1=value1
     ＆name2=value2...
次の表は、これらのクエリ文字列パラメータについて説明しています。
パラメーター 説明
名前空間

namespaceパラメータは、デフォルトの名前空間OOを他の有効なjavascript idに名前変更することによって、同じページに複数の独立したプレーヤコードのコピーを許可します。 たとえば、この埋め込みを使用するとします。

namespace =（ユニークなJavaScript ID)

後で使用できます。

MYPLAYER_1.Player.create（......）MYPLAYER_2.Player.create（......）
プラットフォーム

platformパラメータを使用すると、フラッシュ再生とhtml5再生の間の選択メカニズムを制御できます。 パラメータの形式は次のとおりです。

プラットフォーム=(フラッシュオンリー|フラッシュ|html5フォールバック|
         
         html5-優先度)

Player V3の場合、デフォルトのマルチメディアDefaultはflashです。 パラメータには以下のオプションがあります。

  • フラッシュ - デスクトップ、モバイル、タブレットでのみデフォルトのAdobe Flash再生を有効にします。 このオプションはFlashに設定を設定しますが、html5フォールバックも許可します。
  • フラッシュオンリー - Flashの再生のみ可能です。 Flashが検出されない場合は、エラーメッセージが表示され、視聴者にFlashプラグインのインストールを要求します。
  • html5フォールバック - Flashを使用しますが、html5へのフォールバックを有効にします。
  • html5-優先度 - このパラメータはhtml5を優先再生メカニズムとして設定しますが、Flashフォールバックも可能にします。
    注意： HTML5の再生は、特定のブラウザおよびOS /デバイス設定に対してのみ機能します。 を参照してください サポートセンター → オンラインマニュアル → プレーヤー開発者ガイド → Player V3プラットフォームとブラウザのサポート（廃止予定） 詳しくはトピックを参照してください。
微調整=android-enable-hls 使用 微調整 Web用のAndroidデバイスでHLSを有効にするためのパラメータ。 このパラメーターを使用してAndroidデバイスでHLSを有効にすることはできますが、バージョン4.x以降のデバイスでのHLS再生には一定の制限があります。 Androidデバイスでの操作性の詳細については、「Android WebでのHLSの有効化」を参照してください。
注意： tweaksパラメータを使用するとネイティブ再生の問題が発生する可能性があるため、すべてのAndroidデバイスおよびOSのバージョンで最適な再生エクスペリエンスを保証することはできません。

例

http://player.ooyala.com/v3/123fake6b6774f909c2fa6d23ce5334a?tweaks=android-enable-hls
微調整=html5-force-mp4 使用 微調整 html4プレーヤーでMP5の再生を強制するためのパラメータ。 このパラメータは、hlsをサポートするデバイスであってもhtml5プレーヤーにMP4アセットを再生させるように強制します。
注意： tweaksパラメータを使用するとネイティブ再生の問題が発生する可能性があるため、すべてのAndroidデバイスおよびOSのバージョンで最適な再生エクスペリエンスを保証することはできません。
注意： 従来のプレーヤAPIには、広告サーバと広告ネットワークのタグをOoyalaプレーヤに渡すためのクエリ文字列パラメータがありました。 Player V3の場合、サポートされている広告サーバー/ネットワークと関連タグを渡す仕組みについては、 広告用のPlayer V3埋め込みパラメータ（廃止予定）.
