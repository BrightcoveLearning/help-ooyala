ウォッチリスト名を変更する

このエンドポイントは、指定されたウォッチリストの名前を変更します。
注意： ユーザーの名前を変更することはできません デフォルト ウォッチリスト

HTTPリクエスト

PUT https://player.ooyala.com/watchlist/v3/<watchlist_id>?account_token= <account_token> {name "："新しいウォッチリスト名 "、}

クエリパラメータ

パラメーター 説明 タイプ 必須
api_version でなければなりません v3. 文字列 はい（URL）
watchlist_id 名前を変更する監視リストのID。 これは有効なUIIDである必要があり、ユーザーのアカウントの下に存在している必要があります。 ユーザーの名前を変更することはできません デフォルト ウォッチリスト UUIDv4または文字列 はい（URL）
account_token 特定のエンドユーザに関連付けられているウォッチリストを参照するためにウォッチリストサービスによって使用されます（UUIDによって識別されます）。 詳しくは、 OoyalaアカウントトークンAPI. 文字列 はい（URL）
名 ウォッチリストの新しい名前。 指定できません "デフォルト名前比較では大文字と小文字が区別されます。たとえば、 "MY_WATCHLIST"を "my_watchlist"に変更するとうまくいきます。 文字列 はい（ボディ）

リターンコード

コード 理由 説明
200 OK PUT要求は成功しました。 監視リストは正常に更新されました。
400 リクエストエラー 次の問題が原因で失敗しました。
  • パラメーターが欠落しているかフォーマットが正しくありません。
  • 指定された watchlist_id このユーザーのデフォルトのウォッチリストです。
  • 指定されたウォッチリスト 名 is "デフォルト" （またはその単語の任意のケースのバリエーション）。
  • 指定されたウォッチリスト 名 このユーザーには既に存在します。
401 無断 指定された account_token 無効です。
404 見つかりませんでした サーバーが停止しています。
500 内部サーバーエラー 内部サーバーエラーが発生しました。
