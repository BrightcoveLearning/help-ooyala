ディスカバリープロファイル

Discovery Profiles APIは、Discoveryの推奨設定を構成するための経路を提供します。 このAPIを使用すると、 player_id と profile_id 同じだ。 Backlot UIのプレイヤーのDiscoveryタブの左下に識別子があります。

スラッシュを使用した、推奨される構文を以下に示します。 

[PUT] / v2 / players / player_id / discovery/プロフィール

この古い形式のディスカバリープロファイル構文からの応答は、読みにくい応答を返します。

[PUT] / v2 / players / player_id /発見_プロフィール
