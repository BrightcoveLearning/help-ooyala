はじめに：Android Studioでの環境設定

Eclipse開発環境を使用してSDKをセットアップするには、これらのステップを使用してください。 彼らは私たちと一緒に始める方法についての指示を提供します デフォルト Player Skin（カスタマイズオプションは限られています）を使って BasicPlaybackSampleApp から入手できる このリンク。 私たちと一緒に始めるために カスタマイズ可能な プレイヤースキン（React Nativeベース）は、 OoyalaSkinSampleApp から入手できる このリンク 下記のステップ4で。
ヒント： このスケルトンをすべてのOoyala Mobile SDKアプリケーションの出発点として使用してください。 このリポジトリ内のすべてのアプリケーションは自動的にインポート可能、コンパイル可能、そして実行可能であるべきです。
  1. このリポジトリをコンピュータにクローンします。 Gitクローンhttps://github.com/ooyala/android-sample-apps.git
  2. Android Studioを開く
  3. どちらかを押す 「既存のAndroid Studioプロジェクトを開く」 またはクリック ファイル>開く...
  4. 基本的なプレーヤーのスキンアプリの場合は、に移動します。 android-sample-apps / BasicPlaybackSampleApp。 カスタマイズ可能なプレーヤースキンアプリの場合は、に移動します。 android-sample-apps / OoyalaSkinSampleApp.
  5. 언론사 選択するに電子メールを送るか、もしくは OK
  6. Gradleの実行が完了するのを待ちます。 「app」がビルド構成の1つとして表示されるはずです。
  7. アプリケーションを実行します（実行>アプリを実行） - 実行する接続機器またはシミュレータを選択します。
  8. アプリケーションがロードされると、あなたはから選ぶべきビデオのリストを持つでしょう。 ビデオ再生を見るためにそれらのどれかを選んでください。
