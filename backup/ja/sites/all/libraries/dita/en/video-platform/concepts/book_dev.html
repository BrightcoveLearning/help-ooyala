バックスロット資産管理

このドキュメントでは、Backlot APIを使用して資産を管理する方法について説明しています。

  • Backlot入門
    動画をアップロードしてWebページに埋め込むことで、Backlot UIの使用を開始します。
  • 計画と設定の手引き
    へようこそ Ooyala 計画および設定ガイド
  • Backlot REST APIを使用する
    Ooyala RESTベースのインターフェースを提供します。 バックスロット プラットフォームを提供します。
  • 資産の管理
    Backlotでは、アセットはビデオ、オーディオ、ライブストリームなどのさまざまな種類のコンテンツの一部です。
  • 収益化
    Ooyala 広告やペイウォールを通じてコン​​テンツを収益化することができます。
  • コンテンツ発行
    公開を使用すると、動画や動画のメタデータを複数のWebプロパティやコンテンツネットワークと共有できます。
  • 署名要求のサンプルコード
    リクエストに署名するためのさまざまなプログラミング言語のサンプルコードの集まりです。
  • バックスロットREST API
    プレーヤーのクロム（デザイン要素）をカスタマイズするには、Backlot REST APIを使用してください。 これには、フォント、色、スクラバーバーなどのUI要素が含まれます。
  • ディスカバリーREST APIリファレンス
  • ウーヤラスケジュール
    Ooyala Scheduleを使用すると、仮想アセットのトラックを組み合わせてライブストリームの上で再生できます。
