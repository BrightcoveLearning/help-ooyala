ディスカバリーREST API

これらのディスカバリーAPIを使用するには、ディスカバリー要求を出し、ユーザーのアクション情報を戻してアルゴリズムをトレーニングします。

このようなOoyala Discoveryの推奨事項については、結果が表示されたとき、および推奨された動画がクリックされたときに、コードから要求が送信されます。 パーソナライズされた推奨事項を使用している場合は、 デバイスID リクエストごとに。

  • Ooyala Discovery REST APIリクエストの概要
    Discovery REST APIを使用して、サービスのさまざまな側面を構成し、推奨事項を入手し、その他のユーザー関連のデータを入手します。
  • 推奨コンテンツの傾向
    Ooyala Discoveryのトレンド分析アルゴリズムは、現在あなたのサイトで人気のあるコンテンツを見つけ、その人気を高めるように設計されています。
  • 関連コンテンツの推奨事項
    関連コンテンツアルゴリズムは、現在のアセットに関連するコンテンツを返します。現在のアセットは、WebページまたはApp内に表示できます。
  • 同様の資産の推奨
    ディスカバリーAPIを使用して、同様のビデオに関する推奨事項を提供する方法について説明します。
  • ディスカバリー編集用照会ストリングのパラメーター
    Ooyala Discoveryの推奨事項の結果を調整できるように、OoyalaはベースラインAPIにクエリ文字列パラメータを提供しています。
  • トップタイトル
    トップタイトルのルートを使用して、トップトレンドの動画に関する統計情報を返します。
