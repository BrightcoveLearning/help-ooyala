Androidアプリの権限を設定する

Ooyala Mobile SDK for Androidアプリのマニフェストファイルで権限を設定する方法について説明します。

リリース4.1.0の時点で、あなたのAndroidアプリのマニフェストには以下のパーミッションが設定されている必要があります。

android.permission.INTERNET ネットワークストリームにアクセスするための必須のアクセス許可を提供します。
android.permission.ACCESS_WIFI_STATE 広告に最適なストリームを選択するための必須の権限を提供します。
android.permission.ACCESS_NETWORK_STATE ネットワークステータス情報にアクセスするための必須の権限を提供します。 
<？xml version = "1.0" encoding = "utf-8"？> <マニフェストxmlns：android = "http://schemas.android.com/apk/res/android" package = "com.ooyala.android.sampleapp "android：versionCode =" 1 "android：versionName =" 1.0 "> <uses-sdk android：minSdkVersion =" 9 "/>
       <uses-permission android：name = "android.permission.INTERNET" /> <uses-permission android：name = "android.permission.ACCESS_WIFI_STATE" /> <uses-permission android：name = "android.permission.ACCESS_NETWORK_STATE" / >
   ...
注意： SDKの最小バージョンが9に設定されていることを確認してください。 
<uses-sdk android：minSdkVersion = "9" />
