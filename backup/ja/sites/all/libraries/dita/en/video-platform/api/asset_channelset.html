チャンネルセットアセット（廃止予定）

チャンネルセットは、シングルプレイヤー内のチャンネルの集まりです。

メソッド

Create Asset 
[POST] /v2/assets 
{
  プロパティ

}
List Assets 
[GET] / v2 / assets
Get Channel Set 
[GET] / v2 / assets /channel_set_id
Modify Channel Set 
[パッチ] / v2 / assets /channel_set_id
Delete Channel Set 
[DELETE] /v2/assets/channel_set_id
List Channels in Channel Set 
[GET] / v2 / assets /channel_set_id/並ぶ
Set or Replace Channels 
[PUT] /v2/assets/channel_set_id/lineup
["埋め込みコード"、"埋め込みコード", ...]
Append Channel to Channel Set 
[PUT] /v2/assets/channel_set_id/lineup/埋め込みコード
[PUT] /v2/assets/channel_set_id/lineup/external_id
Delete Channel from a Channel Set 
[DELETE] /v2/assets/channel_set_id/lineup/埋め込みコード
[DELETE] /v2/assets/channel_set_id/lineup/external_id

ルート属性

次の表は、ルートを通じて表現できるすべての属性を示しています。

ルート属性 説明
channel_set_id

The ID of the channel set.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

埋め込みコード

The content ID or embed code of a channel within the channel set.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

external_id

The external ID of a channel within the channel set, if applicable.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

パラメーター 説明 必須？

The maximum number of results to return. If you receive more results than the value you specify, you can get the next page of results with the paging token.

タイプ：文字列

有効値：1〜100

Example: [GET] /v2/assets?limit=5

いいえ
next_page
A URL returned by the previous API request that enables you to page through results.

デフォルト：なし

 いいえ

並べ替えるフィールド。

The following table describes all properties that can be associated with a channel set.

一日中

Recurring only. Specifies whether the content can be viewed during the entire day. If set to false, specify a start and end time.

タイプ：ブール

例

This example creates a channel set: 

[POST] /v2/assets
{
  "name": "My Channel",
  "asset_type": "channel_set"
}

バックスロット 次のような応答を返します。

{
  "asset_type": "channel_set",
  "duration": 0,
  "name": "My Channel",
  "preview_image_url": null,
  "created_at": "2011-08-10T17:45:24+00:00",
  "embed_code": "dvNTVxMjrRktSlb5v",
  "time_restrictions": null,
  "updated_at": "2011-08-10T17:45:24+00:00",
  "external_id": null,
  "hosted_at": null,
  "original_file_name": null,
  "description": null,
  "status": "live"
}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

This example adds the IzNnllMjphu2XF3 そして Y1dTdvMjq9QtOMG へのチャンネル dvNTVxMjrRktSlb5v channel set, replacing the current channels: 

[PUT] /v2/assets/dvNTVxMjrRktSlb5v/lineup
[
  "IzNnllMjphu2XF3",
  "Y1dTdvMjq9QtOMG"
]

バックスロット 次のような応答を返します。

["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
