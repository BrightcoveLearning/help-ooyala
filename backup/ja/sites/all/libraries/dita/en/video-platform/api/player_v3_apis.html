プレーヤV3 JavaScript API関数

プレーヤV3 JavaScript関数を使用してプレーヤの動作と外観をカスタマイズできます。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).

すべてのOoyala APIは、 Ooyala APIドキュメント.

Player JavaScript APIについては、次のURLで詳しく説明されています。 プレーヤJavaScript APIリファレンス.

次の表は、PlayerのJavaScript API関数について簡単に説明しています。また、に含まれる完全な説明へのリンクも含まれています。 プレーヤJavaScript APIリファレンス.
