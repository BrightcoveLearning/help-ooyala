HTML5 API（プレーヤV2は廃止予定）

OoyalaプレーヤーHTML5 APIを使用すると、iOS、Android、およびその他のHTML5をサポートするプラットフォーム用に、Ooyalaプレーヤーとビデオエクスペリエンスの側面をカスタマイズできます。

重要： Ooyala Player V2は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V2はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V2を使用しているお客様は、Player V4に移行する必要があります（を参照）。 プレイヤーV4.

以下のトピックでは、プレーヤHTML5 APIについて説明します。 すべてのAPIを1つのファイルにまとめて表示することも、広告に使用されるAPI、クローズドキャプション、プレーヤーなどの機能カテゴリ別にAPI情報を選択することもできます。 別のトピックには、これらのHTML5 APIの使用方法を示す例が含まれています。

  • HTML5 APIサマリーリスト（プレーヤV2は廃止予定）
    関数/メソッド、オブジェクト、プロパティ、およびイベントごとにグループ化されたすべてのプレーヤHTML5 APIをリストして説明します。
  • HTML5 APIをアセットします（Player V2は廃止予定）
    アセットHTML5 APIを使用して、評価、ビデオ（再生/一時停止、説明）、およびチャンネルを処理できます。
  • プレーヤHTML5 API（プレーヤV2は廃止予定）
    プレーヤ/プレーヤUI HTML5 APIを使用してプレーヤの側面を処理できます。
  • HTML5の例とデモ（Player V2は廃止予定）
    HTML5の例は、iOSベースのデバイス用のHTML5プレーヤーをテストする方法を示しています。
