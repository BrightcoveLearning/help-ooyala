傾きによる再起動の防止

この設定は、視聴者がデバイスの向きを変更したときにビデオが再開されるのを防ぎます。

デバイスの向きが変わったとき（垂直から水平、または水平から垂直）にビデオが再起動しないようにするには、次の手順を実行します。
  1. アプリケーションを開く AndroidManifest.xml ファイル。
  2. 次の属性をに追加します。 アクティビティ 素子： アンドロイド：configChanges = "向き| keyboardHidden".

    例えば： 

    <activity android：name = "。TestOoyalaSDKActivity" android：label = "@ string / app_name"
        アンドロイド：configChanges = "向き| keyboardHidden" >
