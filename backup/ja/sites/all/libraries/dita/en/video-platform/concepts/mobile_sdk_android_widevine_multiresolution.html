Widevineでのマルチ解像度プレイヤーとの連携

Google Widevineで複数の解像度のビデオコンテンツに対してABRをサポートするために、ビデオストリームの最大サイズを設定できます。

モバイルデバイスでマルチ解像度ビデオを最大サイズで再生できるようにするには、以下のコードの断片に示すように、ビデオストリームの最大幅と最大高を設定できます。 選択したストリームは、定義した最大寸法以下の解像度になります。

  1. 延長する DefaultPlayerInfo クラス。

    maxHeight そして maxWidth あなたが奉仕しなければならない最高の解像度に対応するべきです。

  2. 静的を設定します StreamPlayer.DefaultPlayerInfo コードスニペットで、新しいクラスのインスタンスに、 CustomPlayerInfo.
// AndroidクラスCustomPlayerInfoはDefaultPlayerInfoを継承します。{public int getMaxWidth（）{return 1200;} public int getMaxHeight（）{return 700;} public void onCreate（Bundle savedInstanceState）{super.onCreate（savedInstanceState）;} setContentView（R.layout.main）; StreamPlayer.defaultPlayerInfo = new CustomPlayerInfo（）; ...
