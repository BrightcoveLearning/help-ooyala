2012-02-27リリース

アップデートには、iOS SDK、Player APIドキュメント、新しい「Ingestion Guide」、およびサポートセンターの社会的側面が含まれます。

リリース項目は以下のとおりです。

  • iOS SDK - Native SDKを使用すると、iOSモバイルデバイスおよびタブレット用のアプリケーションを構築および公開できます。 詳しくは、「参考文献」ページを参照してください。
  • プレーヤAPIドキュメント - プレーヤAPIドキュメントは書き直され、現在はOoyalaサポートセンターの一部です。 詳細については、 プレーヤV3 JavaScript APIリファレンス（非推奨）.
  • 新しい取り込みガイド - これまで、コンテンツをアップロードするさまざまな方法に関するドキュメントはほとんどありませんでした。 さて、OoyalaはFTP、Aspera、そして他の取り込みオプションの使い方を説明した新しいガイドを持っています。 詳細については、 バックスロット 摂取ガイド.
  • ソーシャルサポートセンター - Ooyalaサポートセンターはソーシャルになりました。 サポートセンターとドキュメントをナビゲートしながら、Facebookを通じてツイート、電子メール、またはページの共有を行うことができます。 さらに、ドキュメントのどのページについてもフィードバックを残すことができます。 詳細については、Backlotユーザーマニュアルを参照して、メインペインの右側にあるアイコンを確認してください。
