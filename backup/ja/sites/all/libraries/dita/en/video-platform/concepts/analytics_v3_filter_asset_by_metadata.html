カスタムメタデータ、期間、または公開日による資産ディメンションのフィルタリング

カスタムメタデータ、事前定義された期間、またはその公開日で資産ディメンションをフィルタ処理できます。 資産 のセクション ビジネス·インテリジェンス のページをご覧ください。

カスタムメタデータに基づいてアセットをフィルタリングするには：
  1. 資産のフィルタアイコンをクリックします。
  2. セレット メタデータ 最初のドロップダウンメニューから。
  3. ここで、フィルタリングしたいカスタムメタデータの値のペアを入力します。
  4. ナビゲーションヘッダーの 適用します.
注意： コンマを使用して、カスタムメタデータの複数のキーと値のペアを追加できます。
事前定義された期間に基づいて資産をフィルタリングするには
  1. 資産のフィルタアイコンをクリックします。
  2. 以下の期間のセレから 戦略 ドロップダウンメニュー：
    • すべての期間
    • 短い長さ（0 - 5分）
    • 中くらいの長さ（5 - 20分）
    • 長尺（20 +分）
  3. ナビゲーションヘッダーの 適用します.
注意： この期間フィルタは、リリース日以降のトラフィックのある資産に対してのみ機能します。
公開日に基づいてアセットをフィルタリングするには：
  1. 資産のフィルタアイコンをクリックします。
  2. ストリーミングを停止する場合は上部の 公開日で絞り込む ボタン。
  3. 日付範囲またはカスタムプリセットを選択してクリック 適用します.
  4. ナビゲーションヘッダーの 適用します 再び。
