カスタムモジュール用のPlayer V3埋め込みパラメータ（廃止予定）

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).

カスタムモジュールの埋め込みパラメータ（キーと値のペア）を OO.Player.create（） 方法。 次の表は、これらのカスタムモジュールパラメータについて説明しています。

パラメーター 説明 プレイヤーの種類
devModuleCategory サードパーティのカスタムモジュールの開発に使用される場合、このパラメータはPlayer.createオプションハッシュの "flashParams"サブハッシュで指定します。 このパラメータを使用して、 'universal'や 'ads-manager'などのカスタムモジュールカテゴリを指定します。

タイプ：ひも

有効な値: 「ユニバーサル」や「広告マネージャ」などのカテゴリ名

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、 'flashParams'：{devModuleCategory： 'universal'、....}、}）;
 フラッシュのみ
devModuleURL サードパーティのカスタムモジュールの開発に使用される場合、このパラメータはPlayer.createオプションハッシュの "flashParams"サブハッシュで指定します。 このパラメータを使用して、サードパーティモジュールのswfをダウンロードできる場所のURLを指定します。

タイプ：ひも

有効な値: サードパーティモジュールのswfが配置されているダウンロードURL

レイアウト

このプラットフォームパラメータを使用してUIレイアウトを制御します。 パラメータの形式は次のとおりです。

レイアウト=(クロムレス)

Player V3の場合、パラメータには次のオプションがあります。

  • クロムレス - デフォルトのプレーヤーコントロールのオンとオフを切り替えます。 クロムレスに設定されている場合、パラメータはクロムと開始画面を削除します。 指定されていない場合、プレーヤーはデフォルトのレイアウトに戻ります。

クロムレスパラメータは、プレーヤUIのコントロールを無効にして、自動再生だけでビデオを再生します。 Ooyala V3プレーヤーでクロムレスレイアウトを実装するには、 layout = 'クロムレス' へのパラメータ <script .... / v3> タグ。 これによりプレーヤコントロールが無効になり、視聴者はコントロールが表示されていなくてもビデオを再生できます。 プレーヤーのコントロールバーを削除すると、再生ボタンが削除されたときに視聴者はビデオを再生できなくなります。 ビデオが自動的に開始されるようにビデオの再生を有効にする必要があります。 をセットする 自動再生 プレーヤがストリームのロードを終了したらすぐにビデオを開始するように1へのパラメータ。

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、レイアウト： 'クロムレス'}）;
