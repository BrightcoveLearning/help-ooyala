ウーヤラIQ Workspace Reference

あなたのワークスペース ウーヤラIQ いくつかの部分があります。

このセクションは、に含まれるすべての機能のリファレンスです。 ウーヤラIQ ユーザーインターフェース。 表示されているワークスペース ウーヤラIQ はいくつかのセクションに分かれており、それぞれが異なる種類のデータ、データのビュー、または機能を扱います。

  • Ooyala IQダッシュボード
    分析ダッシュボードを使用すると、 ウーヤラIQリアルタイムであなたのコンテンツに何が起こっているのかを見るために、あなたは決断を下して直ちに行動を起こすことができます。
  • ビジネスインテリジェンスレポート
    ビジネスインテリジェンスページを使用して、さまざまな方法でフィルタ処理されたデータ内の詳細を表示し、特定の関心点を特定します。
  • ビデオの詳細
    ビデオの詳細を調べるには、[ビデオの詳細]ページを使用します。
