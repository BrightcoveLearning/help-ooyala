Ad Set Fields for Tremor Media

ここで説明するフィールドは、広告ソースがTremor Mediaに設定されている広告セット用です。

これらのフィールドは、広告セットを次のいずれかの広告ソースと統合することに関するものです。
  • 振戦メディアAcudeo
  • 振戦メディアAcudeo Lite
注意： フィールドの正確な意味については、常に広告ソースのドキュメントを確認してください。
フィールド名 説明
広告ポリシーID Acudeo広告ポリシーの一意の識別子
トラッキングピクセルURL 広告資産の配信を監視するためにピクセルを追跡するためのURL
広告掲載順位 広告を再生する場所：プリロール、インストリーム（ミッドロール）、ポストロール（Tremor Media Acudeo Lite）
x個の動画の後に最初の広告を表示 広告の頻度設定
x動画ごとに広告を再生する 広告の頻度設定
