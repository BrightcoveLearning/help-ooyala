Player V3でOO.Player.createを使用して広告セットを割り当てる（廃止予定）

Player V3 enables you to assign ad sets dynamically with OO.Player.create と adSetCode パラメータ。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
Each ad set has a unique Ad Set Code identifier. You must first set up the Ad Set Code in バックスロット, as shown in the first steps below.

To assign the ad set with OO.Player.create と adSetCode パラメータ：

  1. Associate an ad set with an asset (movie) in バックスロット.
  2. Once you create the association, you can get the Ad Set Code. The following figure shows an example of this association and the Ad Set Code that you create in the マネー をクリックします。
    注意： The ad set code identifier is not available by default for security reasons. Please contact your Customer Success Manager or Technical Support to enable this feature for your account.
  3. 使用 OO.Player.create function to generate a player that takes a DOM element and the embed code of the player. In the following script example we name the DOM element playerwrapper and add the standard parameters height and width.
  4. Add the additional adSetCode key value pair to associate an ad set with a particular player and player asset (video). 
     
<スクリプト> var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ： '100％'、幅： '100％'、adSetCode： 'yourAdSetCode'}）; <スクリプト>
