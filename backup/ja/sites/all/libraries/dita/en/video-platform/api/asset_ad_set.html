広告セットをアセットに関連付ける

広告セットをビデオ、ライブストリーム、およびリモートアセットに関連付けることができます。 広告セットの作成については、 バックスロット ユーザーガイド。

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

Associate Ad Set

[PUT] / v2 / assets /asset_id/ad_set/ad_set_id

Get Ad Set for an Asset

[GET]/v2/assets/asset_id/ ad_set

Remove Ad Set from Asset

[DELETE]/v2/assets/asset_id/ ad_set

例

この例では、 PROoCi9B2BwPROoCi9B2Bw123 で設定した広告 IzNnllMjphu2XF3_UgPROoCi9B2rtWs 資産： 

[PUT]/v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/ad_set/PROoCi9B2BwPROoCi9B2Bw123

バックスロット 200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
