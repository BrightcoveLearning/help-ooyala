要求の有効期限を設定する

あなたが設定する必要があります 有効期限が切れます HTTPリクエストのパラメータ。

HTTPリクエストには 有効期限が切れます リクエストが無効になるタイミングを示すパラメータ。

の値 有効期限が切れます 次の例に示すように、parameterは1月からの秒数で、1、1970（ "the epoch"）の後、要求は無効になります。

[GET] https://api.ooyala.com/v2/players/somePlayerId?api_key=someKey&signature=someSignature&expires=1930294539
注意： ASCII以外の文字と ' 文字はエスケープされます。
