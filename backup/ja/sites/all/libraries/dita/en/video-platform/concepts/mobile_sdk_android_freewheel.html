Android上のFreeWheelとの統合

SDKを使用すると、モバイルデバイス向けの豊富なカスタムビデオアプリケーションエクスペリエンスを迅速に開発できます。 SDKには、ビデオの再生、VAST広告およびOoyala広告の表示、Ooyalaコンテンツのクエリなどを行うためのメソッドやクラスが含まれています。
  • Playback
  • プレイリスト/チャンネル - サムネイル
  • インフォ
  • 関連メディア
  • ライブ（新）
  • 広告
  • アナリティクス
  • モバイル使用状況を理解するための完全に統合された分析
  • クローズドキャプション
  • DFXP（現在はTTML）ファイルを介してクローズドキャプションをロードおよび表示するためのAPI
Some key design principles behind Ooyala's SDK for FreeWheel include:
  • Ooyala's FreeWheel Manager object, which incorporates in it all of the necessary functions, so that you do not need to be concerned with working with a wide variety of calls.
  • You can continue to use the basic functions of Ooyala's Mobile SDK, on which Ooyala FreeWheel SDK is built.

あなたが必要なもの

To get started with Ooyala’s Android SDK for FreeWheel, download the following:

  1. Google's Android SDK
  2. Ooyala Mobile SDK for Android (available at Ooyala ダウンロード)
  3. Ooyala FreeWheel SDK for Android (Ooyala FreeWheel Android) (available at Ooyala ダウンロード)
  4. FWAdManager.jar from the FreeWheel website (you need FreeWheel credentials to download the zip)
    注意： While using the latest version of the FreeWheel libraries please make sure the version you are using is compatible with the latest Ooyala integrations.
  5. Androidのメーカー

    注意： Mobile SDKが依存するOoyala Backlotの動画アセットに関連付けられた静的広告タグが必要です。 これらが本番のBacklotアカウントに存在しない場合は、それらをアプリにロードする必要があります。

基本パラメータとFreeWheel OPFモジュール広告セット

To make use of FreeWheel in the Mobile SDK, you must create an ad set of type FreeWheel OPF Module in Backlot. See the Backlot User Guide for details. Ooyala allows you to store FreeWheel-ad-related parameters in a variety of locations. In order of precedence, FreeWheel parameters and their values can be defined in:

  1. アプリ
  2. Internal Ooyala configuration (contact your CSM)
  3. バックスロットで マネー タブ、 広告セット のサブタブ フリーホイールOPFモジュール type of ad set

注意： These parameters must be set either directly in your app or internal-to-Ooyala by your Customer Success Manager:

  • fw_android_player_profile: The defined profile for the player on Android
