ウーヤラライブv3.5.0（2017-12-04）

修正と強化

  • Ooyala Live REST APIドキュメントの更新中です。 更新を行っている間に、APIドキュメントにアクセスするためのオプションがいくつかあります。

    • 最新のLive APIドキュメントはこちらからアクセスできます。 Ooyala Live REST API.

      最新のOoyala Live REST APIドキュメントには、最近公開されたAPIエンドポイントが含まれています。 新しいエンドポイントの説明を閲覧したり、「試してみる」こともできます。 既存のLive APIエンドポイントのドキュメントをこの新しいフォーマットの残業に移行します。

    • 既存のAPIドキュメントはこちらからアクセスできます。 Ooyala Live REST APIリファレンス.

      既存のOoyala Live REST APIリファレンスは、新しいLive APIドキュメント形式へのコンテンツの移行がすべて完了するまで利用できます。 このドキュメントに新しいAPIエンドポイントを追加する予定はありません。

既知の問題点
  • Azureを使用するチャンネルのイベントをトリミングするとき、トリミングコントロールはビデオの10秒単位でトリミングします。
  • イベントの実行中にイベント名を変更した場合、Ooyala Backlotのアーカイブ資産名が最新のイベント名で更新されない可能性があります。
  • 高可用性イベントがスケジュールされた時間に開始されない場合は、まれに断続的に問題が発生します。
