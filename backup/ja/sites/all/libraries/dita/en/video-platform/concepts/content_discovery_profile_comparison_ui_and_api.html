Backlot UIとAPIでの検出プロファイル設定の比較

これは、Backlot UIと検出プロファイル作成用のAPIの比較です。

Backlot UIを使用して検出プロファイルを作成します。 Backlot UIまたはDiscovery APIを使用して検出プロファイルを設定して使用します。 Backlot UIは最も一般的な設定作業には十分ですが、Discovery APIは追加の設定オプションを提供します。

  • UIのフィールドの意味については、 バックスロットUIでの検出の設定.
  • APIのプロパティの正確な意味については、 ディスカバリープロファイル.
