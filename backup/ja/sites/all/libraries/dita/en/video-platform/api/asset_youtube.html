YouTubeアセット

A YouTube asset is a video hosted by YouTube and played through the Ooyala player.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

YouTubeでホストされている動画を再生する利点

  • Ooyalaプレーヤーまたはカスタムプレーヤーを使用して、サイト全体で一貫した外観を維持します。
  • 統合されたカスタマイズされたエクスペリエンスのためにOoyalaの豊富なAPIの使用を活用してください。
  • アカウント内のすべての動画について、Ooyalaの優れた分析機能の洞察を得てその利点を維持します。
  • インターネット最大のビデオライブラリ、Youtubeからあなたのサイトの補足コンテンツを見つけましょう。
  • コンテンツに時間、地域、またはドメインベースのルールを適用します。
  • Ooyalaの直感的で包括的なツールを活用して、YouTubeコンテンツを自分のコンテンツと共に管理、シンジケート、カスタマイズ、および分析します。

動画はBacklotではなくYouTubeでホストされています

YouTubeのビデオはYouTubeから提供されています。 Backlotでは、ビデオは、編集可能なタイトルと説明とともに、そのYouTube URLを含むYouTubeアセットとして表されます。 Backlotでは、アセットタイプは "youtube"です。そのため、我々のドキュメントでは、これらをアセットタイプと呼んでいます。 YouTubeの資産.
したがって：
  • BacklotはYoutubeビデオを取り込んでトランスコードしません。
  • YouTubeの動画は、配信または管理対象コンテンツの割り当てには適用されません。
その結果、すべての動画にYouTubeの透かしが含まれ、プレーヤーの埋め込みサイズに基づいて、YouTubeが提供できる最高の品質で配信されます。

考慮事項

  • BacklotはYouTubeの動画をトランスコードしないため、
    • Backlotはプレビュー画像を生成しません。
    • iOSデバイスではコンテンツを再生できません。
  • YouTubeコンテンツは、常にプレーヤが利用できるとは限りません。 たとえば、動画が削除されたり、YouTubeサイト運営者の規則に基づいて動画が制限されたり、YouTubeへのアクセスが視聴者の会社や機関によってブロックされたりする可能性があります。

YouTube IDs Must Be Unique in Your Account

No two YouTube assets in your account can have the same YouTube ID.
If you attempt to add a YouTube video that has been previously added, you receive the following error message, where asset_id identifies the previously created asset: 
"{" duplicate_youtube_id "：["asset_id"]}"

Get YouTube Settings

[GET]/v2/assets/asset_id/ youtube

Set YouTube Syndication Settings

[PUT] / v2 / assets /asset_id/youtube
   {  
   "youtube_id":"some_youtube_id",
   "youtube_syndication_settings":{  
      "private":true|false,
      "mobile":true|false,
      "embeddable":true|false,
      "keywords":comma-separated-list-of-keywords
   }
}

Create a YouTube Asset

The following example creates a YouTube asset: 
[POST]/v2/assets{  
   "name":"My YouTube Asset",
   "asset_type":"youtube",
   "youtube_id":"a1Y73sPHKxw"
}
Backlot returns a response similar to the following example: 
{  
   "asset_type":"youtube",
   "duration":5,
   "name":"My YouTube Asset",
   "preview_image_url":null,
   "created_at":"2011-08-25T21:01:49+00:00",
   "embed_code":"1saGRyMjoOgjkUSjod_1TQINs_HOQnoU",
   "time_restrictions":null,
   "updated_at":"2011-08-25T21:01:50+00:00",
   "youtube_id":"a1Y73sPHKxw",
   "external_id":null,
   "hosted_at":null,
   "original_file_name":null,
   "description":"The best 5 second clip on the Internet.",
   "status":"live"
}

Get YouTube Settings

The following example gets YouTube settings: 
[GET]/v2/assets/1saGRyMjoOgjkUSjod_1TQINs_HOQnoU/youtube
Backlotは次のような応答を返します。 
{  
   "author":"cregets",
   "youtube_id":"a1Y73sPHKxw",
   "youtube_syndication_settings":{  
      "private":null,
      "mobile":null,
      "embeddable":null,
      "keywords":null
   }
}

Add Keywords to YouTube Syndication Settings

The following example adds keywords to the YouTube syndication settings and makes the player for this video embeddable: 
[PATCH]/v2/assets/1saGRyMjoOgjkUSjod_1TQINs_HOQnoU/youtube{  
   "youtube_syndication_settings":{  
      "private":null,
      "mobile":null,
      "embeddable":true,
      "keywords":[  
         "old",
         "classic"
      ]
   }
}
Backlotは次のような応答を返します。 
{  
   "author":"cregets",
   "youtube_id":"a1Y73sPHKxw",
   "youtube_syndication_settings":{  
      "private":null,
      "mobile":null,
      "embeddable":true,
      "keywords":[  
         "old",
         "classic"
      ]
   }
}
