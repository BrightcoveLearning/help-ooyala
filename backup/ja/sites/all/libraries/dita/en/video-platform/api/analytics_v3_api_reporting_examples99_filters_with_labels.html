グループ化フィルタ

  • 国オーストラリアとともにデータを返す（AU;を参照） 国と場所のコード）と モバイル デバイスタイプ： 
    ...＆filters =（（country == 'AU'））AND（（device_type == 'mobile'））
  • コロンビアの携帯およびタブレット端末： 
    ...＆filters =国== 'CO'、端末タイプ== '携帯電話'、端末タイプ== 'タブレット'
  • コロンビアと米国でのみ携帯： 
    ...＆filters =（（device_type == 'mobile'）AND（country == 'CO'、country == 'US'）））

ラベルによるフィルタリング

Ooyalaラベルはビデオをグループ化するためのメカニズムです。 Ooyala Analyticsでは、ラベルを使用してフィルタリングできます。 ラベルIDを使用してラベルフィルタを識別します。
Ooyala IQでラベルを操作するときは、次の点に注意してください。
  • ラベルと関連付けられている動画間の相関関係は、事前集計または事前集計（ロールアップ）されませんが、APIリクエスト時に解決されます。
  • Ooyalaシステムはラベルの親階層に基づいて集約します。 たとえば、2つのラベル、親 "Cycle World"とその子ラベル "Sport Bikes"を想像してください。 「Sport Bikes」という子供のラベルが付いたビデオは、「Cycle World」という親のラベルにロールアップされます。 ラベルのデータをリクエストした場合、返される指標はそのラベルとその子ラベルのものです。
このリクエストはの合計を返します を ラベルID 123abc456yui123xyzのラベルでグループ化されたメトリック。 
...＆filters = label == '123abc456yui123xyz'
