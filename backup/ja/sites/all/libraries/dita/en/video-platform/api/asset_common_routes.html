すべてのアセットタイプに共通のルートと属性

The basic routes common to all assets have the same syntax, regardless of their type, in the Backlot REST API.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

The common routes for all assets are the create, read, update, and delete (CRUD) operations. Except for creation and updating, which requires certain プロパティ or workflow depending on the asset type, most of these common routes require only the identifier of the asset (asset_id).

Create Asset

[POST]/v2/assets{
   プロパティ

}
Certain asset types have considerable variation in how they are created:
  • ビデオとオーディオ
  • リモートアセット
  • ライブストリーム
  • YouTubeの資産

List All Assets

[GET] / v2 / assets

List Specific Asset

[GET]/v2/assets/asset_id

Modify Asset

[パッチ] / v2 / assets /asset_id

Replace Asset

[POST]/v2/assets/asset_id/置換

Delete Asset

[DELETE]/v2/assets/asset_id

ルート属性

The following table describes the attributes common to all variations of the / v2 / assets routes.

ルート属性 説明
ad_set_id

ID of an ad set. For details, see 広告セット.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg/ad_set/PROoCi9B2BwPROoCi9B2Bw123

asset_id

ID of the asset. This value is referred to by different names depending on where you look for it. In the Backlot API, asset_id is the identifier for a specific asset. asset_id has the same value as the content ID referred to in the Backlot UI (which represents a piece of content). For details, see 資産の管理.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

channel_id (deprecated)

ID or embed code of a channel. For details, see チャンネル（廃止予定）.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

channel_set_id (deprecated)

ID of a channel set. For details, see チャンネル（廃止予定）.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

drm_type

For Digital Rights Management (DRM), the type of DRM system. For details, see デジタル著作権管理.

タイプ：文字列

デフォルト：なし

Valid Values: widevine | playready

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg/drm_attributes/widevine
埋め込みコード

The content ID from Backlot or embed code of a video within a channel. In the Backlot UI, the 埋め込みコード is sometimes referred to as the asset_id または コンテンツID. Please note that in the Backlot UI, the field for embed code refers to the embed snippet that you embed on a web page.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg
external_id External ID of a video within a channel, if applicable. For details, see 外部識別子を使用する.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

注意： nullの動画を追加しないでください external_idつまり、 external_id 値なし（（""）またはの値 "ヌル"。 このようなNULL外部IDは後で検索することはできません。
label_id

ID of a label. For details, see チャンネル（廃止予定）.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg/labels/QtsjklrewsddROoCi9B2B45drs

player_id

ID of a player. For details, see Backlotでプレイヤーを管理する.

タイプ：文字列

デフォルト：なし

Example: /players/b4f1a93b779145b4a9c8ef15f8502345

publishing_rule_id

ID of a publishing rule. For details, see 公開ルール.

タイプ：文字列

デフォルト：なし

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg/publishing_rule/QtsjklrewsddROoCi9B2B45drs
