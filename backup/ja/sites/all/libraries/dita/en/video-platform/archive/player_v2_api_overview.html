V2プレーヤAPI（プレーヤV2は廃止予定）

Ooyala V2プレーヤーAPIリファレンスへようこそ。 このガイドには、Ooyala V2 Player APIを使用して実行できる操作に関する従来の情報が含まれています。

重要： Ooyala Player V2は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V2はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V2を使用しているお客様は、Player V4に移行する必要があります（を参照）。 プレイヤーV4.

このプレーヤAPIリファレンスは、以前のプレーヤAPIサポート用のレガシドキュメントとして提供されています。 従来のプレーヤAPIは、新しいプレーヤAPIのために使用されなくなりました。 このAPIリファレンスでは、アセットなどのカテゴリ別およびActionScriptなどのサポート対象言語別にまとめられた、これらの推奨されないAPIに関する情報を提供します。

注意： 現在のOoyalaプレーヤーは、以前のバージョンのプレーヤーよりも優先されます。 ただし、引き続きレガシープレーヤーをサポートし、お客様の便宜のためにレガシープレーヤーAPIリファレンスドキュメントを含めます。 詳細については、「プレーヤ非推奨APIリファレンスの概要」のセクションを参照してください。
