Twitterカードの高度なプレーヤーオプション

Ooyalaの特別ページを使用して、利用可能なすべてのOoyalaプレーヤーオプションを含む、ビデオをTwitterのプレーヤーカードに埋め込むためのプレーヤーURLを作成できます。

に示すように Ooyala Player用のTwitter <meta>タグの手動取得、Twitterプレーヤーカードでは、基本的なHTML <meta> Ooyalaビデオプレーヤーを定義するタグは次のようになります。 これがOoyalaプレーヤーの最も基本的な呼び出しです。
。 。 。 <meta name = "twitter：player" content = "https://player.ooyala.com/tframe.html?embedCode=videoID＆pbid =playerID"/>。
より高度なプレーヤーオプションを含めるには、URLを作成してから、上で強調表示されている基本的な呼び出しを置き換えます。 基本URLをカスタマイズしたものに置き換えた後は、必ずTwitter Playerカードを本番に移行するための残りの手順に従ってください。

高度なプレーヤーオプションの構文

次のURL構文を調べてください。 の値として使用するためにこのURLを構築できます。 コンテンツ Twitterの属性 twitter：player <meta> 上記のタグ

https://player.ooyala.com/tframe.html?

＆pbid=player_branding_id

tframeでPlayerに埋め込みパラメータを渡す

標準のOoyalaプレーヤー構文では、プレーヤー呼び出しの3番目の引数として特定のランタイム引数（ "オプションハッシュ"とも呼ばれる）を指定できます。 自動再生：true, initialTime：122その他のオプション 埋め込みパラメータ そして 埋め込みパラメータ.

ウーヤラとは tframe.html 同じ埋め込みパラメータを オプション 属性。 あなたは繰り返すことができます ＆オプション 属性と値は何度でも好きなだけ。 唯一の制限は、URLの最大長です。

上記の構文に従って、 オプション 属性は次のようになります。

https://player.ooyala.com/tframe.html?
ec=埋め込みコード
＆pbid =player_branding_id
＆platform =プラットフォーム
＆options [autoplay] = true＆options [initialTime] = 122
ネストされたオプション（つまり、広告タグのようにそれ自体がハッシュであるハッシュキー）は、ドット表記で表されます。 見る 例：プリロール用の複数のIMA広告タグを渡す.

"Ooyala例外：無効なクエリパラメータ"エラーの可能性

tframe.htmlのパラメータが正しくフォーマットまたは構成されていないと、次のエラーメッセージが表示され、プレーヤはロードされません。 
Ooyala例外：無効なクエリパラメータ

Twitter Player Cardを使って本番に入る前に、構文を慎重にチェックしてこの問題を取り除いてください。
