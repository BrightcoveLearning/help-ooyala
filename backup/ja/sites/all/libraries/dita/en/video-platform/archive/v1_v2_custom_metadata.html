カスタムメタデータAPI（廃止予定）

カスタムメタデータAPIは、アセットAPIの一部になりました。 これで、カスタムメタデータまたはアセットを表示、更新、置換、または削除できます。

カスタムメタデータAPIの詳細については、 カスタムメタデータ.

バックスロットAPI v1

http://api.ooyala.com/partner/set_metadata
?pcode=<pcode>
&signature=<signature>
&expires=<expiration>
&embedCode=<embedCode>
&attribute1=<new value>
&attribute2=<new value>
&delete=writer%00copyright

バックスロットAPI v2

[PATCH] / v2 / assets / <asset_id> / metadata {properties}

例

この例では、指定したアセットのカスタムメタデータを作成します。

[PUT] / v2 / assets / IzNnllMjphu2XF3_UgPROoCi9B2rtWs / metadata {"監督"： "ミシェルゴンドリー"、 "年"： "2004"、 "評価"： "R"}

バックスロット 次のような応答を返します。

{"監督"： "ミシェルゴンドリー"、 "年"： "2004"、 "評価"： "R"}

この例では、指定された資産のディレクターを置き換えます。

[PATCH] / v2 / assets / IzNnllMjphu2XF3_UgPROoCi9B2rtWs / metadata {"director"： "Spike Jonze"}

バックスロット 次のような応答を返します。

{"監督"： "スパイクジョーンズ"、 "年"： "2004"、 "評価"： "R"}
