チャンネルおよびチャンネルセットアセット（廃止予定）

A channel is a collection of videos in a single player. A channel set is a collection of channels.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

Common Routes, Attributes, Properties and Query String Parameters

Refer to the following information:
  • Common API routes used by this asset type
  • Global definitions of attributes
  • All properties by asset type
  • Common query mechanism
  • Adding metadata to an asset
  • Adding closed captions
  • Adding URLs for an asset

List Videos in Channel

[GET]/v2/assets/channel_id/並ぶ

Set or Replace Videos

[PUT] / v2 / assets /channel_id/lineup[
   "埋め込みコード"、"埋め込みコード", ...
]

Append Video to Channel

[PUT] / v2 / assets /channel_id/lineup/埋め込みコード
[PUT] / v2 / assets /channel_id/lineup/external_id

Delete Video from a Channel

[DELETE]/v2/assets/channel_id/lineup/埋め込みコード
[DELETE]/v2/assets/channel_id/lineup/external_id

List Channels in Channel Set

[GET]/v2/assets/channel_set_id/並ぶ

Set or Replace Channels

[PUT] / v2 / assets /channel_set_id/lineup[
   "埋め込みコード"、"埋め込みコード", ...
]

Append Channel to Channel Set

[PUT] / v2 / assets /channel_set_id/lineup/埋め込みコード
[PUT] / v2 / assets /channel_set_id/lineup/external_id

Delete Channel from a Channel Set

[DELETE]/v2/assets/channel_set_id/lineup/埋め込みコード
[DELETE]/v2/assets/channel_set_id/lineup/external_id

例

This example creates a channel: 

[POST]/v2/assets{  
   "name":"My Channel",
   "asset_type":"channel"
}

バックスロット 次のような応答を返します。

{  
   "asset_type":"channel",
   "duration":0,
   "name":"My Channel",
   "preview_image_url":null,
   "created_at":"2011-08-10T17:45:24+00:00",
   "embed_code":"dvNTVxMjrRktSlb5v",
   "time_restrictions":null,
   "updated_at":"2011-08-10T17:45:24+00:00",
   "external_id":null,
   "hosted_at":null,
   "original_file_name":null,
   "description":null,
   "status":"live"
}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

This example adds the IzNnllMjphu2XF3 そして Y1dTdvMjq9QtOMG ビデオアセット dvNTVxMjrRktSlb5v channel, replacing the current videos: 

[PUT]/v2/assets/dvNTVxMjrRktSlb5v/lineup[  
   "IzNnllMjphu2XF3",
   "Y1dTdvMjq9QtOMG"
]

バックスロット 次のような応答を返します。

[  
   "IzNnllMjphu2XF3",
   "Y1dTdvMjq9QtOMG"
]
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
