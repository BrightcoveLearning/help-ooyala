パフォーマンス（廃止予定）

資産実績レポートは、アカウントまたは資産レベルでの包括的な実績を示します。

警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、

掲載結果の分析（ビデオ）

[GET] / v2 / analytics / reports /レベル/パフォーマンス/合計/日付範囲 
[GET] / v2 / analytics / reports /アカウント/ performance / videos /日付範囲

掲載結果の分析（Geo）

[GET] / v2 / analytics / reports /レベル/パフォーマンス/国[/国]/日付範囲
[GET] / v2 / analytics / reports /レベル/パフォーマンス/地域/日付範囲
[GET] / v2 / analytics / reports /レベル/パフォーマンス/都市/日付範囲
[GET] / v2 / analytics / reports /レベル/パフォーマンス/国/国/地域[/地域]/日付範囲
[GET] / v2 / analytics / reports /レベル/パフォーマンス/国/国/地域/地域/都市[/シティ]/日付範囲
[GET] / v2 / analytics / reports /レベル/ performance / dmas [/DMA]/日付範囲

掲載結果の分析（ドメインとURL）

[GET] / v2 / analytics / reports /レベル/パフォーマンス/ドメイン[/ドメイン]/日付範囲
[GET] / v2 / analytics / reports /レベル/ performance / domains /ドメイン/ urls [/URL]/日付範囲
[GET] / v2 / analytics / reports /レベル/ performance / urls [/URL]/日付範囲

パフォーマンス分析を入手する（プラットフォームとデバイス）

[GET] / v2 / analytics / reports /レベル/ performance / device_types [/デバイスタイプ]/日付範囲
[GET] / v2 / analytics / reports /レベル/ performance / device_types /デバイスタイプ/プラットフォーム[/プラットフォーム]/日付範囲
[GET] / v2 / analytics / reports /レベル/パフォーマンス/プラットフォーム[/プラットフォーム]/日付範囲
