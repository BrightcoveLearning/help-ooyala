フィードバックの提供

推薦を調整するのを助けるために、ディスプレイと演劇を記録するようにしてください。

推奨事項がプレーヤーのエンドスクリーンに表示されている場合は、何もする必要はありません。 プレイヤーの外におすすめを表示する場合は、必ず各ディスプレイを記録してイベントを再生してください。

ディスプレイを録音する

[POST] / v2 /発見/フィードバック/印象{  
   bucket_info,
   
   デバイスID

}

プレイを録音する

[POST] / v2 /発見/フィードバック/再生{  
    bucket_info,
    
    デバイスID

}
