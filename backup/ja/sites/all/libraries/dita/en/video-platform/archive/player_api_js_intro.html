プレーヤJavaScript API（プレーヤV2は廃止予定）

Ooyala プレーヤー JavaScript APIを使用すると、この一般的なスクリプト言語を使用してOoyala Playerとビデオエクスペリエンスをカスタマイズできます。

重要： Ooyala Player V2は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V2はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V2を使用しているお客様は、Player V4に移行する必要があります（を参照）。 プレイヤーV4.

以下のトピックでは、PlayerのJavaScript APIについて説明します。 すべてのAPIを1つのファイルにまとめて表示することも、広告に使用されるAPI、字幕、プレーヤーなどの機能カテゴリ別にAPI情報を選択することもできます。 別のトピックには、これらのJavaScript APIの使用方法を示す例が含まれています。

  • JavaScript APIサマリーリスト（Player V2は廃止予定）
    関数/メソッド、オブジェクト、プロパティ、およびイベントごとにグループ化されたすべてのPlayer JavaScript APIをリストして説明します。
  • Ads JavaScript API（プレーヤV2は廃止予定）
    広告JavaScript APIは、広告の開始、クリック、完了、スキップなどのイベントに応答するために使用できます。
  • JavaScript APIをアセットします（Player V2は廃止予定）
    アセットActionScript APIを使用して、評価、ビデオ（再生/一時停止、説明）、およびチャンネルを処理できます。
  • ビットレートとバッファのJavaScript API（Player V2は廃止予定）
    ビットレート選択APIを使用すると、開発者はプログラムでバッファ情報を取得したり、ビットレートを取得および設定したりできます。
  • JavaScript APIのキャプションを閉じる（Player V2は非推奨）
    クローズドキャプションJavaScript APIを使用すると、サポートされているクローズドキャプションの取得、クローズドキャプション言語の設定、およびキャプションのタイミングの取得ができます。
  • コードJavaScript APIの埋め込み（Player V2は廃止予定）
    埋め込みコードJavaScript APIを使用して、資産のembedCodeの取得と設定、資産の説明の取得、画像の取得、および資産のメタデータの取得を処理できます。
  • エラーJavaScript API（Player V2は廃止予定）
    エラーJavaScript APIを使用して、エラーコードとエラーテキストを取得できます。
  • プレーヤJavaScript API（プレーヤV2は廃止予定）
    プレーヤー/プレーヤーUI JavaScript APIは、フルスクリーンモード、プレーヤーの状態、音量、再生ヘッド、プレーヤーの音量など、プレーヤーのさまざまな側面を処理するために使用できます。
  • JavaScriptデモと例（Player V2は廃止予定）
    JavaScriptの例は、Ooyalaプレーヤーの埋め込み機能と、プレーヤーAPIを使用してプレーヤーに追加機能を追加する機能を示しています。
