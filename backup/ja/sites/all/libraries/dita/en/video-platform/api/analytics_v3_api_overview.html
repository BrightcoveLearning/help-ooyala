データ保持

Ooyalaはあなたのデータを37ヶ月までしか保持しません。 これは、過去の36月とともに現在の月として計算されます。 たとえば、今日がJan 29、2015の場合、現在の月から現在まで（Jan 1、2015 - Jan 29、2015）と過去36の全月（Jan 1、2012 - Dec 31、2014）のデータがあります。 Ooyalaは、最新の37月の有効なデータについての照会をUIおよびAPIでのみ行い、許可します。 残りのデータはアーカイブされます。 v2 AnalyticsからOoyala IQに移行される履歴データの量については、以下を参照してください。 データ移行（廃止予定）.
