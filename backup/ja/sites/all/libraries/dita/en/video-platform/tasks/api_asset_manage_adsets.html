広告セットを適用する

広告が変化する複数の資産がある場合に広告を管理する最も簡単な方法は、広告セットを使用することです。

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
注意： それらをアセットと関連付ける前に、広告セットを作成する必要があります。 見る 。独自の広告資産、広告セット、および広告ソースを操作する.

広告セットをアセットに関連付けるには、 PATCH とともに / v2 / assets ルート、 / ad_sets 修飾子と広告セットID

使用 / v2 / assets 広告セットを関連付けるためのルート。 次の例では、 cab127d23bf45c38 で設定した広告 しぶき 資産： 
[PATCH]/v2/assets/JxbzdkMjqB/ad_set/cab127d23bf45c38

バックスロット 200応答を返します。
