APIリファレンス

APIリファレンスガイドへようこそ。 これらのトピックには、Backlot、Player、Authorization、およびRights LockerのAPI参照情報が含まれています。

重要： このセクションのトピックは、 Ooyala APIドキュメント のページをご覧ください。

Ooyalaは、ストリーミングビデオプラットフォームのコンテンツをカスタマイズするための豊富なAPIセットを提供しています。 これらのAPIは次のとおりです。

  • バックスロットREST API
    プレーヤーのクロム（デザイン要素）をカスタマイズするには、Backlot REST APIを使用してください。 これには、フォント、色、スクラバーバーなどのUI要素が含まれます。
  • v3 Analytics（Ooyala IQ）API
  • プレーヤV3 JavaScript APIリファレンス（非推奨）
    Player V3 APIは、Player v3機能とメッセージバスイベント処理も提供するUI要素をカスタマイズするためのJavaScript APIです。
  • 認証APIリファレンス
    承認APIは、ユーザーを承認するための一連のアプリケーションプログラミングインターフェイス（API）です。
  • Rights Locker APIリファレンス
    Rights Lockerは、デジタル資産への資格を定義および管理するための一連のアプリケーションプログラミングインターフェイス（API）です。
  • 使用法RESTful API
