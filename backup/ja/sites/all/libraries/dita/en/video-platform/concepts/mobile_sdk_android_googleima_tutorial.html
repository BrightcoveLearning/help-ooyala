AndroidでのIMA Sample Appの実例を見る

サンプルアプリを実行するには、次の手順を使用します。
  1. AndroidサンプルアプリGitリポジトリからIMASampleAppをダウンロードします。 Ooyala Androidサンプルアプリ
  2. 解凍する OoyalaIMASDK-Android.zip ハードドライブ上の場所にファイルします。
  3. Android Studioを開いて選択 既存のプロジェクトを開く.
  4. に移動します / IMASampleAppフォルダを選択 選択する. GradleスクリプトはAndroid Studio IDEでプロジェクトを設定するために実行されます。
  5. 開いた IMAListActivity.java 編集用です。 apps / java / com.ooyala.sample / lists / IMAListActivity.java
  6. アプリに必要な各ビデオの埋め込みコードで選択マップを更新します。 この手順は、次の例に示されています。
    selectionMap = new LinkedHashMap <文字列、PlayerSelectionOption>（）; //埋め込みマップselectionMap.put（ "IMA Ad-Rules Preroll"、new PlayerSelectionOption（ "EzZ29lcTq49IswgZYkMknnU4Ukb9PQMH"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Ad-Rules Midroll"、新しいPlayerSelectionOption（ "VlaG9lcTqeUU18adfd1DVeQ8YekP3H4l"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Ad-Rules Postroll"、新しいPlayerSelectionOption（ "BnaG9lcTqLXQNyod7Yv8eDas3Oog2"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Podded Preroll"、新しいPlayerSelectionOption（ "6wNjE1cDox3G0hQIWxTjsZ3MPUDLSkDY"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Podded Midroll"、新しいPlayerSelectionOption（ "8yNjE1cDodUEfUfp3WNzHkCZCMb2MUP"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Podded Postroll"、新しいPlayerSelectionOption（ "47sNjE1cDoN3ZewFm3ce1238J730BMrEJ"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMA Poddedプレミッドポスト"、new PlayerSelectionOption（ "ZrOTE4cDoXo3sLOWzQPxjS__M-Qk2Co"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMAスキップ可能"、new PlayerSelectionOption（ "FhbGRjbzq32tfaoA8dhfxc3Qs2-RURJfO"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMAプリスキップ、ミッドスキップ、ポストスキップ可能"、new PlayerSelectionOption（ "0NjE10cDpj3nUzYiV8PnFsjC1nEvPQAE"、PreconfiguredIMAPlayerActivity.class））; selectionMap.put（ "IMAアプリケーション構成"、new PlayerSelectionOption（ "EzZ6lcTq29IswgZYkMknnU49Ukb4PQMH"、CustomConfiguredIMAPlayerActivity.class））; \ u9
  7. 開いた PreconfiguredIMAPlayerActivity.java 編集用です。 app / java / com.ooyala.sample / players / PreconfiguredIMAPlayerActivity.java
  8. 置き換える PCODE Ooyala提供のプロバイダIDコードを使用してください。 詳しくは、 あなたのAPIクレデンシャル. この手順は、次のコードサンプルに示されています。

    注意： 以下のままに 新しいPlayerDomain（ドメイン） 変更なし http://www.ooyala.com。 この定数は以下と組み合わせて機能します。 シンジケーション制御 （公開ルール） バックスロット。 ビデオにインターネットドメイン制限を設定している場合 バックスロット （参照 バックスロット ユーザーガイド）、ここでの定数はそれらの許可されたドメインの1つに設定することができます。 設定していない場合 シンジケーション制御定数は効果がありません。
  9. 変更を保存します。
  10. 再生ボタンをクリックして、Android Studioエミュレータでアプリが実行されるのを確認します。
