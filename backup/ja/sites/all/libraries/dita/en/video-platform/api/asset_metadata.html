カスタムメタデータ

どのタイプのアセットにも、最大100個の名前と値のペアを追加できます。

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
The metadata name must follow these restrictions:
  • Maximum length: 128 characters
  • Digits or letters (a to z, A to Z, 0 to 9) and any of the following special characters: - _ ! @ # $ % & ^ * ( ) . ?
  • The following characters must be escaped, as shown:
    文字 Escape Sequence
    ! ％21
    # ％23
    $ ％24
    ^ ％5E

値は2,048文字に制限されています。

Get Custom Metadata

[GET]/v2/assets/asset_id/メタデータ

Delete A Single Custom Metadatum for Player

[パッチ] / v2 / assets /asset_id/metadata{
      "名" : "null"
}

Delete All Custom Metadata for Asset

[DELETE]/v2/assets/asset_id/メタデータ

   "rating":"R"
