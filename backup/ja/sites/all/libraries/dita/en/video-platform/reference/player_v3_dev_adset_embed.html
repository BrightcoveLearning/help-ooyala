プレーヤV3でsetEmbedCodeを使用して広告セットを割り当てる（廃止予定）

You also have the option of assigning ad sets using the setEmbedCode 機能。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
With this function you can set up playlists with different ad sets or set up ad sets for different contexts.
注意： This function requires that you associate an ad set with a movie in バックスロット as shown in Player V3でOO.Player.createを使用して広告セットを割り当てる（廃止予定）.

setEmbedCode function takes an additional オプション parameter that must be an object and must belong to the same provider as the embed code. In the オプション hash, use the adSetCode parameter, as in the following example.

<script>
player.setEmbedCode(embedCode, {
    adSetCode: 'yourAdSetCode'
});
<script>
