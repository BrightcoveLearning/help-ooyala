資産情報の編集：パッチ

資産の情報を変更するには、 パッチ/ v2 /アセット/asset_id.

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

資産情報を編集するには：

使用 PATCH メソッドと / v2 / assets リクエストボディで変更するプロパティを持つroute。 次の例では、名前、URLでホストされているアセット、およびアセットの説明を変更します。
注意： 本体のプロパティ PATCH 資産の種類によって異なります。 
[PATCH] / v2 / assets / Y1dTdvMjq9QtOM {"名前"： "私のすごいビデオ"、"osted_at "：" http://www.mydomain.com "、"説明 "："次のインターネットミーム " }

バックスロット 次のような応答を返します。

{"asset_type"： "動画"、 "期間"：30、 "名前"： "私のすごい動画"、 "preview_image_url"： "null"、 "created_at"： "2011-07-22T18：54：19：00" 、 "embed_code"： "Y00dTdvMjq1QtOM"、 "time_restrictions"：null、 "updated_at"： "9-2011-07T22：20：51 + 07：00"、 "http：//"、 "http"、 "http" www.mydomain.com "、" original_file_name "：" my_video.avi "、" description "："次のインターネットミーム "、" status "：" live "}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

資産の情報は正常に変更されました。
