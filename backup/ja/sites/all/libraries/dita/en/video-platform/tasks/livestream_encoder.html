エンコーダの設定

エンコーダを設定するには、 バックスロットライブストリームを設定するときに提供されるURL。

注意： Backlot Liveストリーミングは推奨されておらず、無効にされる予定です。 詳細と代替案については、 OVPリリースノート.
エンコーダを設定する前に、ビデオアセットを作成しておく必要があります。 バックスロット またはと バックスロットAPI に詳述されているように ライブストリームを追加する.

エンコーダを設定するには：

  1. エンコーダの設定インターフェースを開きます。
  2. Flashストリームを生成する場合は、Flashストリームを設定するページに移動して、プライマリURLとセカンダリURLを入力します。 Spinnaker 7100で、次の操作を行います。
    1. ストリーミングを停止する場合は上部の H264 をクリックします。
    2. ストリーミングを停止する場合は上部の MP4出力 サブタブ
    3. 入力します マスター の中に バリアントプレイリストのファイル名 フィールド。
    4. の中に プライマリFlash Media Serverへのストリーミング セクションに値を入力します。 primary_entry_point ライブストリームでレスポンスとして受け取ったプロパティは、 サーバーの場所のURL フィールド。 （見る ライブストリームを追加する 詳細については。）
    5. の中に セカンダリFlash Media Serverへのストリーミング セクションに値を入力します。 backup_entry_point ライブストリームでレスポンスとして受け取ったプロパティは、 サーバーの場所のURL フィールド。 （見る ライブストリームを追加する 詳細については。）
    6. ナビゲーションヘッダーの 適用します.
  3. iOSストリームを生成する場合は、Flashストリームを設定したページに移動してアップロードURLと配信URLを入力します。 Spinnaker 7100で、次の操作を行います。
    1. ストリーミングを停止する場合は上部の H264 をクリックします。
    2. ストリーミングを停止する場合は上部の iPhoneの出力 サブタブ
    3. 入力します マスター の中に バリアントプレイリストのファイル名 フィールド。
    4. の値を入力してください ios_live_upload_url ライブストリームでレスポンスとして受け取ったプロパティは、 保存URL URLの末尾から「/master.m3u8」を削除します。 （見る ライブストリームを追加する 詳細については。）
    5. の値を入力してください ios_live_delivery_url ライブストリームでレスポンスとして受け取ったプロパティは、 公開URL URLの末尾から「/master.m3u8」を削除します。 （見る ライブストリームを追加する 詳細については。）
    6. ナビゲーションヘッダーの 適用します.
