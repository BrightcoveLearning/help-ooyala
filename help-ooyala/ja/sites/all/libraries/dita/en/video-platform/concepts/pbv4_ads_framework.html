広告マネージャフレームワーク

Player V4用のOoyalaのAd Manager Framework（AMF）は、HTML Web用の統一されたフレームワークです。 このフレームワークにより、広告サーバーはOoyalaのHTML5プレーヤー用の広告マネージャを開発できます。 Ooyalaは、主要な広告管理者がAMFと統合するための広告プラグインを構築しました。

IAB VASTおよびVPAIDフォーマットのプラグインサポート

Ooyala Player V4広告プラグインは、IAB VASTおよびVPAIDフォーマットをさまざまにサポートしています。
プラグイン サポート
Ooyala Pulse広告プラグイン VAST 2.0、VAST 3.0、VPAID 1.0、およびVPAID 2.0フォーマット
フリーホイール広告プラグイン VAST XML
Google IMA Ad Plugin VAST 2.0、VAST 3.0、VPAID 1.0、およびVPAID 2.0のフォーマットで生成されたDFPまたはAdSense広告サーバーからの広告
VASTとVPAIDの広告プラグイン VAST 2.0、VAST 3.0、およびVPAIDフォーマットとLiveRail VAST広告
注意：
  • VASTまたはVPAIDを使用する広告サーバーとしてAdap.tvまたはBrightrollを使用している場合は、VASTおよびVPAIDプラグインを使用してください。
  • VASTまたはVPAIDを使用して広告サーバーとしてDFPまたはAdSenseを使用している場合は、Google IMAプラグインを使用する必要があります。

Ad Manager Frameworkの機能

  • 広告マネージャコントローラ：AMFはプレーヤーにコントローラと呼ばれる 広告マネージャコントローラこれは、広告マネージャがより少ない労力でOoyalaにプラグインできるようにするための基本機能を処理します。
  • 広告プラグイン：AMFを使用すると、広告リクエストから広告の再生まで、すべての第三者広告プラグインが広告セッション全体を管理できます。 このプラグインは、ビデオプレーヤーの上にあるレイヤーとして機能し、広告を表示するのに適切な場合はビデオプレーヤーを一時停止し、広告の中断が完了するとプレーヤーを再開します。
  • 実行時管理：フレームワークには、広告をいつ再生するかを管理し、広告マネージャに制御を渡す機能があります。
  • UIインタラクション：フレームワークはUIインタラクションを処理できます。 たとえば、コントロールバーを表示または非表示にしたり、コントロールバー通知を広告マネージャに送信したりできます。
  • API：フレームワークは、パートナーがOoyalaプレーヤーで使用するためのカスタマイズされたモジュールを開発するのを助けるためのAPIのセットを提供します。
  • 拡張性：パートナーやサイト運営者は、このフレームワークを拡張して、あらゆる広告ソースまたはプロバイダを使用してカスタム広告プラグインを簡単に構築できます。 Ad Manager Frameworkを使用すると、パートナーはカスタムスキン、カスタム広告、および分析モジュールを構築できます。
