コンテンツの発見と推奨事項

Ooyalaのコンテンツ検出機能を使用して、コンテンツの推奨事項、詳細なプログラミングガイド、およびパーソナライズされたコンテンツキューを提供することで、顧客エンゲージメントを高めます。

  • コンテンツの発見と推奨について
    ディスカバリーを使用すると、サイト上の関連性のある関連コンテンツにユーザーを誘導することで、ユーザーの関心を引くことができます。 ディスカバリーは、ビデオの印象や再生から継続的に学習して、動的でパーソナライズされた体験をユーザーに提供します。
  • バックスロットUIでの検出の設定
    パーソナライズされたおすすめを設定する方法と、ビデオの再生が終了または一時停止したときにOoyalaプレーヤーがおすすめを表示する方法を学びます。
  • ディスカバリーインサイトレポート（利用不可）
    Ooyala Discovery Insightsのレポートには、ユーザーエンゲージメントと成長に関する主要業績評価指標（KPI）の詳細なビューが表示されます。
  • プログラム分析インサイトレポート
    Program Analytics Insightレポートは、プログラムの視聴者数やその他の指標をグラフィカルにまとめたものです。
  • ディスカバリーデザインの考慮事項
    プレーヤーのディスカバリーを実装する際には、これらの設計上の問題を考慮してください。
  • 発見のベストプラクティス
    これらのベストプラクティスの推奨事項を考慮して、Discoveryの実装によるメリットを最大化し、サイトまたはアプリへの参加を促進します。
  • ディスカバリープロファイル
    あなたは再利用可能を作成することができます 検出プロファイル これは、プレーヤーに表示されるコンテンツの推奨事項に対するOoyala Discoveryの設定を構成します。
  • 発見フィードバックループ
    ユーザーからOoyala Discoveryに行動活動をフィードすることで、顧客に対する推奨を向上させます。
  • ディスカバリーREST API
    これらのディスカバリーAPIを使用するには、ディスカバリー要求を出し、ユーザーのアクション情報を戻してアルゴリズムをトレーニングします。
