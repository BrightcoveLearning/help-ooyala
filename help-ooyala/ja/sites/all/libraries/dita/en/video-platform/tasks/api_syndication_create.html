シンジケーションを作成する

シンジケーションを作成したら、それをラベルに関連付けることができます。 いずれかのラベルを使用するすべてのビデオとチャンネルは、設定を使用して自動的にシンジケートされます。

シンジケーションを作成するには

  1. 使用 / v2 /シンジケーション 名前とタイプ（詳細、boxee、google、iphone、iphone_abr、itunes、mp4、またはroku）を指定してroute。 次の例では、 "My iPhone Feed"というiPhoneフィードを作成します。 
    [POST] / v2 / syndications {"名前"： "私のiPhoneフィード"、 "タイプ"： "iphone"}
    注意： YouTubeフィードを作成している場合は、YouTubeも指定する必要があります ユーザ名 そして パスワード.

    バックスロット 次のような応答を返します。

    {"name"： "私のiPhoneフィード"、 "include_encodings"：[]、 "destination_url"： ""、 "require_access_key"：false、 "title"： ""、 "include_labels"：[]、 "asset_types"：[ "ビデオ"、 "広告"]、 "syndication_url"： "http://api.ooyala.com/syndications/6b6e72dd71bd4fa/feed?pcode=Y4cG06BlqYMLIjPGyv1NbacSK-x3"、 "ID"： "6b6e72dd71bd4fa"、 "include_all_content"：偽、 "type"： "iphone"、 "description"： ""}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. フィードに動画やその他のアセットを含めるには、ラベルを追加する必要があります。 フィードに関連付けられているアセットにラベルを追加すると、そのアセットが自動的にフィードに追加されます。 次の例では 638aed2a18734375b そして c42b225963ad481b9 へのラベル 6b6e72dd71bd4fa シンジケーション 
    [PATCH] / v2 / syndications / 6b6e72dd71bd4fa {"include_labels"：["638aed2a18734375b"、 "c42b225963ad481b9"]}

    バックスロット 次のような応答を返します。

    {"name"： "My iPhone Feed"、 "include_encodings"：[]、 "destination_url"： ""、 "require_access_key"：false、 "title"： ""、 "include_labels"：["638aed2a18734375b"、 "c42b225963adXNXbX" ]、 "asset_types"：[ "ビデオ"、 "広告"]、 "syndication_url"： "http://api.ooyala.com/syndications/481b9e6dd6bd72fa/feed?pcode=Y71cG4BlqYMLIjPGyv4NbacSK-x06"、 "ID"： "1b3e6dd6bd72fa "、" include_all_content "：false、" type "：" iphone "、" description "：" "}
  3. 提供されているエンコーディングの種類をフィルタするために、デバイスまたはコンテナの制限を指定できます。 次の例では、iPhoneのエンコーディングをMP4のコンテナに制限します。 6b6e72dd71bd4fa シンジケーション 
    [PATCH] / v2 / syndications / 6b6e72dd71bd4fa {"include_encodings"：[{"デバイス"： "iphone"、 "コンテナ"： "mp4"}]}

    バックスロット 次のような応答を返します。

    {"名前"： "私のiPhoneフィード"、 "include_encodings"：[{"デバイス"： "iphone"、 "コンテナ"： "mp4"}]、 "destination_url"： ""、 "require_access_key"：false、 "title "：" "、" include_labels "：[" 638aed2a18734375b "、" c42b225963ad481b9 "]、" asset_types "：["ビデオ "、"広告 "]、" syndication_url "：" http://api.ooyala.com/syndications/syndications/ " 6b6e72dd71bd4fa / feed？pcode = Y4cG06BlqYMLIjPGyv1NbacSK-x3 "、" id "：" 6b6ddXN "、" false "、" ""、 "XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX"、 "" "、" ""、 "" "、" ""、 "："

シンジケーションは正常に作成されました。

注意： ラベル、エンコーディング、およびその他の設定は、シンジケーションの初期作成中に指定できます。
