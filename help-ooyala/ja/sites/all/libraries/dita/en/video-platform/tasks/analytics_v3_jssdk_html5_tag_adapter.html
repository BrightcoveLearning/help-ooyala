Integrating an HTML5 Video Tag with IQ JS SDKを使う

Ooyala HTML5ビデオタグアダプタは、 ウーヤラIQ HTML5 <video />タグイベントを通信するJavaScript SDK ウーヤラIQ。 このアダプタを自分で実装することも、Ooyalaプロフェッショナルサービスがアダプタを有料で実装することもできます。 HTML5ビデオタグプレーヤーをOoyala IQと統合するには、次の手順を実行します。

HTML5ビデオタグアダプタを使用する前に、Ooyalaアセットがサードパーティのプレーヤーで再生可能であることを確認する必要があります。
  1. HTML5ビデオタグでサポートされているエンコードを許可するために、アカウントに正しいエンコードプロファイルが設定されていることを確認してください。 あなたはあなたのアカウントマネージャにあなたの処理プロファイルをチェックする援助を求めることができます。
  2. エンコードプロファイルで新しいファイルをエンコードできるようにする必要がある場合は、新しく生成されたファイルを取得するために、新しいビデオファイルをアップロードするか、ビデオファイルを再エンコードする必要があります。 コンテンツの置き換え（アセットの置き換え）はこのステップでは機能しません。 新しいファイルをアップロードするか、Ooyalaテクニカルサポートでサポートチケットを開いて、同じ埋め込みコードを維持するために手動でアップロードせずにビデオを再処理する必要があります。
  1. でanalytics.jsを参照してください。 https://analytics.ooyala.com/static/analytics.js プレーヤーがホストされているページで これは、プレーヤーがあるページに対して行う必要があります。 <video />要素を変更する他のスクリプトの前にこのスクリプトをロードする必要があります。
    警告： analytics.jsを自分でホストしないでください。 そうしても、バグ修正やアップデートは受けられません。 Ooyalaがホストするバージョンを使ってください。 https://analytics.ooyala.com/static/analytics.js.
  2. ページに<video /> HTML5タグを含めます。
    1. ビデオ要素を変数に入れます。
    2. をセットする data-embedId video要素の属性です。
    3. コー​​ル Ooyala.Analytics.HTML5Reporter.
