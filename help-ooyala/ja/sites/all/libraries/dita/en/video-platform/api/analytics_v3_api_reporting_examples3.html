どの地域がコンテンツトラフィックを促進していますか。

コンテキスト： あなたのコンテンツは複数の国の視聴者に届きます。 あなたはあなたのトラフィックのどの部分があなたのコンテンツが届くそれぞれの地理によって動かされているかを見たいです。

How to Determine This with v3 Analytics

に対してクエリ 国 dimension, filter by ドメイン (domain='exampledomain.com'）と device _type (device_type='tablet'), and sort by the plays_requested メトリック。 
[GET] /v3/analytics/reports/?
report_type=performance
&dimensions=country
&filters=((device_type=='tablet')) AND ((domain=='exampledomain.com'))
&metrics=plays_requested
&start_date=2014-10-10
&end_date=2014-10-20
&sort=plays_requested
&api_key=yourApiKey
https://api.ooyala.com/v3/analytics/reports/?report_type=performance&dimensions=country&amp;filters=%28%28device_type%3D%3D%27tablet%27%29%29+AND+%28%28domain%3D%3D%27exampledomain.com%27%29%29 &metrics=plays_requested&start_date=2014-10-10&end_date=2014-10-20 &sort=plays_requested&api_key=yourApiKey& expires=yourExpiration&signature=yourSignature
