ラベル管理：削除（廃止予定）

バックスロットAPI v1

http://www.ooyala.com/partner/labels
?pcode=<pcode>
&expires=<expiration>
&mode=deleteLabels
&labels=<label1>,<label2>,<label3>
&signature=<signature>

バックスロットAPI v2

[削除] https://api.ooyala.com/v2/labels/ <label_id>

例

次の例では、資産からラベルを削除します。

[DELETE] https://api.ooyala.com/v2/assets/d5751b77a0c24972888bf906734d4522
{
	"name":"Hobbies"
}

バックスロット 200応答を返します。
