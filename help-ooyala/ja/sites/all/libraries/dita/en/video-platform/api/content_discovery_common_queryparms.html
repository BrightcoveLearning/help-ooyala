クエリ文字列パラメータ

これらのクエリ文字列パラメータは、類似/関連アセット、トレンド分析、およびパーソナライズされた推奨事項に対するAPIリクエストで使用できます。

注意： （要求ごとに個別のパラメーターを使用して）アドホック要求を作成するのではなく、ディスカバリー・プロファイルを使用して設定と推奨動作を一元管理することを検討してください。 検出プロファイルは、ユーザーが作成した設定の定義済み構成です。 のGET ルート 詳細については、 ディスカバリープロファイル.

次のクエリ文字列パラメータを使用して、次のリクエストによって返される結果の基本セットを変更します。

  • 同様の動画
  • トレンドと人気コンテンツ
パラメーター 説明 必須？
discovery_profile_id 以前に定義された検出プロファイルの識別子 （DiscoveryのBacklot UIに表示される定義済みプロファイルのプレーヤーIDと同じです。） ディスカバリーデザインの考慮事項.

タイプ：文字列

デフォルト：なし

例： discovery_profile_id=e18ab1da1813483499554ea2z8e67fbd
exclude_labels 結果から除外する最大10個のラベルをコンマで区切って指定します。

タイプ：文字列

デフォルト：なし

例： exclude_labels=hiamZvMjqKnLMr7xnMF-qcTEanQlhCQ1,pmd2xzMjrCg4TpQd6cwz9HP4U-pcFSHH

 いいえ
exclude_videos 結果から除外する最大10個の埋め込みコード（アセットIDまたはコンテンツID）を、コンマで区切って指定します。 10を超える動画を除外する必要がある場合は、編集APIの使用を検討してください。

タイプ：文字列

デフォルト：なし

例： exclude_videos=hiamZvMjqKnLMr7xnMF-qcTEanQlhCQ1,pmd2xzMjrCg4TpQd6cwz9HP4U-pcFSHH

いいえ
filter_by 編集機能の一部で、結果を指定されたラベルに制限します。
注意： で指定されたプロファイルを常にオーバーライドします。 discovery_profile_id パラメータ。

値の中のコンマ（、）は論理ORを示します。

タイプ：文字列

デフォルト：なし

例：

  • filter_by = cat （ラベルに "cat"が含まれています）
  • filter_by =猫、ピアノ （ラベルには「cat」または「piano」が含まれています）
  • filter_by = shared：language （「/ language / english」のように、コンテンツには同じ「language」サブラベルがあります）
  • filter_by = cat、共有：言語 （ラベルには "cat"ラベルまたは同じ言語のサブラベルがあります）
いいえ
制限 返される結果の最大数。
注意： 見る ディスカバリーデザインの考慮事項 このパラメータの誤用について

タイプ：整数

デフォルト：10

有効値：1〜100

例： 制限= 20

いいえ
組合 編集機能の一部で、注入するコンテンツと結果内の位置を指定します。

タイプ：文字列

デフォルト：なし

例：

  • union = next | next_item | 0 （で指定された資産を注入します。 next_item 結果の位置0へのコンテンツメタデータのエントリ）
  • 共用体= n | next_item | 0 （前の例のショートカット）
  • union = next | next_item | 0、union = next | next_highlight | 2 （で指定された資産を注入します。 next_item 結果の位置0およびコンテンツで指定されたアセットへのコンテンツメタデータのエントリ next_highlights 位置2へのコンテンツメタデータのエントリ）
  • 共用体=コレクション| 2012-09_extreme_paint | 1 （2012-09_extreme_paintコレクションから1の位置に宣伝用コンテンツを挿入します）
  • 共用体= c | 2012-09_エクストリームペイント| 1 （前の例のショートカット）
  • 共用体=コレクション|ライブ| 1 （ライブ資産をポジション1に注入します）
  • union = collection |レーベル：BasketballPromo | 0 （BasketballPromoラベルの宣伝コンテンツを0の位置に挿入します）
  • union=collection|label:BasketballPromo|0|2012-10-29|2012-10-31 （BasketballPromoラベルのプロモーションコンテンツを0 / 10 / 29から2012 / 10 / 31の位置2012に挿入します）
 いいえ
where メタデータまたはラベルの値に基づいてアセットを含めるまたは除外する。
注意： where句は、Backlot APIクエリのwhere句と構文的に同じです。
注意： ASCII以外の文字と ' 文字はエスケープされます。

タイプ：文字列

デフォルト：なし

例：

where = metadata.sport = 'soccer' 		いいえ
