画面の停電/ちらつきを防ぐ

この設定は、ビデオの開始時にデバイスの画面が黒くなったりちらついたりするのを防ぎます。

場合によっては、デバイスの画面が単に暗くなることがあります。 これは、SDK自体によるものではなく、基盤となるAndroidシステムによるものであり、おそらく SurfaceView.

デバイスの停電やちらつきを防ぐには、アプリケーションの最初のレイアウトアクティビティに次のXMLを追加します。

<？xml version = "1.0" encoding = "utf-8"？> <RelativeLayout xmlns：android = "http://schemas.android.com/apk/res/android" android：id = "@ + id / ooyala_container "android：layout_width =" fill_parent "android：layout_height =" fill_parent "> <SurfaceView android：layout_width =" 0dp "android：layout_height =" 0dp "/>。 。 ここにあなたの最初のレイアウト。
