Player V3 JavaScript APIパラメータ（廃止予定）

プレーヤV3パラメータは、プレーヤ広告、動作をカスタマイズし、プレーヤスタイルを変更するために使用できます。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
次の一般的な種類のパラメータをPlayer APIに渡すことができます。
  1. クエリ文字列パラメータ
  2. 埋め込みパラメータ
  3. カスタムモジュールの埋め込みパラメータ
次の表は、プレーヤAPIで使用されるパラメータをまとめたものです。
