ラベルを削除する

とともに バックスロットAPIラベルはいつでも削除できます。

To delete labels:

  1. If you don't have the ID of the label to delete, use GET with the / v2 /ラベル ルート。 The following is a GET example. 
    [GET] / v2 / labels

    バックスロット 次のような応答を返します。

    [  
   {  
      "name":"Hobbies",
      "id":"Yh98G7hsaß",
      "full_name":"/Hobbies"
   },
   {  
      "name":"Hockey",
      "id":"82d562682",
      "full_name":"/Hobbies/Hockey",
      "parent_id":"Yh98G7hsaß"
   },
   {  
      "name":"Disc Golf",
      "id":"55514522",
      "full_name":"/Sports/Disc Golf",
      "parent_id":"Yh98G7hsaß"
   }
]
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. Delete a label. The following example deletes the "Disc Golf" label. 
    [DELETE]/v2/labels/55514522

    バックスロット returns a 200 response code.

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
