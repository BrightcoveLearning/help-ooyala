アクティブ共有設定

このセクションでは、埋め込みメタタグを介したアクティブ共有の設定に関する基本的な手順について説明します。

アプリケーションを設定するには

  1. からプログラミング言語のフォルダを探します。 Facebookのコード例（で入手可能） zipダウンロード) zipファイルをダウンロードして、その内容をプロジェクトフォルダにコピーします。
  2. Ooyala Facebook用法ファイルを開く Facebookのコード例（で入手可能） zipダウンロード).
  3. 内容をアプリケーションにコピーしてください。
  4. インポートしたファイルを必ず含めてください。
  5. Insert the return value of the head prefix method as an attribute of the HTML head tag. For example:  <head prefix="og: http://ogp.me/ns# fb: http://ogp.me/ns/fb# video: http://ogp.me/ns/video#">
  6. 挿入する メタタグ HTMLのheadタグの下。 これらのタグの説明については、 Facebookでビデオを共有する。 例えば： 
    <meta property="og:type" content="video.other" />
<meta property="og:title" content="Bhangra Empire - Jashan 2011" />
<meta property="og:description" content="dancing" />
<meta property="og:url" content="iamaurl">
<meta property="og:image" content="http://ak.c.ooyala.com/c1d3AxMzo5_lJK08LHYfpzFF02StTtfk/Y2BrwET1VVpfCiI35jMDoxOjBrOwKdHu" />
<meta property="og:video" content="http://glowing-spring-6591.herokuapp.com/share" />
<meta property="og:video:secure_url" content="https://glowing-spring-6591.herokuapp.com/share" />
<meta property="og:video" content="http://player.ooyala.com/player.swf?embedCode=c1d3AxMzo5_lJK08LHYfpzFF02StTtfk&keepEmbedCode=true&docUrl=http%3A%2F%2Fexample.com" />
<meta property="og:video:secure_url” content=" https://player.ooyala.com/player.swf?embedCode=c1d3AxMzo5_lJK08LHYfpzFF02StTtfk&keepEmbedCode=true " />
	<meta property="og:video:type " content="application/x-shockwave-flash " /> 	 
	<meta property="og:video:height " content="480 " />
    <meta property="og:video:width " content="640 " />
  7. If you want to add a share button, add the following to your page:  <a name="fb_share"></a> <script src="http://static.ak.fbcdn.net/connect.php/js/FB.Share" type="text/javascript"></script>
  8. アプリケーションを実行してテストします。
