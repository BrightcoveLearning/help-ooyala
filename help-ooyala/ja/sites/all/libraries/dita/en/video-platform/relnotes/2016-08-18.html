ソフトウェア機能リリース

eコマースAPI、バージョン3、リリース1

これは、一般化されたオープンeCommフレームワークであるeコマースバージョン3の最初のリリースです。 eコマースバージョン3、リリース1のAPIには、支払い方法の登録と確認、および購読とワンタイムレンタルの作成の機能が含まれています。 詳細については、Ooyalaカスタマーサポートマネージャーにお問い合わせください。

サンプルAPI呼び出し：

  • 支払い方法を登録するには： 
    POST /コマース/ v3 / accounts / payment-methods / token？account_token =＆pcm_id = V＆bv_id = V
  • 支払い方法を確認するには 
    POST / commerce / v3 / accounts / payment-methods？account_token =＆pcm_id = V＆bv_id = V
  • 購読（SVOD）を作成するには： 
    POST / commerce / v3 / accounts / subscriptions？account_token =＆pcm_id = V＆bv_id = V
  • TVODまたはレンタルの一時料金を作成するには 
    POST / commerce / v3 / accounts / onetime-charge？account_token =＆pcm_id = V＆bv_id = V

あなたが使用しているサブスクリプションおよび収益プロバイダはあなたのアカウントのIDを発行します。 上記のコードサンプルでは、​​IDは「V」で示されています。 このリリースでは、購読および収益ソリューションプロバイダーとしてのVindiciaとの統合が維持されています。
