一時停止画面をオンにする

次の要求と応答の例では、PATCHメソッドを使用してプレーヤーIDを使用してプレーヤーを呼び出し、一時停止画面をオンにします。

要求：

[パッチ] / v2 / players /{player_id}/ discovery / profile {"show_pause_screen"： "enable"}

応答：

{"discovery_interface"：{"when_done"：null、 "show_pause_screen"： "true"、 "default_countdown"： "10"、 "hide_thumbnail_title"： "false"}、...
