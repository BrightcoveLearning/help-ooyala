エラーとエラー処理の概要

エラーメッセージを使用して問題を追跡し、トラブルシューティングします。

エラーの原因

エラーは多くの原因から発生します。 探しているエラー情報を見つけやすくするために、エラーを次のように分類しています。
  • ユーザー向け - プレーヤーはプレーヤーUIでこれらのエラーをユーザーに表示します。
  • プレイヤーV3 JavaScript - JavaScriptベースのエラー処理およびエラー追跡機能を提供します。
  • プレイヤーV3イベントベース - Player V3はメッセージバスイベント処理を使用します。
  • モバイルSDK - 当社のモバイルSDKは、当社のPlayer V3 JavaScriptおよびFlashプレーヤーをサポートしています。 それらはいくつかの特定のエラー処理メカニズムを持っています。
  • 摂取 - Ooyalaまたはサードパーティの取り込み方法を使用してコンテンツをアップロードすると、エラーが発生する可能性があります。
  • その他の外部 - Flash Player、サードパーティ広告統合マネージャ、CDN、およびコンテンツ保護メカニズムをサポートしています。 エラーはこれらの原因から発生する可能性があります。 エラーがこれらの原因から発生しているかどうかを確認するためのリンクを提供しています。
次の手順で発生する多くの一般的な問題やエラーのトラブルシューティングを開始できます。
  1. あなたの設定が私達の条件を満たすことを確かめなさい サポートされている標準.
  2. あなたの問題が私たちのうちの1つで答えられているかどうか確認してください よくある質問.
  3. エラーメッセージまたはコードを使用して問題を絞り込んでください。 私たちをチェックしてください ユーザー向けエラー, プレイヤーエラーメッセージ、および プレイヤーエラーイベントメッセージ.
  4. 検索する コミュニティナレッジベース 同様の質問と可能な解決策のために。
  5. ファイルa サポートチケット
