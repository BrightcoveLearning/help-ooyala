ウーヤラディスカバリーキュー（廃止予定）

重要： キューAPIは無効です。 代替オプションについては、Ooyalaの担当者にお問い合わせください。

視聴者のおすすめが検出されたら、Discovery Queue APIを使用して視聴者にそのおすすめを視聴者のキューに追加させます。 動画は動画の所有者（プロバイダ）のみが提供する場合があります。

Discovery Queueのユーザー認証は、OoyalaのRights Lockerに基づいています。 詳細については 権利ロッカー.

ディスカバリーキューAPIメソッドの基本的なシーケンスは次のとおりです。
  1. 視聴者がアイテムをキューに追加したい場合は、 [PUT] / v2 /ディスカバリ/キュー/資産/asset_id そして議論。
  2. ビューアのキューとそのアイテムを表示したい場合は、 [GET] / v2 / discovery / queue 資産のリストを取得してから使用する [GET] / v2 / assets 各資産の詳細を取得する
  3. 視聴者がキューからアイテムを削除したいときは、 [削除] / v2 /ディスカバリ/キュー/asset_id

視聴者のキューにアセットを追加する

この例では、ビューアihqwv9uw1eflbkcqwpouのキューに新しいアセットを追加します。

[PUT] / v2 /発見/キュー/視聴者/アカウントID/アセット/埋め込みコード
バックスロットAPI 以下と同様に応答します。 
"added_at"：1363617641

リストビューアのキュー

次の例では、viewerに属するキュー全体を一覧表示します。 

[GET] / v2 /発見/キュー/視聴者/アカウントID

複数のアセットがキューに追加されていると、応答は次のようになります。 この一連のアセットIDを使って、 [GET] / v2 / assets メタデータを含む個々の資産の詳細を取得します。

["RsYmduMjpNE3RGHOTa6Chm-YoxArCqiw"、 "ZoYmduMjpBbLj3QerkWScewtpPnvmOi2"、 "pxcm9lMzpfyODDtjXNMXdmXmXdmXmXdmXmXdmXmXdmXmXdmXmXmXdmXmxmxmxmxmxmxmxmxxmxxmxmydmil ..."

視聴者のキューから単一のアセットを削除する

次の例では、ビューアihqwv9uw1eflbkcqwpouのキューから項目を削除します。

[削除] / v2 /ディスカバー/キュー/ビューア/アカウントID/アセット/埋め込みコード

呼び出しが成功しても応答本文は返されません。

視聴者のキューからすべてのアセットを削除

次の例では、ビューアihqwv9uw1eflbkcqwpouのキューから項目を削除します。

[削除] / v2 /ディスカバー/キュー/ビューア/アカウントID

呼び出しが成功しても応答本文は返されません。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
