Androidネイティブアプリの検出について

SDKはディスカバリーのサポートを含みます。

DiscoveryManager クラスには、ディスカバリー結果を管理するのに役立つ静的メソッドが含まれています。

それは含まれています コー​​ルバック これには次のものが含まれます。 折り返し電話 ディスカバリー結果を通知するメソッド。 それはによって定義されます 

            public void callback（オブジェクト結果、OoyalaExceptionエラー）;

パラメータを取ります 結果 （ディスカバリー結果のコンテナー） エラー （ OoyalaException 発生する可能性のあるエラーをキャプチャするオブジェクト これらは出力パラメータです。

DiscoveryManager class 以下のメソッドが含まれます。

表1 発見方法
クラスメソッド 説明
getResults この方法を使用して、特定の資産のディスカバリー結果を取得します。 詳細については Android Nativeのディスカバリ結果を取得する.
sendImpression ビデオのおすすめが表示されたときにDiscoveryのフィードバックを送信するには、この方法を使用します。 詳細については Androidネイティブの検出インプレッションを送信する.
sendClick この方法を使用して、推奨ビデオにアクセスしたときに発見フィードバックを送信します。 詳細については Android Nativeのディスカバリークリック通知を送信する.
  • Android Nativeのディスカバリ結果を取得する
  • Android Nativeのディスカバリークリック通知を送信する
  • Androidネイティブの検出インプレッションを送信する
