2013-12-23リリース

アップデートには、Twitterの要件を満たすためのSSLの最適化、サードパーティの広告マネージャの最適化、再生APIの最適化、Twitterの統合の向上、HTML5 iFrameのサポート、モバイルSDKとのGoogle IMA、およびモバイルSDKとのFreeWheelが含まれます。

リリース項目は以下のとおりです。

  • SSL - Twitterの要件を満たすために、宣伝用画像、サーバードメイン、その他のSSL配信の最適化が行われました。

  • 第三者広告マネージャ - 多数の最適化。

  • 再生 - APIの最適化、広告再生の改善、スクラバー画像の更新。

  • Twitter - Twitterとの統合が改善されました。 詳細については、 Twitterプレーヤーカードとの統合 ドキュメンテーション。

  • HTML iFrameのサポート - 詳細については、次のWebサイトを参照してください。 HTML iframeへのHTML5プレイヤーV3の埋め込み（非推奨） ドキュメンテーション。

  • Mobile SDKを使用したGoogle IMA - 詳細については、次のURLを参照してください。 iOS上のGoogle IMAとの統合 そして Android上のGoogle IMAとの統合 ドキュメンテーション。

  • Mobile SDKを使用したFreeWheel - 詳細については、次のWebサイトを参照してください。 Android上のFreeWheelとの統合 ドキュメンテーション。
