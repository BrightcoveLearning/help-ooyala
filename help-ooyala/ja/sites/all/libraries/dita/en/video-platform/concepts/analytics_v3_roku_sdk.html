ウーヤラIQ SDK for Roku Integration

Roku用のOoyala IQ SDKを使用して、RokuチャンネルとOoyala IQを統合できるようになりました。 Roku用Ooyala IQ SDKは、JSONデータをOoyala IQ JSON APIエンドポイントに送信し、どのイベントをいつ送信するかについてのベストプラクティスに自動的に従い、すべての必須フィールドが存在することを確認する、すぐに使えるSDKです。 。

Roku用Ooyala IQ SDKはRoku OS v7 +をサポートします.
注意： Roku用Ooyala IQ SDKは現在Roku Scene Graph APIをサポートしていません。

Ooyala IQ SDK for Rokuを使用している場合は、Rokuの[デバイス]パネルにRokuがOSとして表示されます。 Ooyala IQビジネスインテリジェンス インタフェース。

ホスティング

OoyalaはOoyala IQ SDK for Rokuライブラリをホストしています GitHubの （あなたはライブラリをダウンロードすることができます）。 Roku用のOoyala IQ SDKは、JavaScript SDKを使用した場合とは異なりオンザフライでRokuボックスにダウンロードできないため、Rokuチャネルパッケージの不可欠な部分である必要があります。

実施ステップ

Ooyala IQ SDK for Rokuを無料で自分で統合することも、OoyalaプロフェッショナルサービスでOoyala IQ SDK for Rokuソリューションと有料で統合することもできます。

Roku用のOoyala IQ SDKを実装するには、以下の手順に従います。

  1. Ookuala IQ SDK for Rokuライブラリをダウンロードします。 GitHubの.
  2. Roku用Ooyala IQ SDKをRokuチャンネルと統合します（に説明があります）。 Ooyala IQ RokuライブラリAPIドキュメント).
  3. RokuでRokuアプリケーションを認証してください。
  4. Rokuアプリケーションを顧客に公開します。
  5. Ooyalaが新しいバージョンのOoyala IQ SDK for Rokuをリリースするたびに、ステップ1-4を繰り返します。 新しいリリースが発生するたびにお知らせします。
