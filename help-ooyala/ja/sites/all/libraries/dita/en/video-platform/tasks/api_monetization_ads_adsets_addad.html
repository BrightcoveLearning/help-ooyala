動画広告を広告セットに追加する

広告セットを作成したら、好きなだけビデオを追加できます。

広告セットに広告を追加するには

に投稿する / v2 / ad_set 広告セットの識別子を含むルート / ads 修飾子、およびPOST本文の動画広告の詳細。 次の例では、広告セットを作成し、ユーザー自身のアセットから01M2F0MTr3SWBB9dfAPT5xEsg8Gpyアセットをそれに関連付けます。 
[POST] / v2 / ad_sets / 77506f70fc1e4b08805687980cc4a24d /広告{ "広告"：[{ "ad_source"： "資産"、 "AD_TYPE"： "プリロール"、 "embed_code"： "01M2F0MTr3SWBB9dfAPT5xEsg8Gpy"、 "plays_before"：0、「plays_between "：1、" click_url "：" http://www.myclickurl.com "、" tracking_pixel_urls "：[" http://www.mytrackingpixel.com "]}]}
バックスロットUI 以下と同様に応答します。
