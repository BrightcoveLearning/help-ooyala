リモートアセットのDRM属性

If you package content using Digital Rights Management (DRM) and want Ooyala to serve licenses for that content, you will need to set some attributes per asset.

These attributes need to be applied only to remote assets.

Fetch Attributes for an Asset and Given DRM Type

[GET]/v2/assets/asset_id/drm_attributes/drm_type

Adding or Updating an Attribute

[パッチ] / v2 / assets /asset_id/drm_attributes/drm_type{
   
   プロパティ

}

ルート属性

次の表は、ルートを通じて表現できるすべての属性を示しています。

ルート属性 説明
asset_id 見る すべてのアセットタイプに共通のルートと属性 for a description of this attribute.
drm_type

The type of DRM the asset is associated with.

タイプ：文字列

Valid Values: playready | fps

デフォルト：なし

Example: playready
