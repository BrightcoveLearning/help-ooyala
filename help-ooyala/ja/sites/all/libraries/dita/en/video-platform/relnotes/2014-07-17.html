2014-07-17リリース

注目すべき変更には、iOSおよびAndroidモバイルSDKのv.2.6.0のリリースが含まれます。

ソフトウェア機能リリース

以下は新しいソフトウェアリリース項目です。

iOS SDK v.2.6.0

iOS用Ooyala Mobile SDKのバージョン2.6.0がリリースされました。 バージョン2.6.0の注目すべき変更点は、OoyalaがGoogle IMAライブラリを統合と同時に提供しなくなったことです。 これらのライブラリはGoogle IMAから直接ダウンロードしてください。 私たちは新しいパッケージ化スクリプトに移行しました。 SDKパッケージに欠けているものがあれば、テクニカルサポートに問題を報告してください。

iOS SDK v.2.6.0のフルリリースノート

Android SDK v.2.6.0

Android用Ooyala Mobile SDKのバージョン2.6.0がリリースされました。 バージョン2.6.0の注目すべき変更点は、OoyalaがGoogle IMAライブラリを統合と同時に提供しなくなったことです。 これらのライブラリはGoogle IMAから直接ダウンロードしてください。

私達はまたすべてのOoyalaの顧客に利用可能なVisualOn統合をリリースしました。 詳細については、カスタマーサクセスマネージャーに連絡してください。

Android SDK v.2.6.0のフルリリースノート

新しい“ name_live” Widevine DRMポリシープロパティ

新しいDRMポリシープロパティ「name_live」を使用すると、お客様は、自分のWidevineポータルを使用してトランスコードしたビデオにWidevine DRMポリシーを適用できます。 DRMポリシーは、Widevineライセンスの取得中に適用されます。

ドキュメントリリース項目

以下は、新しいドキュメントのリリース項目です。

  • iOS SDK v.2.6.0

    • iOS用Mobile SDK v.2.6.0 - バージョン2.6.0用のアップデートされたiOS SDKドキュメント。
    • iOS上のGoogle IMAとの統合 - iOS上のGoogle IMAとの統合に関するドキュメントの「必要なもの」セクションを更新しました。 これで、Google Developer WebサイトからGoogle IMA Librariesをダウンロードする必要があります。

  • Android SDK v.2.6.0

    • Android用Mobile SDK v.2.6.0- バージョン2.6.0用のアップデートされたAndroid SDKドキュメント。
    • Android上のGoogle IMAとの統合 - Android上のGoogle IMAとの統合に関するドキュメントの「必要なもの」のセクションを更新しました。 これで、Google Developer WebサイトからGoogle IMA Librariesをダウンロードする必要があります。

  • VAST広告タグの埋め込みパラメータ

    • VAST Adsの埋め込みパラメータ- VAST広告タグの埋め込みパラメータを使用するには、広告セットもBacklotに適用し、広告セットに有効なVAST広告タグを含める必要があるという説明を追加しました。

  • 新しい“ name_live” DRMポリシープロパティ

    • DRMポリシー- 新しい「name_live」プロパティに関するドキュメントを追加しました。

  • クロスデバイスレジューム（XDR）サポート

    • プレイヤー：サポートされているテクノロジとデバイス- クロスデバイスレジュームをサポートしているデバイスに関するドキュメントの更新。

  • 高性能APIエンドポイント説明

    • 高性能APIエンドポイント- 高性能APIエンドポイントの説明を更新しました。
