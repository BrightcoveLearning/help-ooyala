ビジネスインテリジェンスページへのフィルタの適用

フィルタを使用して、[ビジネスインテリジェンス]ページの特定のディメンションにドリルダウンします。 一度に3ディメンションフィルタまで適用できます。

にフィルタを適用するには ビジネス·インテリジェンス ページ：

  1. ディメンションのフィルタ処理されていないビューから始めます。
  2. 寸法パネルの一番右上にあるフィルターアイコンをクリックします。

    アイコンの色が変わり、フィルタリングオプションが表示されます。

  3. 左側に沿って、フィルタリングしたい項目のチェックボックスをクリックします。 項目を選択すると、それらはリストの一番上に移動されてソートされます。
  4. 必要に応じて上部のテキストボックスに追加のフィルタ用語を入力し、Enterキーを押します。 これらの検索語は大文字と小文字を区別しません。 検索語句の結果は、チェックボックスで選択した項目の下に表示されます。
  5. ナビゲーションヘッダーの 適用します. フィルタリングの結果は表形式で表示されます。 フィルタは、ビジネスインテリジェンスページ全体に適用されます。

あなたが取ることができる追加の可能な行動：

  1. 選択からプリセットフィルタを作成するには、 ビジネスインテリジェンスページでのプリセットフィルタの使用.
  2. フィルタをキャンセルするには、 X と言う分野でx 選択したアイテム".
  3. テーブルの右上にある対角線上の矢印をクリックすると、展開されたテーブルビューにデータを表示できます。 見る 拡張テーブルの表示.
  4. ビジネスインテリジェンスページで日付範囲を設定します。.
    注意：
    • Ooyala IQのUIで最大3月（1日）までの日付範囲に31ディメンションフィルタを適用できます。 v3 Analytics APIを使用する.
    • Ooyala IQのUIで最大2年（1日）までの日付範囲に対して366以下のディメンションフィルタを適用できます。 v3 Analytics APIを使用する.

    • 検出情報は、Playerフィルタのみが適用されている場合、またはフィルタが適用されていない場合に使用できます。 他のフィルタ（国、デバイスなど）が使用されている場合、検出データは利用できません。 検出データを表示するには、Player以外のすべてのフィルタの選択を解除するか、すべてのフィルタの選択を解除します。
