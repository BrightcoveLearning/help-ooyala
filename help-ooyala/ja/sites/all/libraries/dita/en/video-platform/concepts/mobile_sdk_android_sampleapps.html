Android Mobile SDKのサンプルアプリケーション

サンプルアプリを使用してSDK APIについて学び、アプリをすばやく起動して実行する。

SDKサンプルアプリはAndroid Studioと互換性があり、次のリポジトリで利用できます。

OoyalaサンプルアプリGit Hub
注意： かつてマスターブランチと呼ばれていたものは、現在 "安定"と呼ばれています。 Ooyalaがサンプルアプリのリリースまたは修正プログラムを公開しない限り、安定版ブランチは変更されません。 リポジトリをすでに複製またはフォークしている場合は、ブランチ名をstableに変更する必要があります。
表1 サンプルアプリケーション
サンプルアプリケーション 説明
高度な再生サンプルアプリ このアプリには、プレーヤレイアウトの変更、カスタムプレーヤコントロールの作成、およびサードパーティの広告プラグインを作成するためのAPIを示すコードが含まれています。
基本再生サンプルアプリ SDKを説明するための基本的なプログラム。 見る 基本的なAndroidアプリの作成.
完全なサンプルアプリ このサンプルは、すべてのサンプルアプリケーションの主なコード例を1つのアプリケーションにまとめたものです。
コンテンツ保護サンプルアプリ このアプリは、利用可能なコンテンツ保護機能の一例です。
Ooyala Pulseサンプルアプリ このアプリは、広告配信のためにOoyala SDKがOoyala Pulseと連携して機能する方法を示しています。
FreeWheelサンプルアプリ このアプリは、Ooyala SDKがFreeWheelと連携して機能する方法を示しています。
Google IMAサンプルアプリ このアプリは、Ooyala SDKがGoogle IMAと連携してどのように機能するかを示しています。
NPAWの統合 このアプリは、Android用のNPAW統合の実装を示しています。
Ooyala APIサンプルアプリ このアプリは、Channel Content APIの実装を示しています。
オプションサンプルアプリ このアプリは、 Options.Builder ネットワークタイムアウトを設定するためのクラスメソッド（ ネットワークタイムアウトについて), キューポイントマーカー, AndroidアセットのTV評価を有効にするそして、VisualOn設定を使って作業します。 AndroidアプリのデフォルトのVisualOnライブラリを上書きする).
Secure Playerサンプルアプリ このアプリは、HLS Secure Playerの使用方法を示しています。 このアプリを実行するには、まずあなたのOoyala担当者が提供するHLSデコードライブラリを正しくインストールする必要があります。 提供されているライブラリには、IDEエミュレータでアプリをテストする前に実行する必要があるシェルスクリプトが含まれています。 このスクリプトはLinuxまたはMac端末で実行するように設計されています（ Cygwin Windowsの場合）。
注意： このアプリはIDEエミュレータでは実行できません。 このアプリをテストするには、接続されているデバイスを使用するようにIDEを設定します。
VisualOnサンプルアプリ このアプリは、ライブストリーミングとVODのためのVisualOn固有のライブラリとSecure Playerの使用方法を示しています。 このアプリを実行するには、まずあなたのOoyala担当者が提供するVisualOn HLSデコードライブラリを正しくインストールする必要があります。 提供されているライブラリには、IDEエミュレータでアプリをテストする前に実行する必要があるシェルスクリプトが含まれています。 このスクリプトはLinuxまたはMac端末で実行するように設計されています（ Cygwin Windowsの場合）。 詳細については VisualOnの統合について.
注意： このアプリはIDEエミュレータでは実行できません。 このアプリをテストするには、接続されているデバイスを使用するようにIDEを設定します。
