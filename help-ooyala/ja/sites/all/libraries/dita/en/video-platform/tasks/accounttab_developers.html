APIキーと秘密

バックスロットUI ユーザーアカウントに割り当てられているAPIキーとシークレットを表示します。 これらの認証情報は、HTTPリクエストに署名したり、 スクラッチパッド。 詳細については、を参照してください。 あなたのAPIクレデンシャル.

API ping URL

APIのアップロード後にファイルのトランスコードが完了したときにGET要求を受け入れるサービスのURLを指定できます。 GETリクエストには、embed_code（アセットに割り当てられたID）と関連情報が含まれています。 複数のURLをコンマ区切りリストで指定できます。 詳しくは、 API ping URL.
