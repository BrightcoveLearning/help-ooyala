CDNキャッシング層は、各応答のコピーを約5分間保持します。 要求を行うとき、CDNは将来的に同じキャッシュ結果を返すためにあなたの正確な要求をキャッシュします。 要求の一部を変更すると、CDNはそれを新しい要求として扱います。 High Performance APIから最高のパフォーマンスを引き出すには、request expiresパラメータが "cache friendly"であることを確認してください。 つまり、短すぎません。 次のRubyスニペットでは、要求は常に明日の初めに期限切れになります。 

 今日= Date.todayの有効期限= Time.mktime（today.year、today.month、today.day）+ 86400

埋め込みのFlashまたはiOSアプリケーションで使用するURLを生成している場合は、将来有効期限が切れるURLを生成できます。
