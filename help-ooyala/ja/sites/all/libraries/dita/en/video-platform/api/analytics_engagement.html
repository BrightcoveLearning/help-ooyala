エンゲージメント（非推奨）

エンゲージメントレポートには、ビデオの再生中の視聴者数が表示されます。これにより、視聴者が興味を失うポイントを特定し、広告掲載の効果を見つけ、全体的な効果を評価できます。

警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、

エンゲージメント分析（動画）

[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/合計/日付範囲

Engagement Analytics（Geo）を入手する

[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/国[/国]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/地域/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/都市/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/国/国/地域[/地域]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/国/国/地域/地域/都市[/シティ]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/ dmas [/DMA]/日付範囲

Engagement Analytics（ドメインとURL）を入手する

[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/ドメイン[/ドメイン]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/ドメイン/ドメイン/ urls [/URL]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/婚約/のURL [/URL]/日付範囲

エンゲージメントアナリティクスを入手する（プラットフォームとデバイス）

[GET] / v2 / analytics / reports / asset /asset_id/ Engagement / device_types [/デバイスタイプ]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/ Engagement / device_types /デバイスタイプ/プラットフォーム[/プラットフォーム]/日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/エンゲージメント/プラットフォーム[/プラットフォーム]/日付範囲

segment_watched 配列は各2 1 / 2％セグメントで何人の視聴者がビデオを見ていたかを表します。 各セグメントバケットは、コンテンツの2 1 / 2％用です。

percentage_watched 配列は、指定された割合のビデオを視聴した視聴者数を表します。 この例では、305はビデオの0 - 2 1 / 2％を視聴し、67ビューアは97 1 / 2 - 100％を視聴しました。 の percentage_watched 結果は線形ではありません。 ビデオの50％を視聴したユーザーは、前半、後半、25％ポイントから75％ポイントなどを視聴した可能性があります。
