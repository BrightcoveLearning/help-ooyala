デフォルトのカウントダウンを設定する

ディスカバリーの推奨事項は、推奨動画のシーケンスとして提供されます。 ビデオが再生された後、 次に 次のビデオのタイトルと共にプレーヤーに表示されます。 プロファイル内のこの設定は、2つのビデオ間の時間を指定します。

注意： 次に 広告再生中は表示されません。

次の要求と応答の例は、使用方法を示しています。 PATCH 方法と default_countdown デフォルトのカウントダウンを10秒に設定するプロパティ。

要求：

[PATCH] / v2 / players / {player_id} / discovery / profile {"discovery_interface"：{"default_countdown"： "10"}

応答：

{"discovery_interface"：{
"default_countdown"： "10"、 "show_pause_screen"： "false"、 "hide_thumbnail_title"： "false"、 "when_dosne"： ""}、...
