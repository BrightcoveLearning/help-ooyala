Player V3埋め込みパラメータ（廃止予定）

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
埋め込みパラメータを OO.Player.create（） 方法。 これらのパラメータは次のとおりです。
  • CSSスタイルの設定（ 幅 そして 高さ)
  • サポートされているFlash属性（ Wモード)
  • その他のパラメータ（広告サーバーや広告ネットワークアカウントのタグ（高度な広告の追跡とターゲティングに使用されるタグなど））
次の表は、キーと値のペアとして表されるこれらの埋め込みパラメータのいくつかを説明しています。 OO.Player.create（） 方法。 特に明記しない限り、この表のパラメータはHTML5プレーヤーとFlashプレーヤーの両方に使用できます。

自動再生

ロード時にアセット（ビデオまたはオーディオ）の自動再生を有効にします。 これは、再生/一時停止コントロールを持たないUIや、コンテンツをすぐに再生する条件がある場合に便利です。

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、自動再生：true、....}）;
enableChannels チャンネルモードでFlashビデオの読み込みを有効にします。 このパラメータは、チャネルに対して下位互換性を提供します。 デフォルトでは、フラグはfalseに設定されています。 embedChannelsパラメータはHTML5の互換性を処理し、Flashチャンネルの使いやすさを保証します。

HTMLの場合:

  • 設定する enableChannels：true チャンネルの最初のビデオまたはチャンネルセットの最初のチャンネルの最初のビデオを再生する
  • 設定する enableChannels：false エラー画面を表示します。

フラッシュの場合：

  • 設定する enableChannels：true チャンネルをプレーヤーに渡して通常通りに実行します（これはパフォーマンスに影響を与える可能性があります）。
  • 設定する enableChannels：false エラー画面を表示します。
例: 
var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、enableChannels：true、....}）;
initialTime このパラメータを使用して、特定の時点でコンテンツの再生を開始する初期時間を秒単位で設定します。 このパラメータは、iOSベースのデバイスの検索を有効にするために使用できます。 このパラメーターをCross-Device Resumeと一緒に使用して、特定のユーザーの位置を探すこともできます。

タイプ整数

有効な値秒単位の時間

例：initialTime：10

OO.ready（function（）{window.pp = OO.Player.create（ 'playerV3Container'、 'embed_code'、{幅：480、高さ：360、initialTime：10}）;
インストリーム このパラメータを使用して、In-Stream広告サーバーまたはネットワークタグをOoyalaプレーヤーに渡します。 このパラメータは、In-Stream URLを表すキーと値のペアを取ります。

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、 'インストリーム'：{tagUrl： 'some url'、....}、}） ;
ローカル

このパラメータを使用して、言語とクローズドキャプションのオプションを設定または上書きします。 次の例では、ロケールパラメータを使用してヘブライ語のローカライズプレーヤーを作成しています。 iOSではクローズドキャプションがサポートされているため、リモートアセット用のIOS Webにクローズドキャプションデータを追加することはできません。

例: 
var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、ロケール： 'he_IL'、....}）;
ループ このパラメータを使用して連続再生を有効にします。 loopをtrueに設定すると、再生が開始されると、ユーザーが再生を停止するかブラウザを閉じるまで再生が継続されます。 setEmbedCodeを使用して埋め込みコードを設定した場合も、動作は同じです。 自動再生がtrueの場合、埋め込みコードが設定されるとすぐに、プレーヤの以前の状態（ビデオの再生中/一時停止中/停止中）に関係なく再生がすぐに開始されます。 自動再生およびループのパラメータがsetEmbedCodeを介して渡されない場合は、既存の値が使用されます（以前のsetEmbedCodeの呼び出しで設定された可能性があります）。

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、ループ：true、....}）;
onCreate このパラメータを使用して、イベントメッセージを監視し、アクションを実行します。 このパラメータを使用すると、プレーヤが完全に作成される前にメッセージバスからのイベントメッセージを購読することができます。 それはあなたがその完全な作成の前にプレイヤーを修正することを可能にします。

呼び出されると、 onCreate：function（プレーヤー）：

  • 追加モジュール（カスタム、3rdパーティー、またはその他のタイプ）がないか調べます。

  • これらの追加モジュールがメッセージバスに接続できるようにします。

  • 各モジュールに起動を通知するメッセージをメッセージバスに送信します。

あなたが呼び出す必要があります onCreate 何かが起こる前に。 それ以外の場合は、既存のモジュール、追加のモジュール、または他社製のモジュールはメッセージバスに接続されず、初期化されません。

例

    OO.Player.create（ 'playerwrapper'、embedCode、{onCreate：function（player）{player.subscribe（ '*'、 'myPage'、function（eventName）{}）;}}）;

onCreateとPlayerのV2コールバック

Player V2でイベントを処理するには、コールバック関数を定義してから、コールバックパラメータを使用してその名前をembedタグに渡します。 ただし、Player V3でイベントを処理するには、 onCreate に機能 OO.Player.create（） を呼び出してから、すべてのメッセージを登録します。 Player V3イベントの詳細については、 HTML5 Player V3のイベントモデル（非推奨）。 Player V2コールバックとパラメータについては、 プレーヤクエリ文字列パラメータ（プレーヤV2は廃止予定） トピック。 Player V2は推奨されていません。

プリロード このパラメータを使用して、ビデオのプリロードを無効にします。これはデフォルトで行われます。 さまざまな理由でロードを無効にしたい場合があります。
  • 同じページに複数の動画を読み込むときに読み込み時間を短縮する
  • 動画の再生頻度が低いと予想される場合の読み込みコストを回避するため

例:

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、プリロード：false、....}）;
showAdMarquee
注意： このパラメータはFlash専用です。

広告再生中に広告マーキーを表示するか非表示にするかを指定します。 showAdMarqueeがないかtrueに設定されている場合、広告がプレーヤで再生されている間に広告マーキーが表示されます。 このグローバルパラメータは、Ooyalaでサポートされているすべての広告タイプで機能します。

有効な値： 本当| 偽
  • True：広告の再生中にadMarqueeが表示されます。
  • False：広告の再生中にadMarqueeは非表示になります。

デフォルト：本当

例： 
var player = OO.Player.create（ 'playerwrapper'、embedCode、{showAdMarquee：true}）;
showInAdControlBar

広告の再生中にコントロールバーを表示するかどうかを指定します。 showInAdControlBarがないかtrueに設定されている場合、広告がプレーヤで再生されている間にコントロールバーが表示されます。 このグローバルパラメータは、Ooyalaでサポートされているすべての広告タイプで機能します。

注意： このパラメータはFlashおよびHTML5デスクトップで利用可能です。

有効な値： 本当| 偽
  • True：広告の再生中にcontrolBarが表示されます。
  • False：広告の再生中にcontrolBarは非表示になります。

デフォルト：本当

例： 
var player = OO.Player.create（ 'playerwrapper'、embedCode、{showInAdControlBar：false}）;

注意： このパラメータはFlashデスクトップ専用です。

このオプションのパラメータは、TVレーティング透かしが表示されるプレーヤー上の位置を指定します。

有効な値左上| 右上| 左下| 右下

デフォルト： 左上

例： 
var player = OO.Player.create（ 'playerwrapper'、embedCode、{tvRatingsPosition： "左上"}）;
tvRatingsTimer

注意： このパラメータはFlashデスクトップ専用です。

最初の再生時、および広告が中断されてから動画が再開されるたびに、プレーヤーのTVレーティングの透かしの長さを秒単位で指定します。 TVレーティング透かしをアセットに適用する場合は、このパラメータは必須です。

有効な値：いつも| 決して some_number
  • always：透かしはビデオの間表示されます（広告の再生中は表示されません）。
  • never：透かしは表示されません。
  • some_number：some_numberは、テレビのレーティングをプレーヤーに表示し続ける秒数に置き換えます。

デフォルト： 決して

注意： TVレーティング透かしをアセットに適用するには、TVレーティング透かしを指定したいアセットごとにBacklotにカスタムメタデータを含める必要があります。 見る Flash Playerへのテレビ格付け透かしの追加 TVレーティング透かしをアセットに適用する方法の詳細については、

例： 
var player = OO.Player.create（ 'playerwrapper'、embedCode、{tvRatingsTimer： "15"}）;

var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ：100％、幅：100％、useFirstVideoFromPlaylist：false、....}; ooyalaPlayer = OO.Player.create（ 'playerContainer'、 '埋め込みコード'、playerConfiguration）; ;））;
