視聴者にあなたのアプリを許可させる

動画を視聴者のタイムラインに表示するには、各ユーザーがアプリケーションを承認する必要があります。

The following describes the user login process:

注意：Each viewer only has to do this once.

  1. The customer accesses your site.
  2. You provide a button that the user clicks for the connected Facebook experience or you block access unless the customer logs in.
  3. The Facebook Login page appears.
  4. The user clicks Log In with Facebook. Video play information is recorded to the user’s timeline.
