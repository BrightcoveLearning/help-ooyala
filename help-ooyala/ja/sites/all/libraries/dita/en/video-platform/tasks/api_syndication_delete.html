シンジケーションを削除する

シンジケーションはいつでも削除できます。 新しいコンテンツはシンジケーションに送信されず、フィードは削除されます。

シンジケーションを削除するには

deleteを使って シンジケーション ルート。 次の例では、 ZnbWVpMjqToT1W シンジケーション 
[削除] / v2 /シンジケーション/ ZnbWVpMjqToT1W

バックスロット 200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

シンジケーションは正常に削除されました。
