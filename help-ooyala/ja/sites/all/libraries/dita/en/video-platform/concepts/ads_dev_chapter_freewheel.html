Player V3フリーホイール広告の統合

The following documentation describes the FreeWheel integration with Player V3. For documentation on the FreeWheel integration with Player V4, see フリーホイール広告プラグイン.

Freewheel Ads

FreeWheel is an ad server and ad management platform. Using the FreeWheel ad manager, publishers are able to segment their ad inventory, define ad targeting rules, and take advantage of the FreeWheel MRM network capabilities.

Supported Ad Positions - Flash Player V3

Ads are supported on V3 Flash Players and the mobile SDKs for iOS and Android for the following ad positions:
  • プリロール
  • ミッドロール
  • ポストロール
  • クリックスルー

Supported Ad Positions - HTML5 Player V3

Ads are supported on V3 HTML5 players for the following ad positions:
  • プリロール
  • ポストロール
  • クリックスルー

広告とFirefoxのプライベートブラウジング

For consumers using プライベートブラウジングウィンドウ in Firefox, some ads may fail to load because privacy mode blocks certain kinds of tracking URLs.
