プレーヤV4メッセージバスイベント

メッセージバスイベントを使用して、動画から広告の再生までのプレーヤイベントを購読または発行します。

すべてのOoyala APIは、 Ooyala APIドキュメント.

Player V4 JavaScript API、およびプレーヤイベントは、次のURLで表示されます。 プレーヤJavaScript APIリファレンス.

プレーヤーイベントは、イベントバスによって発行される既定のイベントです。 あなたのモジュールはこれらのイベントのどれでもそして全てを購読することができます。 メッセージバスによって発行されるこれらすべてのデフォルトイベントのリストを取得するには、メッセージバスを使用できます。 申し込む（） 関数： 
player.mb.subscribe（ "*"、 'example'、function（eventName）{}）;
