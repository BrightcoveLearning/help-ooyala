AndroidアセットのTV評価を有効にする

Use the following steps to enable TV ratings for an Android video.
  1. Backlot UIで適切なカスタムメタデータを使ってムービーを設定します。 このステップについては、以下のトピックを参照してください。
    • TVの評価バックスロットメタデータ設定
    • カスタムメタデータの追加
  2. 作る FCCTVRatingConfiguration FCC評価の表示方法を設定するパラメータの概要を説明するオブジェクト。
  3. 作る オプション 設定時に使用するオブジェクト OoyalaPlayer.
  4. を初期化する OoyalaPlayerViewController そして追加 オプション パラメータとしてのオブジェクト
  5. 使用 setEmbedCode method and play the asset to see the TV ratings appear as configured in your FCCTVRatingsConfiguration オブジェクト。

次のコード例は、上記の手順の結果です。

FCCTVRatingConfiguration.Builder builder = new
  FCCTVRatingConfiguration.Builder();
  FCCTVRatingConfiguration tvRatingConfiguration =
    builder.setPosition(Position.TopLeft).setDurationSeconds(5)
    .build();
  Options options = new      
  Options.Builder().setTVRatingConfiguration
    (tvRatingConfiguration).build();
  OoyalaPlayerLayoutController playerLayoutController = new
  OoyalaPlayerLayoutController( playerLayout, PCODE, new
    PlayerDomain(DOMAIN), DefaultControlStyle.AUTO, null, options);
  playerLayoutController.setEmbedCode
    ("5od253bzo-9DTMTY5q45pX-PRRXa5c4d");
