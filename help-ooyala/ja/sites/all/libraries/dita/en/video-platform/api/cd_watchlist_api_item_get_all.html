ウォッチリストからすべてのアイテムを入手する

このエンドポイントは、指定されたウォッチリストに属するすべてのアイテムを取得します。

HTTPリクエスト

GET https://player.ooyala.com/watchlist/v3/<watchlist_id>/items?page_number=<123>&page_size=<10>&account_token= <アカウントカウント>

クエリパラメータ

パラメーター 説明 タイプ 必須
api_version でなければなりません v3. 文字列 はい（URL）
watchlist_id 取得するアイテムを含むウォッチリストのID。 ユーザーのアカウントに関連付けられている既存のウォッチリストにリンクされている有効なUUIDである必要があります。 指定しない場合、アイテムはユーザーのデフォルトのウォッチリストから取得されます。 UUIDv4または文字列 いいえ（URL）
ページ番号 どのページのアイテムを返すかを指定します。 0以上の整数値でなければなりません。
注意： ページは、インデックス0から始まる番号が付けられています。
 文字列 はい（URL）
ページサイズ 1ページあたりの項目数を指定します。 指定した場合は、1以上の整数値でなければなりません。 指定しない場合、デフォルトのページサイズは10アイテムです。 文字列 いいえ（URL）
account_token 特定のエンドユーザに関連付けられているウォッチリストを参照するためにウォッチリストサービスによって使用されます（UUIDによって識別されます）。 詳しくは、 OoyalaアカウントトークンAPI. 文字列 はい（URL本体）

返されたメタデータ

Ooyalaがサポートするメタデータストアを使用している場合は、次のメタデータが返されます。
注意： ウォッチリストサービスは、アセットメタデータストアとの同期を自動的に維持する責任を負いません。 メタデータストアでアセットが削除または変更された場合、これによって自動的に（ウォッチリストの範囲内で）このアセットへのすべての参照が削除または変更されることはありません。
フィールド 説明
位置 リスト内の項目の配置
created_at リストが作成された時刻。
updated_at リストが最後に更新された日時。
リーディングシート それぞれのリスト asset_id リストに含まれています。
メタデータ 資産に関するすべてのメタデータ。
注意： 使用しているサポートされているアセットメタデータストアによっては、追加のメタデータが返されることがあります。

リターンコード

コード 理由 説明
200 OK GETリクエストは成功しました。
400 リクエストエラー 次の問題が原因で失敗しました。
  • パラメーターが欠落しているかフォーマットが正しくありません。
  • 指定された watchlist_id 空ではなく、存在しません。
401 無断 指定された account_token 無効です。
404 見つかりませんでした サーバーが停止しています。
500 内部サーバーエラー 内部サーバーエラーが発生しました。
