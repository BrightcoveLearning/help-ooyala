ラッパーSDK / APIライブラリ

さまざまな言語用のSDKがいくつかあります。

Ooyalaは、Backlot RESTベースAPI用の使いやすい "ラッパー"ソフトウェア開発キット（SDK）をさまざまなプログラミング言語で提供しています。 たとえば、新しいリクエストを作成して署名するには、次のように簡単です（Java）。

OoyalaAPI api = new OoyalaAPI（ "apikey"、 "秘密鍵"）;
ラッパーSDKは、GitHubから次の言語で入手できます。
  • Java
  • PHP
  • Python
  • ルビー
  • C#
  • 目標C

さらに、（ラッパーSDKを除く）多数のプログラミング言語でシグニチャを計算し、リクエストに署名するためのサンプルコードも利用できます。 見る 署名要求のサンプルコード.
