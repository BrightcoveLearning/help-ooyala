Ooyalaプレーヤーの言語設定

注意： これは、Ooyala Web Playerスキンのみの言語を設定します。 デフォルトの言語トラックのプレーヤー設定もあります（ビデオ素材の マルチオーディオ 言語トラック）とデフォルト 字幕 言語。

Player V4（HTML5）とAndroidおよびiOSのプレーヤスキンをローカライズできます。 デフォルト skin.json Githubの例に含まれるファイルは、プレーヤーのデフォルトのローカライズファイルを参照します。

"ローカライゼーション"：{"defaultLanguage"： "en"、 "availableLanguageFile"：[{"言語"： "en"、 "languageFile"： ""、 "androidResource"： "skin-config / languageFiles / en.json"、 "iosResource"： "en"}、{"言語"： "es"、 "languageFile"： ""、 "androidResource"： "skin-config / languageFiles / es.json"、 "iosResource"： "es"}、 {"言語"： "zh"、 "言語ファイル"： ""、 "androidResource"： "スキン設定/言語ファイル/ zh.json"、 "iosResource"： "zh"} {"言語"： "ja"、 " languageFile "：" "、" androidResource "：" skin-config / languageFiles / ja.json "、" iosResource "：" ja "}]

デフォルト言語の変更

言語設定を変更するには、skin.jsonファイルの言語の値を必要な値に上書きします。 作成したローカライズファイルをホストする必要があります。

目的の言語ファイルが自己ホスト型の場合は、上書きでパスの場所を参照してください。

例（Webのみ）：

{"スキン"：{"インライン"：{"ローカライズ"：{"defaultLanguage"： "de"、 "availableLanguageFile"：[{"言語"： "de"、 "languageFile"： "https://example.com /de.json "、//カスタム言語ファイルがホストされているURL}]}}}}

ブラウザの言語設定に基づいてプレーヤーの言語を動的に設定する

これを実現するには、新しいブール値を使用します。 useBrowserLanguage ページレベルパラメータ - 追加したとき useBrowserLanguage：true を skin.json 設定を変更すると、デフォルト設定が上書きされ、現在のブラウザの言語設定に基づいてプレーヤの言語が設定されます。
