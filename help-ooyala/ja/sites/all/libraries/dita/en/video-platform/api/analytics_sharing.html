共有（非推奨）

共有レポートには、アカウント内のすべてのコンテンツまたは個々のアセットに対する共有操作（Facebook、Twitterなど）の数が表示されます。

警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、

ビデオ分析を入手

[GET] / v2 / analytics / reports /アカウント/共有/ビデオ/日付範囲   
[GET] / v2 / analytics / reports /アカウント/共有/ビデオ/asset_id/日付範囲     
[GET] / v2 / analytics / reports / asset /asset_id/共有/合計/日付範囲

ジオアナリティクスを入手 

[GET] / v2 / analytics / reports /レベル/共有/国/ [国]/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/地域/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/都市/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/国/国/地域/ [地域]/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/国/国/地域/地域/都市/ [シティ]/日付範囲
[GET] / v2 / analytics / reports /レベル/ sharing / dmas / [DMA]/日付範囲

ドメインとURLを取得する

[GET] / v2 / analytics / reports /レベル/ sharing / domains / [ドメイン]/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/ドメイン/ドメイン/ urls [/URL]/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/ urls [/URL]/日付範囲

プラットフォームとデバイスの分析を取得する 

[GET] / v2 / analytics / reports /レベル/ sharing / device_types / [デバイスタイプ]/日付範囲
[GET] / v2 / analytics / reports /レベル/ sharing / device_types /デバイスタイプ/ platform / [プラットフォーム]/日付範囲
[GET] / v2 / analytics / reports /レベル/共有/プラットフォーム/ [プラットフォーム]/日付範囲
