クローズドキャプションファイルの削除

クローズドキャプションファイルはいつでも削除できます。

クローズドキャプションファイルを削除するには：

DELETEを使う / v2 / assets 、資産ID、および / closed_captions 修飾子 次の例では、からクローズドキャプションを削除します。 しぶき 資産： 
[削除] / v2 / assets / JxbzdkMjqB / closed_captions

バックスロット 200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
