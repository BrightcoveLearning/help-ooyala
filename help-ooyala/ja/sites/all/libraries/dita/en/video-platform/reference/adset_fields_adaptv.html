Ad Set Fields for Adap.tv

ここで説明するフィールドは、広告ソースがAdap.tvに設定されている広告セット用です。

注意： フィールドの正確な意味については、常に広告ソースのドキュメントを確認してください。
フィールド名 説明
発行者キー Adap.tvサイト運営者ID
キーワード 動画を識別するキーワードのカンマ区切りリスト
カテゴリー 「スポーツ」などのビデオを識別するカテゴリのカンマ区切りのリスト。
ゾーン ゾーンを使用してサイトをセグメント化し、特定の地域をターゲットにすることができます。 ゾーンIDとも呼ばれ、Adap.tvプラットフォームのコンソール設定で（zid）セットアップされます。
コンテキスト 広告システムに渡されるデータ（通常はエンコードされている）
コンパニオンID コンパニオン広告を表示するためのHTMLページの<DIV>タグのID
トラッキングピクセルURL 広告資産の配信を監視するためにピクセルを追跡するためのURL
親トピック 広告ソースと統合するための広告セットの作成
関連タスク
独自の広告資産、広告セット、および広告ソースを操作する
