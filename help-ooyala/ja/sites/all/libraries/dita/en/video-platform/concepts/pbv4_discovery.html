Player V4でコンテンツを発見する

ディスカバリーを使用すると、サイト上の関連性のある関連コンテンツにユーザーを誘導することで、ユーザーの関心を引くことができます。 ディスカバリーは、ビデオの印象や再生から継続的に学習して、動的でパーソナライズされた体験をユーザーに提供します。
ディスカバリーを使用すると、以下のことをお勧めできます。
  • 現在の時間、日、週、または月のコンテンツの人気または速度に基づいた動画。
  • 選択した動画に似た利用パターンの動画。

アカウントの検出を有効にする

Ooyala Backlotアカウントで検出を明示的に有効にする必要があります。 まだ有効になっていない場合は、Ooyalaアカウント担当者に連絡して、サイトの検出機能を有効にしてください。

検出の設定

You configure discovery settings using the Discover tab in Backlot. For instructions, see バックスロットUIでの検出の設定.

Adding the Discovery Plugin to an HTML Page

注意： DiscoveryとPlaylists（ThemeBuilderを使用して作成された）はそれぞれ画面の動作を終了させるので、それらを同時に使用することはできません。 Webページにプレーヤーを埋め込むときは、次のものを使用できます。 どちら ディスカバリーまたはプレイリスト。ただし両方ではありません。
To use Discovery with Player V4:
  • If you use Backlot to generate your HTML embed code, and you choose the V4 HTML5標準プレーヤー埋め込みコード（推奨） option, the Discovery plugin is automatically included and should not be explicitly added to the web page where you launch the player. For details, see バックスロットでのプレーヤー埋め込み設定.
  • あなたが V4 HTML5プレーヤー埋め込みコード（詳細） option in Backlot, or if you manually create the HTML embed code yourself, you need to add the Discovery plugin (discovery_api.min.js) to your web page so that the Discovery plugin loads before the player is created.
Discovery functionality requires the playerBrandingId you specify on the page.

Customizing the Discovery User Interface

In addition to using Backlot (see バックスロットUIでの検出の設定), you can customize the Discovery interface by modifying settings in the skin.json config file or in CSS properties.

Discovery settings in skin.json

You can change Discovery interface settings in skin.json:
  • To customize visual elements in the Discovery screen (such as the title), change settings under discoveryScreen.
  • To display the Discovery screen when playback ends, under the endScreen settings, specify "screenToShowOnEnd": "discovery".
  • To display the Discovery screen when playback pauses, under the pauseScreen settings, specify "screenToShowOnPause": "discovery".
  • To configure Discovery buttons, under ボタン、 変更 発見 button settings for desktopContent そして desktopContent.
  • To configure the Discovery icon, under アイコン、 変更 発見 icon settings.
詳細については、を参照してください。 skin.jsonを使用したPlayer V4スキンのカスタマイズ.

Discovery settings in CSS

You can customize the Discovery interface by modifying settings under the discoveryScreen element. See CSSを使用したプレーヤV4外観のカスタマイズ 手順を参照してください。

Customizing Video Recommendations

You can change settings that Ooyala's video recommendations engine uses to generate recommendations for consumers. See バックスロットUIでの検出の設定 手順を参照してください。

Discovery and Autoplay

  • Beginning with Player V4 version 4.12.6, Discovery defaults to autoplay, even when the main video is not set to autoplay on the page. To disable the automatic playing of Up Next or Discovery videos after the main video has played, set playerParams.autoPlay to false and specify playerParams.autoPlayUpNextVideosOnly as false. See プレーヤV4のページレベルパラメータ.
  • For Player V4 version 4.11.14 and earlier, Discovery defaulted to autoplay only if the embedded player was set to autoplay.
