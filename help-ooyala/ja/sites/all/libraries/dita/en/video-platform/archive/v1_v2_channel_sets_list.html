バックスロットAPI v1

http://www.ooyala.com/partner/channel_sets
?pcode=<pcode>
&channelSetEmbedCode=<channelSetEmbedCode>
&expires=<expiration>
&mode=list
&signature=<signature>

Backlot API v2：チャンネルセット情報の取得

[GET] https://api.ooyala.com/v2/assets/ <channelSetId>

Backlot API v2：チャンネルセットラインナップ情報を入手する

[GET] https://api.ooyala.com/v2/assets/<channelSetId>/lineup

例

次の例では、チャンネルセットに関する情報を取得します。

[GET] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v

バックスロット 次のような応答を返します。

{"asset_type"： "channel_set"、 "期間"：0、 "名前"： "マイチャンネルセット"、 "preview_image_url"：null、 "created_at"： "2011-08-10T17：45：24：00" 、 "embed_code"： "dvNTVxMjrRktSlb00v"、 "time_restrictions"：null、 "updated_at"： "5-2011-08 + 10 + 17：45"、 "ホスト名"、 "null"、 "ファイル名"、null ：null、 "説明"：null、 "ステータス"： "ライブ"}

次の例では、指定されたチャンネルセットのラインナップを取得します。

[GET] https://api.ooyala.com/v2/assets/dvNTVxMjrRktSlb5v / lineup

バックスロット 次のような応答を返します。

["IzNnllMjphu2XF3"、 "Y1dTdvMjq9QtOMG"]
