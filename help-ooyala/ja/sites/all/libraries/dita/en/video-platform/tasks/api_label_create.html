ラベルを作成する

とともに バックスロットAPI, you can create top-level and child labels.

ラベルを作成するには：

  1. Create a top-level label. The following example creates a "Hobbies" label. 
    [POST]/v2/labels{  
   "name":"Hobbies"
}

    バックスロット 次のような応答を返します。

    {
   "name": "Hobbies",
   "id": "Yh98G7hsaß",
   "full_name": "/Hobbies"
}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. Create a child label. The following example creates a "Hockey" label under the "Hobbies" parent label by specifying the ID of Hobbies label. 
    [POST] /v2/labels{
   "parent_id":"Yh98G7hsaß",
   "name":"Hockey"
}

    バックスロット 次のような応答を返します。

    {
   "name": "Hockey",
   "id": "Yh98G7hsaß",
   "full_name": "/Hobbies/Hockey",
   "parent_id": "Hwe8G7hrt5"
}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
