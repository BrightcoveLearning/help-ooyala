プレイリストを置き換える

APIリクエストで指定した新しいコンテンツで既存のプレイリストを置き換えることができます。

注意： The new playlist maintains playlist_id of the old playlist.

To replace a playlist:

1. Perform a のGET オン プレイリスト playlists route to view the existing playlist. The following example retrieves an existing playlist from a given playlist_id (in this example, playlist_id=625622dfd22c47e9b7dcd0e8d1843638):

[GET]/v2/playlists/625622dfd22c47e9b7dcd0e8d1843638

Backlotは次のような応答を返します。

{  
   "sort_by":"custom",
   "limit":null,
   "type":"movie",
   "id":"625622dfd22c47e9b7dcd0e8d1843638",
   "items":[  

   ],
   "created_at":"2013-07-10T18:24:32Z",
   "name":"New Movie Playlist",
   "updated_at":"2013-07-10T18:35:17Z",
   "provider_id":"61670"
}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

2。 使用 PUT とともに プレイリスト route to replace the content in an existing playlist with new information. The following example replaces an existing playlist with new content.

[PUT]/v2/playlists/625622dfd22c47e9b7dcd0e8d1843638{  
   "name":"Super New Movie Playlist",
   "type":"movie",
   "items":[  
      "M0YWNmODqougN5VrA1FIk02F3CH7Mq_d",
      "81YWNmODqT-3NfNatUGgJq9_XAR6blq8"
   ]
}

Backlotは次のような応答を返します。

{  
   "name":"Super New Movie Playlist",
   "updated_at":"2013-07-11T18:24:32Z",
   "limit":"null", 
   "id":"625622dfd22c47e9b7dcd0e8d1843638",
   "items":[  
      "M0YWNmODqougN5VrA1FIk02F3CH7Mq_d",
      "81YWNmODqT-3NfNatUGgJq9_XAR6blq8"
   ],
   "type":"movie",
   "created_at":"2013-07-10T18:24:32Z",
   "provider_id":"12345",
   "sort_by":"custom"
}
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
