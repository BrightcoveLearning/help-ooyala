Ooyala Player用のTwitter <meta>タグの手動取得

TwitterでTwitterカードを正しく設定するには、各ビデオに関するメタデータを提供する必要があります。 この情報はHTMLの形式です <meta> タグ このセクションでは、これらを手動で生成する方法について説明します。 <meta> 特定の動画のタグ。 あなたはこのプロセスを自動化したいでしょう。 FlashコンテンツはTwitterではサポートされていません。TwitterではHTML5コンテンツが必要です。

サンプルコードをWebページに追加するには

  1. にログインします。 バックスロットUI. バックスロットUI 開きます。
  2. Video Cloud Studioで 管理 のページをご覧ください。
  3. ビデオを選択します。
  4. ストリーミングを停止する場合は上部の 埋め込む のページをご覧ください。
  5. コンテンツIDを書き留めます。
  6. In your web browser, make the following request:  
    http://player.ooyala.com/twitter/meta/player_id/content_id

    where player_id プレーヤID content_id バックスロットのコンテンツID 埋め込む この動画のタブです。

    空白のページが表示されます。

     
  7. この空白ページのHTMLソースを表示します。 ほとんどのブラウザでは、ページを右クリックして ページソースを表示する. 生成された <meta> タグが表示されます。 例えば： 
    <meta name = "twitter：カード" content = "player"> <meta name = "twitter：title" content = "123 MarvelのThe Avengers.mov" /> <meta name = "twitter：説明" content = "video" > <meta name = "twitter：image" content = "http://ak.c.ooyala.com/B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt/Ut_HKthath5eww8X4xMDoxOjBhOzV3Va" tw> "meta =" twitter：imagehttps://player.ooyala.com/tframe.html?ec=B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt&pbid=MzZiMzc1ZDUzZGVlYmMxNzA3Y2MzNjBk"> <meta name =" twitter：player：width "content =" 435 "> <meta name =" twitter：player：height "content =" 251 ">

    注意： 広告タグなど、プレーヤ用の高度なオプションがある場合は、上記で強調表示されている基本URLを置き換える必要があります。 見る Twitterカードの高度なプレーヤーオプション.

  8. 返品をコピー <meta> テキストエディタにタグを付けます。
  9. の最後に以下を追加してください <meta> タグ： 
    <meta name = "twitter：site" content = "@ username">

    where ユーザ名 あなたのTwitter IDです。

  10. テストするには <meta> タグは、次のURLを開きます。 https://dev.twitter.com/docs/cards/validation/validator. [Twitterカードのプレビュー]ページが表示されます。
  11. メタをTwitterに入力 標準タグ フィールドとクリック プレビューカード. 

    Twitterがプレビューを返します。

  12. 次の手順に進んでください。
