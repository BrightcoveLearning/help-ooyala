Query String Parameters in Player V3 (Deprecated)

Modify the player with query string parameters.

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
You add these types of query parameters to the http://player.ooyala.com/v3/.. .. <script> tag. These query string parameters include the following:
パラメーター 説明
名前空間=(uniqueJavaScriptID) 使用 名前空間 parameter to place multiple independent copies of a player on a page. See details in Renaming the Default Namespace.
プラットフォーム=( flash-only | flash | html5-fallback | html5-priority ) 使用 プラットフォーム parameter to control the selection mechanism between Flash and HTML5 playback. See details in Player V3でパラメータを使用してデフォルトのビデオプラットフォームを設定する（廃止予定）.
微調整=android-enable-hls 使用 微調整 parameter to enable HLS on your Android devices for the web. Although you can use this parameter to enable HLS on Android devices, certain limitations exist for HLS playback on 2.x and 4.x devices. For more information about operability on Android devices, see "Enabling HLS on Android Web."
注意： As the use of the tweaks parameter can involve native playback issues, we cannot guarantee optimal playback experience on every Android device and OS version.

例

http://player.ooyala.com/v3/123fake6b6774f909c2fa6d23ce5334a?tweaks=android-enable-hls
注意： The legacy Player API had query string parameters for passing tags of ad server and ad networks to the Ooyala player. For Player V3, the supported ad servers/networks and the mechanisim for passing related tags is detailed in 広告用のPlayer V3埋め込みパラメータ（廃止予定）.
