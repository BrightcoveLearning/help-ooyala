分析フレームワーク

Analytics Frameworkは、HTML Web用の統一されたフレームワークです。 このフレームワークにより、分析プロバイダーはOoyalaのHTML5プレーヤー用の分析プラグインを開発することができます。 パートナーやパブリッシャーはこのフレームワークを拡張して、任意の分析ソースまたはプロバイダーとカスタム分析プラグインを簡単に構築できるため、Ooyala Player V4を他のサードパーティ分析プロバイダーと統合することができます。

Analyticsフレームワークを使用して構築されたプラグイン

  • OoyalaはAnalytics Frameworkを使用して構築しました プラグイン Adobe Analytics（Omniture）の機能をPlayer V4と統合する。
  • ニースの人々の仕事 Analytics Frameworkを使用して構築した プラグイン YOUBORA分析機能をPlayer V4と統合する。
  • コムスコア Analytics Frameworkを使用して構築した プラグイン comScore分析機能をPlayer V4と統合する。

オープンソースのHTML JavaScript Analytics Framework API

オープンソースのHTML5 JavaScript Analytics Framework APIを使用すると、サードパーティの分析プロバイダはPlayer V4にプラグインできます。 詳細については、 Githubのhtml5-analytics-plugins そして Analytics Framework API.
