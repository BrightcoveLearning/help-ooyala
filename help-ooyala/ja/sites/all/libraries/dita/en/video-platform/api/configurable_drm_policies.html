DRMポリシー

特定のDRMポリシーまたは自分のアカウントに関連付けられているすべてのポリシーを表示できます。

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

DRMポリシーの一覧表示

[GET] / v2 / drm_policies

DRMポリシーを表示する

[GET] / v2 / drm_policies /drm_policy_id

プロパティ

以下のプロパティーは、DRMポリシーに固有のIDと名前を与えます。

プロパティ 説明 必須？
id

ポリシーのID

タイプ：文字列

Example: "86ff97ae7c81495eacbd9a01feff0f22"

はい
名

DRMポリシーの名前 非ラテン文字はUTF-8でURIエンコードされている必要があります。

タイプ：文字列

例：「HDポリシー」

はい

ＤＲＭポリシー内に、１つまたは複数のＤＲＭ技術（ＰｌａｙＲｅａｄｙ、Ｗｉｄｅｖｉｎｅなど）に対して特定のポリシーを含めることができる。 次の表は、各タイプのDRMテクノロジのプロパティを示しています。

PlayReadyのプロパティ

以下のプロパティは、Microsoft PlayReadyポリシー定義に関連付けることができます。

プロパティ 説明 必須？
playready_policy

PlayReadyポリシーのプロパティを指定します。

タイプ：コンテナ

デフォルト：none

はい
analog_video

アナログビデオが保護される最小ビットレートを指定します。

タイプ：整数

例：150

デフォルト：none

親：playready_policy

 いいえ
analog_video_extension_guid

Macrovisionなど、保護されたアナログコンテンツの再生を許可されている拡張テクノロジを指定します。 拡張テクノロジを指定するGUIDと構成データ値は、カンマ区切りです。

タイプ：文字列

Example: C3FD11C6-F8B7-4d20-B008-1DB17D61F2DA,3

デフォルト：none

親：playready_policy

 いいえ
compressed_digital_video

圧縮デジタルビデオを保護する最小ビットレートを指定します。

タイプ：整数

例：500

デフォルト：none

親：playready_policy

 いいえ
有効期限

ライセンス期間（秒）。

タイプ：整数

例：120

デフォルト：60

親：playready_policy

 いいえ
first_play_expiration

クライアントが最初にコンテンツを再生してからライセンスが期限切れになるまでの秒数。

タイプ：整数

例：120

デフォルト：none

親：playready_policy

 いいえ
uncompressed_digital_video

非圧縮デジタルビデオを保護する最小ビットレートを指定します。

タイプ：整数

例：300

デフォルト：none

親：playready_policy

 いいえ

フラッシュアクセスプロパティ

次のプロパティは、Adobe Flash Accessポリシー定義に関連付けることができます。

プロパティ 説明 必須？
flashaccess_policy

Flash Accessポリシーのプロパティを指定します。

タイプ：コンテナ

デフォルト：none

はい
analog_output_protection

使用するアナログ出力保護の種類を指定します。

タイプ：文字列

有効な値は以下のとおりです。 no_protection | required_acp | required_all | required_cgmsa | use_if_available_acp | use_if_available_all | use_if_available_cgmsa

例：no_playback

デフォルト：none

親：flashaccess_policy

 いいえ
caching_duration

ライセンスがクライアントのデバイスにキャッシュされる期間を秒単位で指定します。 クライアントに有効期限が切れていないキャッシュされたライセンスがある場合は、Flash Accessから新しいライセンスを取得する代わりにそれが使用されます。

タイプ：整数

例：3600

デフォルト：ライセンスは無期限にキャッシュされます

親：flashaccess_policy

 いいえ
digital_output_protection

使用するデジタル出力保護の種類を指定します。

タイプ：文字列

有効な値は以下のとおりです。 no_protection | 必須 use_if_available

例：no_protection

デフォルト：none

親：flashaccess_policy

 いいえ
minimum_security_level
コンテンツへのアクセスに必要なセキュリティレベルを指定します。 （通常、DRMシステムのセキュリティレベルで開発環境と本番環境の違いが規定されています。）
注意： このプロパティは常に10000に設定するか、空白のままにします。

タイプ：整数

例：10000

デフォルト：none

親：flashaccess_policy

 いいえ

Widevineモジュラー特性

次のプロパティをGoogle Widevine Modularポリシー定義に関連付けることができます。

プロパティ 説明 必須？
widevine_modular_policy

Widevine Modularポリシーのプロパティを指定します。

タイプ：コンテナ

デフォルト：none

いいえ
can_persist

オフラインで使用するためにライセンスを不揮発性ストレージに保持できることを示します。

タイプ：ブール

例：true

有効な値：true、false

デフォルト：false

親：widevine_modular_policy

 いいえ
遊べる

コンテンツの再生が許可されていることを示します。

タイプ：ブール

例：true

有効な値：true、false

デフォルト：false

親：widevine_modular_policy

 いいえ
can_renew

ライセンスの更新が許可されていることを示します。 trueの場合、ライセンスの有効期間はハートビートによって延長できます。

タイプ：ブール

例：true

有効な値：true、false

デフォルト：false

親：widevine_modular_policy

 いいえ
cgms

Widevine ModularアセットすべてにCGMS保護があるかどうかを示します。 CGMS保護のレベルには、[自由にコピー]、[1回コピー]、または[コピーしない]があります。

タイプ：文字列

例：フリー

有効な値：free、once、never

デフォルト：none

親：widevine_modular_policy

 いいえ
HDCP

すべてのWidevine ModularアセットにどのバージョンのHDCP保護を使用するかを指定します。

タイプ：文字列

例：V2

有効値：V1、V2

デフォルト：none

両親：widevine_modular_policy

 いいえ
license_duration_seconds

この特定のライセンスの時間枠を示します。 値0は、期間に制限がないことを示します。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
playback_duration_seconds

ライセンス期間内に再生が開始されると表示される時間枠を示します。 値0は、期間に制限がないことを示します。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
renewal_delay_seconds

license_start_timeから更新が最初に試行されるまでの秒数を示します。 このフィールドは、can_renewがtrueの場合にのみ使用できます。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
renewal_recovery_duration_seconds

更新の試行中に再生を続行できるが、ライセンスサーバーに関するバックエンドの問題のために失敗した期間を示します。 値0は、期間に制限がないことを示します。 このフィールドは、can_renewがtrueの場合にのみ使用できます。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
renewal_retry_interval_seconds

後続のライセンス更新要求間の遅延を秒単位で示します。 このフィールドは、can_renewがtrueの場合にのみ使用できます。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
renewal_server_url

このライセンスに対するすべてのハートビート（更新）要求が送信される先のURLを示します。 このフィールドは、can_renewがtrueの場合にのみ使用できます。

タイプ：文字列

例： ""

有効値：有効なURL

デフォルト：none

親：widevine_modular_policy

 いいえ
renew_with_usage

使用開始時にライセンスが更新のために送信されることを示します。 このフィールドは、can_renewがtrueの場合にのみ使用できます。

タイプ：ブール

例：true

有効な値：true、false

デフォルト：true

親：widevine_modular_policy

 いいえ
rent_duration_seconds

再生が許可されている間のタイムウィンドウを示します。 値0は、期間に制限がないことを示します。

タイプ：整数

例：0

有効な値：秒数

デフォルト：0

親：widevine_modular_policy

 いいえ
