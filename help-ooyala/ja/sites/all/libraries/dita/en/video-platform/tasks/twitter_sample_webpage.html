サンプルWebページの作成

テスト中に、サンプルWebページを作成します。 最終的には、このプロセスを自動化したいと思うでしょう。

サンプルWebページを作成するには

  1. テキストエディタを開きます。
  2. Webページを作成してください。 例えば： 
    <!DOCTYPE html>
   <html>
    <head>
      <title>My Page</title>
    </head>
    <body>
      <h1>My Heading</h1>
      <p>My paragraph.</p>
    </body>
  </html>
  3. 貼り付け <meta> で生成したタグ Ooyala Player用のTwitter <meta>タグの手動取得 に <HEAD> of the page. For example:  
    <!DOCTYPE html> 
<html>
<head>
  <title>My Page</title>
  <meta name="twitter:card" content="player">
  <meta name="twitter:title" content="123 Marvel's The Avengers.mov" />
  <meta name="twitter:description" content="video">
  <meta name="twitter:image" content="http://ak.c.ooyala.com/B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt/Ut_HKthATH4eww8X4xMDoxOjBhOzV3Va">
  <meta name="twitter:player" content="https://player.ooyala.com/tframe.html?ec=B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt&pbid=MzZiMzc1ZDUzZGVlYmMxNzA3Y2MzNjBk">
  <meta name="twitter:player:width" content="435">
  <meta name="twitter:player:height" content="251">
  <meta name="twitter:site" content="@username">
</head>
<body>
  <h1>My Heading</h1>
  <p>My paragraph.</p>
</body>
</html>
  4. Webページを公開Webサイトの場所にコピーします。 承認のためにTwitterに提供できるように、URLを書き留めます。
