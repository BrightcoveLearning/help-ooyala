YouTubeとのシンジケート

YouTubeシンジケーション（つまり、YouTubeに動画をエクスポート）の場合 バックスロット ビデオのソースファイルを自動的にYouTubeに直接アップロードします。これにより、すべてのトランスコードとプレビュー画像の生成が処理されます。

一方向でシンジケーションが発生する

YouTubeとのシンジケートは「一方向」です。 バックスロット YouTubeにプッシュされます。
YouTubeの設定とコンテンツは Studio上ではサポートされていません。 引き戻した バックスロット.

YouTubeアカウントごとに1つのYouTubeシンジケート

1つのYouTubeアカウントにはYouTubeシンジケーションを1つだけ含めることができます。 たとえば、YouTubeアカウントabc@abc.comとxyz@abc.comがある場合、abc@abc.comにはYouTubeシンジケートがあり、xyz@abc.comには別のYouTubeシンジケートがある必要があります。

YouTubeアカウントの制限

Ooyalaは、YouTubeに配信するビデオファイルのサイズと期間に制限を設けていません。 これらの制限は、YouTubeとの関係（YouTubeアカウント設定）によって決まります。 内のメッセージを無視してください バックスロットUI 下に示された。

YouTubeによる変更処理

YouTubeでは、以前にアップロードされた（またはシンジケートされた）ビデオのコンテンツをあなたに置き換えることはできません。 代わりに、YouTubeは新しい動画コンテンツに新しい識別子（ID）を割り当てます。 この設計のため、資産や設定に多くの変更を加えます バックスロット 次のような新しい動画識別子をYouTubeに作成させます。
  • ビデオの名前を変更する
  • 説明など、その他の詳細を変更する
  • 動画のメタデータを変更する
  • ビデオからラベルを削除する バックスロットUI またはフィード自体を編集してラベルを削除します。 そのラベルに以前関連付けられていたビデオはYouTubeから削除されます。
  • プライバシー設定。 YouTubeでプライバシー設定を変更した場合、 バックスロット YouTubeを更新すると、アカウントに関する統計情報を含め、YouTubeに対する以前の変更は失われます。

    バックスロット YouTubeのプライバシー設定は、公開と非公開の2つのみです。 YouTubeシンジケーションを作成すると、デフォルトであなたのビデオはYouTubeで公開されます。 以下の最後の手順で詳しく説明するように、このデフォルトを変更できます。

    プライバシー設定を変更しても、シンジケーションに追加されたビデオにのみ影響します。 変更後。 以前にYouTubeに配信されたビデオのプライバシー設定を変更するには、個々のアセットの設定を 管理 ページ 細部 次の例に示すように、特定の資産のサブタブ。
  • YouTubeからのCAPTCHAチャレンジの処理
    あなたが悪いパスワードでYouTubeシンジケーションを数回作成しようとした場合、あなたはCAPTCHAチャレンジに直面するかもしれません。
  • YouTubeシンジケートのプライバシー設定およびその他の考慮事項
    Youtubeでシンジケートするアセットのデフォルトのプライバシー設定を設定できます。
