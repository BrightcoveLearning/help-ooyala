ActionScript APIサマリーリスト（Player V2は廃止予定）

関数/メソッド、オブジェクト、プロパティ、およびイベントごとにグループ化されたすべてのPlayer ActionScript APIを一覧表示して説明します。

重要： Ooyala Player V2は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V2はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V2を使用しているお客様は、Player V4に移行する必要があります（を参照）。 プレイヤーV4.

このセクションでは、ActionScriptインターフェイスを介して利用できるさまざまなActionScript API関数、メソッド、プロパティ、およびイベントへのクラス参照を提供します。

パッケージcom.ooyala.api

パブリッククラスPlayerはUIComponentを拡張します

ActionScriptビットレート選択API

ビットレート選択APIを使用すると、開発者はプログラム的に現在のバッファサイズを取得したり、ビットレートを取得および設定したりできます。 ビットレートを選択するために、2つの異なるワークフローを提供します。

  • その品質（自動、低、中、または高）に基づいてビットレートを選択します。 たとえば、ビットレート品質を高に設定した場合、プレーヤーはどちらが最高のエンコーディングかを決定します。
  • kbps数（オーディオとビデオの両方のエンコーディングの合計）に基づいてビットレートを選択します。 たとえば、使用可能なビットレートが[600 kbps、1000 kbps、3000 kbps、4000 kbps]のビデオを考えます。 ビットレート品質をこれらの値のいずれかに設定できます。
