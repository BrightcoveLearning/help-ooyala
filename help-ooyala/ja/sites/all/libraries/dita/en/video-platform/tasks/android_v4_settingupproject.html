Android用Skinサンプルアプリケーションの使い方

注意： Androidのバージョンが4.1以上で、最新バージョンのAndroid Studioがインストールされていることを確認してください。

サンプルアプリを使い始めて、コンパイルしたSDKを自分のアプリケーションで試してみてください。 ソースコードSDKは、あなたのアプリケーションに直接リンクできるAndroid Studioプロジェクトです。 ソースコードSDKはどちらのローカルホスティングも使用できます or JSBundle

  1. クローン android-sample-apps リポジトリ。
  2. Android Studioでプロジェクトを開きます。

    ナビゲーションヘッダーの 既存のAndroid Studioプロジェクトを開く or フィレット > 開いた.

  3. に移動します OoyalaSkinSampleApp ディレクトリ。 クリック 選択する or OK.
  4. Gradleの実行が完了するのを待ちます。 君は見るべきだ アプリ ビルド構成の1つとして表示されます。
  5. ナビゲーションヘッダーの ラン > 「アプリ」を実行をクリックして、接続先のデバイスまたはアプリを実行するシミュレータを選択します。

独自のテスト資産を追加する

あなたはあなた自身のテスト資産でコンパイルされたSDKを試すことができます。

他のすべてのSDKサンプルアプリケーションと同様に、自分の埋め込みコードを使用してListActivityを変更して、動画の動作を確認できます。 あなたの資産が機能するためにはアプリケーションの追加設定が必要になるかもしれないことに注意してください。

  1. OoyalaSkinSampleAppを開きます。
  2. src / main / java / com / ooyala / sample / lists / OoyalaSkinListActivity.javaを変更します。
  3. アプリケーションを再実行してください。

BasicPlaybackSampleAppを試してください。

BasicPlaybackSampleAppを使用してコンパイル済みのSDKを試すこともできます。 BasicPlaybackSampleAppを使い始めるには、を参照してください。 Androidスキンを使用した既存のアプリケーションの更新.

プレイヤースキンを修正する

skin-config / skin.json 設定ファイルは、プレーヤUIの外観を定義するJSONファイルです。 それを変更してサンプルアプリケーションに配置することができます。 見る プレーヤV4外観のカスタマイズ 詳細については。
