関連コンテンツの推奨事項

関連コンテンツアルゴリズムは、現在のアセットに関連するコンテンツを返します。現在のアセットは、WebページまたはApp内に表示できます。

注意： （要求ごとに個別のパラメーターを使用して）アドホック要求を作成するのではなく、ディスカバリー・プロファイルを使用して設定と推奨動作を一元管理することを検討してください。 検出プロファイルは、ユーザーが作成した設定の定義済み構成です。 のGET ルート 詳細については、 ディスカバリープロファイル.

関連コンテンツについて

関連コンテンツアルゴリズムは、最初はメタデータを使用して関係を判断しますが、ユーザーの行動に基づいて結果を最適化します。 たとえば、アルゴリズムはアイスホッケー、アイススケート、およびZamboniビデオの間の関係を見つけるかもしれません。 時が経てば、アイスホッケーのビデオやザンボニのビデオを見たり、アイススケートやザンボニのビデオを見たり、ザンボニのビデオだけを見たりする人もいます。

関連コンテンツの結果を表示すると、コンテンツ間の関係の一部が明らかになっています。 しかし、それほど明白ではないかもしれないものがたくさんあります。 たとえば、スポーツビデオの場合、関連コンテンツアルゴリズムは次のようにグループ化します。

  • ハイライト動画
  • NG集
  • スポーツを超えたトップパフォーマンス
  • エクストリームスポーツの最高

これらのテーマは明白ですが、他のそれほど明確ではないテーマが現れるかもしれません。 さらに、結果は動的です。 ユーザーが時間の経過とともにさまざまなコンテンツを視聴するにつれて、おすすめ動画も変わります。
