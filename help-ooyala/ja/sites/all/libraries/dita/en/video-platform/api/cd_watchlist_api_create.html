ウォッチリストを作成する

このエンドポイントはコンシューマ用の新しいウォッチリストを作成し、その32文字のUUIDを返します。 すべてのユーザーは デフォルト 自動的に作成されるウォッチリスト。 このリクエストを使用して追加のウォッチリストを作成します。

HTTPリクエスト

POST https://player.ooyala.com/watchlist/v3?account_token= <account_token> {"name"： "新しいウォッチリストの名前"、}

クエリパラメータ

パラメーター 説明 タイプ 必須
api_version でなければなりません v3. 文字列 はい（URL）
account_token 特定のエンドユーザに関連付けられているウォッチリストを参照するためにウォッチリストサービスによって使用されます（UUIDによって識別されます）。 詳しくは、 OoyalaアカウントトークンAPI. 文字列 はい（URL本体）
名 作成する監視リストのわかりやすい名前。
  • 長さが3から64文字のストリングでなければなりません。 空の文字列にはできません。 英数字と特殊文字（アンダースコア、スペース、句読点など）を使用できます。
  • を指定 名 これはこのユーザーに固有のものです。 重複した名前は無効です。 比較では大文字と小文字が区別されるため、 "my_watchlist"と "MY_WATCHLIST"は異なると見なされます。
  • 指定できません "デフォルト"（または場合に応じて任意の変形） 名 パラメータ すべての消費者には自動的に作成されるデフォルトのウォッチリストがあります。
 文字列 はい（ボディ）

リターンコード

コード 理由 説明
201 OK POST要求は成功しました。 新しいウォッチリストが正常に作成されました。
400 リクエストエラー 次の問題が原因で失敗しました。
  • パラメーターが欠落しているかフォーマットが正しくありません。
  • 指定されたウォッチリスト 名 is "デフォルト" （またはその単語の任意のケースのバリエーション）。
  • 指定されたウォッチリスト 名 このユーザーには既に存在します。
401 無断 指定された account_token 無効です。
404 見つかりませんでした サーバーが停止しています。
500 内部サーバーエラー 内部サーバーエラーが発生しました。

収益

パラメーター 説明
watchlist_id 新しく作成されたウォッチリストのID。 32文字のUUID
