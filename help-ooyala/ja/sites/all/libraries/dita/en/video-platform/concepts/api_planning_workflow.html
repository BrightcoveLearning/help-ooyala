コンテンツのアップロードと管理

コンテンツの整理方法を決定したら、動画をアップロードして管理する準備が整いました。

ビデオをアップロードして設定するには：

  1. ビデオをアップロードしてください。 動画をアップロードするには、 動画をアップロードする.
  2. クローズドキャプションを追加します。 詳細については、 クローズドキャプションの管理.
  3. カスタムメタデータを追加します。 詳細については、 カスタムメタデータの管理.
  4. 詳細を更新してプレビュー画像を変更します。 詳細については、 ビデオ情報の編集 そして プレビュー画像の管理.
  5. コンテンツをどこで、いつ、どのデバイスで表示できるかを定義する公開ルールを追加します。 公開ルールを追加するには、 資産への公開ルールの割り当て。 動画、チャンネル、チャンネルセットで共有できる公開ルールを作成するには、 公開ルール.
  6. 広告や広告セットを関連付けます。 広告を関連付けるには、 広告を管理する。 広告または広告セットを作成するには、 広告.
  7. 必要に応じて、ペイウォールを指定してください。 詳細については、 ペイウォール統合.
  8. 公開してください。 Webサイトにビデオを埋め込むには、 手動でアセットを埋め込む。 シンジケートのためにビデオを1つ以上のラベルに関連付けるには、 ラベルを使った作業。 シンジケーションを設定するには、 シンジケーション.
