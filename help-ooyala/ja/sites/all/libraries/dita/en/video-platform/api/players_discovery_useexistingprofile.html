以前に作成した検出プロファイルを使用する

Discovery APIリクエストで既存の検出プロファイルを利用するには、クエリ文字列パラメータを追加します。 discovery_profile_id = profile_id。 ザ profile_id 以前に作成されたプロファイルの識別子です。 このクエリ文字列パラメータは、リクエストに含まれる他のクエリ文字列パラメータに優先します。

次のリクエストとレスポンスの例は、パーソナライズされた推奨のリクエストを示しています。 discovery_profile クエリ文字列パラメータ。

[GET] / v2 / discover / trending？discovery_profile_id = {profile_id}
