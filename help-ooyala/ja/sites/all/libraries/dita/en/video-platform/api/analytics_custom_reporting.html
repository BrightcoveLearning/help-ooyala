カスタム分析：レポーティング（廃止予定）

カスタム分析レポートには、設定したタグの掲載結果情報が表示されます。

警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、

カスタムアナリティクスを入手

[GET] / v2 / analytics / reports / account / performance / tags / {  
   日付範囲
[GET] / v2 / analytics / reports / asset /asset_id/ performance / tags [  
   日付範囲
] [GET] / v2 / analytics / reports / asset /asset_id/ performance / tags /tag_id[  
     
   日付範囲

]

例

この例では、1日分のアカウントのカスタム分析を表示順に返します。 

[GET]/v2/analytics/reports/account/performance/tags/2011-08-01...2011-08-02?order_by=displays
注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
