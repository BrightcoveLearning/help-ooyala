iOS用Ooyala Mobile SDKを使用する

The Ooyala Mobile SDK for iOS provides APIs to manage and configure:

  • レイアウト
  • イベント
  • エラー
  • クローズドキャプション
  • ローカライゼーション
  • レイアウトコントロール
    DefaultControlsSource directory contains the default Ooyala player skin.
  • イベントでの作業
    プレーヤオブジェクトのイベント通知を監視する必要があります。
  • オフライン再生：iOS用ダウンロード（DTO）およびレンタル（DTR）
  • iOS Mobileのエラー処理
    プレーヤオブジェクトのエラー通知を監視する必要があります。
  • Ooyalaプレーヤーオプションの設定
  • ユーザインタフェースのローカライズ
    UIテキストにカスタムローカライズ文字列を設定できます。
  • iOSネイティブアプリの検出について
  • iOSアプリのパフォーマンスモニタリングの実装
