Flash PlayerへのTVレーティングの透かしの追加V3（廃止予定）

Add a TV rating watermark to assets using player parameters and custom metadata in Backlot.

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).

注意： この機能はFlashデスクトップでのみ利用可能です。

Ooyala Flashプレーヤーは今 FCC対応 表示用 テレビの評価 associated with broadcast content. TV ratings are applied at the asset level, and each asset can only have one TV rating. The TV rating watermark appears when the video starts playing and will appear again when the video resumes after ad playback.

注意： The TV rating watermark will not appear during ad playback.

Custom Metadata to Add to Backlot per Asset

Populate the following metadata in Backlot as custom metadata for each asset you want to give a TV rating watermark.

カスタムメタデータ 説明 有効な値 必須 例
精進する 視聴率または推奨視聴者を指定します。

TV-X

Where X can be one of the following: Y | Y7 | G | PG | 14 | MA

Go to the FCC site to see descriptions of each rating.

 はい テレビ：“ TV-PG”
TVsubratings

Specifies content label or nature of content.

注意： tvsubratings can only be applied to Y7, PG, and 14 TV ratings.
V | S | L | D | FV
  • V: violence
  • S: sexual situations
  • L: coarse or crude indecent language
  • D: suggestive dialogue FV: fantasy violence

注意： For multiple sub ratings, values should be comma or semicolon separated with no spaces between values.

 いいえ tvsubratings:

”D,S”
Tvratingurl

（オプション）

 クリックスルーURL。 A URL link. いいえ tvratingurl = "http://www.parentstv.org/ptc/parentsguide/tvratings2.html"

プレイヤーパラメータ

Apply the following parameters at the page level.

tvRatingsTimer: This required parameter specifies the duration of the TV rating watermark on the player (in seconds) on initial play and every time the video resumes from Ad breaks. The default value of this parameter is "never", but other valid values include "always" and "some_number".
  • always：透かしはビデオの間表示されます（広告の再生中は表示されません）。
  • never：透かしは表示されません。
  • some_number：some_numberは、テレビのレーティングをプレーヤーに表示し続ける秒数に置き換えます。

tvRatingsPosition: This optional parameter specifies the position on the player where the TV rating will appear. By default this value is set to "top-left", but other valid values include "top-right", "bottom-left", and "bottom-right".

例

The following example shows how to set the tvRatingsPosition そして tvRatingsTimer player parameters.

var player = OO.Player.create('playerwrapper', embedCode, {
    tvRatingsPosition: "top-left",
    tvRatingsTimer: "15"
});
