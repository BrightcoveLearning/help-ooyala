When you create a new asset in the Backlot REST API, the asset_type property in the request body can specify any of the following values.

  • ビデオ
  • ad: An ad is simply a video, marked as an ad in the バックスロットUI.
  • オーディオ
  • remote_asset
  • ユーチューブ: A video destined for YouTube
  • live_stream (deprecated)
  • live_audio (deprecated)
  • チャンネル (deprecated)
  • channel_set (deprecated)
