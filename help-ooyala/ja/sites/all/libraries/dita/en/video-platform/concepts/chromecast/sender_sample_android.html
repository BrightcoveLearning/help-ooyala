Android用サンプル送信者アプリケーション

Ooyalaは、Ooyala Chromecast統合の送信側コンポーネントと受信側コンポーネントの実用バージョンを実行するAndroid用のサンプル送信側アプリケーションを提供しています。

必須条件

To perform the following steps to run the Android sample app, you must have:
  • Android Studioを実行しているコンピューター。
  • chromecastモジュールがインストールされている受信機。
  • An Android device running Android SDK 4.x or newer with the Chromecast App installed and paired with the receiver. This process is well documented by following the set up steps that appear when the app is opened on the device.

Running the Sample Sender Application for Android

注意： If you are repeating these steps, make sure the app is removed from your device before beginning the second pass.
  1. クローン Ooyala SDK for Android Sample Apps リポジトリ。
  2. Android Studioを開きます。
  3. 選択します 既存のAndroid Studioプロジェクトを開く:
  4. Navigate to android-sample-apps, select ChromeCastSampleApp をクリックし 選択する for OS or OK Windowsの場合
  5. Wait for the Gradle execution to complete. The sample app opens in the IDE.
  6. 確認する プロジェクト tab is active.
  7. 実行を選択します。
  8. Choose the connected device from the selection menu and click OK.
    The app is running on the device:
  9. 放送アイコン（右上隅）をタップしてChromecast受信機を選択します。
  10. Tap to select the correct Chromecast receiver from the list:
  11. 受信機に送信するビデオをタップして選択します。
アプリはレシーバーで再生されています。
