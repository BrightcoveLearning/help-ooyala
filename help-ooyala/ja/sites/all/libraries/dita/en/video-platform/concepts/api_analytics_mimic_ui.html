ここにいくつかの分析データのさまざまなグラフィック表示 ANALYZE ページ上 バックスロットUI 統計グラフィックの基礎となるデータを返すAPIリクエストと照合されます。

パフォーマンス：国別

In バックスロットUI あなたはあなたの資産のパフォーマンスのいくつかの異なるビューをすることができます。 たとえば、左側の選択リストでは、次に示すように国別のパフォーマンスを表示できます。

すべての動画についてこの表示の基礎となるデータを取得するための呼び出し。

[GET]/v2/analytics/reports/account/performance/countries/2012-01-01...2012-11-11

日付による株式

同様に、タイムラインビューを見ると、「共有」（ソーシャルネットワークでの動画の投稿または共有）の傾向を見ることができます。 以下のデータは1日を表しています。

1年中この表示の基礎となるデータを取得するための呼び出し。

[GET]/v2/analytics/reports/account/sharing/videos/2012-01-01...2012-12-31
