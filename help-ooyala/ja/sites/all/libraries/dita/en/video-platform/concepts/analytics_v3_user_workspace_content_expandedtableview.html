拡張テーブルの表示

二重矢印アイコンを使用して、テーブルレイアウトにデータを表示します。

ディメンションデータを表形式で表示する場合は、展開アイコン（）の寸法パネルの右上隅に ビジネス・インテリジェンス のページをご覧ください。

注意： ビジネスインテリジェンスページへのフィルタの適用 展開された表を見る前に。

表示されたテーブルでは、次のことができます。
  • 表の下部にあるページオプションを使用して、データのさまざまなページを調べます。
  • テーブル列を追加または削除します。 デフォルトでは、すべてのメトリックがテーブルに表示されています。
    1. ナビゲーションヘッダーの テーブル列の編集.
    2. テーブルからメトリック列を削除するには、非表示にするメトリックの横にあるチェックボックスをオフにします。 選択された指標のみがテーブルの列として表示されます。
    3. テーブルにメトリック列を追加するには、表示するメトリックの横にあるチェックボックスをオンにします。
    4. CSV形式にエクスポートするには、をクリックします CSVにエクスポート。 エクスポートされたテーブルには、最大1,000個の項目を含めることができます。
  • 列の境界線をドラッグして列幅を調整します。
  • 列タイトルをクリックしてデータの順序を調整します（たとえば、データを昇順または降順に並べることができます）。
  • 使用 で表示 [地理]パネルと[デバイス]パネルのメニューで、エクスポートするデータの分類を選択します。
    • [地理]パネルで、[表示：BY COUNTRY]、[表示：BY DMA]、または[表示：BY STATE]を選択します。
    • デバイスパネルでは、Show：BY DEVICEまたはShow：BY OSを選択します。
