iOSの基本的なクローズドキャプション機能

A simple programming example shows the basic functions on iOS to get and set the closed caption (CC) language.

注意： Be sure you have setup the required DFXP (now TTML) files for all your videos, as detailed バックスロットに必要なクローズドキャプションの設定. This simple example shows some of the basic function calls for programming with closed captions on iOS:
  • Shown first here is a check for the OOOoyalaPlayerCurrentItemChangedNotification event, which in this context indicates that the video and its associated DFXP (now TTML) file have been loaded. (Event programming is not required for working with closed captions, but is shown here only as a useful feature. If you are interested in more details about event programming, see イベントでの作業.)
