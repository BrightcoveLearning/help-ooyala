ディスカバリプロファイルの更新

Backlot UIでプロファイルを作成したら、ディスカバリー・プロファイル・プロパティーを使用してそれを更新することができます。 APIリクエスト文字列にはプレーヤーIDが必要です。 次のリクエストとレスポンスの例は、すべての利用可能なプレイヤーをリストするリクエストと、それに関連付けられた各プレイヤーを含むレスポンスを示しています。 player_id.

要求：

[GET] / v2 / players

応答：

{"items"：[{"id"： "{player_id}"、" name "：" ForProfiles "、"透かし "：{" alpha "：" 1.0 "、" image_url "：null、" scaling_percentage "：null、" click_url "：null、" position "："右下 " 、 "scaling_option"： "no_scale"}、...
