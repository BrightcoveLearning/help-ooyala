プレーヤV3のVAST Ads埋め込みパラメータ（廃止予定）

VAST広告サーバーまたはネットワークタグをOoyalaプレーヤーに渡します。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
VASTは、リニア広告（動画広告）、ノンリニア広告（オーバーレイ広告）、コンパニオン広告を表すためにxmlを使用する広告フォーマットです。 埋め込みパラメータを使用してVAST広告サービス関連のタグをプレーヤに渡すことができます。
必須条件
  • OoyalaのVAST広告セットを使用するには、サポートセンターにログインして、この広告ソースをアカウントに追加するように要求するチケットを送信してください。 有効にすると、プレーヤー用の広告セットを作成できるようになります。
  • VAST広告タグの埋め込みパラメータを使用するには、広告セットもBacklotに適用する必要があり、広告セットに有効なVAST広告タグを含める必要があります。
現時点では、プレロール広告、ミッドロール広告、ポストロール広告のサポートはプラットフォームによって異なります。 各プラットフォームでどの広告掲載順位がサポートされているかを確認するには、 クライアントサイド広告統合.

VASTパラメータ

使用 広大 VAST広告サーバーまたはネットワークタグをOoyalaプレーヤーに渡すためのパラメータ。 このパラメータは、広告サーバーまたはネットワークのURLを表すキーと値のペアを取ります。 また使用することができます 広大 広告のURLを変更します。 広告URLが 広大 このパラメータを設定すると、バックスロットで再生するように設定されたときに広告が再生されます（プリロール、ミッドロール、またはポストロール）が、再生される広告コンテンツは新しい広告URLで上書きされます。 このパラメータはFlashおよびHTML5用です。 次の例では、 広大 パラメータ。 
var videoPlayer = OO.Player.create（ 'playerwrapper'、 'embed_code'、{高さ： '100％'、幅： '100％'、 'vast'：{tagUrl： 'some url'、...}}） ;

vastAdsパラメータ

注意： このパラメーターはHTML5専用です。

使用 広大な広告 Backlotに適用されたVAST広告または広告セットをプログラムで上書きするためのパラメータ。 この機能を使用すると、VAST広告をアセットに追加したり、プレーヤーの埋め込みパラメータからVAST広告のコンテンツと位置（プリロール、ポストロール、ミッドロール）を更新したりできます。 広告がに設定されている場合 広大な広告、Backlotのプレーヤーに関連付けられている広告および広告セットは完全に無視されます。

広大な広告 広告で使用されるすべてのパラメータを指定する必要があるコンテナです。
  • type：広告タイプ "広大"に設定します。
  • URL：VAST広告コンテンツへのリンク。
  • 周波数：VAST広告を呼び出す頻度 任意の正の整数に設定してください。
  • first_shown：VAST広告が表示されるまでにいくつの動画を再生する必要があるか。 任意の正の整数に設定してください。
  • 時間：VAST広告が再生されるまでの時間（ミリ秒）。 任意の正の整数に設定してください。
    • プレロール広告を作成するには、時間を0に設定します。
    • ミッドロール広告を作成するには、アセットの時間内で時間を他の数値に設定します。
    • ポストロール広告を作成するには、時間を正確に1000000000に設定します。
次の例は、プレロール広告とミッドロール広告の設定方法を示しています。 広大な広告. 
var player = OO.Player.create（ 'playerwrapper'、embedCode、{'vastAds'：[{type： "vast"、url： "http://pubads.g.doubleclick.net/gampad/ads?sz=640x360&iu = / 6062 / iab_vast_samples / skippable＆ciu_szs = 300x250,728x90＆impl = s＆gdfp_req = 1＆env = vp＆output = xx_vast2＆amp;時間：＆lt; =長さ：X：長さ： URL：「http://pubads.g.doubleclick.net/gampad/ads?sz=1x1&iu=/0/iab_vast_samples/skippable&ciu_szs=0x640x360&impl=s&gdfp_req=6062&env=vp&output=xml_vast300&unviewed_position_start=250,728&url=[referrer_url]&correlator=[timestamp] "、頻度：90、first_shown：1、時間：2}]}）;
