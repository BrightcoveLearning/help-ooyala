チャネルセットAPI（廃止予定）

チャンネルセットAPIはチャンネルセットラインナップの一覧表示と設定を可能にします。

注意： このソフトウェアは非推奨です。 最新版を使用してください。

「リスト」、「割り当て」、「作成」の3つのモードがあります。 Query APIと同様に、Channel Set APIとのやり取りは、署名されたRESTスタイルのリクエストを介して行われます。 http://api.ooyala.com/partner/channels.

必須パラメータ

  • pcode - プロバイダコード
  • expires - エポックからの経過秒数での照会有効期限（00：00：00 1 / 1 / 1970 GMT）
  • モード - の1つ リスト, 割り当てます, 作ります
  • signature - リクエストのデジタル署名 署名は、署名生成規則とともにSHA-256を使用して計算する必要があります。

リストモード

追加の必須パラメーター：channelSetEmbedCode

すべてのチャンネルセットコンポーネントを一覧表示します。

例：

http://www.ooyala.com/partner/channel_sets?pcode=<pcode>&channelSetEmbedCode=<channelSetEmbedCode>&expires=<expiration>&mode=list&signature=<signature>

結果：

<？xml version = "1.0" encoding = "UTF-8"？> <channelSet> <embedCode> 9scDM426kElm5ddWAExjw3uq6R <title>新しいチャンネル</atus> <stit>埋め込み> > w7adOyOiMjsW-4yCgrY58IOer92v_hAp </embedCode> <title>デモチャンネル</title> <description> </description> <status>ライブ</status> <content_type>チャンネル</content_type> <uploadedAtloaded> > 3 </length> <size> 1217036203 </size> <updatedAt> 130766 </updatedAt> </channel> </channelSet>

割り当てモード

追加の必須パラメーター：channelSetEmbedCode、channelEmbedCodes

チャネルセット埋め込みコードに、チャネル埋め込みコードのカンマ区切りリスト（または単一のチャネル埋め込みコード）をラインアップとして割り当てます。 他のチャンネルセットの埋め込みコードは無効です。 このモードを使用すると、既存のチャンネルセットのラインナップが上書きされます。 成功した場合は 'OK'、失敗した場合はエラーメッセージを含むテキストファイルを返します。

例： 

http://www.ooyala.com/partner/channel_sets?pcode=<pcode>&channelSetEmbedCode=<channelSetEmbedCode>&channelEmbedCodes=<embedCode1>,<embedCode2>&expires=<expiration>&mode=assign&signature=<signature>

作成モード

追加の必須パラメーター：title

与えられたタイトルでBacklotにチャンネルセットを作成します。

例：

http://www.ooyala.com/partner/channel_sets?pcode=<pcode>&expires=<expiration>&mode=create&title=<channelSetTitle>&signature=<signature>

結果：

<？xml version = "1.0" encoding = "UTF-8"？> <channelSet> <embedCode> R2NTZkMTpJ0i-YaH8cPuKwS_yHJgri7O </embedCode> <タイトル> live <タイトル> </tit> / channelSet>
