アクセスキーをオンにする

シンジケーションのMRSSフィードのアクセスキーをオンにすることができます。

シンジケーションのアクセスキーを有効にするには

目的のシンジケーションに対して、PATCH要求を使用して「require_access_key」を「true」に設定します。 次の例では、シンジケーションのアクセスキーをオンにします。 6b6e72dd71bd4faシンジケーションID 
[PATCH] / v2 / syndications / 6b6e72dd71bd4fa {"require_access_key"：true}
"require_access_key"がtrueに設定されていると、フィードのsyndication_urlに移動しても機能しなくなります。 フィードにアクセスするには、 "access_key = ..."クエリ文字列パラメータをsyndication_urlに追加する必要があります。 
http://api.ooyala.com/v2/syndications/syndication_id/feed?
pcode=yourPcode＆access_key = accessKey

バックスロット 次のような応答を返します。
