カスタムメタデータの削除

メタデータエントリはいつでも削除できます。

注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

カスタムメタデータを削除するには：

アセットのメタデータを削除するには、 DELETE とともに / v2 / assets ルート、アセットID、および /メタデータ 修飾子 次の例では、からカスタムメタデータを削除します。 JxbzdkMjqBEsO 資産。 
[削除] / v2 / assets / JxbzdkMjqBEsO / metadata

バックスロット 200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
メタデータが削除されます。
