シンジケーションを編集する

シンジケーションはいつでも編集できます。 変更は数分以内に行われます。

シンジケーションを編集するには

  1. 使用 シンジケーション シンジケーションに関する情報を取得するための経路。 次の例では、getを実行して資産の現在の設定を取得します。 6b6e72dd71bd4fa IDを指定します。 
    [GET]/v2/syndications/6b6e72dd71bd4fa
    注意： すべてのシンジケーションに関する情報を入手するには、要求にIDを指定しないでください。

    バックスロット 次のような応答を返します。

    {"name"： "My iPhone Feed"、 "include_encodings"：[]、 "destination_url"： ""、 "require_access_key"：false、 "title"： ""、 "include_labels"：["638aed2a18734375b"、 "c42b225963adXNXbX" ]、 "asset_types"：[ "ビデオ"、 "広告"]、 "syndication_url"： "http://api.ooyala.com/syndications/481b9e6dd6bd72fa/feed?pcode=Y71cG4BlqYMLIjPGyv4NbacSK-x06"、 "ID"： "1b3e6dd6bd72fa "、" include_all_content "：false、" type "：" iphone "、" description "：" "}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. PATCH要求で変更を加えます。 次の例では、名前、URLでホストされている、および説明を変更します。 
    [PATCH] / v2 / syndications / ZnbWVpMjqToT1W {"名前"： "私の素晴らしいiPhoneフィード"、 "説明"： "私のiPhoneフィードは素晴らしいです（2）。" }

    バックスロット 次のような応答を返します。

    {"name"： "私の素晴らしいiPhoneフィード"、 "include_encodings"：[]、 "destination_url"： ""、 "require_access_key"：false、 "title"： ""、 "include_labels"：["638aed2a18734375b"、 "c42b225963bXN" "]、" asset_types "：["動画 "、" ad "]、" syndication_url "：" http://api.ooyala.com/syndications/481b9dd6bd6fa/feed?pcode=Y72BlqYMLIXPX "" xx "XXXXXXXXXXXXX" " "71b4e4dd06bd1fa"、 "include"、 "type"： "iphone"、 "description"： "iPhoneフィードが素晴らしい（3）です。" }
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.

シンジケーションは正常に編集されました。
