tvOS用モバイルSDK

Ooyala tvOS SDKはHLSとMP4アセットの基本的な再生を提供し、FairPlay DRMとOoyala分析をサポートします。

注意： React NativeはtvOSではサポートされていません。

必須条件

  • OoyalaのtvOS SDKを使用する前に、iOS用のOoyalaモバイルSDKをダウンロードしてセットアップしてください。 Ooyala ダウンロードここに記述されているように）： iOS用モバイルSDK.
  • アプリを実行するtvOSは、バージョン9以上でなければなりません。
あなたが必要なもの

OoyalaのtvOS SDKを使い始めるには、をダウンロードして開きます。 OoyalaSDK-tvOS.zip そして OoyalaSkinSDK-tvOS.zip ファイル。

これら二つのファイルは OoyalaTVSDK.framework そして OoyalaTVSkinSDK.framework ライブラリ。

Ooyala tvOS SDKを統合する

Xcodeを起動し、Xcodeプロジェクトフォルダを開き、ターゲットをクリックします。
  1. コピー OoyalaTVSDK.framework そして OoyalaTVSkinSDK.framework あなたのライブラリフォルダに。
  2. ナビゲーションヘッダーの ビルド設定：選択 サポートされているプラ​​ットフォーム > tvOS そして ベースSDK > 最新のtvOS.
  3. ナビゲーションヘッダーの 構築フェーズ 展開する バイナリとライブラリのリンク。 その OoyalaTVSDK.framework そして OoyalaTVSkinSDK.framework ここにあなたのアプリケーションとリンクしています。
  4. プロジェクトをtvOSシミュレータまたはApple TVデバイスでビルドして実行します。
  5. (任意Ooyala tvOS SDKが正しく統合されていることを確認するには、Ooyala tvOSサンプルアプリをビルドして実行します。

Ooyala TVOSSampleAppは、AppleのtvOSを統合してApple TVデバイス上でアプリケーションを実行する方法を示しています。 サンプルアプリを試すには、にアクセスしてください。 https://github.com/ooyala/ios-sample-apps/tree/stable/PlaybackLab/TVOSSampleApp.

Ooyala tvOSプレーヤーのUIはユニークで、カスタマイズ可能なオープンソースのプラグインが含まれています。 UIをカスタマイズするには

  2. 削除する OoyalaTVSkinSDK.framework あなたのプロジェクトから。
  3. 加えます OoyalaTVSkinSDK あなたがマスタープロジェクトに分岐したプロジェクト。

tvOSコアSDK使用上の注意

ウーヤラ tvOSコアSDK の直接ポートです。 iOSコアSDK。 両方のSDKのコードベースはほぼ同じです。

tvOS Skin SDKは、特にtvOSプラットフォーム用にゼロから構築されたUIを提供します。 tvOSはリアクトネイティブをサポートしていないため、iOS React-Native Skin SDKはtvOSに移植できません。

Skin SDKは、ネイティブスキンレポジトリで完全にオープンソースです。 https://github.com/ooyala/native-skin/tree/stable/sdk/tvOS。 独自のUIを作成するには、リポジトリをフォークし、Ooyalaコントロールをベースラインとして使用し、必要に応じてコードを拡張します。

tvOS Core SDKでは、 ViewControllerを 他のSDKで行われる方法とは大きく異なります。 インスタンス化しないでください OoyalaPlayer を作成し、 OoyalaPlayerViewController 小学校の子供の頃 ViewControllerを。 代わりに、 OOOoyalaTVPlayerViewController そしてその新しいものに分節する ViewControllerを.

たとえば、ヘッダーファイルには次のコードが含まれています。

#import <UIKit / UIKit.h> #import <OoyalaTVSkinSDK / OOOoyalaTVPlayerViewController.h> @interface FullscreenPlayerViewController：OOOoyalaTVPlayerViewController @end
実装ファイルには次のコードが含まれています。
self.pcode = self.option.pcode; self.playerDomain = self.option.domain; self.player = [[OOOoyalaPlayer alloc] initWithPcode：self.pcodeドメイン：[[OOPlayerDomain alloc] initWithString：self.playerDomain]]; [[NSNotificationCenter defaultCenter] addObserver：self selector：@selector（notificationHandler :) name：nil object：self.player]; [self.player setEmbedCode：self.option.embedCode]; [セルフプレイ]

サンプルアプリとのやり取り：Apple TVと対話するには、Apple TV Remoteを使用する必要があります。 Xcodeで、Xcodeでアプリをビルドして実行したとき、Apple TVでアプリを実行したとき シミュレータ Macのデスクトップでは、 ハードウェア > Apple TV Remoteを表示する.

アプリを制御するには、次のようにApple TV Remoteを使用します。

  • ビデオを再生または一時停止するには、 再生/一時停止 ボタン、または ハードクリック タッチパッドを押してクリックするまで（押し）ます。

  • 前方または後方にシークするには： ソフトクリック リモコンの両側にあるタッチパッド。

  • 一時停止中にビデオを前後にスクラブするには、タッチパッドを左または右にスワイプします。

インラインプレーヤーの使用を避ける：tvOS用のサンプルアプリには1つ含まれていますが、以下の理由から「インラインプレーヤー」を完全にはサポートしていません。

  • インラインプレーヤーは、プレーヤーとやり取りするユーザーエクスペリエンスの問題を引き起こします。
  • 常にフルスクリーンでビデオを表示することをお勧めします。
