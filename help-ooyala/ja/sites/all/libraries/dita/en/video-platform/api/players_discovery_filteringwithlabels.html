ラベルを使ったフィルタリング

次の要求と応答の例は、プロファイルの推奨事項に関連付けられているラベルを変更する方法を示しています。 次の例では、 PATCH メソッド from_labels ラベルセットのIDを使用して、そのラベルに基づいて推奨事項を提供します。

注意： ラベルは、サブアカウントからマスターアカウントのプレイヤーに割り当てることができます。
注意： すべてのラベルIDのリストを取得するには、次の要求を使用します。
[GET] / v2 / labels

要求：

[PATCH] / v2 / players / {player_id} / discovery / profile {"discovery_scope"：{"from_labels"：["a77e7871246c4d20b923b4559822d5be"]}}}}

応答：
