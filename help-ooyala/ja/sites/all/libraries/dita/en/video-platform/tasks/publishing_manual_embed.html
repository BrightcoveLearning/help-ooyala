バックスロットでのプレーヤー埋め込み設定

プレーヤーを作成した後（参照 Backlotでプレイヤーを作成する）、埋め込み設定を構成できます。 このプレーヤーに関連付けられているすべての動画に変更が自動的に適用されます。
注意： あなたのウェブサイトのコードに残っているページレベルのプレイヤーパラメータはまだ優先されます。 見る プレーヤーのV4設定の優先順位.
プレーヤV4の場合も参照してください。 基本的な埋め込み情報 そして WebページへのプレーヤーV4の埋め込み.

プレイヤーの埋め込みサブタブ

Backlot UIでプレーヤーの埋め込み設定を構成するには、埋め込みサブタブに移動します。
  1. Backlotにログインします。 https://backlot.ooyala.com/.
  2. [管理]タブを選択します。
  3. リストから資産を選択します。
  4. 埋め込みタブを選択します。
このタブは2つのセクションで構成されています。
  • プレーヤーの設定
  • 埋め込みコードを生成する

設定するプレーヤーを選択してください

[プレーヤーの設定]の[カスタムプレーヤー]で、構成するプレーヤーをリストから選択します。 プレーヤーのバージョンによって、どの埋め込みコードのオプションを利用できるかが決まります。

プレーヤー設定を構成する

選択したプレーヤーの基本的な埋め込み設定を構成します。
設定 説明
プレイヤー（カスタムまたはデフォルト） 設定するプレーヤーを選択してください。
コンテンツが埋め込まれているパーマリンク コンテンツが埋め込まれているURL。
コンテンツID 選択したコンテンツのバックスロットID。
プレーヤーID 選択したプレーヤーのバックスロットID。

埋め込みコードの生成

選択したプレーヤーの埋め込みコードを生成します。
設定 説明
サイズ プレイヤーウィンドウのサイズ。 次のいずれかの設定
  • 敏感な （Player V4のみ） - 埋め込まれているWebコンテナに基づいて自動的にサイズ変更されます
  • プリセットサイズ - 選択したアセットのエンコードサイズを含む
  • カスタム
カスタムサイズを選択した場合は、寸法を指定します。
  • 高さ
  • 幅
  • アスペクト比を維持
自動再生
  • アセットが読み込まれたらすぐに自動的に再生を開始するには、（チェック）を選択します。
  • ユーザーがアセットの再生を開始するために[再生]ボタンをクリックするように要求するには、オフ（オフ）にします。
自動繰り返し
  • 再生が終了したらすぐに自動的にアセットの再生（ビデオのみ）を開始するには（オン）を選択します。
  • ユーザーが[再生]ボタンをクリックしてアセットの再生を再開するように要求するには、オフ（オフ）にします。
埋め込みコードの種類 リストボックスから埋め込みコードの種類を選択します。 利用可能なオプションは、選択したプレーヤーに関連付けられているプレーヤーのバージョンによって異なります。
  • V4 HTML5標準プレーヤー埋め込みコード（推奨）:

    以下のプラグインを自動的に含みます。

    • コアプレーヤー（core.min.js)
    • プレイヤースキン（html5-skin.min.js, html5-skin.min.css） ご了承ください skin.json 自動的には含まれないため、BacklotからプレーヤーのUI設定を読み取ることができます。
    • HLSとDASH用のBitmovinプラグイン（bit_wrapper.min.js)
    • HLSとMP4のメインビデオプラグイン（main_html5.min.js)
    • 広告プラグイン（Pulse、Google IMA、Freewheel、VAST）
    • ディスカバリープラグイン（discovery_api.min.js)
    この埋め込みコードを選択すると、これらのプラグインをWebページに追加しなくなります（重複したプラグインの埋め込みにより予期しない結果が生じる可能性があるため、現在インラインの場合は削除する必要があります）。 他のすべてのプラグイン（たとえば、分析プラグイン）の場合は、Webページにプラグインを明示的に追加する必要があります。
  • V4 HTML5プレーヤー埋め込みコード（詳細） プラグインは含まれていません。 Webページで使用するすべてのプラグインを指定しなければなりません。
    注意： これは、Maylot 4、Backlotの31より前に利用可能になったPlayer V2017埋め込みコードと同じです。
  • V4 HTML5プレーヤーiFrame：プレーヤーをHTML iFrameに埋め込むときにこのオプションを選択します。 詳しくは、 プレーヤV4をHTML iframeに埋め込む.
  • V3 HTML埋め込みコード or V3 HTML（HTTPS）埋め込みコード：プレイヤーV3のみ。 どちらもURLに "//"を使用します。
  • V2 Flash埋め込みコード（非推奨） or V2フラッシュ（HTTPS）埋め込みコード（廃止予定）：プレイヤーV2のみ。 どちらもURLに "//"を使用します。
注意： Flash埋め込みコードを使用すると、iOSデバイスはビデオを見ることができなくなります。 ビデオをAppleデバイスで表示できるようにするには、HTML埋め込みコードを使用してください。
注意： オーディオアセットにはHTMLが必要です。
生成された埋め込みコード このタブでの他の選択に基づいてBacklotが生成するコード（HTMLフォーマット）を埋め込みます。 クリック コピー 埋め込みコード全体をクリップボードにコピーするためのボタン。 公開するページにコンテンツを貼り付けます。 HTMLにパブリッシュする場合は、コンテンツがHTML内にあることを確認してください。 <BODY> 素子。 Player V4については、 WebページへのプレーヤーV4の埋め込み.
