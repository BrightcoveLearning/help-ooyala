API Requests: v2アナリティクス and v3 Analytics Comparison (Deprecated)

これらの例は、いくつかの一般的に使用されるクエリを示しています。 v2アナリティクス v3 Analytics（およびそれに相当するもの）ウーヤラIQ).
警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、
読みやすくするために：
  • v3 クエリパラメータ値はURLエンコードされていません。
  • 変数値の前には$記号が付きます。
  • 一部の要求は複数行にまたがっています。
  • "＆metrics = metrics" デフォルトではすべてのメトリックが返されるため、構文から除外されます。 レポートに返す特定の指標を定義する "＆metrics = yourDesiredMetrics".
ここでの例は次のとおりです。
  • 総寸法
  • 1日の内訳の合計
  • ジオディメンション
  • デバイスとプラットフォームの寸法

総寸法

v2のURL 構文 v3 同等の 
/ v2 / analytics / reports /アカウント/パフォーマンス/合計/ $ date_range
 
/ v3 / analytics / reports /？report_type =パフォーマンスと開始日= $開始日と終了日= $終了＆api_key = yourApiKey
https://api.ooyala.com/v3/analytics/reports/?report_type=performance&start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&api_key=yourApiKey&expires=yourExpiration&signature=yourSignature

1日の内訳の合計

v2のURL 構文 v3 同等の 
/ v2 / analytics / reports /アカウント/パフォーマンス/合計/ $ date_range？breakdown_by = day
 
/ v3 / analytics / reports /？report_type =パフォーマンスとtime_segment = day＆start_date = $ start＆end_date = $ end＆api_key = yourApiKey
https://api.ooyala.com/v3/analytics/reports/?report_type=performance&time_segment=day&start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&api_key=yourApiKey&expires=yourExpiration&signature=yourSignature

ジオディメンション

v2のURL 構文 v3 同等の 
/ v2 /分析/レポート/資産/ $ asset_id /パフォーマンス/国/ $国/地域/ $地域/ $ date_range
 
 / v3 / analytics / reports /？report_type =パフォーマンスとディメンション=資産、国、地域とフィルタ=資産== 'asset_id'、国== '国'、地域== '地域'＆start_date = $ startとend_date = $ end＆api_key = yourApiKey
https://api.ooyala.com/v3/analytics/reports/?report_type=performance&dimensions=asset,country,region&filters=asset==’asset_id’, country==’country’,region==’region’&start_date=YYYY-MM-DD&end_date=YYYY-MM-DD &api_key=yourApiKey&expires=yourExpiration&signature=yourSignature

デバイスとプラットフォームの寸法

v2のURL 構文 v3 同等の 
/ v2 / analytics / reports / asset / $ asset_id / performance / device_types / $ device_type / platform / $ platform / $ date_range
 
/ v3 / analytics / reports /？report_type =パフォーマンスとディメンション=資産、device_typeとフィルタ=（（device_type == '$ device_type'））AND（（asset_id == '$ asset_id'））＆start_date = $ startとend_date = $ end＆api_key = yourApiKey
https://api.ooyala.com/v3/analytics/reports/?report_type=performance&dimensions=asset,device_type&filters=%28%28device_type%3D%3D%27device_type%27%29%29+AND+%28%28asset_id%3D%3D%27$asset_id%27%29%29 &start_date=YYYY-MM-DD&end_date=YYYY-MM-DD &api_key=yourApiKey&expires=yourExpiration&signature=yourSignature
