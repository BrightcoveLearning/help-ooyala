プレーヤV3メッセージバスイベント（非推奨）

メッセージバスイベントを使用して、動画から広告の再生までのプレーヤイベントを購読または発行します。

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).

すべてのOoyala APIは、 Ooyala APIドキュメント.

プレーヤJavaScript APIとプレーヤイベントは、次のURLで表示されます。 プレーヤJavaScript APIリファレンス.

プレーヤーイベントは、イベントバスによって発行される既定のイベントです。 あなたのモジュールはこれらのイベントのどれでもそして全てを購読することができます。 メッセージバスによって発行されるこれらすべてのデフォルトイベントのリストを取得するには、メッセージバスを使用できます。 申し込む（） 関数： 
player.mb.subscribe（ "*"、 'example'、function（eventName）{}）;
注：プレーヤー用のプレーヤーAPIとイベント（FlashとHTML5の両方）を使用する必要があります。 既存の既存のOoyalaプレーヤーAPIがある場合、それらは現在のOoyala V3プレーヤーによって「ラップ」されているOoyala Flashプレーヤーに対して引き続き機能することができます。
