Android用のカスタムコントロールの実装

Androidの再生用にカスタムコントロールでプレーヤーを設定する方法を学びます。

OoyalaのAndroid SDKには、デフォルトのフルスクリーンコントロールとインラインコントロールをオーバーライドするためのツールが用意されています。 次の手順では、カスタムコントロールを実行する方法について説明します。 OoyalaPlayer.

注意： あなたが使用している場合 OptimizedOoyalaPlayerLayoutController インラインコントロールをオーバーライドするだけです。 詳しくは、Ooyala Mobile SDK for Android APIの資料を参照してください。
  1. Ooyala Mobile SDKフォルダには、Default Controlsという名前のディレクトリが含まれています。 このフォルダから次のファイルをプロジェクトのソースフォルダにコピーします。
    • Images.java
    • OptimizedOoyalaPlayerLayoutController
    • DefaultOoyalaPlayerInlineControls.java
  2. 次のコントロールファイルの名前を変更して、カスタムフルスクリーンコントロールとインラインコントロールのコードベースを作成します。
    • OptimizedOoyalaPlayerLayoutController.java （例えば MyLayoutController.java)
    • DefaultOoyalaPlayerInlineControls.java MyInlineControls.java)
  3. ベースコードを編集して、カスタムのインラインコントロールとフルスクリーンコントロールを作成します。
    注意：

    カスタムコントロールの実装の詳細については、Ooyala Mobile SDK for iOS APIのドキュメントを参照してください。 サンプルアプリリポジトリでカスタムコントロールの例を見ることができます。

    Ooyala AdvancedPlaybackSampleApp

  4. インスタンス化 OoyalaPlayerLayout プロジェクトアクティビティコードで、新しいカスタムコントロールファイルを追加します。 この手順は、次のコード例に示されています。 
     OoyalaPlayerLayout playerLayout =（OoyalaPlayerLayout）findViewById（R.id.ooyalaPlayer）; MyLayoutController playerLayoutController = new MyLayoutController（playerLayout、PCODE、new PlayerDomain（DOMAIN））; player = playerLayoutController.getPlayer（）; MyInlineControls ic = new MyInlineControls（player、playerLayout）; playerLayoutController.setInlineControls（ic）;

    これで、カスタムコントロールがOoyala Playerで使用されます。 これを確認するには、カスタムコントロールにビューを追加/削除するか、ログを追加します。
