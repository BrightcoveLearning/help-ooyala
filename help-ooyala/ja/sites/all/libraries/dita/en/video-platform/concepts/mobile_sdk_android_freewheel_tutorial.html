Android上のFreeWheelサンプルアプリの実例を見る

Use the following steps to configure and run the FreeWheel sample app..
  1. Get the SDK and open the sample app in your Eclipse environment.
    1. Download the Ooyala FreeWheel SDK for Android (available at Ooyala ダウンロード).
    2. 解凍する OoyalaFreeweheelSDK-Android.zip ファイル。
    3. サブディレクトリに移動します OoyalaFreeWheelSDK-Android/FreeWheelSampleApp.
    4. Open the following file in Eclipse: 
      src/ooyala/com/android/freewheelsampleapp/FreeWheelSampleAppActivity.java
  2. 更新する FreewheelSampleAppActivity class to include your own PCODE. This is the Ooyala-supplied provider ID code. For details, see あなたのAPIクレデンシャル.

    The following code snippet shows the PCODE variable to modify:

    public class FreewheelSampleAppActivity extends Activity {
  final String PCODE  = "XXXXXXXXXXXXXXXX";
  final String DOMAIN = "http://www.ooyala.com";
...
    注意： 出て DOMAIN 変更なし http://www.ooyala.com。 この定数は以下と組み合わせて機能します。 シンジケーション制御 （公開ルール） バックスロット。 ビデオにインターネットドメイン制限を設定している場合 バックスロット （参照 バックスロット ユーザーガイド）、ここでの定数はそれらの許可されたドメインの1つに設定することができます。 設定していない場合 シンジケーション制御定数は効果がありません。
  3. 更新する embedMap hash with your FreeWheel assets. 
    embedMap = new HashMap < String, String>();
  embedMap.put("Freewheel Preroll", "XXXXXXXXXXXXXXXX");
  embedMap.put("Freewheel Midroll","XXXXXXXXXXXXXXXX");
  embedMap.put("Freewheel Postroll","XXXXXXXXXXXXXXXX");
  embedMap.put("Freewheel PreMidPost","XXXXXXXXXXXXXXXX");
  4. 変更を保存します。
  5. から ラン メニューを選択 ラン. The Android Virtual Device (AVD) selector appears.
  6. Select a virtual device or attach real hardware to your system.
  7. Click the play button to view the video.
