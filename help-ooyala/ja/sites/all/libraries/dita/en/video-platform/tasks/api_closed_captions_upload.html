クローズドキャプションファイルのアップロードと表示

After preparing a DFXP (now TTML) closed captions file, you can upload it to バックスロット.

注意： If you need to edit an already uploaded closed captions file, change it on your computer and then upload the updated file.

To upload a closed captions file:

  1. Create a DFXP (now TTML) closed captions file.
  2. To add closed captions, use PUT with the / v2 / assets route, the asset ID, and the / closed_captions 修飾子 The following example adds closed captions to the しぶき 資産： 
    [PUT]/v2/assets/JxbzdkMjqB/closed_captions
   <DFXP file contents>

    バックスロット 200応答を返します。

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  3. To view the contents of the closed captions file, use のGET とともに / v2 / assets ルート、アセットID、および / closed_captions 修飾子 The following example gets closed captions for the しぶき 資産： 
    [GET]/v2/assets/JxbzdkMjqB/closed_captions

    You are redirected to the file through a HTTP 302 redirect. The location header contains the redirect URL.

    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
