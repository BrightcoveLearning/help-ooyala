Player V4スキンとReact Application Framework

スキンコードをフォークしてより複雑なカスタマイズを行う予定の場合は、OoyalaとFacebookのReactアプリケーションフレームワークとの統合方法を理解する必要があります。

ReactとWebアプリケーション

OoyalaプレイヤーV4 for HTML5 標準を使用します 反応する 動的な単一ページのWebアプリケーションを構築するための実装。 Player V4は、コントローラとコンポーネントを使ってReactのコード構造を実装します。 主なロジックを提供するコントローラーは、すべてのネットワーク呼び出し、状態管理、およびデータベースからのアイテムの取得を行います。 コントローラは他の関連コンポーネントに状態を渡します。 コンポーネントは高度な機能を持つ抽象化された要素です。 これらはUI要素と関連ロジックから構成されています。 例としては、HTMLプレーヤコントローラバー、スタート画面、およびウィジェットがあります。 導入については、Reactのを参照してください。 こんにちは世界 ドキュメンテーション。
