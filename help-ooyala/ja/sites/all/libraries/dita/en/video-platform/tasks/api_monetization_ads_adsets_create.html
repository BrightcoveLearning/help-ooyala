広告セットを作成する

広告セットはいくつでも作成できますが、1つの動画に割り当てることができる広告セットは1つだけです。

注意： Ooyalaでホストされている広告（広告ソースが "My Ads"に設定されている）では、インストリーム広告とミッドロール広告は現在サポートされていません。 [広告ソース]が[マイ広告]に設定されている広告セットの広告位置として[インストリーム]または[ミッドロール]を選択しないでください。

広告セットを作成するには

  1. 使用 / v2 / ad_sets POST本文に含める広告の詳細を使用してルーティングします。 次の例では、広告セットを作成し、ユーザー自身のアセットから01M2F0MTr3SWBB9dfAPT5xEsg8Gpyアセットをそれに関連付けます。 
    [POST] / v2 / ad_sets {"名前"： "新しい広告セット"、 "デフォルト"：true、 "広告"：[{"ad_source"： "資産"、 "ad_type"： "preroll"、 "embed_code" ： "01M2F0MTr3SWBB9dfAPT5xEsg8Gpy"、 "plays_before"： "0"、 "plays_between"：1、 "click_url"、 "http://www.myclickurl.com"、 "トラッキング" [pixel] [track] pixel [pixel] [track] pixel [pixel] [track] pixel [pixel] }]}
    バックスロットUI 以下と同様に応答します。 
    {"名前"： "私の新しい広告セット"、 "ID"： "77506f70fc1e4b08805687980cc4a24d"、 "広告"：[]}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
  2. とPATCHを使用して、広告セットの名前を変更します。 / v2 / ad_sets 目的の広告セットのルートと識別子。 
    [PATCH] / v2 / ad_sets / 77506f70fc1e4b08805687980cc4a24d {"name"： "広告セット#9"、}
    バックスロットUI 以下と同様に応答します。 
    {"名前"： "広告セット#9"、 "ID"： "77506f70fc1e4b08805687980cc4a24d"、}
    注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
