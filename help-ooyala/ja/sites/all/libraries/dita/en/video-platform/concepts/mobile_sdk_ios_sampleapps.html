iOS Mobile SDKのサンプルアプリケーション

Mobile SDKサンプルアプリケーションを使用して堅牢なモバイルアプリケーションを作成する方法を学びます。

SDKサンプルアプリはXcodeと互換性があり、次のリポジトリで入手できます。

OoyalaサンプルアプリGit Hub
注意： かつてマスターブランチと呼ばれていたものは、現在 "安定"と呼ばれています。 Ooyalaがサンプルアプリのリリースまたは修正プログラムを公開しない限り、安定版ブランチは変更されません。 リポジトリをすでに複製またはフォークしている場合は、ブランチ名をstableに変更する必要があります。
表1 サンプルアプリケーション
サンプルアプリケーション 説明
高度な再生 このアプリには、プレーヤレイアウトの変更、カスタムプレーヤコントロールの作成、およびサードパーティの広告プラグインを作成するためのAPIを示すコードが含まれています。
基本的な再生 SDKを説明するための基本的なプログラム。 見る はじめに：iOSサンプルアプリケーション.
完全なサンプル これは、カスタムナビゲーション、さまざまなパラメータに基づくビデオのロード、さまざまなビデオ再生オプションなど、最も充実した機能を備えた複雑なサンプルです。
コンテンツ保護 このアプリは、利用可能なコンテンツ保護機能の一例です。
ウーヤラパルス このアプリは、広告配信のためにOoyala SDKがOoyala Pulseと連携して機能する方法を示しています。 このサンプルアプリケーションは、次の場所にあります。 iOS用Ooyalaプレーヤーv4 Pulse SDK 2.xの統合.
フリーホイール このアプリは、Ooyala SDKがFreeWheelと連携して機能する方法を示しています。 このサンプルアプリケーションは、Ooyala SDK for FreeWheel iOS zipファイルに含まれています。 Ooyala ダウンロード.
IMA このアプリは、Ooyala SDKがGoogle IMAと連携してどのように機能するかを示しています。 このサンプルアプリケーションは、Google IMA iOS用のOoyala SDKに含まれています。 Ooyala ダウンロード).
NPAWの統合 このアプリは、iOS用のNPAW統合の実装を示しています。 見る https://github.com/ooyala/ios-sample-apps/tree/stable/NPAWSampleApp.
ウーヤラAPI このアプリは、Channel Content APIの実装を示しています。
オプション このアプリは、 OOOptions 設定用オブジェクト キューポイントマーカー そして iOSアプリのTVレーティングを有効にする.
