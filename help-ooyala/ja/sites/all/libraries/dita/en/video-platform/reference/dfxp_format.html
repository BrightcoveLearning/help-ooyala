サポートされているクローズドキャプションDFXP（現在はTTML）形式

Closed caption files that you upload to Backlot must be in a supported format. The DFXP format (now TTML) is a popular supported format.

注意： Ooyala supports other closed caption formats. For details, see サポートされているクローズドキャプション.
注意： The Ooyala Web Player utilizes WebVTT. If a TTML file is uploaded, the Ooyala Player will utilize a WebVTT file that is converted from the TTML file, which will remove styling.

Supported DFXP Format

注意： DFXP (now TTML) is used as a familiar term for closed caption formats. However, the specification is now called TTML.
Among the DFXP formats, Ooyala supports only the DFXP v2006 format. For more information about the DFXP format that Ooyala supports, go to the official DFXP site at http://www.w3.org/TR/2006/WD-ttaf1-dfxp-20060427/.
注意： Ooyalaモバイルアプリ（Ooyala iOSおよびAndroid SDKを使用して開発されたアプリ）で再生するには、TTML（以前のDFXP）形式が必要です。 Player V4 Webは、iOSおよびAndroidデバイスのブラウザを含むすべてのブラウザ環境で、追加の字幕フォーマットをサポートしています。

デフォルトのスタイリングプロパティ

Flashプレーヤーの場合、DFXPファイルの次のスタイルプロパティがデフォルトとして使用されます（DFXP字幕ファイルで指定されている場合）。 OoyalaはDFXP v2006フォーマットのみをサポートします。
  • Language
  • テキストの配置
  • テキストの背景色（tts：backgroundColor)
    注意： テキストの背景は、ttsとペアになっている場合にのみデフォルトで表示されます。showBackground = "always"
  • テキストの色 （tts：色)
  • フォントサイズ （tts：fontSize)
  • フォントファミリー （tts：fontFamily)
  • フォントスタイル（tts：fontStyle)
  • フォントの太さ（tts：fontWeight)
  • 不透明度、すなわちテキストボックスの背景またはウィンドウの背景に適用可能（tts：不透明度)
注意： テキスト方向（tts：方向）、Flashは言語に基づいてテキストの方向を自動的に調整します。

Supported DFXP Tags

Ooyala supports all tags from the 2006 DFXP standard, listed below. Other tags are not supported.

タグ 例 追加情報
カラー tts:color="green" Ooyala only supports this in the top-level tag that wraps all other captions.
fontSize tts:fontSize="18"
textOutline textOutline="green 2px 0px" Applies at the paragraph level only, not in subtags.
fontFamily tts:fontFamily="serif" Supports fonts installed only by the end-user or generic fonts.
textAlign tts:textAlign="right" Re-aligns within the closed caption box only. This does not move the box.
fontStyle tts:fontStyle="italic" Oblique style is not supported.
fontWeight tts:fontWeight="bold"
lineHeight tts:lineHeight="32"
不透明度 tts:opacity="0.50" Sets opacity of the background bounding box color.

キャッシュされた設定がないブラウザでは、プレーヤーとアセットにDFXP設定がロードされると、上記のリストにマークされているすべての属性がDFXPファイルで提供されているデフォルト設定で適用されます。 エンドユーザーがUIインターフェイスを介してCC設定を変更すると、DFXP設定はすべての設定に対して上書きされ、プレーヤーのデフォルトおよびユーザー設定に置き換えられます。 Flashプレーヤーのデフォルトは以下の表のとおりです。

DFXPスタイリングプロパティ デフォルトのFlash Player設定 追加情報

テキストの配置 センター 有効な値 左 | センター | 右
テキストの背景色（tts：背景) なし
テキストの色 （tts：色) 白

DFXP Example

The following is an example of a DFXP (now TTML) file with seven captions in English (xml:lang="en") and one caption in French (xml:lang="fr").

注意： On the first line or in the appropriate <DIV>s, be sure to change the value of the xml:lang attribute to match your target language; for example, change en 〜へ de for German.
<?xml version="1.0" encoding= "UTF-8" ?>
<tt xml:lang="en" xmlns="http://www.w3.org/ns/ttml" xmlns:tts="http://www.w3.org/2006/04/ttaf1#styling">
<head>
  <styling>
    <style xml:id= "defaultCaption"
      tts:fontSize= "12"
      tts:fontFamily= "sansSerif"
      tts:fontWeight= "normal"
      tts:fontStyle= "normal"
      tts:color= "white"
      tts:backgroundColor= "black"
      tts:showBackground= "always"
      tts:textAlign= "left"
    />

    <style xml:id= "styleTwo"
      tts:fontFamily= "serif"
      tts:color= "blue"
    />

    <style xml:id= "styleThree"
      tts:color= "black"
      tts:backgroundColor= "white"
      tts:fontStyle= "italic"
    />

    <style xml:id= "styleFour" style= "styleTwo"
      tts:textOutline= "red 2px 0px"
    />

    <style xml:id= "styleFive" style= "styleTwo"
      tts:color= "red"
      tts:fontWeight= "bold"
    />

  </styling>
</head>
<body xml:id= "thebody">
  <div style= "defaultCaption" xml:lang= "en">
    <p begin= "0:00:00.00" end= "0:00:04.00">This is my 1st caption</p>
    <p begin= "0:00:05.00" end= "0:00:09.00" style= "styleTwo">This is my 2nd caption</p>
    <p begin= "0:00:10.00" end= "0:00:14.00" style= "styleThree">This is my 3rd caption</p>
    <p begin= "0:00:15.00" end= "0:00:19.00" style= "styleFour">This is my 4th caption</p>
    <p begin= "0:00:20.00" end= "0:00:24.00">This is my 5th caption
      <span style= "styleFive">This is a sub-caption</span>
    </p>
    <p begin= "0:00:25.00" end= "0:00:29.00">This is my 6th caption</p>
    <p begin= "0:00:25.00" end= "0:00:29.00" style= "styleTwo">This is my 7th caption</p>
  </div>
  <div style= "defaultCaption" xml:lang= "fr">
    <p begin= "0:00:00.00" end= "0:00:34.00">Bonjour</p>
  </div>
</body>
</tt>

Closed Caption Validator

We are also providing a Ooyala クローズドキャプションバリデータ to check your DFXP/TTML documents for your convenience. However, please note that we provide Ooyala-specific DFXP support. Even if you follow the w3c document definition for DFXP/TTML, it might not be supported. If you run into any issues with your DFXP/TTML files, check with your Ooyala representative or Technical Support.
