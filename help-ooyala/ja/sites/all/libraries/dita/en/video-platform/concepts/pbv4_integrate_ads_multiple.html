複数の広告マネージャを統合する

1ページに複数の広告プラグイン（Freewheel、VAST、Google IMAの広告プラグインのみ）をロードして使用できます。 たとえば、3つすべてがページにロードされていて、そのページに複数の埋め込みプレーヤーがある場合、プレーヤーAとプレーヤーBは両方ともIMAを使用でき、プレーヤーCはフリーホイールを使用でき、プレーヤーDはVASTを使用できます。
注意： Pulseプラグインを使用したい場合は、代わりにiFrameを使用することを検討してください。 あなたがpulse pluginをロードして、それに広告を与えないならば、あなたは予想外の行動を得る。
見る 広告マネージャフレームワーク 概要と 広告を統合する 紹介のために。

ステップ1：ページにプラグインをロードする

以下の要件を考慮してください。
  • フリーホイール広告、VAST広告、およびGoogle IMA広告プラグインのみを使用している限り、1つのページに複数の広告プラグインを読み込むことができます。
  • プラグインはすべて同じプレーヤV4バージョンであるべきです。
  • 1ページにロードできるのは、同じ広告プラグインの1つのバージョンだけです。 たとえば、同じページに2つの異なるバージョンのFreewheelプラグインをロードすることはできません。
  • そのページで使用する予定のプラグインのみをロードしてください。
広告プラグインをロードするには、（ロード後に）次のスクリプトを追加します。 core.min.jsプレーヤーをロードしているページに）。 たとえば、3つの広告プラグインをすべてロードするには、次のようにします。 
<script language =“ javascript "src ="url_where_hosted/freewheel.min.js "> </script> <スクリプト言語=“ javascript" src = "url_where_hosted/ad_manager_vast.min.js "> </script> <スクリプト言語=“ javascript" src = "url_where_hosted/google_ima.min.js "> </script>
交換します url_where_hosted リソースをホストしているURLを指定します。

ステップ2：ページへのプレイヤーのロードと設定

以下の要件を考慮してください。
  • ページ上の複数の埋め込みプレーヤーが同じ広告プラグインにアクセスできます。
  • ただし、埋め込みプレーヤーごとに使用できる広告プラグインは1つだけです。 たとえば、プレーヤーAとプレーヤーBはIMAのみを使用でき、IMAとVASTは使用できません。
該当する広告マネージャをページで有効にしたら、次のことができます。 一般的な広告設定を構成する 埋め込まれたパラメータを使って。 次の説明に従って、広告マネージャ固有の設定を構成することもできます。
  • VAST広告とVPAID広告の統合
  • フリーホイール広告の統合
  • Google IMA広告の統合
