HTML5プレーヤーV3 - Supported Browsers

HTML5プレーヤーV3 works with the following browsers on all Ooyala supported devices:

ブラウザ Player Support

  • Prior Firefox versions: Flash playback only

マイクロソフトエッジ（Windows 10） Flash and HTML5 playback (MP4).

HTML5プレーヤーV3 - Supported Platforms

  • Windows Desktop - Windows 10, Windows 8, Windows 7; Internet Explorer v9+, Microsoft Edge for Windows 10
