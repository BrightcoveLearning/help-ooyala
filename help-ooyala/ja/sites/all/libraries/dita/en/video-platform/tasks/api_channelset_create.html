チャネルセットの作成（廃止予定）

チャンネルセットを作成したら、チャンネルを追加できます。

注意： チャンネルとチャンネルセットは廃止予定で、無効になる予定です。 詳細と代替案については、 OVPリリースノート.
注意： Backlot REST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.

チャンネルセットを作成するには：

用途 POST そして / v2 / assets とともに asset_type に設定 チャンネルセット リクエストボディ内 次の例では、 "My Channel Set"チャンネルセットを作成しています。 
[POST] / v2 / assets {"name"： "マイチャンネルセット"、 "asset_type"： "channel_set"}

バックスロット 次のような応答を返します。

{"asset_type"： "channel_set"、 "期間"：0、 "名前"： "マイチャンネルセット"、 "preview_image_url"：null、 "created_at"： "2011-08-10T18：20：03：00" 、 "embed_code"： "IwYTVxMjqLtYHmwqy"、 "time_restrictions"：null、 "updated_at"： "00-2011-08T10：18：20 + 04：00"、 "external_id"、 "null"、 "元のファイル"、 "hostname"。 ：null、 "説明"：null、 "ステータス"： "ライブ"}

チャネルセットは正常に作成されました。
