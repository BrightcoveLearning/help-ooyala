アクセスキーの作成

特定のシンジケーションに対してアクセスキーを作成できます。

アクセスキーを作成するには

使用 / v2 / syndications / syndication_id / access_keys アクセスキーの名前を指定します。 次の例では、 "My Access Key"アクセスキーを作成しています。 
[POST] / v2 / syndications / b839b27a4e844a5c9 / access_keys {"名前"： "私のアクセスキー"}
バックスロット 次のような応答を返します。 
{"名前"： "自分のアクセスキー"、 "access_key"： "12345678-abcd-abcd-12ab-12345678"}
