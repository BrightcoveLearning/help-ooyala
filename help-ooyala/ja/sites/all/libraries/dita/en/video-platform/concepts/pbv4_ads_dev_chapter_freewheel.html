フリーホイール広告プラグイン

使用 freewheel.js ad plugin to load FreeWheel ads in Player V4. FreeWheel is an ad server and ad management platform. Using FreeWheel, publishers are able to segment their ad inventory, define ad targeting rules, and take advantage of the FreeWheel MRM network capabilities. The Ooyala Ad Manager Controller (part of the 広告マネージャフレームワーク) handles when ads play and how video ads are rendered for FreeWheel ads. FreeWheel controls overlay ads.

Ooyala supports the FreeWheel ad manager on the following platforms for Player V4 for this release:
  • HTML5 web
  • iOSモバイルSDK

サポートされている広告掲載順位

Ads served through the FreeWheel ad server (VAST XML) are supported on V4 HTML5 players and the iOS and Android mobile SDKs for the following ad positions:
  • プリロール
  • ミッドロール
  • ポストロール
  • スキップ可能
  • クリックスルー
  • ポッド
  • オーバーレイ

広告とFirefoxのプライベートブラウジング

使用している消費者のために プライベートブラウジングウィンドウ Firefoxでは、プライバシーモードが特定の種類のトラッキングURLをブロックするため、一部の広告が読み込めないことがあります。
