プレーヤV3のプレーヤスクラバーイメージのアップロード

You can upload an image to show as viewers scrub through a video for Player V3.

注意： Ooyalaプレーヤー用のREST APIコマンドについて詳しくは、以下を参照してください。 バックスロットAPIリファレンス.
重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).

Upload Scrubber Image

[PUT] / v2 / players /player_id/scrubber_image
   <file_contents>

Get Scrubber Image for a Player

[GET] / v2 / players /player_id/scrubber_image

Remove Scrubber Image from a Player

[削除] / v2 / players /player_id/scrubber_image

ルート属性

次の表は、ルートを通じて表現できるすべての属性を示しています。

ルート属性 説明
player_id

The ID of the player.

タイプ：文字列

デフォルト：なし

Example: /players/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

例

This example uploads a scrubber image for a player: 

[PUT]/v2/assets/IzNnllMjphu2XF3_UgPROoCi9B2rtWs/scrubber_image

バックスロット 200応答を返します。

注意： Ooyala Scratchpadでアカウントの認証情報を使用してコードサンプルを試してみてください。 スクラッチパッドを起動するには、にアクセスしてください。 Ooyala APIスクラッチパッド。 スクラッチパッドの使用方法については、 スクラッチパッドを使った練習の依頼.
