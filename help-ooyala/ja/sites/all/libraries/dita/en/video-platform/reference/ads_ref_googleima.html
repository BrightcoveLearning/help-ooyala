プレーヤV3のGoogle IMA Ads埋め込みパラメータ（廃止予定）

重要： Ooyala Player V3は非推奨であり、予定されている予定です。 無効 2018-01-31について その日以降、Player V3はあなたのビデオまたはオーディオコンテンツを再生しなくなります。 まだPlayer V3を使用しているお客様は、Player V4に移行する必要があります（を参照）。 Player V3からの移行 そして プレイヤー移行に関するFAQ).
埋め込みパラメータを使用してGoogle IMA広告サービス関連のタグをプレーヤに渡すことができます。
注意： OoyalaのGoogle IMA広告マネージャを使用するには、サポートセンターにログインして、この広告ソースをアカウントに追加するように要求するチケットを送信してください。 有効にすると、プレーヤー用の広告セットを作成できるようになります。
広告の掲載順位は、デスクトップデバイスとモバイルデバイスでは多少異なります。 見る クライアントサイド広告統合 プラットフォーム別の広告掲載サポートの包括的なリストについては。

Google IMAパラメータ

使用 google-ima-ads-manager Google IMA広告サーバーまたはネットワークタグをOoyalaプレーヤーに渡すためのパラメータ。 このパラメータは、広告サーバーまたはネットワークのURLを表すキーと値のペアを、次に説明する他のパラメータと共に受け取るコンテナです。

adTagUrlパラメータをBacklot URLを上書きするように設定します。

のパラメータの1つ google-ima-ads-manager 次の形式のキーと値のペアです。 adTagUrl： "いくつかのURL".

有効な値： 任意のURL

例えば： 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、 "google-ima-ads-manager"：{adTagUrl： "some url"、 ...}、}）;
キー値 "some url"をレスポンスを含む実際のgoogle-ima広告タグに変更する必要があります。 adTagキーと値のペアのこのURL値は、広告サーバーまたはネットワークのURLを表します。 このadTag URLはBacklotのものを上書きします。 例えば：
adTagUrl： "http://myAdServer.com/myGoogle-ima-ads-manager"
Backlotから取得したURLの末尾に追加のadTagパラメータを追加できます。

Flashプレーヤーでノンリニア広告の[閉じる]ボタンを非表示にするには、useStyledNonLinearAdsパラメータを設定します。

注意： このパラメータはFlash専用です。
あなたが設定できるもう一つのパラメータは useStyledNonLinearAdstrueに設定すると、ノンリニア広告の閉じるボタンが表示されます。 このパラメータはオプションです。デフォルトでは、広告を閉じるボタンはノンリニア広告の場合に表示されます。

有効な値： はい| 本当| いいえ| 偽

デフォルト： true

次の例に示すように、このパラメータをfalseに設定して広告を閉じるボタンを非表示にします。 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、 "google-ima-ads-manager"：{adTagUrl： "some url"、 useStyledNonLinearAds：false、...}、}）;

showInAdControlBarを使用して広告再生中にプレーヤーコントロールバーを表示または非表示にする

あなたが使用することができます showInAdControlBar 広告再生中にプレーヤーコントロールバーを表示/非表示にします。

有効な値： はい| 本当| いいえ| 偽

デフォルト： いいえ

注意： あり 'showInAdControlBar'とも呼ばれる一般的なプレーヤーパラメータ。 両方のshowInAdControlBarパラメータを同時に使用しないでください。 推奨される方法は、「google-ima-ads-manager」ハッシュの外側で一般プレーヤーのshowInAdControlBarパラメータを使用することです。
例えば： 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、 "google-ima-ads-manager"：{showInAdControlBar：true、... ;}}）;

showAdMarqueeを使用して広告再生中に広告マーキーを表示または非表示にする

用途 showAdMarquee 広告再生中に広告マーキーを表示または非表示にします。 falseまたはnoに設定すると、追加のadMarqueeは非表示になります。 true、yes、または省略に設定した場合、広告の再生中にadMarqueeが表示されます。

有効な値： 本当| 偽

デフォルト： true

例えば： 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、 "google-ima-ads-manager"：{showAdMarquee：true、... ;}}）;

playWhenAdClickを使用して広告がクリックされた後の広告再生動作を設定する

If playWhenAdClick [はい]に設定すると、クリックに関係なく広告の再生が続行されます。

有効な値： はい| 本当| いいえ| 偽

デフォルト： いいえ

注意： Flashの場合、広告コントロールなしでplayWhenAdClickがfalseに設定されている場合、ユーザーが広告をクリックしてリダイレクトされた後、ユーザーが2回目に広告をクリックしたときに広告の再生が続行されます。 これにより、広告の一時停止イベントと広告再生イベントの両方に対して広告URLが開くという望ましくない動作が発生することがわかります。 Google IMAチームと協力してこの問題を解決しています。 それ以外の場合、広告の再生は一時停止します。
例えば： 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、 "google-ima-ads-manager"：{playWhenAdClick：true、... ;}}）;

additionalAdTagParametersをAppend More Parametersに設定します。

あなたが使用することができます additionalAdTagParameters 人口統計ターゲティングを広告に追加するなど、広告処理に機能を追加する。 これらの追加された広告タグパラメータは、Backlotで広告タグがどのように設定されるかに依存します。 また、追加のadTagUrlパラメータのキーと値のペアはadTagUrlパラメータの一部であり、キーと値のペアを含む辞書の形式である必要があります。 google-imaがサポートしている任意の内部キー/値ペアを使用できます。 Google IMAのドキュメントには、このサイトのこれらのパラメータが記載されています。 https://support.google.com/dfp_premium/answer/1068325?hl=en。 これらのパラメータを使用すると、広告タグの最後に追加されます。 追加されたパラメータ BacklotからのURLの内容を上書きしないでください。
注意： 広告タグ URL BacklotのURLを上書きします。
これらのパラメータの使用を開始するには、次のように指定します。 
additionalAdTagParameters：{"parameter1"： "value1"、 "parameter2"： "value2"}

Backlotで、広告タグのURLを次のように指定した場合、http://www.example.com/example?param1=test結果として得られる要求は、次のようになります。 "www.example.com/example?param1=test&cust_params=env%3Dmobilevcmshost%26siteview%3D1%26pth%3Djsbin.com&vid=code&pod=2"

このリクエストは次の例のようになります。

"google-ima-ads-manager"：{additionalAdTagParameters：{cust_params： "env％3Dmobilevcmshost％26siteview％3D1％26pth％3Djsbin.com"、vid： "2"}

cust_params

この値ペアを使用すると、カスタムターゲティングパラメータをOoyalaプレーヤーに渡すことができます。 cust_paramsには任意の種類のキー値を設定できます。 例えば：

このパラメータはビデオアセットの一意の識別子であり、OoyalaビデオアセットIDの値（埋め込みコード）を使用する必要があります。 あなたはBacklotからあなたのビデオ埋め込みコードを得ることができます。
注意： この値は、DFPを使用してコンテンツメタデータをターゲティングするために必要です。
例えば：

この値を使用して、ビデオストリーム内の広告の位置を指定できます。

adRequestTimeoutパラメータ

動画の再生を続ける前に、adRequestTimeoutを設定してタイムアウトを設定します。 このパラメータはFlashおよびHTML5用です。

次の形式でキーと値のペアを使用します adRequestTimeout： "何らかの整数" 広告リクエストのタイムアウトを設定してビデオの再生を継続するのにかかる時間をミリ秒単位でカスタマイズします。 ミリ秒全体を使用することをお勧めします（例：2001ミリ秒の代わりに2000.5ミリ秒）。

有効な値： 1000以上の任意の整数

デフォルト： 3000

例えば： 
var videoPlayer = OO.Player.create（ "playerwrapper"、 "embed_code"、{高さ： "100％"、幅： "100％"、adRequestTimeout：1000 "google-ima-ads-manager"：{adTagUrl： "some url "、...}、}）;
