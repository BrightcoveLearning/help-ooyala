動的マニフェスト

Microsoft Azure Media Servicesを使用してアップロードされたアセットを保存している場合は、事前定義済みのフィルタを使用して、トランスコーディング用のカスタマイズされた動的なマニフェストを生成できます。 フィルタは、特定のデバイスで最高の視聴体験を提供するためにオーディオとビデオのレンディションのサブセットを指定するために最も一般的に使用されるサーバー側のルールです。 たとえば、アセットのトランスコード時にHDとSDのレンディションのみが作成されるように要求するようにフィルタを設定できます。

動的マニフェストは、視聴者が、フィルタが関連付けられているビデオのストリーミングを要求したときに作成されます。 これは他の種類のOoyalaマニフェストファイルと比較されます。これらのファイルはアセットと一緒にアップロードされ、すべての表示要求に対して同じままです（「 CSVマニフェストファイル or XMLマニフェストファイル） アセットは、静的マニフェストファイルと動的マニフェストフィルタの両方を持つことができます。

動的マニフェストは、アセットのデフォルトのマニフェストファイルで定義できるものよりも柔軟性が必要な場合に役立ちます。 動的マニフェストを使用すると、次のことができます。

  • さまざまな種類のデバイスで顧客の視聴体験を適切に制御
  • CDNの使用を減らす

動的目録の設定

動的マニフェストを有効にするには、Ooyalaのサポート担当者に連絡して必要なフィルタ条件を指定してください。 Ooyalaチームはあなたのアカウント用にフィルタを設定し、あなたにフィルタの名前を送ります。

以下の点から、希望する出力を指定できます。

  • ビットレート（シングルビットレートまたはビットレートの範囲）
  • オーディオストリーム専用のビットレート
  • ビデオストリーム専用のビットレート

フィルタの使用

フィルタを有効にするには、Ooyala PlayerをWebページに埋め込むとき、またはiOS SDKまたはAndroid SDKを呼び出してモバイルアプリでビデオを表示するときにフィルタ名を含めます。

例：Webプレーヤー

var playerParam = {"dynamicFilters"： "filterA、filterB、filterC"}; OO.Player.Create（DIV、embed_code、playerParam）;

例：Android SDK

List <String> dynamicFilters = Arrays.asList（ "filterA"、 "filterB"、 "filterC"）; Options options = new Options.Builder（）。setDynamicFilters（dynamicFilters）.build（）; player = new OoyalaPlayer（pcode、新しいPlayerDomain（ドメイン）、null、options）;

例：Swift付きiOS SDK

オプションをさせてください：OOOptions = OOOptions（）！ options.dynamicFilters = ["filterA"、 "filterB"、 "filterC"、 "filterD"] player = OOOoyalaPlayer（pcode：pcode、domain：ドメイン、embedTokenGenerator：nil、options：options）！

例：Objective-C付きiOS SDK

OOOptions * options = [OOOptions new]; options.dynamicFilters = [[NSArray alloc] initWithObjects：@ "filterA"、@ "filterB"、@ "filterC"、nil]; OOOoyalaPlayer * ooyalaPlayer = [[OOOoyalaPlayer alloc] initWithPcode：self.pcodeドメイン：[[OOPlayerDomain alloc] initWithString：self.playerDomain] options：options];

制限事項

1つの資産に複数のフィルターを含めることができます。資産ごとに最大3つ（3）のフィルターを含めることができます。

エラー処理

embedパラメータまたはSDK呼び出しのフィルタ名が定義済みのフィルタと一致しない場合（たとえば、コードに誤字がある場合など）、フィルタは適用されず、ビデオは通常、フィルタが要求されていない場合と同様にトランスコードします。
