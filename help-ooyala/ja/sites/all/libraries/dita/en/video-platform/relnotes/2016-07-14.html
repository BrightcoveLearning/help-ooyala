2016-07-14リリース

このリリースの注目すべき内容はOoyalaプレイヤーのバージョン4.5.7を含みます。

ソフトウェア機能リリース

プレーヤーV4リリース - バージョン4.5.7

このPlayerリリースの新機能

  • セキュアトークン用のAkamaiライブおよびリモートアセットのエンコードのサポート。
  • サーバーサイド広告挿入を使用した場合の広告要求に対するキャッシュ無効化のサポート。 '[CACHEBUSTING]'マクロを持つURLはランダムな文字列に置き換えられます。 この機能のデフォルト設定はtrueに設定されており、常にキャッシュバスティングを行います。 キャッシュバスティングを無効にするには、プレーヤパラメータに次の変数を設定する必要があります。 
    {"ssai-pulse-ads-manager"：{"cacheBuster"：false}}

デフォルトのプレイヤーリンク

以前のバージョンのPlayer V4から最新のPlayer V4バージョンに更新するには、次の手順を実行します。
  1. すべての新しいプレーヤーリソースをダウンロードしてホストします（コアプラグイン、 スキンプラグイン、CSS、フォント、および画像リソース, ビデオプラグイン, 広告プラグイン, 分析プラグインなど）を追加して、これらの新しいファイルを指すようにリソースを更新します。
  2. 独自のバージョンのskin.jsonを使用して独自のPlayerスキンをカスタマイズした場合は、最新のskin.jsonファイルをダウンロードし、このファイルの変更内容をカスタマイズしたskin.jsonファイルとマージします。
  • Core Player（必須）：http://player.ooyala.com/static/v4/stable / 4.5.7 / core.min.js
  • プレイヤーのスキン：
    • http://player.ooyala.com/static/v4/stable/4.5.7/skin-plugin/html5-skin.min.js
    • http://player.ooyala.com/static/v4/stable/4.5.7/skin-plugin/skin.json
    • http://player.ooyala.com/static/v4/stable/4.5.7/skin-plugin/html5-skin.min.css
  • ビデオプラグイン（少なくとも1つは必須）
    • HLSおよびMP4用のメインビデオプラグイン：http://player.ooyala.com/static/v4/stable / 4.5.7 / video-plugin / main_html5.min.js
    • HDS用のOSMF Flashプラグイン：http://player.ooyala.com/static/v4/stable / 4.5.7 / video-plugin / osmf_flash.min.js
    • HLSおよびDASH用のBitmovinプラグイン：http://player.ooyala.com/static/v4/stable / 4.5.7 / video-plugin / bit_wrapper.min.js
    • Akamai HDビデオプラグイン：http://player.ooyala.com/static/v4/stable / 4.5.7 / video-plugin / akamaiHD_flash.min.js
  • 広告プラグイン
    • Ooyala Pulse広告プラグイン：https://player.ooyala.com/static/v4/stable / 4.5.7 / ad-plugin / pulse.min.js
    • VASTおよびVPAID広告プラグイン：https://player.ooyala.com/static/v4/stable / 4.5.7 / ad-plugin / ad_manager_vast.min.js
    • Google IMA広告プラグイン：https://player.ooyala.com/static/v4/stable / 4.5.7 / ad-plugin / google_ima.min.js
    • FreeWheel広告プラグイン：https://player.ooyala.com/static/v4/stable / 4.5.7 / ad-plugin / freewheel.min.js

強化された利用API

VOD GB配信用の埋め込みコードレベルの内訳を含む、いくつかの新しいUsage APIが追加されました。これにより、埋め込みコードごとのVOD GB配信を毎日追跡することができます。 最新を見る 利用APIリファレンス 詳細については。

ドキュメントリリース

  • 埋め込みパラメータを使用したプレーヤV4広告設定の構成：新しいプレイヤーパラメータに関する詳細を追加しました ssai-pulse-ads-manager.cacheBuster.
  • 利用APIリファレンス：以下の新しいUsage APIを追加しました。 
    http://api.ooyala.com/v3/billing/usage/report/provider/{pcode}
http://api.ooyala.com/v3/billing/usage/reports/asset/{embed_code}/delivery/total/{date_range}
http://api.ooyala.com/v3/billing/usage/reports/account/delivery/total/{date_range}
