Androidアプリのパフォーマンスモニタリングを実装する

アプリケーションのパフォーマンスに関する統計レポートを作成したい場合は、 PerformanceMonitor APIです。

  1. Add the following import statement to the code for your class: 
    import com.ooyala.android.performance.PerformanceMonitor;
  2. 加える PerformanceMonitor field to your class: 
    private PerformanceMonitor performanceMonitor
  3. あなたのインスタンス化 PerformanceMonitor 使用して getStandardMonitor API： 
    performanceMonitor = PerformanceMonitor.getStandardMonitor( player )
  4. To generate a report after having set embed codes and played assets, build a statistics snapshot and call the generateReport API： 
    Log.d( TAG, performanceMonitor.buildStatisticsSnapshot().generateReport() );
見ます PerformanceStatisticsSnapshot 取得できる統計についての情報のためのAPI。
