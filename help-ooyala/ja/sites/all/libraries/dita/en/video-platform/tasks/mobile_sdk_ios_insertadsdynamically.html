動的に広告を挿入する

再生中に動画に広告を動的に挿入するようにアプリを設定するには、次の手順を使用します。
  1. で概説されている手順を使用する はじめに：Xcodeで環境を設定する プロジェクトフレームワークを作成します。
  2. Video Cloud Studioで ViewController.m Xcodeで編集するためのファイル。
  3. 次のインポート文を追加します。
    • #import <OoyalaSDK / OOVASTAdSpot.h>
    • #import <OoyalaSDK / OOManagedAdSpot.h>
    • #import <OoyalaSDK / OOManagedAdsPlugin.h>
    • #import <OoyalaSDK / OOOoyalaAdSpot.h>
    注意： <OoyalaSDK / OOVASTAdSpot.h> このチュートリアルのVAST実装のみが必要です。 <OoyalaSDK / OOOyalaAdSpot.h> このチュートリアルのOoyala Ad実装にのみ必要です。
  4. 加えます insertAd に機能性 notificationHandler.
    注意： insertAd 後に呼ばれなければならない OOOoyalaPlayerStateReadyそれ以外の場合、挿入された広告は実行時に削除されます。

    • VAST広告の場合、この手順は次のコード例で説明されています。

      重要： 次の例では、 VastAdURL あなたのコンテンツに適したURLを入力してください。
       - （void）notificationHandler：（NSNotification *）通知{NSString * name = notification.name; if（_ooyalaPlayerViewController.player.state == OOOoyalaPlayerStateReady）{
        [_ooyalaPlayerViewController.player.managedAdsPlugin insertAd：[[OOVASTAdSpot割り当て] initWithTime：@0 clickURL：nilトラッキングURL：nil vastURL：[NSURL URLWithString：@ "VastAdUrl"]]];}

    • Ooyala広告の場合、この手順は次のコード例で説明されています。

      重要： 次の例では、 OoyalaEmbedCode あなたのコンテンツに適した埋め込みコードを使って。
       - （void）notificationHandler：（NSNotification *）通知{NSString * name = notification.name; if（_ooyalaPlayerViewController.player.state == OOOoyalaPlayerStateReady）{   
        [_ooyalaPlayerViewController.player.managedAdsPlugin insertAd：[[OOOoyalaAdSpot]] initWithTime：@0 clickURL：nil trackingURL：nil embedCode：@ "OoyalaEmbedCode"

          api：_ooyalaPlayerViewController.player.api]];}
