v2 / v3 Fundamental Difference: Multidimensional Analysis (Deprecated)

v3 Analytics（ウーヤラIQ）、データは多次元的に格納され提示される。

警告： v2 Analytics APIは非推奨になりました。 見る ウーヤラIQアナリティクス 新しいv3 Analytics APIの詳細については、

In v2アナリティクス、データは一次元的に（すなわち、一次元的に）示されている。 これはつまり v2 一度に調査できるデータの種類は1つだけです。 たとえば、ドメイン別、地理別、デバイス別などです。

In v3アナリティクスただし、データは多次元で表示されます（つまり、1つ以上の次元）。 これが意味するのは、一度に複数のデータセットの分析を実行できるということです。 たとえば、地理的またはドメインタイプ別にのみデータを見るのではなく、 v3アナリティクス あなたは、地理学とドメイン（これら二つの次元の間の交差）によってデータを見ることができます。 一度に3ディメンションフィルタまで適用できます。

v3 Analytics（ウーヤラIQ）、多次元分析はGUIとRESTベースのAPIの両方でサポートされています。
