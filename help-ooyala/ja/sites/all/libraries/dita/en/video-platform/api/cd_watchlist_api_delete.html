ウォッチリストを削除する

このエンドポイントは指定されたウォッチリストを削除します。
注意： ユーザーを削除することはできません デフォルト ウォッチリスト

HTTPリクエスト

削除https://player.ooyala.com/watchlist/v3/<watchlist_id>?account_token= <account_token>

クエリパラメータ

パラメーター 説明 タイプ 必須
api_version でなければなりません v3. 文字列 はい（URL）
account_token 特定のエンドユーザに関連付けられているウォッチリストを参照するためにウォッチリストサービスによって使用されます（UUIDによって識別されます）。 詳しくは、 OoyalaアカウントトークンAPI. 文字列 はい（URL）
watchlist_id 削除する監視リストのID。 これは有効なUIDでなければならず、ユーザーのアカウントに存在していなければなりません。 ユーザーを削除することはできません デフォルト ウォッチリスト UUIDv4または文字列 はい（URL）

リターンコード

コード 理由 説明
200 OK DELETE要求は成功しました。 監視リストを削除しました。
400 リクエストエラー 次の問題が原因で失敗しました。
  • パラメーターが欠落しているかフォーマットが正しくありません。
  • 指定された watchlist_id このユーザーのデフォルトのウォッチリストです。
  • 指定されたウォッチリスト 名 is "デフォルト" （またはその単語の任意のケースのバリエーション）。
  • 指定されたウォッチリスト 名 このユーザーには存在しません。
401 無断 指定された account_token 無効です。
404 見つかりませんでした サーバーが停止しています。
500 内部サーバーエラー 内部サーバーエラーが発生しました。
