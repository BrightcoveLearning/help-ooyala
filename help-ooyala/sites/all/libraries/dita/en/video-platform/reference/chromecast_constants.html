Modifying Chromecast Constants

You can modify the following constants in the custom receiver code and see the changes on your screen.

Important: This topic contains older information and has been deprecated. For the latest information about Ooyala integration with Chromecast, see Chromecast Integration.
Constants Default Value Description
SPLASH_SCREEN_TIMEOUT_MILLIS 5 Minutes Sets the splash screen timeout trigger time. If the splash screen is shown for the set number of minutes, disconnect back to the Chromecast home screen.
PAUSE_STATE_TIMEOUT_MILLIS 5 Minutes Sets the pause state timeout trigger time. If the video is paused for the set number of minutes, disconnect back to the Chromecast home screen.
spinnerScale 10% of the image size This value will scale the width and height of the spinner.
desiredOpacity 0 (invisible) Sets the opacity of the user interface control elements to be invisible in a fade effect.
originalOpacity 1 (visible) Sets the opacity of the user interface control elements to be visible
secondsToTransition 1 Second The time it takes for all the screen elements to fade out.
secondsToScrubberTransition 5 Seconds If the video is seeking from a playing state, only the scrubber elements (progress bar, duration, playhead time) will be displayed, and fade out in the set amount of time.
millisToFade 500 Milliseconds Sets a delay and timeout before fading starts when the video starts playing.
