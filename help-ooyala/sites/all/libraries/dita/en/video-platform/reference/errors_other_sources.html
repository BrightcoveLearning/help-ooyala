Player Errors from Other Sources (Deprecated)

The player displays errors that originate from other sources.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).

Player V3 can return errors that originate from non-Ooyala systems. For example, the player can return errors from Adobe Flash. If your error is not one that is provided in our lists of errors, you can check other sources. We provide the following list as a convenience for tracking down Flash Player errors.

Adobe Flash Player Errors

Ooyala general returns errors as text strings. If you are using a Flash player or an Adobe content protection product, you might see an error code. Some Flash Player error messages look like this:

Flash Player Error 2048
Error #1502--Flash Player

You can lookup this code at Adobe's website:

  • Flash Errors Database
  • Adobe Flash Platform Run-Time Errors
  • flash.error Package
  • Flash Player Help
