Message Bus API

Communicate with the message bus using publish, subscribe, and other functions.

Note: If you plan to develop code that will report and/or take action on specific player events, please consider utilizing our Ooyala Player Analytics Framework instead of the Message Bus API. The Analytics Framework streamlines and filters player events, making it simpler for developers to use.

The Ooyala Player exposes several functions for communication with the message bus. See the Player JavaScript API Reference for more information.
