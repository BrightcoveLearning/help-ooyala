Creating a Sample Web Page

During testing, you will want to create a sample web page. Eventually, you will want to automate this process.

To create a sample web page:

  1. Open a text editor.
  2. Create a web page. For example: 
    <!DOCTYPE html>
   <html>
    <head>
      <title>My Page</title>
    </head>
    <body>
      <h1>My Heading</h1>
      <p>My paragraph.</p>
    </body>
  </html>
  3. Paste the <meta> tags you generated in Manually Getting Twitter <meta> Tags for Ooyala Player into the <head> of the page. For example:  
    <!DOCTYPE html> 
<html>
<head>
  <title>My Page</title>
  <meta name="twitter:card" content="player">
  <meta name="twitter:title" content="123 Marvel's The Avengers.mov" />
  <meta name="twitter:description" content="video">
  <meta name="twitter:image" content="http://ak.c.ooyala.com/B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt/Ut_HKthATH4eww8X4xMDoxOjBhOzV3Va">
  <meta name="twitter:player" content="https://player.ooyala.com/tframe.html?ec=B5aDU0NjrcNyJKEEAPvqsFAKuJ-VLxLt&pbid=MzZiMzc1ZDUzZGVlYmMxNzA3Y2MzNjBk">
  <meta name="twitter:player:width" content="435">
  <meta name="twitter:player:height" content="251">
  <meta name="twitter:site" content="@username">
</head>
<body>
  <h1>My Heading</h1>
  <p>My paragraph.</p>
</body>
</html>
  4. Copy the web page to a location on your public web site. Make note of the URL so you can give it to Twitter for approval.
