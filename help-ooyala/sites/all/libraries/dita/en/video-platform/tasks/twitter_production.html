Approach to Automating Creation of Twitter Web Pages

Although you can generate your code at any time, we recommend generating it at the time you create your video.

The following general steps are an approach to automating the creating of Twitter Player Cards in production. Exact steps depend entirely on your own processes and programmers.

  1. A producer uploads a video to your CMS.
  2. The upload triggers a program that gets the <meta> tags detailed in Manually Getting Twitter <meta> Tags for Ooyala Player.
  3. The program embeds the <meta> tags inside the <head> of a web page.
  4. The program performs any other content preparation tasks.
  5. The program or producer sets the status of the content to "live" in Backlot.
