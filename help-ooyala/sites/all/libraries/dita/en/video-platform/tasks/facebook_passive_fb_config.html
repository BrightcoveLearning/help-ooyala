Configuring the Facebook App

After creating a Facebook App, you must configure it.

To configure the Facebook app:

  1. Open http://developers.facebook.com/apps with a web browser and log in. The Apps page appears.
  2. Select an app. Information about the app appears.
  3. Click Edit App. The Basic settings page appears.
  4. Click Open Graph. The Open Graph: Getting Started page appears.
  5. In the People Can fields, start typing watch and select watch Video from the list box. Then, start typing video and select video Video from the list box.
  6. Click Get Started. Facebook begins building the app.
