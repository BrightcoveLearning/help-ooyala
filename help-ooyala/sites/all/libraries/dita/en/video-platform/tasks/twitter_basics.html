Creating an Ooyala Player Twitter Card

These are the basics steps to create a Twitter Player Card with a video and the Ooyala player.

  • Manually Getting Twitter <meta> Tags for Ooyala Player
    For Twitter to correctly set up your Twitter Card, you need to provide metadata about each video; this information is in the form of HTML <meta> tags. This section describes how to manually generate these <meta> tags for any specific video. You will want to automate this process. Note that Flash content is no longer supported by Twitter, which requires HTML5 content.
  • Creating a Sample Web Page
    During testing, you will want to create a sample web page. Eventually, you will want to automate this process.
  • Requesting Approval from Twitter
    To participate in the Twitter Cards program, follow Twitter's online instructions. Make sure to include the link to the web page you created in the previous procedure.
  • Approach to Automating Creation of Twitter Web Pages
    Although you can generate your code at any time, we recommend generating it at the time you create your video.
