Editing Custom Metadata

You can edit metadata at any time.

To edit metadata:

  1. Log in to the Backlot UI. The Backlot UI opens.
  2. Open the MANAGE page. The MANAGE page opens with a list of content.
  3. Select an asset. The asset is selected with the Details tab selected.
  4. Click the Custom Metadata tab.
  5. For each metadata entry to add, enter a Name and Value. Each entry must have a unique name. If a piece of metadata requires multiple values, you could use a delimiter such as a pipe (|) or a comma (,). For example: Value: Action | Adventure | Suspense
  6. For each metadata entry to delete, click its delete icon.
