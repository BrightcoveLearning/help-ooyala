Editing Video Information

You can change information about a video at any time.

To edit video information:

  1. Use the /v2/assets route. The following example performs a get to obtain the current settings for the asset with the Y1dTdvMjq9QtOM ID. 
    [GET] /v2/assets/Y1dTdvMjq9QtOM
    Note: To get information about all assets, do not specify the asset ID in the request.

    Backlot returns a response similar to the following.

    {
  "asset_type": "video",
  "duration": 30,
  "name": "My Video",
  "preview_image_url": null,
  "created_at": "2011-07-22T18:54:19+00:00",
  "embed_code": "Y1dTdvMjq9QtOM",
  "time_restrictions": null,
  "updated_at": "2011-07-22T20:51:07+00:00",
  "external_id": null,
  "hosted_at": null,
  "original_file_name": "my_video.avi",
  "description": null,
  "status": "live"
}
    Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.
  2. Make any changes with a PATCH request. The following example changes the name, hosted at URL, and description. 
    [PATCH] /v2/assets/Y1dTdvMjq9QtOM
{
  "name": "My Awesome Video",
  "hosted_at": "http://www.example.com",
  "description": "The next Internet meme."
}

    Backlot returns a response similar to the following.

    {
  "asset_type": "video",
  "duration": 30,
  "name": "My Awesome Video",
  "preview_image_url": null,
  "created_at": "2011-07-22T18:54:19+00:00",
  "embed_code": "Y1dTdvMjq9QtOM",
  "time_restrictions": null,
  "updated_at": "2011-07-22T20:51:07+00:00",
  "external_id": null,
  "hosted_at": "http://www.example.com",
  "original_file_name": "my_video.avi",
  "description": "The next Internet meme.",
  "status": "live"
}
    Note: Try out the code samples using your account credentials in the Ooyala Scratchpad. To launch the Scratchpad, go to Ooyala API Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad.

The video is successfully edited.
