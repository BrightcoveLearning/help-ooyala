Enabling TV Ratings for iOS Apps

Use the following steps to enable TV ratings for an iOS app.
  1. Configure your movies with the proper custom metadata in the Backlot UI. See the following topics for information about this step:
    • TV Ratings Backlot Metadata Configurations
    • Adding Custom Metadata
  2. Create an OOFCCTVRatingConfiguration object to outline the parameters that configure how FCC Ratings are shown.
  3. Create an OOOptions object to use when configuring the OoyalaPlayer.
  4. Initialize the OoyalaPlayerViewController and add the OOOptions object as a parameter.
  5. Use the setEmbedCode method and play the asset to see the TV ratings appear as configured in your OOFTVRatingsConfiguration object.

The follow code example is the result of the above steps:

OOFCCTVRatingConfiguration *tvRatingTopLeft =
  [[OOFCCTVRatingConfiguration alloc]
  initWithDurationSeconds:DURATION
  position:OOFCCTvRatingsPositionTopLeft
  scale:OOFCCTVRATINGCONFIGURATION_DEFAULT_SCALE
  opacity:OOFCCTVRATINGCONFIGURATION_DEFAULT_OPACITY]

  OOOptions *ooOptions = [[OOOptions alloc]
    initWithTVRatingsConfiguration:tvRatingTopLeft];
    self.ooyalaPlayerViewController = [[OOOoyalaPlayerViewController alloc]
                                     initWithPcode:PCODE
                                     domain:[[OOPlayerDomain alloc]
    initWithString:PLAYERDOMAIN]
                                     embedTokenGenerator:nil
                                     controlType:OOOoyalaPlayerControlTypeInline
                                     options:ooOptions];
