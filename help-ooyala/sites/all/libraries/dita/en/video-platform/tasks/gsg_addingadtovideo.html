Applying an Ad Set to a Video

You can monetize videos by applying one or more advertisements.

To apply an ad set to a video, do the following:

  1. Log in to the Backlot UI and open the MANAGE page.
  2. Select the video you uploaded.
  3. Click the MONETIZE subtab.
  4. Select the ad set you created from the Ad Set list box. Ads from the ad set will be played each time the video is viewed.
