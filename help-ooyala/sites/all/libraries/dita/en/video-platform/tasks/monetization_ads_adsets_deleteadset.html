Deleting an Ad Set

You can delete an ad set at any time. All settings for the ad set will be deleted and cannot be recovered.

To delete an ad set:

  1. Log in to the Backlot UI. The Backlot UI opens.
  2. Open the MONETIZE page.
  3. Click the Ad Sets tab.
  4. Locate an ad set and click its delete icon. You are prompted for confirmation.
  5. Click YES.
