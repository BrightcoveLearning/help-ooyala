Managing the Preview Image

After you upload an asset, you can specify the image to display before viewers click the play button.

To manage the preview image:

  1. Log in to the Backlot UI and open the MANAGE page.
  2. To only view specific asset types, select the asset type under CONTENT TYPES in the left pane.
  3. Select an asset.
  4. Click Edit Preview Image. The Edit Preview Image dialog box appears.
  5. Click UPLOAD AN IMAGE and select an image.
    Note: The player preview shown on the Details subtab uses Player V4.
  6. When you are finished, click Close.
