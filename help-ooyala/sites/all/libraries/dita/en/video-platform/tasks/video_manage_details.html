Managing Video Details

The Details enables you to configure metadata and specify a player.

To manage video details:

  1. Log in to the Backlot UI and open the MANAGE page.
  2. To only view video files, select VIDEO under CONTENT TYPES in the left pane.
  3. Select a video.
  4. Make any changes in the TITLE and DESCRIPTION fields.
  5. Click the Embed tab.
  6. Select the player in which the video is played from the Custom Player list box.
  7. Specify the URL where the content is embedded in the Permalink where content is embedded field (e.g., http://www.mysite.com/mymusic).
