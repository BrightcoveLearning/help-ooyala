2013-10-16 Release

Updates include improved error handling for third party ads manager, improvements to the reseller portal, enhancements to player preview images, updated OPF modules, updated HD2, improved share via email, and updated uploading default to HTTPS.

The following are release items:

  • Third Party Ads Manager—Improved error handling.

  • Reseller Portal—Numerous performance and UI improvements.

  • Player Preview Images—Numerous performance enhancements for processing and display.

  • OPF Modules—Updated FreeWheel, Conviva, and Omniture modules.

  • HD2—Updated for improved Widevine support.

  • Share via Email—Improved viewer experience.

  • Uploading—HTTPS now set to default for uploading.
