2016-07-12 Release

Notable contents of this release include syndication of videos to Facebook and deprecation of the v2 Analytics API.

Facebook Syndication

New Features and Enhancements

  • With Facebook syndication, publishers can now easily set up a feed that automatically publishes selected videos in the Backlot CMS to a Facebook timeline, page, group or event.

v2 Analytics API Deprecation

As planned, we have officially deprecated the Ooyala v2 Analytics API. Going forward, do not use the Analytics v2 API. Please develop against the new v3 Analytics API (Ooyala IQ API). For more information on the v3 Analtyics API, see Ooyala IQ Analytics .

Documentation Releases

  • Syndication to Facebook: Added support for syndicating videos to Facebook from Backlot.
  • All v2 Analytics API documentation has been deprecated. See Ooyala IQ Analytics for more information about the v3 Analytics API.
