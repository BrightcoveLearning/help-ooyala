2013-12-23 Release

Updates include SSL optimizations to meet Twitter requirements, third party ads manager optimizations, playback API optimizations, improved Twitter integration, HTML5 iFrame support, Google IMA with the mobile SDK, and FreeWheel with the mobile SDK.

The following are release items:

  • SSL—Promotional images, server domain, and other SSL delivery optimizations were made to meet Twitter requirements.

  • Third Party Ads Manager—Numerous optimizations.

  • Playback—API optimizations, ad playback improvements, and scrubber image updates.

  • Twitter—Improved Twitter integration. For more information, refer to the Integration with Twitter Player Cards documentation.

  • HTML iFrame Support—For more information, refer to the Embedding an HTML5 Player V3 in an HTML iframe (Deprecated) documentation.

  • Google IMA with the Mobile SDK—For more information, refer to the Integration with Google IMA on iOS and Integration with Google IMA on Android documentation.

  • FreeWheel with the Mobile SDK—For more information, refer to the Integration with FreeWheel on Android documentation.
