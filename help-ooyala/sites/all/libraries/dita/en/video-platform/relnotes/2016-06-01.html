2016-06-01 Release

Notable contents of this release include an update to the closed caption behavior in the V3 ASP Flash Player and a bug fix for the mobile SDK for Android.

Software Feature Releases

V3 Flash Player Closed Caption Update

With today's release, for all streams except live streams, the Ooyala V3 ASP (ActionScript) Flash player will only show the closed caption button if a dfxp file/cuepoints are available for the stream. For 607/708 closed captions for live streams, you need to work with your account manager to apply backend settings to show the closed caption button at the movie level (movie attribute) or provider level (provider attribute).

Mobile SDK for Android v4.14.1

Resolved Issues

This release fixes an issue with concurrent streams and authorization heartbeats.
