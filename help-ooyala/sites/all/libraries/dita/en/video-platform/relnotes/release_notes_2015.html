2015 Video Platform Release Notes

Release Notes from 2015.

  • 2015-12-30 Release
    Notable contents of this release include the new Ooyala Live portal and the release of the mobile SDKs for iOS and Android v.4.9.0.
  • 2015-12-16 Release
    Notable changes for this release include new and enhanced features in the Ooyala IQ UI and new metrics in the IQ API.
  • 2015-12-07 Release
    A notable change for this release is the release of v.4.8.0 of the mobile SDKs for iOS and Android.
  • 2015-12-02 Release
    Notable changes for this release include new third-party player reporting functionality in Ooyala IQ and a new Community login process.
  • 2015-11-19 Release
    Notable changes for this release include new URL reporting functionality in Ooyala IQ and NPAW integration updates.
  • 2015-11-16 Release
    Notable changes for this release include the release of the mobile SDKs for iOS and Android v.4.7.0.
  • 2015-11-11 Release
    Notable changes for this release include the new Usage API, integration with Adways Studio, and integration with Avid Technology.
  • 2015-10-21 Release
    Notable changes for this release include the release of the mobile SDKs for iOS and Android v.4.6.0.
  • 2015-10-15 Release
    Notable changes for this release include the new Discovery Insights features in Ooyala IQ and new Ooyala IQ documentation.
  • 2015-10-07 Release
    The notable changes in this release are the new Watchlist REST API to help manage Watchlists, and new integration capabilities with NewZulu's FileMobile and Media Factory products.
  • 2015-09-30 Release
    Notable changes for this release include iOS9 certification for the HTML5 Player V3 and mobile SDK for iOS and an update about CORS enablement.
  • 2015-09-18 Release
    Notable changes for this release include updated support for VAST on the Ooyala mobile SDKs and updated Aspera documentation.
  • 2015-09-11 Release
    Notable changes for this release include updated documentation for Theme Builder, closed captions, and Player v3.
  • 2015-09-01 Release
    Notable changes for this release include the release of the mobile SDK for iOS v.4.4.1.
  • 2015-08-24 Release
    Notable changes for this release include the release of the mobile SDK for iOS and Android v.4.4.0.

  • 2015-07-24 Release
    Notable changes for this release include new documentation about player states and events.
  • 2015-07-22 Release
