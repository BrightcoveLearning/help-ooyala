2016-04-15 Release

Notable contents of this release include a fix for Live channel creation with ad insertion.

Software Feature Releases

Ooyala Live Release - Versoin 1.2.1

Resolved Issues

  • There was an issue where a non-admin user was unable to create a channel with Ad insertion. This issue has been resolved.

Documentation Releases

  • Basic Embedding Information and Publishing a Video: Added details on how to retrieve the New Ooyala Player (V4) Embed Code from Backlot.
  • Migrating to Player V4: Added a note about the scenarios where the ad control bar and ad marquee settings applied in the Player V4 skin.json file will be purposefully ignored to avoid blocking ad interactivity.
