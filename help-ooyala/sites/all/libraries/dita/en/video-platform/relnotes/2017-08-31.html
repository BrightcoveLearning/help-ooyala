2017-08-31 Release

Documentation Updates

  • Page-level plugin overrides for the standard embed code: A new topic describes how, if you use the standard embed code, you can override (on the web page) the default plugins by specifying a list of only the exact plugins you want to load. See Specifying Plugins for the Standard Embed Code.
  • Ooyala Player Token with Player V4: A new topic in the Ooyala Help Center tells how to use the Ooyala Player Token with Player V4 to provide content protection. See Ooyala Player Token for Player V4.
