2015-06-17 Release

Software Feature Releases and Updates

Discovery Feedback Loop

The Ooyala Discovery Feedback Loop routes are enhanced to include a new parameter for defining the placement of recommendations in the player. For more information, see Discovery Feedback Loop.

Documentation Releases

  • Discovery Feedback Loop and Using the Feedback Loop Routes: Updated the Discovery Feedback Loop documentation to include descriptions of the parameters available for use with the Feedback Loop routes and discuss the use of the optional ui_tag parameter.

  • Labels: Added a description of best practices for using labels with Theme Builder.

  • Limiting Concurrent Streams per Viewer: Added information about how the heartbeat feature is configurable.
  • Ad Integration FAQ: Added this new topic to provide information about integrating with ad managers.
  • Syndication with Podcast: Updated descriptions of the syndication fields.
  • Supported Browsers: Updated the supported browsers list to reflect the browsers currently supported. Due to the deprecation of the Chrome Frame plugin, Internet Explore has been removed from the list.

  • Player V3 Functional Support Across Environments (Deprecated): Updated the Analytics Support table to reflect the supported analytics platforms.

  • v3 Analytics Best Practices: Added the recommendation that you should run one query at a time with the v3 Analytics API for optimal performance.
