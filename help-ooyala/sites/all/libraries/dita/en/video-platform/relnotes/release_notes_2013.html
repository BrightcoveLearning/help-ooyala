2013 Video Platform Release Notes

Release Notes from 2013.

  • 2013-12-23 Release
    Updates include SSL optimizations to meet Twitter requirements, third party ads manager optimizations, playback API optimizations, improved Twitter integration, HTML5 iFrame support, Google IMA with the mobile SDK, and FreeWheel with the mobile SDK.
  • 2013-10-16 Release
    Updates include improved error handling for third party ads manager, improvements to the reseller portal, enhancements to player preview images, updated OPF modules, updated HD2, improved share via email, and updated uploading default to HTTPS.
  • 2013-07-03 Release
    Updates include the enhanced Third Party Ads Manager, updated the Discovery Profile, closed caption improvements, UI fixes in the Up Next preview screen, optimized data visualization for Ooyala Now, improved metadata retrieval, fixed HLS playback issues, and enabled Uploader V2 for all accounts.
  • 2013-03-26 Release
    Updates include the release of Ooyala Schedule, Hook, and Rights Locker.
