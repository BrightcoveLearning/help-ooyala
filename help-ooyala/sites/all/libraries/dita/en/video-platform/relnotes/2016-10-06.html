2016-10-06 Release

Notable contents of this release include Ooyala Live release 2.3.0 and enhancements to the Player V4 documentation.

Software Feature Releases

Ooyala Live Release - Version 2.3.0

New Features and Enhancements

  • You can now stream prerecorded videos from Ooyala Backlot. To do this, you create a playlist of videos in Ooyala Live, add videos to the playlist, associate the playlist with a channel, and run the channel. For more information, see Stream Prerecorded Videos.

Documentation Releases

  • Player V4 and Flash-free Playback. Instructions for enabling HLS on web browsers without the use of Flash. Requires the bit_wrapper plugin for DASH and HLS (bit_wrapper.min.js). See Using Flash-free Playback with Player V4.
  • Updated Player Migration FAQ. See Player Migration FAQ.
