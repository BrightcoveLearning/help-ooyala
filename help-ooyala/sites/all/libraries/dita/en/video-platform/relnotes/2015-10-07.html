2015-10-07 Release

The notable changes in this release are the new Watchlist REST API to help manage Watchlists, and new integration capabilities with NewZulu's FileMobile and Media Factory products.

Software Feature Releases

Watchlists

The Watchlist Service powers the 'Add to Watchlist' action and related consumer experiences. Video publishers can allow registered consumers to save videos for watching at a later time. Supported API operations include managing Watchlists and their associated items (videos).

Ooyala-NewZulu Integration

Ooyala integration with NewZulu's FileMobile and Media Factory products allow users to seamlessly combine NewZulu's content collection, curation, and workflow capabilities with Ooyala analytics, advertising, and video services. NewZulu's FileMobile product provides desktop and mobile users with an easy to use, responsive video upload experience.
