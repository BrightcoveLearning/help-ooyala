Labels Management: Rename (Deprecated)

Backlot API v1

http://www.ooyala.com/partner/labels
?pcode=<pcode>
&expires=<expiration>
&mode=renameLabel
&oldlabel=<oldlabel>
&newlabel=<newlabel>
&signature=<signature>

Backlot API v2

[PUT] https://api.ooyala.com/v2/labels/<label_id>
{
  “name” : “<new name>”
}

Example

The following example renames a label:

[POST] https://api.ooyala.com/v2/labels/d5751b77a0c24972888bf906734d4522
{
	"name": "Sports"
}

Backlot returns a response similar to the following:

{
  "name": "Sports",
  "id": "d5751b77a0c24972888bf906734d4522",
  "full_name": "/Sports",
  "parent_id": null
}
