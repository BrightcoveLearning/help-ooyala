HTML5 APIs (Player V2 Deprecated)

Ooyala Player HTML5 APIs enable you to customize aspects of the Ooyala player and video experience for iOS, Android and other HTML5-supporting platforms.

Important: Ooyala Player V2 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V2 will no longer play your video or audio content. Customers still using Player V2 need to migrate to Player V4 (see Player V4.

The following topics describe the Player HTML5 APIs. You can view all of the API in a single file, or select the API information by functional categories such as APIs used for ads, close captioning, Player and so on. A separate topic contains examples that illustrate the use of these HTML5 APIs.

  • HTML5 API Summary List (Player V2 Deprecated)
    Lists and describes all Player HTML5 APIs grouped by Functions/Methods, Objects, Properties and Events.
  • Assets HTML5 APIs (Player V2 Deprecated)
    Assets HTML5 APIs can be used to handle ratings, videos (play/pause, describe), and channels.
  • Player HTML5 APIs (Player V2 Deprecated)
    Player/Player UI HTML5 APIs can be used to handle aspects of the player.
  • HTML5 Example and Demo (Player V2 Deprecated)
    The HTML5 Examples demonstrates how to test the HTML5 Player for an iOS-basedd device.
