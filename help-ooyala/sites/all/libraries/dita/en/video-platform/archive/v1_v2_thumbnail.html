Thumbnail Query API (Deprecated)

The thumbnail query API is now part of the asset API.

You can perform a GET against an asset to return the automatically generated preview images, preview image files you specified, or preview image URLs you specified.

Backlot API v1

http://api.ooyala.com/partner/thumbnails
?pcode=<pcode>
&embedCode=<embedCode>
&range=0-25
&resolution=600x400
&expires=<expiration>
&signature=<signature>

Backlot API v2

[GET] https://api.ooyala.com/v2/assets/<embedCode>/{generated_preview_images | preview_image_files | preview_image_urls)

Example

This example returns the generated preview images:

[GET] https://api.ooyala.com/v2/assets/Ut_HKthATH4eww8X5hMDoxOjBrOw-uIx/generated_preview_images
