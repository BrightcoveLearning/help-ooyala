Label Management API (Deprecated)

The new label management API has more features and is less complex than the Backlot API v1.

The old label API provided modes that enabled you to:

  • List labels
  • Create labels
  • Delete labels
  • Assign labels
  • Unassign labels
  • Rename labels

The new label API enables you to list, create, rename, and delete labels. Assigning and unassigning labels is now part of the assets API.
