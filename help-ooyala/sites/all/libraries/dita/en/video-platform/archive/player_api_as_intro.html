Player ActionScript APIs (Player V2 Deprecated)

Ooyala Player ActionScript APIs enable you to customize the Adobe Flash-based player and video experience.

Important: Ooyala Player V2 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V2 will no longer play your video or audio content. Customers still using Player V2 need to migrate to Player V4 (see Player V4.

The following topics describe the Player ActionScript APIs. You can view all of the APIs in a single file, or select the API information by functional categories such as APIs used for ads, close captioning, Player and so on. A separate topic contains examples that illustrate the use of these ActionScript APIs.

  • ActionScript API Summary List (Player V2 Deprecated)
    Lists and describes all Player ActionScript APIs grouped by Functions/Methods, Objects, Properties and Events.
  • Ads ActionScript APIs (Player V2 Deprecated)
    Ads ActionScript APIs can be used to respond to events such as starting, clicking, completing and skipping ads.
  • Assets ActionScript APIs (Player V2 Deprecated)
    Assets ActionScript APIs can be used to handle ratings, videos (play/pause, describe), and channels.
  • Bitrate and Buffer ActionScript APIs (Player V2 Deprecated)
    The Bitrate Selection API allows developers to programmatically obtain current buffer size, as well as get and set bitrates.
  • Closed Captioning ActionScript APIs (Player V2 Deprecated)
    Closed Captioning ActionScript APIs enable you to get supported closed captions, set closed caption languages, and get the caption timing.
  • Embed Code ActionScript APIs (Player V2 Deprecated)
    Embed Code ActionScript APIs can be used to handle getting and setting the embedCode for assets, getting asset descriptions, getting images and fetching asset metadata.
  • Error ActionScript APIs (Player V2 Deprecated)
    Error ActionScript APIs enable you to get error codes, the associated error message, and player state.
  • Player ActionScript APIs (Player V2 Deprecated)
    Player/Player UI ActionScript APIs can be used to handle aspects of the player such as fullscreen mode, player state, volume, playhead, player volume and video play length and status.
  • ActionScript Examples (Player V2 Deprecated)
    ActionScript Examples demonstrate the embedding capability of the Ooyala player and the capability to add additional features to the player using the Player API.
