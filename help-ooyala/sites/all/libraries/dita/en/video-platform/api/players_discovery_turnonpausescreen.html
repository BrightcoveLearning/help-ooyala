Turning on the Pause Screen

The following request and response examples use the PATCH method to call a player using the player ID and turn on the pause screen.

Request:

[PATCH]/v2/players/{player_id}/discovery/profile
{
"show_pause_screen": "enable"
}

Response:

{ "discovery_interface": {
"when_done": null,
"show_pause_screen": "true",
"default_countdown": "10",
"hide_thumbnail_title": "false"
},
...
