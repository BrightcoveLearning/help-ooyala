Channel Set Assets (Deprecated)

A channel set is a collection of channels in a single player.

Methods

Create Asset 
[POST] /v2/assets 
{
  properties
}
List Assets 
[GET] /v2/assets
Get Channel Set 
[GET] /v2/assets/channel_set_id
Modify Channel Set 
[PATCH] /v2/assets/channel_set_id
Delete Channel Set 
[DELETE] /v2/assets/channel_set_id
List Channels in Channel Set 
[GET] /v2/assets/channel_set_id/lineup
Set or Replace Channels 
[PUT] /v2/assets/channel_set_id/lineup
["embed_code","embed_code", ...]
Append Channel to Channel Set 
[PUT] /v2/assets/channel_set_id/lineup/embed_code
[PUT] /v2/assets/channel_set_id/lineup/external_id
Delete Channel from a Channel Set 
[DELETE] /v2/assets/channel_set_id/lineup/embed_code
[DELETE] /v2/assets/channel_set_id/lineup/external_id

Route Attributes

The following table describes all attributes that can be expressed through the route.

Route Attribute Description
channel_set_id

The ID of the channel set.

Type: String

Default: None

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

embed_code

The content ID or embed code of a channel within the channel set.

Type: String

Default: None

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

external_id

The external ID of a channel within the channel set, if applicable.

Type: String

Default: None

Example: /assets/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg

Properties

The following table describes all properties that can be associated with a channel set.

Property Description Required?
all_day

Recurring only. Specifies whether the content can be viewed during the entire day. If set to false, specify a start and end time.

Type: Boolean

Valid Values: true | false

Default: true

Example: true

Parent: time_restrictions

No
asset_type

Type of asset.

Type: String

Valid Values: video | ad | audio | remote_asset | youtube | live_stream | live_audio | channel | channel_set | channel_set

Default: video

Example: "video"

Yes
created_at

Read only. Date and time the video was processed.

Type: DateTime

Default: None

Example: "2011-04-25T15:32:57+00:00"

No
description

Description of the asset.

Type: String

Default: None

Example: "Season 22 Opening Game"

No
duration

Length of the video, in milliseconds.

Type: Integer

Default: None

Conditions: Required for remote assets.

Example: "120"

Conditional
embed_code

Read only. Returns the content ID of the asset that was assigned by Backlot.

Type: String

Default: None

Example: "IzNnllMjphu2XF3_UgPROoCi9B2Bwrde"

No
end_date

End date of the restriction.

Type: DateTime (range) or Date (recurring)

Default: None (does not end)

Example: "2011-08-09T21:21:49-07:00" or 2011-08-09

Parent: time_restrictions

No
end_time

Recurring only. Specifies the end time when content can play.

Type: Time

Default: 23:59:59

Example: 21:00:00

Parent: time_restrictions

No
external_id

A custom identifier you define that you can use instead of the content ID (embed_code).

Type: String

Default: None

Conditions: Required for videos.

Example: "my_movie.avi"

Conditional
hosted_at

The permanent URL at which you embed the asset.

Type: String

Default: None

Conditions: Required for videos.

Example: "http://example.com"

Conditional
name

Name of the asset.

Type: String

Default: None

Example: "My Movie"

Yes
original_file_name

Name of the file that was specified when uploading the asset.

Type: String

Default: None

Conditions: Required for videos.

Example: "my_movie.avi"

Conditional
preview_image_url

Read only. URL where the preview image is located. To upload a preview image, see Managing the Preview Image.

Type: String

Default: None

Example: "http://ak.c.ooyala.com/IzNnllMjphu2XF3_UgPROoCi9B2BwtSg/Ut_HKthATH4eww8X5hMDoxOjBrOw-rtx"

Yes
recurring_days

Recurring only. List of days which the content can be viewed.

Type: Array

Valid Values: "SUN" | "MON" | "TUE" | "WED" | "THU" | "FRI" | "SAT"

Default: None (all days)

Example: ["SUN", "FRI", "SAT"]

Parent: time_restrictions

No
start_date

Start date of the restriction.

Type: DateTime (range) or Date (recurring)

Default: None (starts immediately)

Example: "2011-05-09T21:21:49-07:00" or 2011-05-09

Parent: time_restrictions

No
start_time

Recurring only. Specifies the start time when content can play.

Type: Time

Default: 0:00:00

Example: 20:00:00

Parent: time_restrictions

No
status

Returns the status of the video.

Type: String

Valid Values: Processing | Uploading | Live

Default: None

Example: "Uploading"

No
time_restrictions

Specifies time restrictions for the asset.

Type: Container

Default: None

No
type

Specifies whether this is a recurring or one-time range restriction.

Type: String

Valid Values: "range" | "recurring"

Default: None (no restriction)

Example: "range"

Parent: time_restrictions

 No
updated_at

Read only. Date and time the video was last modified.

Type: DateTime

Default: None

Example: "2011-04-25T15:32:57+00:00"

No

Examples

This example creates a channel set: 

[POST] /v2/assets
{
  "name": "My Channel",
  "asset_type": "channel_set"
}

Backlot returns a response similar to the following:

{
  "asset_type": "channel_set",
  "duration": 0,
  "name": "My Channel",
  "preview_image_url": null,
  "created_at": "2011-08-10T17:45:24+00:00",
  "embed_code": "dvNTVxMjrRktSlb5v",
  "time_restrictions": null,
  "updated_at": "2011-08-10T17:45:24+00:00",
  "external_id": null,
  "hosted_at": null,
  "original_file_name": null,
  "description": null,
  "status": "live"
}
