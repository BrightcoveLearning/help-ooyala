HTML5-based Player V3 Error Messages (Deprecated)

From the OO.EVENTS.ERROR event, the Ooyala Player can show specific error messages.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).
You can use the Ooyala V3 Player's Message Bus to handle errors by subscribing to or intercepting the OO.EVENTS.ERROR event. For a list of errors, see the Player JavaScript API Reference for more information. The following table also contains a complete list of errors:
