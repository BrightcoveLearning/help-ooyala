Backlot

You can work with Backlot to manage ads and ad sets.

  • Uploading a Video Advertisement
    Before you can add a video advertisement to a video, you must upload the video advertisement.
  • Creating an Ad Set
    To add advertisements to videos, you must first create an ad set.
  • Applying an Ad Set to a Video
    You can monetize videos by applying one or more advertisements.
  • Deleting an Ad from an Ad Set
    You can delete an ad from an ad set at any time. If you are just changing a video, it might be easier to edit the ad settings.
  • Deleting an Ad Set
    You can delete an ad set at any time. All settings for the ad set will be deleted and cannot be recovered.
