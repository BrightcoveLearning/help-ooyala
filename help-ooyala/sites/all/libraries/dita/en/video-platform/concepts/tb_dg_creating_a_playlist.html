Creating a Playlist

A playlist consists of a list of movies. Playlists can be asset-based or label-based.

Note:
  • The maximum number of videos you can add to a playlist is 64.
  • The maximum number of labels you can associate with a playlist is 16.
  • Allow some time for the playlist you create to be propagated through the Ooyala systems. If you are doing a demonstration, it is best to use playlists you have created previously.

To create a playlist:

Use the playlists route. The following example creates a playlist with movies.

[POST]/v2/playlists{
   "name": "Test Movie Playlist",
   "type": "movie"
}

Backlot returns a response similar to the following:

{  
   "name":"Test Movie Playlist",
   "updated_at":"2013-07-10T18:24:32Z",
   "limit":null   ", 
   "id":"625622dfd22c47e9b7dcd0e8d1843638",
   "items":[  

   ],
   "type":"movie",
   "created_at":"2013-07-10T18:24:32Z",
   "provider_id":"12345",
   "sort_by":"custom"
}
Note:

Try out the code samples using your account credentials in the Ooyala Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad. To launch the scratchpad, go to Ooyala API Scratchpad.

The playlist is successfully added.

Note: Playlists are not visible by default. If you do not enable pods for your player, playlists are not visible to users. To make playlists visible to users, enable pods by associating the playlist and pod with a player.
