Integrations

The Mobile SDK for iOS integrates with the following third-party solutions:

Table 1. Third Party Integrations
Third Party Solution Description
Ooyala Pulse Ooyala offers an add-on to the Mobile SDK to integrate with the Ooyala Pulse ad server. For more information see Integration with Ooyala Pulse on iOS.
FreeWheel Ooyala offers an add-on to the Mobile SDK to integrate with the FreeWheel ad service. For more information see Integration with FreeWheel on iOS.
Omniture Ooyala offers an add-on to the Mobile SDK to integrate with Omniture analytics. For more information see Integration with Omniture on iOS.
Google IMA Ooyala offers an add-on to the Mobile SDK to integrate with Google IMA ad service. For more information see Integration with Google IMA on iOS.
