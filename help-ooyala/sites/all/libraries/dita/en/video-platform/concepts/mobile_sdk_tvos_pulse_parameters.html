Parameters for Ooyala Pulse on tvOS

The following table lists the parameters you can set in the Ooyala Pulse tvOS ad plugin for Player V4 and the relationship between the different locations where these parameters are read from, if not set in the integration.

Note: take these rules into account when reading the table:
  • In general, embedded parameters override custom metadata set on individual assets in Backlot UI, custom metadata overrides ad set parameters set in Backlot UI, and ad set parameters override content metadata from assets in Backlot UI.
  • (1) indicates the location where the base value for that particular parameter is read from.
  • (2) indicates that values set here are merged with the values set for the equivalent parameter from the column(s) to the right of the current column.
