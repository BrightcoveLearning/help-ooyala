About Creating and Managing Profiles with the Backlot UI

You can use the Backlot UI to create discovery profiles to use with defined players or on Discovery API requests.

Although the profiles that you create in the Backlot UI appear to be intrinsically linked with individual players, the Player ID at the bottom of the page is the same as the programmatic profile_id that you use on the individual recommendation API requests, as shown in Managing Discovery Profiles with the Discovery API.
Note: Discovery must be enabled for your account for the Discovery tab to appear on the user interface. For more information about enabling Discovery, contact your Ooyala representative.
