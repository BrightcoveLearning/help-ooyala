Ads Integration with Backlot and Backlot REST API

You can use the Backlot UI and Backlot REST API to integrate ad sources with your content.

You can perform most of the tasks needed to integrate an ad or ad source module using the Ooyala Backlot UI and/or the Backlot REST API. The following topics describe how to use Backlot and the Backlot REST API to add, create, delete and perform other ad integration tasks.
