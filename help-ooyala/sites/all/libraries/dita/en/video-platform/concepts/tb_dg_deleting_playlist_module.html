Deleting a Playlist/Pods Module's Association with a Player

You can delete the association of a Playlist/Pods module with a player.

To delete the association of a Playlist/Pods module with a player:

Use DELETE with the players route. The following example deletes the association between the module (id=d42a1c8d4f91437c88db1c25cb8d6d04) and the player (id=e18ab1da1813483499554ea2d8e67fbd). This does not delete the player or the playlist/pod, but it does remove the association between the two. 

[DELETE]/v2/players/e18ab1da1813483499554ea2d8e67fbd/third_party_modules/d42a1c8d4f91437c88db1c25cb8d6d04

Backlot returns a 200 response.

Note:

Try out the code samples using your account credentials in the Ooyala Scratchpad. For information about using the Scratchpad, see Practice Making Requests with the Scratchpad. To launch the scratchpad, go to Ooyala API Scratchpad.
