Theme Builder Customization for Player V3

Build playlists and customize the playback experience for Player V3

Use Theme Builder to build asset-based or label-based playlists, use complex templates to build a custom playback experience, and create an embed that contains the playlists you construct.
Note: Theme Builder is designed to work with both Ooyala Player V4 and Player V3. There are some differences in the way you configure playlists for Player V4, as described in Using Playlists in Player V4.
