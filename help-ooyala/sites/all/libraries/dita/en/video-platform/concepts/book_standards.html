Supported Standards

Welcome to Supported Standards Guide. These topics describe the requirements and standards that Ooyala supports.

  • Supported Ingest Formats
    Before uploading content to Backlot, make sure you are using a supported file format and following the best practices for source files.
  • Supported Output Formats
    Backlot supports a large number of popular output formats.
  • Supported Content Types
    Ooyala supports these popular content types.
  • Supported Closed Captions
    Before uploading closed caption files to Backlot, verify that you are using a supported language and file format.
  • Supported Closed Caption DFXP (now TTML) Format
    Closed caption files that you upload to Backlot must be in a supported format. The DFXP format (now TTML) is a popular supported format.
  • Supported Ad Providers
    Ooyala supports third-party ad networks (ad providers) that support Video Ad Serving Template (VAST).

  • Player V3 Maximum Bit Rate for the Flash Player (Deprecated)
    Generally, the smaller the embed width of a video, the more likely the video is being served by a device that has a less powerful CPU or a lower bandwidth connection.
