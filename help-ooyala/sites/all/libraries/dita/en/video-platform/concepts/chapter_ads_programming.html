Player V3 Ads Integration Programming

Ooyala supports integration with a number of ad sources using simple JavaScript embedded parameters.

You can use the Backlot UI and Backlot REST API to upload ads and ad sets. You also need to add a few lines of JavaScript to associate the ads with a particular ad source such as Google IMA or FreeWheel. The following topics discuss the general process for adding JavaScript to your web page for ads integration. For details about how to integrate with each of the ad source modules that Ooyala supports, see Integrating Ad Managers with Player V3.
