About Cocoa Pods

Learn about use Cocoa Pods to manage library dependencies in your iOS mobile apps.

Using Cocoa Pods can improve and streamline the management of library dependencies for Ooyala iOS Mobile apps. With CocoaPods the app no longer needs to have libraries downloaded and installed in the project. To use CocoaPods you must have Ruby installed on your system.
