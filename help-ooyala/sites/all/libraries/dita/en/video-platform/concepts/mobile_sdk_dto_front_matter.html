Introduction to Offline Playback: Download to Own (DTO) and Rentals (DTR)

The Ooyala iOS and Android SDKs provide users with the ability to download video assets so that they can be played offline. In order to offer this capability in an iOS and Android apps, you will need to use the following:

Offline Playback Parameters

Entitlement parameters, such as if the asset is allowed to be downloaded, as well as how long the asset can be played once downloaded, are defined using the Ooyala Rights Locker API.

Note: Ooyala currently supports defining Offline Playback entitlement parameters only through these APIs.
