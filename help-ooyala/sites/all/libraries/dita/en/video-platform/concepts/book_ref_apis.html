API References

Welcome to the API Reference Guides. These topics contain API reference information for Backlot, Player, Authorization, and Rights Locker.

Important: The topics in this section are being migrated to the Ooyala API Docs page.

Ooyala offers a rich set of APIs to customize your streaming video platform content. These APIs include:

  • Backlot REST API
    Use the Backlot REST API to customize the chrome (design elements) of your player. This includes UI elements such as fonts, colors, scrubber bar and so forth.
  • v3 Analytics (Ooyala IQ) API
  • Player V3 JavaScript API Reference (Deprecated)
    The Player V3 API is a JavaScript API for customizing your Player v3 functionality and UI elements that also provides message bus event handling.
  • Authorization API Reference
    Authorization API is a set of application programming interfaces (APIs) for authorizing users.
  • Rights Locker API Reference
    Rights Locker is a set of application programming interfaces (APIs) for defining and managing entitlements to digital assets.
  • Usage RESTful API
