Using the Date Range Comparison Feature on the Business Intelligence Page

The Ooyala IQ Date Range Comparison functionality allows you to easily add a comparison date range on the Business Intelligence page to see how user engagement has changed. For example, you can find insights like:

  • How does last week compare to the week before it?
  • How does September this year compare to September last year?
  • How does a time-bound sport event compare to another sport event that happened some time ago?

To add a comparison date range, click the Add comparison date range button next to the selected date period. By default, the most recent previous period is selected. You can customize the comparison period by clicking the comparison date range to set another start date. The end date is calculated automatically based on the length of the selected period.

Once a comparison period has been added, the Ooyala IQ Business Intelligence page updates to include several comparisons.

The time series card is updated to include the total change in the left side column. You can also see comparisons with dashed lines throughout the time series. To isolate a metric, hover over it on the left-hand side of the view.

The asset card is also updated to include comparisons. Please note that only assets that were active during the full period (including the comparison period) will have comparisons added. This logic was implemented to avoid cases where there is massive uplift because the asset was not active for the full comparison period.

The bottom four cards for device, geography, player and traffic source are also updated to include comparisons.

Comparisons are made as percentages since it is sometimes difficult to quickly find the difference of two very large numbers. You can, however, hover the percentages throughout the user interface to get the actual number.

Note: Important: The date compare feature is not available in the Ooyala IQ API. You will have to either include the full date range or make two separate requests, combine the results and calculate the change.
