Configuring the Ooyala Player for Multi-Audio Playback

Ooyala Web Player

  • No special changes needed
  • A page-level parameter can be used to change the default audio track. This is useful to set the default audio track based on the browser language setting.

For more information on configuring the Ooyala player, please see our Player documentation.

Ooyala SDKs

  • No special changes needed
  • Default language will be set by metadata in the stream (or order).
Note: To set the default audio language at the page level, use the audioLanguage page-level parameter, and set the value to the track's ISO-639 language ID. For example: {"audioLanguage":"ger"}
