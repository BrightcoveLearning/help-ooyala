Assigning Ad Sets Dynamically in Player V3 (Deprecated)

The player enables you to assign ad sets dynamically.

Important: Ooyala Player V3 is deprecated and is scheduled to be disabled on 2018-01-31. After that date, Player V3 will no longer play your video or audio content. Customers still using Player V3 need to migrate to Player V4 (see Migrating from Player V3 and Player Migration FAQ).

You can assign an ad set to a specific embed code using the setEmbedCode function. With this setting, you can override a Backlot ad set with one that you set dynamically at runtime. Instead of having static ad content, you can adjust the ad content in response to any number of demographic factors.

There are two ways to assign ad sets dynamically

  • OO.Player.create Player V3 JavaScript function and the adSetCode parameter.
  • Embedded parameters.
