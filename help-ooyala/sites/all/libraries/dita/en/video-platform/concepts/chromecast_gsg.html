Getting Started with Chromecast

Important: This topic contains older information and has been deprecated. For the latest information about Ooyala integration with Chromecast, see Chromecast Integration.
The following are overview steps on how to integrate with Ooyala’s sender and default receiver applications. Details on each step are explained in the sections that follow.
  • Enable the Ooyala Chromecast Integration by submitting a ticket requesting that the "Chromecast Default Receiver" third party module be added to your account. You need to provide Technical Support with your player ID at this time. For information on how to get your player ID, see Basic Tutorial for the HTML5 Player V3 (Deprecated). For the custom receiver, request the "Chromecast Custom Receiver” module.
  • Integrate with Ooyala and the Cast SDK using a sender application for Android or iOS.
  • Integrate with Ooyala and the Cast SDK using the default receiver application.
  • Cast a video.
Note: The iOS or Android sample applications show a sample integration between Ooyala and the Cast SDK.
