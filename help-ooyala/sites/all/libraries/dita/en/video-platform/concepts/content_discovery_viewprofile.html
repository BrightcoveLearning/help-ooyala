Viewing a Discovery Profile

Note: Use the Backlot UI to create a discovery profile (see Creating Discovery Profiles Using the Backlot UI). A default discovery profile is created automatically.
To view an existing discovery profile, use the following GET request.
[GET]/v2/players/profile_id/discovery/profile

The settings shown in the following response example are the default settings.
