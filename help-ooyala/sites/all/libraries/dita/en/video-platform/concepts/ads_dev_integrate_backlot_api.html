Backlot REST API

You can work with the Backlot REST API to manage ads and ad sets.

  • Ad Sets
    Ad sets are ad configurations that control the ad experience of an asset during playback.
  • Creating Ad Sets for Integrating with Ad Sources
    Instead of using your own assets, you can work with ad networks.
  • Associate Ad Sets with Assets
    You can associate ad sets with videos, live streams, and remote assets. For information about creating ad sets, refer to the Backlot User Guide.
