Customizing Player Information

The Android Mobile SDK allows you to programmatically customize the player information to add or remove support for media formats. The PlayerInfo object contains information about the supported player formats. You can override its interface and use the Options.Builder() method to create the OoyalaPlayer:
PlayerInfo playerInfo = new CustomerPlayerInfo();
Options options = new Options.Builder().setPlayerInfo(playerInfo).build();
player = new OoyalaPlayer(PCODE, domain, options);
