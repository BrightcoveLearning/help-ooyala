Channels (Deprecated)

You can create and manage channels.

Note: Channels and channel sets have been deprecated and are scheduled to be disabled. For details and alternatives, see the OVP Release Notes.

A channel is a collection of videos in a single player.

By default, all videos within a channel appear in the left pane of the channel window. A preview image of the selected video appears in the right pane.

Note: You can have up to 64 videos per channel.
  • Creating a Channel (Deprecated)
    Once a channel is created, you can add videos.
  • Managing Channel Details (Deprecated)
    After you create a channel, you can configure metadata and specify a player.
  • Adding Videos to a Channel (Deprecated)
    After you create a channel, you can start adding videos.
  • Managing the Preview Image
    After you upload an asset, you can specify the image to display before viewers click the play button.
  • Managing Monetization
    You can monetize your content using advertisements or create a pay-per-view experience using paywalls.
  • Managing Publishing Rules (Deprecated)
    After you create a channel, you can specify any publishing rules. If both a video and a channel have publishing rules, Backlot evaluates both rule sets for each video and uses the more restrictive settings.
  • Configuring Player Embed Settings in Backlot
  • Associating an Asset with a Label
    After you associate a video, audio file, channel, remote asset, or channel set with a label, Backlot automatically publishes it based on how you configured syndication.
  • Deleting a Channel (Deprecated)
    You can delete channels at any time.
